# v0.4.0 Phase 1: Foundation Types Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `HirType`, `HirPattern`, `ExprId`, `HirItem` and generic-aware `HirFn` to the HIR layer, restructuring `node.rs` into a `node/` module directory — all purely additive with zero behavior change.

**Architecture:** Split `node.rs` into 7 focused files under `node/`. Add the full `HirType` enum (with `Generic`, `Tuple` variants), `HirPattern` enum, `ExprId` newtype, and `HirItem` enum that generalizes the current `Hir.fns` list into a proper item arena. Update the two consumers (`lower.rs`, `codegen.rs`) to use the new structure. No new behavior — every existing test passes unchanged.

**Key differences from naive plan (derived from actual codebase):**
- `node.rs` is ~55 LOC but contains `HirStmt`, `HirExpr` with 9 variants (not 5), `If`/`Println`/`Assert`/`StrLit` — all must be preserved exactly
- `lower()` takes `(&Ast, &Interner)` — signature must not change
- `lower.rs` handles `StrLit`, `If`, `Call` (println/assert builtins), `BlockItem::Stmt` — all must be preserved
- `codegen.rs` imports `HirStmt` and handles `If`/`Println`/`Assert`/`StrLit` — all must be preserved
- `glyim-diag` is NOT a dependency of `glyim-hir` — must be added for `Span` in item types
- `pipeline.rs` calls `glyim_hir::lower(&ast, &interner)` in 2 places — no change needed since signature is preserved
- `compile_to_ir()` calls `glyim_hir::lower(&out.ast, &out.interner)` — no change needed

**Tech Stack:** Rust, existing crate structure (glyim-hir, glyim-codegen-llvm, glyim-parse)

## File Structure

Before this plan, `crates/glyim-hir/src/` has:
```
lib.rs      — re-exports from node and lower
lower.rs    — AST → HIR lowering (takes &Ast, &Interner)
node.rs     — HirExpr (9 variants), HirStmt, HirFn, Hir, HirBinOp, HirUnOp
```

After this plan:
```
lib.rs           — re-exports from node and lower (updated with new types)
lower.rs         — AST → HIR lowering (updated: type_params, Hir { items })
node/
  mod.rs         — pure re-export hub
  id.rs          — ExprId newtype
  ty.rs          — HirType enum
  pattern.rs     — HirPattern enum
  item.rs        — HirItem, Hir, HirStructDef, HirEnumDef, HirImplDef, HirExternBlock, HirVariant, HirVariantField, HirExternFn
  expr.rs        — HirExpr (all 9 variants), HirStmt, HirBinOp, HirUnOp (exact copy from node.rs)
  fn_def.rs      — HirFn (with new type_params field)
```

No other files in any other crate change.

use glyim_interner::Symbol;

#[derive(Debug, Clone, PartialEq, Eq)]

    Fn(HirFn),
    Struct(HirStructDef),
    Enum(HirEnumDef),
    Impl(HirImplDef),
    ExternBlock(HirExternBlock),

pub struct HirStructDef {
    pub name: Symbol,
    pub type_params: Vec<Symbol>,
    pub fields: Vec<(Symbol, HirType)>,
    pub span: Span,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct HirEnumDef {
    pub name: Symbol,
    pub type_params: Vec<Symbol>,
    pub variants: Vec<HirVariant>,
    pub span: Span,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct HirVariant {
    pub name: Symbol,
    pub fields: Vec<HirVariantField>,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum HirVariantField {
    Unnamed(HirType),

use glyim_interner::Interner;
use glyim_parse::{BinOp, BlockItem, ExprKind, Item, StmtKind, UnOp};
use crate::{Hir, HirBinOp, HirExpr, HirFn, HirItem, HirStmt, HirUnOp};

pub fn lower(ast: &glyim_parse::Ast, interner: &Interner) -> Hir {
    let mut fns = vec![];
    for item in &ast.items {
        match item {
            Item::Binding { name, value, .. } => {
                if let ExprKind::Lambda { params, body } = &value.kind {
                    fns.push(HirFn { name: *name, type_params: vec![], params: params.clone(), body: lower_expr(&body.kind, interner) });
                }
            }
            Item::FnDef { name, params, body, .. } => {
                let param_syms: Vec<_> = params.iter().map(|(sym,_)| *sym).collect();
                fns.push(HirFn { name: *name, type_params: vec![], params: param_syms, body: lower_expr(&body.kind, interner) });
            }
            Item::Stmt(_stmt) => {}
            Item::Use(_) => {}
        }
    }
    Hir { items: fns.into_iter().map(HirItem::Fn).collect() }
}

fn lower_expr(expr: &ExprKind, interner: &Interner) -> HirExpr {
    match expr {
        ExprKind::IntLit(n) => HirExpr::IntLit(*n),
        ExprKind::StrLit(s) => HirExpr::StrLit(s.clone()),
        ExprKind::Ident(sym) => HirExpr::Ident(*sym),
        ExprKind::Binary { op, lhs, rhs } => HirExpr::Binary {
            op: lower_binop(op.clone()),
            lhs: Box::new(lower_expr(&lhs.kind, interner)),
            rhs: Box::new(lower_expr(&rhs.kind, interner)),
        },
        ExprKind::Unary { op, operand } => HirExpr::Unary {
            op: lower_unop(op.clone()),
            operand: Box::new(lower_expr(&operand.kind, interner)),
        },
        ExprKind::Lambda { params: _, body } => lower_expr(&body.kind, interner),
        ExprKind::Block(items) => {
            let stmts: Vec<HirStmt> = items.iter().map(|item| match item {
                BlockItem::Expr(e) => HirStmt::Expr(lower_expr(&e.kind, interner)),
                BlockItem::Stmt(s) => match &s.kind {
                    StmtKind::Let { name, mutable, value } => HirStmt::Let {
                        name: *name, mutable: *mutable, value: lower_expr(&value.kind, interner),
                    },
                    StmtKind::Assign { target, value } => HirStmt::Assign {
                        target: *target, value: lower_expr(&value.kind, interner),
                    },
                },
            }).collect();
            HirExpr::Block(stmts)
        }
        ExprKind::If { condition, then_branch, else_branch } => HirExpr::If {
            condition: Box::new(lower_expr(&condition.kind, interner)),
            then_branch: Box::new(lower_expr(&then_branch.kind, interner)),
            else_branch: else_branch.as_ref().map(|e| Box::new(lower_expr(&e.kind, interner))),
        },
        ExprKind::Call { callee, args } => {
            if let ExprKind::Ident(sym) = &callee.kind {
                let name = interner.resolve(*sym);
                match name {
                    "println" => {
                        let arg = args.first().map(|a| lower_expr(&a.kind, interner)).unwrap_or(HirExpr::IntLit(0));
                        return HirExpr::Println(Box::new(arg));
                    }
                    "assert" => {
                        let cond = args.first().map(|a| lower_expr(&a.kind, interner)).unwrap_or(HirExpr::IntLit(0));
                        let msg = args.get(1).map(|a| lower_expr(&a.kind, interner)).map(Box::new);
                        return HirExpr::Assert { condition: Box::new(cond), message: msg };
                    }
                    _ => {}
                }
            }
            HirExpr::IntLit(0) // fallback for non-builtin calls
        }
    }
}

fn lower_binop(op: BinOp) -> HirBinOp {
    match op {
        BinOp::Add => HirBinOp::Add, BinOp::Sub => HirBinOp::Sub, BinOp::Mul => HirBinOp::Mul,
        BinOp::Div => HirBinOp::Div, BinOp::Mod => HirBinOp::Mod, BinOp::Eq => HirBinOp::Eq,
        BinOp::Neq => HirBinOp::Neq, BinOp::Lt => HirBinOp::Lt, BinOp::Gt => HirBinOp::Gt,
        BinOp::Lte => HirBinOp::Lte, BinOp::Gte => HirBinOp::Gte, BinOp::And => HirBinOp::And,
        BinOp::Or => HirBinOp::Or,
    }
}
fn lower_unop(op: UnOp) -> HirUnOp { match op { UnOp::Neg => HirUnOp::Neg, UnOp::Not => HirUnOp::Not } }
