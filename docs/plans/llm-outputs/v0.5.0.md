# v0.5.0 Phase 1: Foundation — Attributes, Alloc Shims, DWARF Line Tables Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `#[test]` attribute parsing, heap allocation shim declarations, `--debug`/`--release` CLI flags, and DWARF line table generation so debuggers can show function names and source locations.

**Architecture:** Parser gains `#[ident]` and `#[ident(key=val)]` attribute nodes on items. Codegen emits `malloc`/`free` extern declarations. DIBuilder creates per-function `DISubprogram` entries. CLI gains `--debug` and `--release` flags controlling optimization level and DWARF emission.

**Non-goals (deferred):** Variable-level DWARF, test harness binary, `glyim_alloc`/`glyim_free` wrapper implementations, source-line mapping (needs spans on HIR nodes from v0.3.0 type checker work).

**API design constraint:** All new APIs are additive. `Codegen::new` gains a sibling `Codegen::with_debug`. `pipeline::run` gains a sibling `pipeline::run_with_mode`. Zero existing call sites change. All 17 integration tests pass without modification.

**Tech Stack:** Rust, inkwell 0.9 DIBuilder, LLVM 22 DWARF, existing crate structure

fn parse_attributes(&mut self) -> Vec<Attribute> {
    let mut attrs = vec![];
    while self.tokens.at(SyntaxKind::Hash) {
        let hash_tok = match self.tokens.bump() {
            Some(t) => t,
            None => break,
        };
        let start = hash_tok.start;
        if let Err(e) = self.tokens.expect(SyntaxKind::OpenBracket) {
            self.errors.push(e);
            break;
        }
        let name_tok = match self.tokens.expect(SyntaxKind::Ident) {
            Ok(t) => t,
            Err(e) => { self.errors.push(e); break; }
        };
        let name = name_tok.text.to_string();

        let mut args = vec![];
        if self.tokens.eat(SyntaxKind::LParen) {
            loop {
                if self.tokens.at(SyntaxKind::RParen) {
                    self.tokens.bump();
                    break;
                }
                if self.tokens.is_eof() {
                    break;
                }
                let key_tok = match self.tokens.expect(SyntaxKind::Ident) {
                    Ok(t) => t,
                    Err(e) => { self.errors.push(e); self.tokens.eat(SyntaxKind::RParen); break; }
                };
                let key = key_tok.text.to_string();
                let value = if self.tokens.eat(SyntaxKind::Eq) {
                    match self.tokens.peek() {
                        Some(val_tok) => {
                            let val_str = val_tok.text.to_string();
                            self.tokens.bump();
                            Some(val_str)
                        }
                        None => None,
                    }
                } else {
                    None
                };
                args.push(AttributeArg {
                    key,
                    value,
                    span: Span::new(key_tok.start, key_tok.end),
                });
                self.tokens.eat(SyntaxKind::Comma);
            }
        }
        let end_tok = match self.tokens.expect(SyntaxKind::CloseBracket) {
            Ok(t) => t,
            Err(e) => {
                self.errors.push(e);
                Attribute { name, args, span: Span::new(start, self.tokens.peek().map_or(start, |t| t.start)) }
                    .pipe(|attr| { attrs.push(attr); break; })

    pub fn new(
        module: &inkwell::module::Module<'ctx>,
        file_path: &str,

            true,
            DWARFSourceLanguage::C,
            file_path,
            ".",
            "Glyim 0.5.0",
            false,
            "",
            0,
            "",
            DWARFEmissionKind::LineTablesOnly,
            0,

            file,
            "Glyim 0.5.0",
            false,
            "",
            0,
            "",
            DWARFEmissionKind::LineTablesOnly,
            false,
            false,
        );

        Some(Self { dibuilder, compile_unit, file })

    pub fn create_function_subprogram(
        &self,
        name: &str,
        line: u32,
    ) -> Option<inkwell::debug_info::DISubprogram<'ctx>> {

            self.file,
            None, // void return

            self.compile_unit, // scope
            name,
            None,              // linkage name
            self.file,
            line,
            fn_type,
            true,              // is_local_to_unit
            true,              // is_definition
            line,              // scope line
            DIFlags::Zero,
            false,             // is_optimized
        ))
    }

    pub fn finalize(&self) {

    }
