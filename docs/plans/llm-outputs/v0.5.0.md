# v0.5.0 Phase 1: Foundation — Attributes, Alloc Shims, DWARF Module Metadata

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `#[test]` attribute parsing, heap allocation shim declarations, `--debug`/`--release` CLI flags, and DWARF module-level metadata so debuggers can show function names in backtraces.

**Architecture:** Parser gains `#[ident]` and `#[ident(key=val)]` attribute nodes on items. Codegen emits `malloc`/`free` extern declarations. DIBuilder creates module-level DWARF metadata (DICompileUnit, DIFile). CLI gains `--debug` and `--release` flags controlling optimization level. Debug info is emitted via `DebugInfoGen` with auto-finalize on drop.

**Non-goals (deferred):** Per-function DISubprograms (inkwell API uncertain), per-instruction debug locations (no spans on HIR nodes), `glyim_alloc`/`glyim_free` wrapper implementations, variable-level DWARF, test harness binary.

**DWARF scope decision:** Module-level metadata only. `gdb` shows function names in backtraces via symbol table. No source-line mapping (no spans on HIR nodes). This matches the plan's stated non-goals. Per-function metadata will be added when spans are available from the v0.3.0 type checker work.

**Inkwell API uncertainty:** The `create_compile_unit` parameter order in inkwell 0.9 with llvm22-1 is uncertain. The plan writes the best guess and marks the location for iteration. All DWARF tests verify compilation succeeds, not IR content.

**Corrections from original draft:**
- Bug 1: `parse_item` must include `KwExtern` and `At` branches (original draft silently dropped them)
- Bug 2: Removed — Phase 4 will handle `with_target`; Phase 1 only defines `with_debug`
- Bug 3: Removed — IR generation is target-independent; only `write_object_file` validates triples
- DWARF: Simplified from per-function DISubprograms to module-level-only metadata. Avoids all uncertain inkwell API calls (`set_subprogram`, `create_function` with linkage)
- DWARF test: Checks for `"Dwarf Version"` in module flags instead of `!dbg` (more reliable across LLVM versions)

**Tech Stack:** Rust, inkwell 0.9 DIBuilder, LLVM 22 DWARF line tables, existing crate structure

```rust
fn parse_attributes(&mut self) -> Vec<Attribute> {
    let mut attrs = vec![];
    while self.tokens.at(SyntaxKind::Hash) {
        let hash_tok = match self.tokens.bump() {
            Some(t) => t,
            None => break,
        };
        let start = hash_tok.start;
        if let Err(e) = self.tokens.expect(SyntaxKind::OpenBracket) {
            self.errors.push(e);
            break;
        }
        let name_tok = match self.tokens.expect(SyntaxKind::Ident) {
            Ok(t) => t,
            Err(e) => { self.errors.push(e); break; }
        };
        let name = name_tok.text.to_string();

        let mut args = vec![];
        if self.tokens.eat(SyntaxKind::LParen) {
            loop {
                if self.tokens.at(SyntaxKind::RParen) {
                    self.tokens.bump();
                    break;
                }
                if self.tokens.is_eof() {
                    break;
                }
                let key_tok = match self.tokens.expect(SyntaxKind::Ident) {
                    Ok(t) => t,
                    Err(e) => { self.errors.push(e); self.tokens.eat(SyntaxKind::RParen); break; }
                };
                let key = key_tok.text.to_string();
                let value = if self.tokens.eat(SyntaxKind::Eq) {
                    match self.tokens.peek() {
                        Some(val_tok) => {
                            let val_str = val_tok.text.to_string();
                            self.tokens.bump();
                            Some(val_str)
                        }
                        None => None,
                    }
                } else {
                    None
                };
                args.push(AttributeArg { key, value, span: Span::new(key_tok.start, key_tok.end) });
                self.tokens.eat(SyntaxKind::Comma);
            }
        }

        let end = match self.tokens.expect(SyntaxKind::CloseBracket) {
            Ok(t) => t.end,
            Err(e) => { self.errors.push(e); self.tokens.peek().map_or(start, |t| t.start) }
        };
        attrs.push(Attribute { name, args, span: Span::new(start, end) });
    }
    attrs
}
```

```rust
fn parse_item(&mut self) -> Option<Item> {
    let attrs = self.parse_attributes();
    match self.tokens.peek()?.kind {
        SyntaxKind::KwFn => self.parse_fn_def_with_attrs(attrs),
        SyntaxKind::KwStruct => self.parse_struct_def(),
        SyntaxKind::KwEnum => self.parse_enum_def(),
        SyntaxKind::KwExtern => self.parse_extern_block(),
        SyntaxKind::KwLet => self.parse_let_stmt().map(Item::Stmt),
        SyntaxKind::KwUse => self.parse_use_item().map(Item::Use),
        SyntaxKind::At => self.parse_macro_def(),
        SyntaxKind::Ident => {
            if self.tokens.peek2().is_some_and(|t| t.kind == SyntaxKind::Eq) {
                self.parse_binding_with_attrs(attrs)
            } else {
                None
            }
        }
        _ => None,
    }
}
```

```rust
fn parse_fn_def_with_attrs(&mut self, attrs: Vec<Attribute>) -> Option<Item> {
    self.tokens.bump()?; // consume 'fn'
    let name_tok = match self.tokens.expect(SyntaxKind::Ident) { Ok(t) => t, Err(e) => { self.errors.push(e); return None; } };
    let name = self.interner.intern(name_tok.text);
    let name_span = Span::new(name_tok.start, name_tok.end);
    if let Err(e) = self.tokens.expect(SyntaxKind::LParen) { self.errors.push(e); return None; }
    let mut params = vec![];
    while !self.tokens.at(SyntaxKind::RParen) {
        let tok = match self.tokens.expect(SyntaxKind::Ident) { Ok(t) => t, Err(e) => { self.errors.push(e); break; } };
        params.push((self.interner.intern(tok.text), Span::new(tok.start, tok.end)));
        self.tokens.eat(SyntaxKind::Comma);
    }
    if let Err(e) = self.tokens.expect(SyntaxKind::RParen) { self.errors.push(e); }
    let body = self.parse_block_expr()?;
    Some(Item::FnDef { name, name_span, params, body, attrs })
}
```

```rust
fn parse_binding_with_attrs(&mut self, attrs: Vec<Attribute>) -> Option<Item> {
    let name_tok = self.tokens.bump()?;

    let name_span = Span::new(name_tok.start, name_tok.end);
```

```rust
use inkwell::context::Context;

/// Emit extern declarations for malloc and free.
/// Does NOT redeclare abort (already in emit_runtime_shims).
pub fn emit_alloc_shims(module: &inkwell::module::Module<'_>) {
    let i8_type = module.get_context().i8_type();
    let i64_type = module.get_context().i64_type();
    let void_type = module.get_context().void_type();
    let ptr_type = i8_type.ptr_type(inkwell::AddressSpace::from(0u16));

    let malloc_ty = ptr_type.fn_type(&[i64_type.into()], false);
    module.add_function("malloc", malloc_ty, None);

    let free_ty = void_type.fn_type(&[ptr_type.into()], false);
    module.add_function("free", free_ty, None);
}
```

```rust
pub fn run_with_mode(input: &Path, mode: BuildMode) -> Result<i32, PipelineError> {
    // Copy of run() body using mode for Codegen::with_debug
    let source = fs::read_to_string(input)?;
    let mut parse_out = glyim_parse::parse(&source);
    if !parse_out.errors.is_empty() {
        let rendered = glyim_diag::render_diagnostics(
            &source,
            &input.to_string_lossy(),
            &parse_out.errors.iter().map(|e| {
                let span = match e {
                    glyim_parse::ParseError::Expected { span, .. } => Some(glyim_diag::Span::new(span.0, span.1)),
                    glyim_parse::ParseError::UnexpectedEof { .. } => None,
                    glyim_parse::ParseError::ExpectedExpr { span, .. } => Some(glyim_diag::Span::new(span.0, span.1)),
                    glyim_parse::ParseError::Message { span, .. } => Some(glyim_diag::Span::new(span.0, span.1)),
                };
                glyim_diag::Diagnostic::error(e.to_string()).with_span_opt(span)
            }).collect::<Vec<_>>(),
        );
        eprintln!("{rendered}");
        return Err(PipelineError::Parse(parse_out.errors));
    }
    let hir = glyim_hir::lower(&parse_out.ast, &mut parse_out.interner);
    let context = Context::create();
    let mut codegen = Codegen::with_debug(&context, parse_out.interner, !mode.is_release());
    codegen.generate(&hir).map_err(PipelineError::Codegen)?;
    let tmp_dir = tempfile::tempdir()?;
    let obj_path = tmp_dir.path().join("output.o");
    codegen.write_object_file(&obj_path).map_err(PipelineError::Codegen)?;
    let exe_path = tmp_dir.path().join("glyim_out");
    link_object(&obj_path, &exe_path)?;
    let status = Command::new(&exe_path).status().map_err(PipelineError::Run)?;
    Ok(status.code().unwrap_or(1))
}

pub fn build(input: &Path, output: Option<&Path>) -> Result<PathBuf, PipelineError> {
    Self::build_with_mode(input, output, BuildMode::Debug)
}

pub fn build_with_mode(input: &Path, output: Option<&Path>, mode: BuildMode) -> Result<PathBuf, PipelineError> {
    // Copy of build() body using mode for Codegen::with_debug
    let source = fs::read_to_string(input)?;
    let (hir, _ir, interner) = compile_to_hir_and_ir(&source)?;
    let output = output.map(|p| p.to_path_buf()).unwrap_or_else(|| {
        let stem = input.file_stem().unwrap_or_default().to_string_lossy().to_string();
        PathBuf::from(stem)
    });
    let tmp_dir = tempfile::tempdir()?;
    let obj_path = tmp_dir.path().join("output.o");
    let context = Context::create();
    let mut codegen = Codegen::with_debug(&context, interner, !mode.is_release());
    codegen.generate(&hir).map_err(PipelineError::Codegen)?;
    codegen.write_object_file(&obj_path).map_err(PipelineError::Codegen)?;
    link_object(&obj_path, &output)?;
    Ok(output)
}
```

```rust
use std::fmt;

/// Parsed contents of a `glyim.toml` [package] section.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct PackageManifest {
    pub name: String,
    pub version: String,
}

/// Errors from manifest parsing.
#[derive(Debug, Clone)]
pub enum ManifestError {
    /// The TOML is syntactically invalid.
    Parse(String),
    /// The `[package]` table is missing.
    MissingSection(&'static str),
    /// A required field is missing or has the wrong type.
    MissingField(&'static str),
}

impl fmt::Display for ManifestError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            Self::Parse(msg) => write!(f, "invalid TOML in glyim.toml: {msg}"),
            Self::MissingSection(section) => write!(f, "missing [{section}] section in glyim.toml"),
            Self::MissingField(field) => write!(f, "missing required field '{field}' in glyim.toml"),
        }
    }
}

impl std::error::Error for ManifestError {}

/// Parse a `glyim.toml` string into a `PackageManifest`.
pub fn parse_manifest(toml_str: &str) -> Result<PackageManifest, ManifestError> {
    let value: toml::Value = toml_str
        .parse()
        .map_err(|e| ManifestError::Parse(e.to_string()))?;

    let package = value
        .get("package")
        .ok_or(ManifestError::MissingSection("package"))?;

    let name = package
        .get("name")
        .and_then(|v| v.as_str())
        .ok_or(ManifestError::MissingField("package.name"))?
        .to_string();

    let version = package
        .get("version")
        .and_then(|v| v.as_str())
        .unwrap_or("0.0.0")
        .to_string();

    Ok(PackageManifest { name, version })
}
```

```rust
/// Walk from `start` upward looking for `glyim.toml`.
/// Returns the directory containing `glyim.toml`, or None.
pub fn find_package_root(start: &Path) -> Option<PathBuf> {
    let mut current = if start.is_file() {
        start.parent()?.to_path_buf()
    } else {
        start.to_path_buf()
    };
    loop {
        if current.join("glyim.toml").exists() {
            return Some(current);
        }
        if !current.pop() {
            return None;
        }
    }
}
```

```rust
pub fn build_package(
    package_dir: &Path,
    output: Option<&Path>,
    mode: BuildMode,
) -> Result<PathBuf, PipelineError> {
    let manifest_path = package_dir.join("glyim.toml");
    let toml_str = fs::read_to_string(&manifest_path).map_err(|e| {
        if e.kind() == std::io::ErrorKind::NotFound {
            PipelineError::Manifest(crate::manifest::ManifestError::MissingSection("package"))
        } else {
            PipelineError::Io(e)
        }
    })?;
    let _manifest = crate::manifest::parse_manifest(&toml_str).map_err(PipelineError::Manifest)?;

    let main_path = package_dir.join("src").join("main.g");
    if !main_path.exists() {
        return Err(PipelineError::Manifest(crate::manifest::ManifestError::MissingField(
            "src/main.g",
        )));
    }

    Self::build_with_mode(&main_path, output, mode)
}
```

```rust
    interner: &glyim_interner::Interner,

                }
            }
            tests.push(TestFunction {
                name: interner.resolve(*name).to_string(),
                ignored: is_ignored,
            });
        }
    }
```

```rust
pub fn generate_for_tests(
    &mut self,
    hir: &Hir,
    test_names: &[String],
    ignored_names: &[String],
) -> Result<(), String> {
    crate::runtime_shims::emit_runtime_shims(self.context, &self.module);
    crate::alloc::emit_alloc_shims(&self.module);

    let should_panic: std::collections::HashSet<String> = test_names
        .iter()
        .filter_map(|item| {
            if let Item::FnDef { attrs, name, .. } = item {
                attrs.iter().any(|a| a.name == "should_panic")
            } else {
                false
            }
        })
        .cloned();

    for item in &hir.items {
        match item {
            glyim_hir::item::HirItem::Fn(f) => {
                let name = self.interner.resolve(f.name);
                // Skip user's main — the harness provides it
                if name == "main" {
                    continue;
                }
                self.codegen_fn(f)?;
```

```rust
fn emit_test_harness(
    &mut self,
    active_names: &[String],
    ignored_names: &[String],
    should_panic: &std::collections::HashSet<String>,
) {
    let i32_type = self.i32_type;
    let main_type = i32_type.fn_type(&[], false);
    let main_fn = self.module.add_function("main", main_type, None);
    let entry = self.context.append_basic_block(main_fn, "entry");
    self.builder.position_at_end(entry);

    // Print header
    let count = active_names.len() + ignored_names.len();
    let count_str = count.to_string();
    let header = format!("running {} tests\n", count_str);
    self.emit_printf_call(&header);

    // alloca for any_failed flag
    let any_failed = self
        .builder
        .build_alloca(i32_type, "any_failed")
        .unwrap();
    self.builder
        .build_store(any_failed, i32_type.const_int(0, false))
        .unwrap();

    let ignored_set: std::collections::HashSet<&str> = ignored_names.iter().map(|s| s.as_str()).collect();

    for test_name in active_names {
        // Print "test {name} ... "
        let msg = format!("test {} ... ", test_name);
        self.emit_printf_call(&msg);

        if ignored_set.contains(test_name.as_str()) {
            self.emit_printf_call("ignored\n");
            continue;
        }

        let is_should_panic = should_panic.contains(test_name.as_str());

        // Call the test function
        let fn_val = match self.module.get_function(test_name) {
            Some(f) => f,
            None => {
                self.emit_printf_call("ERROR: test function not found\n");
                self.builder
                    .build_store(any_failed, i32_type.const_int(1, false))
                    .unwrap();
                continue;
            }
        };
        let result = self
            .builder
            .build_call(fn_val, &[], "test_result")
            .unwrap()
            .try_as_basic_value()
            .left()
            .unwrap()
            .into_int_value();

        // should_panic: panic returns 0, non-zero = fail
        if is_should_panic {
            let is_zero = self
                .builder
                .build_int_compare(
                    IntPredicate::EQ,
                    result,
                    self.i64_type.const_int(0, false),
                    "is_zero",
                )
                .unwrap();
            if is_zero {
                self.emit_printf_call("FAILED\n");
                self.builder
                    .build_store(any_failed, i32_type.const_int(1, false))
                    .unwrap();
            } else {
                self.emit_printf_call("ok\n");
            }
        } else {
            let is_fail = self
                .builder
                .build_int_compare(
                    IntPredicate::NE,
                    result,
                    self.i64_type.const_int(0, false),
                    "is_fail",
                )
                .unwrap();
            if is_fail {
                self.emit_printf_call("FAILED\n");
                self.builder
                    .build_store(any_failed, i32_type.const_int(1, false))
                    .unwrap();
            } else {
                self.emit_printf_call("ok\n");
            }
        }
    }

    // Print summary
    let n_passed = active_names.len() - ignored_set.len();
    let n_ignored = ignored_names.len();
    let summary_msg = format!(
        "\ntest result: {}",
        if self.builder.get_insert_block().is_none() { "ok" } else { "FAILED" },
        ". {} passed; {} failed; {} ignored\n",
        n_passed, n_failed, n_ignored,
    );
    self.emit_printf_call(&summary_msg);

    // return *any_failed
    let failed_val = self
        .builder
        .build_load(i32_type, any_failed, "failed_val")
        .unwrap()
        .into_int_value();
    self.builder.build_return(Some(&failed_val)).unwrap();
}

impl Drop for Codegen<'ctx> {
    fn drop(&mut self) {
        if let Some(ref di) = self.debug_info {
            di.finalize();
        }
    }
}
```

# v0.5.0 Phases 3–6: Test Harness, Test Runner, Pipeline Integration, CLI Subcommand

These four phases implement the `#[test]` execution pipeline end-to-end: codegen emits a test harness `main`, a runner module collects test functions from AST attributes, the pipeline wires parsing→collection→codegen→execution, and the CLI gains a `test` subcommand.

**Dependency chain:** Phase 1 (attributes + alloc shims) must complete first. Phase 2 (manifest) is independent. Phases 3→4→5→6 are sequential.

**Key architectural decision:** Attributes live on the AST but NOT on the HIR (lowering uses `..` to discard them). Test function names are collected from the AST *before* lowering and passed as `&[String]` to `generate_for_tests`. This avoids modifying the HIR or lowering pass.

**Key architectural decision:** `#[should_panic]` is NOT supported in Phases 3–6. The harness treats return-0 as pass, return-nonzero as fail. A test that calls `abort()` (via assert failure) kills the process — the harness output is truncated. Plan 7 adds `#[should_panic]` awareness.

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `generate_for_tests()` to Codegen that emits all user functions (skipping `main`), then generates a test-harness `main` that calls each test function, checks return values, prints pass/fail per test, and returns non-zero if any test failed.

**What the generated IR looks like:**
```llvm
; User's test function (unchanged)
define i64 @check() { ... }

; Test harness main (generated)
define i32 @main() {
entry:
  call i32 @printf(%s, "running 1 tests\n")
  %any_failed = alloca i32
  store i32 0, i32* %any_failed
  call i32 @printf(%s, "test check ... ")
  %result = call i64 @check()
  %is_fail = icmp ne i64 %result, 0
  br i1 %is_fail, label %fail, label %pass
fail:
  call i32 @printf(%s, "FAILED\n")
  store i32 1, i32* %any_failed
  br label %next
pass:
  call i32 @printf(%s, "ok\n")
  br label %next
next:
  call i32 @printf(%s, "\ntest result: ok. 1 passed; 0 failed; 0 ignored\n")
  %failed = load i32, i32* %any_failed
  ret i32 %failed
}
```

```rust
            format!("test.str.{}", n)
        };
        let global = self.module.add_global(arr_type, Some(AddressSpace::from(0u16)), &name);
        let mut elems: Vec<_> = bytes.iter()
            .map(|b| i8_type.const_int(*b as u64, false))
            .collect();
        elems.push(i8_type.const_int(0, false)); // null terminator
        let arr = unsafe { inkwell::values::ArrayValue::new_const_array(&arr_type, &elems) };
        global.set_initializer(&arr);
        global.set_constant(true);
        global.set_linkage(inkwell::module::Linkage::Private);
        let zero = self.context.i32_type().const_int(0, false);
        unsafe {
            self.builder.build_gep(arr_type, global.as_pointer_value(), &[zero, zero], "str_ptr")
        }.map_err(|e| e.to_string())
    }

    /// Call printf(fmt_ptr, str_ptr). Both must be valid pointers.
    fn call_printf(
        &self,
        fmt_ptr: inkwell::values::PointerValue<'ctx>,
        str_ptr: inkwell::values::PointerValue<'ctx>,
    ) -> Result<(), String> {
        let printf = self.module.get_function("printf")
            .ok_or("printf not declared (missing runtime_shims?)")?;
        self.builder.build_call(printf, &[fmt_ptr.into(), str_ptr.into()], "printf")
```

```rust
    /// then emit a test-harness `main` that calls each test function.
    ///
    /// `test_names` is a list of function names (resolved strings) that should be
    /// treated as test entries. These names are collected from AST attributes
    /// before HIR lowering.
    pub fn generate_for_tests(&mut self, hir: &Hir, test_names: &[String]) -> Result<(), String> {
        crate::runtime_shims::emit_runtime_shims(self.context, &self.module);
        crate::alloc::emit_alloc_shims(&self.module);
        types::register_builtin_enums(self);

        for item in &hir.items {
            match item {
                glyim_hir::item::HirItem::Fn(f) => {
                    let name = self.interner.resolve(f.name);
                    if name == "main" {
                        continue; // Test harness provides its own main
                    }
                    function::codegen_fn(self, f)?;
                }
                glyim_hir::item::HirItem::Struct(s) => types::codegen_struct_def(self, s),
                glyim_hir::item::HirItem::Enum(e) => types::codegen_enum_def(self, e),
                glyim_hir::item::HirItem::Extern(_) => {}
                glyim_hir::item::HirItem::Impl(_) => {}
            }
        }

        self.emit_test_harness(test_names)?;
        Ok(())
    }
```

```rust
    /// Emit a `main` function that acts as a test harness.
    ///
    /// For each test name: prints "test {name} ... ", calls the function,
    /// prints "ok\n" if it returns 0 or "FAILED\n" if non-zero.
    /// Returns non-zero exit code if any test failed.
    ///
    /// Known limitation: if a test calls abort() (e.g. via failed assert),
    /// the process exits immediately and the harness output is truncated.
    /// #[should_panic] support is deferred to Phase 7.
    fn emit_test_harness(&mut self, test_names: &[String]) -> Result<(), String> {
        use inkwell::{AddressSpace, IntPredicate};

        if test_names.is_empty() {
            return Err("no test functions to generate harness for".into());
        }

        let i32_type = self.i32_type;
        let i64_type = self.i64_type;
        let zero32 = i32_type.const_int(0, false);
        let zero64 = i64_type.const_int(0, false);
        let one32 = i32_type.const_int(1, false);

        // Create main function
        let main_type = i32_type.fn_type(&[], false);
        let main_fn = self.module.add_function("main", main_type, None);
        let entry = self.context.append_basic_block(main_fn, "entry");
        self.builder.position_at_end(entry);

        // Create "%s" printf format string
        let fmt_ptr = self.create_c_string_global("%s")?;

        // Print header
        let header = format!("running {} tests\n", test_names.len());
        let header_ptr = self.create_c_string_global(&header)?;
        self.call_printf(fmt_ptr, header_ptr)?;

        // alloca for any_failed flag
        let any_failed = self.builder.build_alloca(i32_type, "any_failed")
            .map_err(|e| e.to_string())?;
        self.builder.build_store(any_failed, zero32)
            .map_err(|e| e.to_string())?;

        for test_name in test_names {
            // Print "test {name} ... "
            let msg = format!("test {} ... ", test_name);
            let msg_ptr = self.create_c_string_global(&msg)?;
            self.call_printf(fmt_ptr, msg_ptr)?;

            // Call the test function (non-main functions return i64)
            let fn_val = self.module.get_function(test_name)
                .ok_or_else(|| format!("test function '{}' not found in module", test_name))?;
            let call_result = self.builder.build_call(fn_val, &[], "test_result")
                .map_err(|e| e.to_string())?;
            let result_val = call_result.try_as_basic_value().left()
                .ok_or_else(|| format!("test function '{}' returned void, expected i64", test_name))?
                .into_int_value();

            // Compare return value to zero
            let is_fail = self.builder.build_int_compare(
                IntPredicate::NE, result_val, zero64, "is_fail"
            ).map_err(|e| e.to_string())?;

            // Create basic blocks for pass/fail/next
            let pass_bb = self.context.append_basic_block(main_fn, "test_pass");
            let fail_bb = self.context.append_basic_block(main_fn, "test_fail");
            let next_bb = self.context.append_basic_block(main_fn, "test_next");

            self.builder.build_conditional_branch(is_fail, fail_bb, pass_bb)
                .map_err(|e| e.to_string())?;

            // Fail path: print FAILED, set flag
            self.builder.position_at_end(fail_bb);
            let fail_ptr = self.create_c_string_global("FAILED\n")?;
            self.call_printf(fmt_ptr, fail_ptr)?;
            self.builder.build_store(any_failed, one32)
                .map_err(|e| e.to_string())?;
            self.builder.build_unconditional_branch(next_bb)
                .map_err(|e| e.to_string())?;

            // Pass path: print ok
            self.builder.position_at_end(pass_bb);
            let ok_ptr = self.create_c_string_global("ok\n")?;
            self.call_printf(fmt_ptr, ok_ptr)?;
            self.builder.build_unconditional_branch(next_bb)
                .map_err(|e| e.to_string())?;

            // Continue to next test
            self.builder.position_at_end(next_bb);
        }

        // Print summary
        let summary = format!(
            "\ntest result: ok. {} passed; 0 failed; 0 ignored\n",
            test_names.len()
        );
        let summary_ptr = self.create_c_string_global(&summary)?;
        self.call_printf(fmt_ptr, summary_ptr)?;

        // Return any_failed value
        let failed_val = self.builder.build_load(i32_type, any_failed, "failed_val")
            .map_err(|e| e.to_string())?
            .into_int_value();
        self.builder.build_return(Some(&failed_val))
            .map_err(|e| e.to_string())?;

        Ok(())
    }
```

```rust
pub fn compile_to_ir_tests(source: &str, test_names: &[String]) -> Result<String, String> {
    let out = glyim_parse::parse(source);

    let mut cg = Codegen::new(&ctx, interner, vec![]);
    cg.generate_for_tests(&hir, test_names)?;
    Ok(cg.ir_string())
}
```

```rust
/// A test function collected from the AST.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct TestFunction {
    pub name: String,
    pub ignored: bool,
}

/// Result of a single test execution.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum TestResult {
    Passed,
    Failed,
    Ignored,
}

/// Summary of a test run, with per-test results.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct TestRunSummary {
    pub results: Vec<(String, TestResult)>,
}

impl TestRunSummary {
    pub fn passed(&self) -> usize {
        self.results.iter().filter(|(_, r)| *r == TestResult::Passed).count()
    }

    pub fn failed(&self) -> usize {
        self.results.iter().filter(|(_, r)| *r == TestResult::Failed).count()
    }

    pub fn ignored(&self) -> usize {
        self.results.iter().filter(|(_, r)| *r == TestResult::Ignored).count()
    }

    pub fn total(&self) -> usize {
        self.results.len()
    }

    pub fn exit_code(&self) -> i32 {
        if self.failed() > 0 { 1 } else { 0 }
    }

    pub fn format_summary(&self) -> String {
        let passed = self.passed();
        let failed = self.failed();
        let ignored = self.ignored();
        format!(
            "\ntest result: {}. {} passed; {} failed; {} ignored",
            if failed == 0 { "ok" } else { "FAILED" },
            passed,
            failed,
            ignored,
        )
    }
}

/// Collect `#[test]` function names from the AST.
///
/// `filter_name`: if Some, only include tests whose name matches exactly.
/// `include_ignored`: if false, skip `#[ignore]` tests.
pub fn collect_test_functions(
    ast: &Ast,
    interner: &Interner,
    filter_name: Option<&str>,
    include_ignored: bool,
) -> Vec<TestFunction> {
    let mut tests = Vec::new();
    for item in &ast.items {
        if let Item::FnDef { attrs, name, .. } = item {
            let is_test = attrs.iter().any(|a| a.name == "test");
            if !is_test {
                continue;
            }
            let is_ignored = attrs.iter().any(|a| a.name == "ignore");
            if is_ignored && !include_ignored {
                continue;
            }
            let resolved = interner.resolve(*name).to_string();
            if let Some(ref filter) = filter_name {
                if resolved != *filter {
                    continue;
                }
            }
            tests.push(TestFunction {
                name: resolved,
                ignored: is_ignored,
            });
        }
    }
    tests
}
```

# Phase 5: Pipeline Test Integration

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `run_tests()` and `run_tests_package()` to the pipeline that parse source, collect test names from AST, lower to HIR, type-check, codegen with test harness, compile, link, execute, parse stdout to build a `TestRunSummary`.

**Prerequisites:** Phases 1, 3, 4 complete.

**Non-goals:** `--ignored`/`--filter` passthrough (Phase 7 adds these by updating the call sites), structured JSON output, test timeout, parallel execution.

**Key design:** The test harness prints lines like `test check ... ok` and `test other ... FAILED`. `run_tests` captures stdout, parses these lines, and builds individual `TestResult` entries. This gives accurate per-test results even though the harness itself only returns a single exit code.

**Known limitation:** If a test calls `abort()` (assert failure), the process exits immediately. The captured stdout will be truncated (missing the summary line). In this case, `run_tests` returns `Err(PipelineError::Run(...))` since the process was killed by a signal. This is acceptable for v0.5.0.

```rust
/// Parse test harness stdout into individual test results.
///
/// Expected format per line: `test {name} ... ok` or `test {name} ... FAILED`
/// Lines that don't match are ignored.
fn parse_test_output(stdout: &str) -> Vec<(String, crate::test_runner::TestResult)> {
    use crate::test_runner::TestResult;
    let mut results = Vec::new();
    for line in stdout.lines() {
        let line = line.trim();
        // Match "test {name} ... ok"
        if let Some(rest) = line.strip_prefix("test ") {
            if let Some(name_end) = rest.find(" ... ") {
                let name = &rest[..name_end];
                let status = &rest[name_end + 5..]; // after " ... "
                if status == "ok" {
                    results.push((name.to_string(), TestResult::Passed));
                } else if status == "FAILED" {
                    results.push((name.to_string(), TestResult::Failed));
                }
                // Other status strings are ignored (future: "ignored", etc.)
            }
        }
    }
    results
}
```

```rust
/// Run `#[test]` functions in a single .g file.
///
/// Returns a TestRunSummary with per-test results parsed from harness stdout.
/// If the process is killed (e.g. abort), returns a Run error.
pub fn run_tests(input: &Path) -> Result<crate::test_runner::TestRunSummary, PipelineError> {
    run_tests_with_options(input, None, false)
}

/// Internal: run tests with optional filter and ignored flags.
/// Phase 7 updates call sites to pass these through from CLI.
fn run_tests_with_options(
    input: &Path,
    filter_name: Option<&str>,
    include_ignored: bool,
) -> Result<crate::test_runner::TestRunSummary, PipelineError> {
    let source = format!("{}\n{}", PRELUDE, fs::read_to_string(input)?);
    let mut parse_out = glyim_parse::parse(&source);
    if !parse_out.errors.is_empty() {
        let rendered = glyim_diag::render_diagnostics(
            &source,
            &input.to_string_lossy(),
            &parse_out
                .errors
                .iter()
                .map(|e| {
                    let span = match e {
                        glyim_parse::ParseError::Expected { span, .. } => {
                            Some(glyim_diag::Span::new(span.0, span.1))
                        }
                        glyim_parse::ParseError::UnexpectedEof { .. } => None,
                        glyim_parse::ParseError::ExpectedExpr { span, .. } => {
                            Some(glyim_diag::Span::new(span.0, span.1))
                        }
                        glyim_parse::ParseError::Message { span, .. } => {
                            Some(glyim_diag::Span::new(span.0, span.1))
                        }
                    };
                    glyim_diag::Diagnostic::error(e.to_string()).with_span_opt(span)
                })
                .collect::<Vec<_>>(),
        );
        eprintln!("{rendered}");
        return Err(PipelineError::Parse(parse_out.errors));
    }

    // Collect test function names from AST (before lowering, since HIR drops attrs)
    let test_fns = crate::test_runner::collect_test_functions(
        &parse_out.ast,
        &parse_out.interner,
        filter_name,
        include_ignored,
    );
    if test_fns.is_empty() {
        return Err(PipelineError::Codegen(
            "no #[test] functions found".into(),
        ));
    }

    // Separate ignored from active (harness only calls active tests)
    let active_names: Vec<String> = test_fns
        .iter()
        .filter(|t| !t.ignored)
        .map(|t| t.name.clone())
        .collect();
    let ignored_names: Vec<String> = test_fns
        .iter()
        .filter(|t| t.ignored)
        .map(|t| t.name.clone())
        .collect();

    if active_names.is_empty() {
        // All tests are ignored — return summary with only ignored entries
        let results: Vec<_> = ignored_names
            .iter()
            .map(|n| (n.clone(), crate::test_runner::TestResult::Ignored))
            .collect();
        return Ok(crate::test_runner::TestRunSummary { results });
    }

    // Lower to HIR
    let hir = glyim_hir::lower(&parse_out.ast, &mut parse_out.interner);

    // Type check
    let mut typeck = TypeChecker::new(parse_out.interner.clone());
    if let Err(errs) = typeck.check(&hir) {
        return Err(PipelineError::TypeCheck(errs));
    }

    // Codegen with test harness
    let context = Context::create();
    let mut codegen = Codegen::new(&context, parse_out.interner, typeck.expr_types.clone());
    codegen
        .generate_for_tests(&hir, &active_names)
        .map_err(PipelineError::Codegen)?;

    // Compile and link
    let tmp_dir = tempfile::tempdir()?;
    let obj_path = tmp_dir.path().join("output.o");
    codegen
        .write_object_file(&obj_path)
        .map_err(PipelineError::Codegen)?;
    let exe_path = tmp_dir.path().join("glyim_test_out");
    link_object(&obj_path, &exe_path)?;

    // Execute and capture stdout
    let output = Command::new(&exe_path)
        .output()
        .map_err(PipelineError::Run)?;
    let stdout = String::from_utf8_lossy(&output.stdout);

    // Parse per-test results from stdout
    let mut results = parse_test_output(&stdout);

    // Add ignored entries (they weren't run, so not in stdout)
    for name in &ignored_names {
        results.push((name.clone(), crate::test_runner::TestResult::Ignored));
    }

    // If process was killed by signal, treat all active tests as failed
    if !output.status.success() && output.status.code().is_none() {
        for (_, ref mut result) in results.iter_mut() {
            if *result == crate::test_runner::TestResult::Passed {
                *result = crate::test_runner::TestResult::Failed;
            }
        }
    }

    Ok(crate::test_runner::TestRunSummary { results })
}
```

```rust
/// Run `#[test]` functions in a package directory (reads `glyim.toml`, finds `src/main.g`).
pub fn run_tests_package(
    package_dir: &Path,
) -> Result<crate::test_runner::TestRunSummary, PipelineError> {
    let main_path = package_dir.join("src").join("main.g");
    if !main_path.exists() {
        return Err(PipelineError::Io(std::io::Error::new(
            std::io::ErrorKind::NotFound,
            format!("{} not found", main_path.display()),
        )));
    }
    run_tests(&main_path)
}
```
