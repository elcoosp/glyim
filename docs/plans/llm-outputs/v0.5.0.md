# v0.5.0 Phase 1: Foundation — Attributes, Alloc Shims, DWARF Line Tables Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `#[test]` attribute parsing, heap allocation shims, `--debug`/`--release` CLI flags, and DWARF line table generation so stack traces show function names and source locations.

**Architecture:** Parser gains `#[ident]` and `#[ident(key=val)]` attribute nodes on items. Codegen emits `malloc`/`free` declarations and wrapper shims. DIBuilder creates per-function `DISubprogram` entries with source locations. CLI gains `--debug` and `--release` flags that control optimization level and DWARF emission.

**Tech Stack:** Rust, inkwell DIBuilder, LLVM DWARF, existing crate structure

```rust
fn parse_attributes(&mut self) -> Vec<Attribute> {
    let mut attrs = vec![];
    while self.tokens.at(SyntaxKind::Hash) {
        let hash_tok = self.tokens.bump().unwrap(); // #
        if let Err(e) = self.tokens.expect(SyntaxKind::OpenBracket) {
            self.errors.push(e);
            break;
        }
        let name_tok = match self.tokens.expect(SyntaxKind::Ident) {
            Ok(t) => t,
            Err(e) => { self.errors.push(e); break; }
        };
        let name = name_tok.text.to_string();

        let mut args = vec![];
        if self.tokens.eat(SyntaxKind::LParen) {
            loop {
                if self.tokens.at(SyntaxKind::RParen) { self.tokens.bump(); break; }
                let key_tok = match self.tokens.expect(SyntaxKind::Ident) {
                    Ok(t) => t,
                    Err(e) => { self.errors.push(e); break; }
                };
                let key = key_tok.text.to_string();
                let value = if self.tokens.eat(SyntaxKind::Eq) {
                    let val_tok = match self.tokens.peek() {
                        Some(t) => {
                            let t = *t;
                            self.tokens.bump();
                            t.text.to_string()
                        }
                        None => break,
                    };
                    Some(value)
                } else {
                    None
                };
                args.push(AttributeArg { key, value, span: Span::new(key_tok.start, key_tok.end) });
                self.tokens.eat(SyntaxKind::Comma);
            }
        }
        if let Err(e) = self.tokens.expect(SyntaxKind::CloseBracket) {
            self.errors.push(e);
        }
        attrs.push(Attribute {
            name,
            args,
            span: Span::new(hash_tok.start, hash_tok.end),
        });
    }
    attrs
}
```

pub enum BuildMode {

impl BuildMode {

        match self {
            BuildMode::Debug => inkwell::OptimizationLevel::None,
            BuildMode::Release => inkwell::OptimizationLevel::Aggressive,

    pub fn from_str(s: &str) -> Self {
        match s {
            "release" => BuildMode::Release,
            _ => BuildMode::Debug,
        }
    }
}

        line: u32,
    ) -> Option<DISubprogram<'ctx>> {
        let void_type = self.dibuilder.create_basic_type(
            self.dibuilder.create_void_type(),
            0,
            "void",
            DIFlags::Zero,
        ).ok()?;

        let fn_type = self.dibuilder.create_subroutine_type(
            self.compile_unit.get_file(),
            Some(void_type),
            &[],
            DIFlags::Zero,
        );

        Some(self.dibuilder.create_function(
            self.compile_unit.get_file().as_debug_info_scope(),
            name,
            None,
            self.compile_unit.get_file(),
            line,
            fn_type,
            false,
            true,  // is_local_to_unit
            line,
            DIFlags::Zero,
            false,
        ))
    }

    /// Finalize debug info. Must be called after all functions are codegen'd.
    pub fn finalize(&self) {
        self.dibuilder.finalize();
    }
}
