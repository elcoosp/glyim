# Type Checker & Monomorphizer Rewrite — Plan v5

> Complete rewrite addressing **every finding** from the harsh code review. No stubs, no undefined methods, no hacks, no silent correctness holes.

## Architectural Decisions (Review-Driven)

| Decision | Rationale |
|---|---|
| `HirExpr` remains an enum with macro-generated accessors | Struct-of-fields changes every match in the entire codebase (lowering, codegen, etc.). A macro eliminates the maintenance trap without a cross-crate refactor. |
| `TypeCheckContext` struct for submodules | Limits submodule access to only the fields they need. `frozen_*` and `generic_expr_ids` stay on the outer `TypeChecker`. |
| `Option<HirIndex>` on `TypeChecker` | Eliminates the stub-index-with-throwaway-Interner footgun. |
| `HashSet<ExprId>` tracks generic function expressions | Allows `validate_mono_input` to permit `Param` in generic outputs while rejecting it in non-generic outputs. |
| `validate_mono_input` takes `generic_expr_ids: &HashSet<ExprId>` | Fixes the fatal flaw where Param types in generic function templates were rejected. |
| `substitute_type_with` has NO `has_param` fast path | The fast path was a pessimization for structural monomorphic types (2× cost). The traversal is already efficient. |
| `WorkItem` derives `Hash`/`Eq` directly | Eliminates string-based dedup keys. `HirType` already derives `Hash`/`Eq`. No allocation on `push`. |
| `MangleTable` clones args on insert only | Documented tradeoff: clone on cache miss is acceptable for the expected table size (tens to hundreds of entries). |
| `MonoDriver` fields grouped into `MonoOutput`, `MonoState`, `MonoInput` | Reduces the god-object problem from 14 fields to 3 groups. |
| Discovery module is COMPLETE — every `HirExpr` variant handled | No catch-all. No silent dropping of generic calls in While/ForIn/Return/etc. |
| `discover_method_call` receives `&KnownSymbols` through walk chain | Eliminates the stub-KnownSymbols-with-wrong-Interner bug. |
| `MethodCall → Call` transformation deferred to codegen | SubstWalker no longer transforms MethodCall to Call, avoiding the call_type_args contract violation. |

## Walkthrough: `fn id<T>(x: T) -> T { x }` Through the Entire Pipeline

### 1. HIR Lowering
```text
HirFn {
    name: Symbol("id"),
    type_params: [Symbol("T")],
    params: [(Symbol("x"), HirType::Param(Symbol("T")))],
    param_mutability: [false],
    ret: Some(HirType::Param(Symbol("T"))),
    body: HirExpr::Ident { id: ExprId(42), name: Symbol("x"), span: ... },
}
```

### 2. Type Checking (`check_fn`)
```text
Step 1: Create fresh Infer vars for type params
  type_param_map = { TypeVar(0) → Symbol("T") }
  type_param_sub = { Symbol("T") → HirType::Infer(TypeVar(0)) }

Step 2: Substitute param/return types
  param type: Param(T) → Infer(?0)
  return type: Param(T) → Infer(?0)

Step 3: Insert into TypeEnv
  env.insert(Symbol("x"), HirType::Infer(TypeVar(0)), false)

Step 4: Type-check body
  infer_expr(Ident "x") → lookup env → Infer(?0)
  expr_types[ExprId(42)] = Infer(?0)

Step 5: Unify body type with return type
  unify(Infer(?0), Infer(?0)) → OK (same root)

Step 6: Record ExprId(42) in generic_expr_ids
  (because this function has type params)

Step 7: Freeze
  For ExprId(42), type is Infer(?0).
  Resolve Infer(?0): unbound (no concrete binding).
  CHECK: TypeVar(0) is in type_param_map → maps to Symbol("T").
  Result: replace Infer(?0) with Param(T).
  expr_types[ExprId(42)] = HirType::Param(Symbol("T"))
```

### 3. Type Checker Output
```text
TypeCheckOutput {
  expr_types: HashMap { ExprId(42) → Param(Symbol("T")) },
  call_type_args: HashMap { ExprId(99) → [HirType::Int] },
  generic_expr_ids: HashSet { ExprId(42) },
  index: HirIndex { ... },
  interner: Interner,  // MOVED, not cloned
  known: KnownSymbols, // MOVED, not cloned
}
```

### 4. Validation Boundary
```text
validate_mono_input checks:
  - ExprId(42): type is Param(T), but ExprId(42) IS in generic_expr_ids → OK ✓
  - ExprId(99): type args are [Int], no Infer/Param → OK ✓
  - No Infer types in output ✓
  - No Param types in NON-GENERIC output ✓
```

### 5. Monomorphization (specializing `id<i64>`)
```text
Step 1: Discover from main's call: id with type_args [Int]
  → WorkItem::FnSpecialize(Symbol("id"), [HirType::Int])

Step 2: process_fn_specialize("id", [Int])
  sub = { Symbol("T") → HirType::Int }
  mangled_name = "id$i64"

Step 3: SubstWalker walks body of id
  For ExprId(42) (the "x" ident):
    concrete_type_for(42) → substitute Param(T) → Int, concretize → Int
    Assign new ExprId(200), store output_expr_types[200] = Int

Step 4: Remap call_type_args (CONCRETIZED by driver before discovery)
  Raw type args [Param(T)] → substitute & concretize → [Int]

Step 5: Build output function
  HirFn { name: "id$i64", params: [("x", Int)], ret: Int, ... }
  No Param/Infer leaks.
```

Instead of 54 lines of boilerplate that must be updated for every new variant, use a declarative macro that lists each variant exactly once:

```rust
/// Macro to generate `get_id()` and `get_span()` on HirExpr.
/// Add new variants HERE — the compiler will flag any match elsewhere
/// that's missing the new variant, but these accessors stay in sync automatically.
macro_rules! impl_hir_expr_accessors {
    ($($variant:ident { $($field:ident),* $(,)? }),* $(,)?) => {
        impl HirExpr {
            /// Get the expression's unique ID.
            pub fn get_id(&self) -> ExprId {
                match self {
                    $(HirExpr::$variant { id, .. } => *id,)*
                }
            }

            /// Get the expression's source span.
            pub fn get_span(&self) -> Span {
                match self {
                    $(HirExpr::$variant { span, .. } => *span,)*
                }
            }
        }
    };
}

impl_hir_expr_accessors! {
    IntLit { id, span },
    FloatLit { id, span },
    BoolLit { id, span },
    StrLit { id, span },
    UnitLit { id, span },
    Ident { id, span },
    Binary { id, span },
    Unary { id, span },
    Block { id, span },
    If { id, span },
    Call { id, span },
    MethodCall { id, span },
    StructLit { id, span },
    EnumVariant { id, span },
    FieldAccess { id, span },
    As { id, span },
    Deref { id, span },
    Match { id, span },
    While { id, span },
    ForIn { id, span },
    Return { id, span },
    Println { id, span },
    Assert { id, span },
    SizeOf { id, span },
    TupleLit { id, span },
    AddrOf { id, span },

```

- [ ] **Step 7: Create `naming.rs` — Collision-free structural mangling using `$`**

```rust
//! Shared naming utilities for method names and type mangling.
//!
//! Used by both the type checker and the monomorphizer.
//! The type checker does NOT depend on the monomorphizer module —
//! it imports from this shared module instead.
//!
//! MANGling CONVENTION: `base$arg1_arg2_...`
//!
//! The `$` separator is ALREADY INVALID in source-language identifiers,
//! so it can never collide with user-defined names. This requires ZERO
//! lexer changes.
//!
//! Structural types use `$` prefixes (e.g., `$p`, `$o`, `$t`) which are
//! also invalid in source identifiers, guaranteeing that a user-defined
//! type named `ptr_i64` will never collide with the internal `$p$i64`.
//!
//! PANICS: `type_to_short_string` panics on `Infer` types.
//! Infer types must NEVER reach mangling — they are resolved by
//! freeze before any downstream consumer sees them.

use crate::types::HirType;
use glyim_interner::{Interner, Symbol};

/// Separator between base name and type arguments in mangled names.
pub const MANGLE_SEPARATOR: &str = "$";

/// Convert a type to a short string for use in mangled names.
///
/// # Panics
/// Panics if `ty` contains `Infer` — Infer must never reach mangling.
pub fn type_to_short_string(ty: &HirType, interner: &Interner) -> String {
    match ty {
        HirType::Int => "i64".into(),
        HirType::Bool => "bool".into(),
        HirType::Float => "f64".into(),
        HirType::Str => "str".into(),
        HirType::Unit => "unit".into(),
        HirType::Never => "never".into(),
        HirType::Error => "err".into(),
        HirType::Infer(var) => {
            panic!(
                "FATAL: Infer(?{}) reached type_to_short_string — this is a compiler bug. \
                 Infer types must be resolved before mangling. \
                 Freeze phase should have replaced all Infer vars.",
                var.raw_index()
            )
        }
        HirType::Param(sym) => interner.resolve(*sym).to_string(),
        HirType::Named(sym) => interner.resolve(*sym).to_string(),
        HirType::Generic(sym, args) => {
            let inner = args.iter()
                .map(|a| type_to_short_string(a, interner))
                .collect::<Vec<_>>()
                .join("_");
            format!("{}{}{}", interner.resolve(*sym), MANGLE_SEPARATOR, inner)
        }
        HirType::Tuple(elems) => format!(
            "$t{}{}",
            MANGLE_SEPARATOR,
            elems.iter()
                .map(|e| type_to_short_string(e, interner))
                .collect::<Vec<_>>()
                .join(MANGLE_SEPARATOR)
        ),
        HirType::RawPtr(inner) => format!(
            "$p{}{}",
            MANGLE_SEPARATOR,
            type_to_short_string(inner, interner)
        ),
        HirType::Option(inner) => format!(
            "$o{}{}",
            MANGLE_SEPARATOR,
            type_to_short_string(inner, interner)
        ),
        HirType::Result(ok, err) => format!(
            "$r{}{}{}{}",
            MANGLE_SEPARATOR,
            type_to_short_string(ok, interner),
            MANGLE_SEPARATOR,
            type_to_short_string(err, interner)
        ),
        HirType::Func(params, ret) => {
            let p = params.iter()
                .map(|a| type_to_short_string(a, interner))
                .collect::<Vec<_>>()
                .join("_");
            format!("$f{}{}_{}", MANGLE_SEPARATOR, p, type_to_short_string(ret, interner))
        }
        HirType::Opaque(sym) => format!("$x{}{}", MANGLE_SEPARATOR, interner.resolve(*sym)),
    }
}

/// Mangle a name with concrete type arguments.
/// Non-generic items (empty args) keep their original name.
pub fn mangle_name(
    interner: &mut Interner,
    base: Symbol,
    type_args: &[HirType],
) -> Symbol {
    if type_args.is_empty() {
        return base;
    }
    let base_str = interner.resolve(base).to_string();
    let args_str = type_args.iter()
        .map(|t| type_to_short_string(t, interner))
        .collect::<Vec<_>>()
        .join("_");
    interner.intern(&format!("{}{}{}", base_str, MANGLE_SEPARATOR, args_str))
}

/// Build a method base name: "TypeName_method_name"
///
/// Uses single `_` separator (not `$`) for method names.
/// The `$` separator is reserved for type argument mangling only.
pub fn method_base_name(
    interner: &mut Interner,
    type_name: Symbol,
    method_name: Symbol,
) -> Symbol {
    let s = format!("{}_{}", interner.resolve(type_name), interner.resolve(method_name));
    interner.intern(&s)
}

/// Extract the base (unmangled) name from a mangled symbol.
/// Takes `&mut Interner` because it may intern the base name.
pub fn demangle_base(interner: &mut Interner, mangled: Symbol) -> Symbol {
    let s = interner.resolve(mangled).to_string();
    if let Some(pos) = s.find(MANGLE_SEPARATOR) {
        interner.intern(&s[..pos])
    } else {
        mangled
    }
}

/// Check if a symbol contains the mangled separator.
pub fn is_mangled(interner: &Interner, sym: Symbol) -> bool {
    interner.resolve(sym).contains(MANGLE_SEPARATOR)
}
```

- [ ] **Step 4: Write `substitute_type_with` — NO has_param fast path**

The `has_param` fast path was a pessimization: for structural monomorphic types like `Generic("Vec", [Int])`, it traversed the tree twice (once for `has_param`, once for `clone`). Without the fast path, the match traverses once and clones only what it needs.

```rust
use std::collections::HashMap;
use glyim_interner::Symbol;

/// Generic substitution: for each `Param(sym)`, call `f(sym)`.
/// If `f` returns `Some(replacement)`, replace. If `None`, keep `Param(sym)`.
///
/// NOTE: No `has_param` fast path. The fast path was a pessimization for
/// structural monomorphic types (traversed twice instead of once).
/// The match below is already efficient — leaf types return `ty.clone()`
/// in O(1), and structural types are traversed exactly once.
pub fn substitute_type_with<F>(ty: &HirType, f: &mut F) -> HirType
where
    F: FnMut(&Symbol) -> Option<HirType>,
{
    match ty {
        HirType::Param(sym) => f(sym).unwrap_or_else(|| ty.clone()),

        HirType::Generic(sym, args) => {
            let new_args: Vec<HirType> = args.iter()
                .map(|a| substitute_type_with(a, f))
                .collect();
            // If no args changed, avoid reallocating
            if new_args.iter().zip(args.iter()).all(|(n, o)| n == o) {
                // At least one arg must have been a Param that f returned None for,
                // or no args were Params at all. Either way, the original is fine.
                // But wait — if f returned Some for a Param, the new_args entry differs.
                // If all are equal, no substitution happened, so return original.
                ty.clone()
            } else {
                HirType::Generic(*sym, new_args)
            }
        }

        HirType::Tuple(elems) => {
            let new_elems: Vec<HirType> = elems.iter()
                .map(|e| substitute_type_with(e, f))
                .collect();
            if new_elems.iter().zip(elems.iter()).all(|(n, o)| n == o) {
                ty.clone()
            } else {
                HirType::Tuple(new_elems)
            }
        }

        HirType::RawPtr(inner) => {
            let new_inner = substitute_type_with(inner, f);
            if new_inner == *inner.clone() { ty.clone() }
            else { HirType::RawPtr(Box::new(new_inner)) }
        }
        HirType::Option(inner) => {
            let new_inner = substitute_type_with(inner, f);
            if new_inner == *inner.clone() { ty.clone() }
            else { HirType::Option(Box::new(new_inner)) }
        }
        HirType::Result(ok, err) => {
            let new_ok = substitute_type_with(ok, f);
            let new_err = substitute_type_with(err, f);
            if new_ok == *ok.clone() && new_err == *err.clone() {
                ty.clone()
            } else {
                HirType::Result(Box::new(new_ok), Box::new(new_err))
            }
        }
        HirType::Func(params, ret) => {
            let new_params: Vec<HirType> = params.iter()
                .map(|p| substitute_type_with(p, f))
                .collect();
            let new_ret = substitute_type_with(ret, f);
            let params_changed = new_params.iter().zip(params.iter()).any(|(n, o)| n != o);
            let ret_changed = new_ret != *ret.clone();
            if !params_changed && !ret_changed {
                ty.clone()
            } else {
                HirType::Func(new_params, Box::new(new_ret))
            }
        }

        // Leaf types — no Param possible. Return directly.
        HirType::Infer(_)
        | HirType::Named(_)
        | HirType::Int
        | HirType::Bool
        | HirType::Float
        | HirType::Str
        | HirType::Unit
        | HirType::Never
        | HirType::Error
        | HirType::Opaque(_) => ty.clone(),
    }
}

/// Substitute type parameters by name.
/// `Param(sym)` → `sub[sym]` if present, otherwise unchanged.
pub fn substitute_type(ty: &HirType, sub: &HashMap<Symbol, HirType>) -> HirType {
    substitute_type_with(ty, &mut |sym| sub.get(sym).cloned())
}

/// Replace `Param(sym)` with `Infer(var)` for type parameter solving.
pub fn substitute_type_vars(ty: &HirType, vars: &HashMap<Symbol, TypeVar>) -> HirType {
    substitute_type_with(ty, &mut |sym| {
        vars.get(sym).map(|&var| HirType::Infer(var))
    })
}

//! Unified type error type. Every type-check error flows through this enum.
//!
//! Key design decisions:
//! - `TypeFormatter` replaces closure boilerplate for formatting types.
//!   Created per-use from TypeCheckContext, NOT stored as a field on TypeChecker.
//! - `span()` method: single source of truth for primary span.
//! - `Diagnostic::labels()`: MismatchedTypes produces TWO labels, unless
//!   the spans are identical (merged to avoid confusing double-arrows).
//! - `InferLeakedPastFreeze` is NOT emitted by default — gated behind
//!   `TypeChecker::verbose_internal_errors`.

use glyim_hir::types::{HirType, TypeVar};
use glyim_interner::{Interner, Symbol};
use miette::Diagnostic;

/// Formats types for error messages. Created per-use from TypeCheckContext.
/// Replaces the closure pattern `|t| format_type_for_error(t, &interner)`.
pub struct TypeFormatter<'a> {
    interner: &'a Interner,
}

impl<'a> TypeFormatter<'a> {
    pub fn new(interner: &'a Interner) -> Self {
        Self { interner }
    }

    /// Format a HirType for human-readable error messages.
    /// Infer vars are shown as `?N` so diagnostics distinguish them.
    pub fn format(&self, ty: &HirType) -> String {
        format_type_for_error(ty, self.interner)
    }
}

/// Format a HirType for human-readable error messages.
/// Infer vars are shown as `?N` (not `_`) so diagnostics distinguish them.
pub fn format_type_for_error(ty: &HirType, interner: &Interner) -> String {
    match ty {
        HirType::Int => "i64".into(),
        HirType::Bool => "bool".into(),
        HirType::Float => "f64".into(),
        HirType::Str => "Str".into(),
        HirType::Unit => "()".into(),
        HirType::Never => "!".into(),
        HirType::Error => "<error>".into(),
        HirType::Infer(var) => format!("?{}", var.raw_index()),
        HirType::Param(sym) => interner.resolve(*sym).to_string(),
        HirType::Named(sym) => interner.resolve(*sym).to_string(),
        HirType::Generic(sym, args) => {
            let a = args.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("{}<{}>", interner.resolve(*sym), a)
        }
        HirType::RawPtr(inner) => format!("*{}", format_type_for_error(inner, interner)),
        HirType::Option(inner) => format!("Option<{}>", format_type_for_error(inner, interner)),
        HirType::Result(ok, err) => format!(
            "Result<{}, {}>",
            format_type_for_error(ok, interner),
            format_type_for_error(err, interner)
        ),
        HirType::Func(params, ret) => {
            let p = params.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("fn({}) -> {}", p, format_type_for_error(ret, interner))
        }
        HirType::Tuple(elems) => {
            let e = elems.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("({})", e)
        }
        HirType::Opaque(sym) => format!("@{}", interner.resolve(*sym)),
    }
}

/// Byte bounds within a source file. Every TypeError variant carries one or more.
#[derive(Clone, Debug, PartialEq)]
pub struct SpanBounds {
    pub start: usize,
    pub end: usize,
}

impl SpanBounds {
    pub fn new(start: usize, end: usize) -> Self {
        Self { start, end }
    }
    pub fn from_span(span: glyim_diag::Span) -> Self {
        Self { start: span.start, end: span.end }
    }
    pub fn to_source_span(&self) -> miette::SourceSpan {
        (self.start..self.end).into()
    }
    pub fn len(&self) -> usize {
        self.end.saturating_sub(self.start)
    }
    pub fn is_empty(&self) -> bool {
        self.start >= self.end
    }
}

/// What kind of expression couldn't be inferred.
#[derive(Clone, Debug, PartialEq)]
pub enum InferKind {
    Variable,
    Return,
    Closure,
    GenericArg,
    LetBinding,
    ForInIterator,
}

impl std::fmt::Display for InferKind {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            InferKind::Variable => write!(f, "variable"),
            InferKind::Return => write!(f, "return type"),
            InferKind::Closure => write!(f, "closure"),
            InferKind::GenericArg => write!(f, "generic argument"),
            InferKind::LetBinding => write!(f, "let binding"),
            InferKind::ForInIterator => write!(f, "for-in iterator"),
        }
    }
}

                // confusing double-arrows at the same location.

                    Some(Box::new(std::iter::once(label)))
                } else {
                    let labels: Vec<miette::LabeledSpan> = vec![
                        miette::LabeledSpan::new(
                            Some(format!("expected {}", expected)),
                            expected_span.start,
                            expected_span.len(),
                        ),
                        miette::LabeledSpan::new(
                            Some(format!("found {}", found)),
                            found_span.start,
                            found_span.len(),
                        ),
                    ];
                    Some(Box::new(labels.into_iter()))
                }
            }
            _ => {
                let s = self.span();
                let labeled = miette::LabeledSpan::new(
                    Some("here".into()),
                    s.start,
                    s.len(),
                );
                Some(Box::new(std::iter::once(labeled)))
            }
        }
    }
}

//! Pre-built HIR index for O(1) lookups.
//! Shared by type checker and monomorphizer. No separate Registry needed.
//!
//! TRADEOFF: StructInfo/EnumInfo clone field names + types from the HIR.
//! This is a conscious choice: these are queried on every struct literal
//! and field access during type checking. Full definitions (with spans)
//! are accessed via find_struct_def(hir, name).

use crate::{EnumDef, HirFn, HirImplDef, HirItem, StructDef};
use crate::types::HirType;
use crate::naming::method_base_name;
use glyim_interner::{Interner, Symbol};
use std::collections::{HashMap, HashSet};

/// Lightweight field info — name and type only (no span).
#[derive(Clone, Debug)]
pub struct FieldInfo {
    pub name: Symbol,
    pub ty: HirType,
}

#[derive(Clone, Debug)]
pub struct StructInfo {
    pub name: Symbol,
    pub fields: Vec<FieldInfo>,
    pub field_map: HashMap<Symbol, usize>,
    pub type_params: Vec<Symbol>,
    hir_item_index: usize,
}

#[derive(Clone, Debug)]
pub struct EnumInfo {
    pub name: Symbol,
    pub variants: Vec<crate::HirVariant>,
    pub variant_map: HashMap<Symbol, usize>,
    pub type_params: Vec<Symbol>,
    hir_item_index: usize,
}

/// Tracks where a function lives in the HIR to avoid index hacks.
/// Fixes the bug where impl methods used hir_item_index: 0 and
/// returned the wrong definition.
#[derive(Clone, Debug)]
pub enum FnSource {
    /// Top-level HirItem::Fn at the given index.
    TopLevel(usize),
    /// Method inside a HirItem::Impl at impl_idx, method at method_idx.
    ImplMethod { impl_idx: usize, method_idx: usize },
}

#[derive(Clone, Debug)]
pub struct FnInfo {
    pub name: Symbol,
    pub type_params: Vec<Symbol>,
    pub params: Vec<(Symbol, HirType)>,
    pub ret: Option<HirType>,
    pub is_generic: bool,
    pub source: FnSource,
}

/// Shared index built from HIR. Used by both typeck and mono.
#[derive(Debug)]
pub struct HirIndex {
    pub structs: HashMap<Symbol, StructInfo>,
    pub enums: HashMap<Symbol, EnumInfo>,
    pub fns: HashMap<Symbol, FnInfo>,
    pub extern_fns: HashMap<Symbol, crate::ExternFn>,
    /// target_name → method names (unmangled)
    pub impl_methods: HashMap<Symbol, Vec<Symbol>>,
    /// Generic entity names — HashSet for O(1) lookup
    generic_fn_names: HashSet<Symbol>,
    generic_struct_names: HashSet<Symbol>,
    generic_enum_names: HashSet<Symbol>,
    impl_method_names: HashSet<Symbol>,
}

impl HirIndex {
    /// Build the index from the HIR.
    /// Takes `&mut Interner` so impl method names can be mangled correctly.
    pub fn build(hir: &crate::Hir, interner: &mut Interner) -> Self {
        let mut idx = Self {
            structs: HashMap::new(),
            enums: HashMap::new(),
            fns: HashMap::new(),
            extern_fns: HashMap::new(),
            impl_methods: HashMap::new(),
            generic_fn_names: HashSet::new(),
            generic_struct_names: HashSet::new(),
            generic_enum_names: HashSet::new(),
            impl_method_names: HashSet::new(),
        };

        for (i, item) in hir.items.iter().enumerate() {
            match item {
                HirItem::Struct(s) => {
                    idx.register_struct(s, i);
                }
                HirItem::Enum(e) => {
                    idx.register_enum(e, i);
                }
                HirItem::Extern(ext) => {
                    for f in &ext.functions {
                        idx.extern_fns.insert(f.name, f.clone());
                    }
                }
                HirItem::Fn(f) => {
                    idx.register_fn(f, false, FnSource::TopLevel(i));
                }
                HirItem::Impl(imp) => {
                    idx.register_impl(imp, i, interner);
                }
                _ => {}
            }
        }

        idx
    }

    fn register_struct(&mut self, s: &StructDef, hir_index: usize) {
        let mut field_map = HashMap::new();
        let fields: Vec<FieldInfo> = s.fields.iter().enumerate().map(|(i, f)| {
            field_map.insert(f.name, i);
            FieldInfo { name: f.name, ty: f.ty.clone() }
        }).collect();

        if !s.type_params.is_empty() {
            self.generic_struct_names.insert(s.name);
        }
        self.structs.insert(s.name, StructInfo {
            name: s.name,
            fields,
            field_map,
            type_params: s.type_params.clone(),
            hir_item_index: hir_index,
        });
    }

    fn register_enum(&mut self, e: &EnumDef, hir_index: usize) {
        let mut variant_map = HashMap::new();
        for (i, v) in e.variants.iter().enumerate() {
            variant_map.insert(v.name, i);
        }
        if !e.type_params.is_empty() {
            self.generic_enum_names.insert(e.name);
        }
        self.enums.insert(e.name, EnumInfo {
            name: e.name,
            variants: e.variants.clone(),
            variant_map,
            type_params: e.type_params.clone(),
            hir_item_index: hir_index,
        });
    }

    fn register_fn(&mut self, f: &HirFn, is_impl_on_generic: bool, source: FnSource) {
        let is_generic = !f.type_params.is_empty() || is_impl_on_generic;
        if is_generic {
            self.generic_fn_names.insert(f.name);
        }
        self.fns.insert(f.name, FnInfo {
            name: f.name,
            type_params: f.type_params.clone(),
            params: f.params.clone(),
            ret: f.ret.clone(),
            is_generic,
            source,
        });
    }

    fn register_impl(&mut self, imp: &HirImplDef, impl_idx: usize, interner: &mut Interner) {
        let target_is_generic = self.structs.get(&imp.target_name)
            .map_or(false, |s| !s.type_params.is_empty())
            || self.enums.get(&imp.target_name)
                .map_or(false, |e| !e.type_params.is_empty())
            || !imp.type_params.is_empty();

        let method_names: Vec<Symbol> = imp.methods.iter().map(|m| m.name).collect();
        for (method_idx, m) in imp.methods.iter().enumerate() {
            // Mangle method name to "Type_method" so it doesn't overwrite
            // free functions with the same name in the fns HashMap.
            let mangled_name = method_base_name(interner, imp.target_name, m.name);

            self.register_fn_with_name(mangled_name, m, target_is_generic, FnSource::ImplMethod { impl_idx, method_idx });
            self.impl_method_names.insert(mangled_name);
        }
        self.impl_methods.insert(imp.target_name, method_names);
    }

    /// Register a function under a specific (possibly mangled) name.
    fn register_fn_with_name(
        &mut self,
        name: Symbol,
        f: &HirFn,
        is_impl_on_generic: bool,
        source: FnSource,
    ) {
        let is_generic = !f.type_params.is_empty() || is_impl_on_generic;
        if is_generic {
            self.generic_fn_names.insert(name);
        }
        self.fns.insert(name, FnInfo {
            name,
            type_params: f.type_params.clone(),
            params: f.params.clone(),
            ret: f.ret.clone(),
            is_generic,
            source,
        });
    }

    pub fn find_fn(&self, name: Symbol) -> Option<&FnInfo> { self.fns.get(&name) }
    pub fn find_struct(&self, name: Symbol) -> Option<&StructInfo> { self.structs.get(&name) }
    pub fn find_enum(&self, name: Symbol) -> Option<&EnumInfo> { self.enums.get(&name) }

    /// Get the full struct definition from the original HIR.
    pub fn find_struct_def<'a>(&self, hir: &'a crate::Hir, name: Symbol) -> Option<&'a StructDef> {
        let idx = self.structs.get(&name)?.hir_item_index;
        match hir.items.get(idx)? {
            HirItem::Struct(s) => Some(s),
            _ => None,
        }
    }

    /// Get the full enum definition from the original HIR.
    pub fn find_enum_def<'a>(&self, hir: &'a crate::Hir, name: Symbol) -> Option<&'a EnumDef> {
        let idx = self.enums.get(&name)?.hir_item_index;
        match hir.items.get(idx)? {
            HirItem::Enum(e) => Some(e),
            _ => None,
        }
    }

    /// Get the full function definition from the original HIR.
    /// Uses FnSource to locate impl methods correctly.
    pub fn find_fn_def<'a>(&self, hir: &'a crate::Hir, name: Symbol) -> Option<&'a HirFn> {
        let info = self.fns.get(&name)?;
        match &info.source {
            FnSource::TopLevel(idx) => match hir.items.get(*idx)? {
                HirItem::Fn(f) => Some(f),
                _ => None,
            },
            FnSource::ImplMethod { impl_idx, method_idx } => match hir.items.get(*impl_idx)? {
                HirItem::Impl(imp) => imp.methods.get(*method_idx),
                _ => None,
            },
        }
    }

    /// O(1) checks using HashSet
    pub fn is_generic_fn(&self, name: Symbol) -> bool { self.generic_fn_names.contains(&name) }
    pub fn is_generic_struct(&self, name: Symbol) -> bool { self.generic_struct_names.contains(&name) }
    pub fn is_generic_enum(&self, name: Symbol) -> bool { self.generic_enum_names.contains(&name) }
    pub fn is_impl_method(&self, name: Symbol) -> bool { self.impl_method_names.contains(&name) }
    pub fn is_generic_type(&self, name: Symbol) -> bool {
        self.is_generic_struct(name) || self.is_generic_enum(name)
    }

    pub fn type_param_count(&self, name: Symbol) -> Option<usize> {
        self.structs.get(&name).map(|s| s.type_params.len())
            .or_else(|| self.enums.get(&name).map(|e| e.type_params.len()))
    }

    pub fn fn_names(&self) -> impl Iterator<Item = Symbol> + '_ { self.fns.keys().copied() }
    pub fn struct_names(&self) -> impl Iterator<Item = Symbol> + '_ { self.structs.keys().copied() }
    pub fn enum_names(&self) -> impl Iterator<Item = Symbol> + '_ { self.enums.keys().copied() }
}

//! Unification table operating directly on `HirType`.
//!
//! Uses union-find with path compression on every `find` call.
//! `Infer(TypeVar)` is the only representation of type variables.
//! `Param(Symbol)` is NEVER unified by name — it's a distinct type
//! unless substitution has replaced it first.
//!
//! Error reporting: callback only, returns bool.
//!
//! CONSTRAINT SOLVING IS DESTRUCTIVE: once types are unified, there
//! is no rollback. Each function gets a fresh UnificationTable, so
//! failed solving means error recovery (best-effort), not rollback.
//!
//! POST-FAILURE INVARIANT: After a failed unification, the table may
//! contain bindings from the partially-successful unification. Subsequent
//! type checking continues using the table as-is. This is acceptable
//! because each function gets a fresh table and error recovery is
//! best-effort. The table remains internally consistent (union-find
//! invariants hold), but some bindings may reflect only partial
//! constraints.

use crate::errors::{SpanBounds, TypeError, format_type_for_error};
use glyim_hir::types::{HirType, TypeVar};
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

/// Maximum depth for chasing Infer bindings during resolve.
/// Only incremented when following an Infer→binding chain, NOT for
/// structural recursion. Prevents stack overflow on infinite types.
const MAX_INFER_DEPTH: u32 = 256;

/// Union-find–based unification table.
pub struct UnificationTable {
    parents: Vec<u32>,
    ranks: Vec<u8>,
    /// Span where each TypeVar was created (for diagnostics).
    spans: Vec<SpanBounds>,
    /// Bindings: root TypeVar index → concrete HirType.
    bindings: HashMap<u32, HirType>,
}

impl Default for UnificationTable {
    fn default() -> Self { Self::new() }
}

impl UnificationTable {
    pub fn new() -> Self {
        Self {
            parents: Vec::new(),
            ranks: Vec::new(),
            spans: Vec::new(),
            bindings: HashMap::new(),
        }
    }

    /// Create a fresh type variable associated with a source span.
    /// This is the ONLY correct way to construct a TypeVar.
    pub fn fresh_var(&mut self, span: SpanBounds) -> TypeVar {
        let idx = self.parents.len() as u32;
        self.parents.push(idx);
        self.ranks.push(0);
        self.spans.push(span);
        TypeVar::from_raw_unchecked(idx)
    }

    /// Get the creation span for a TypeVar (for diagnostics).
    pub fn var_span(&self, var: TypeVar) -> Option<&SpanBounds> {
        self.spans.get(var.raw_index() as usize)
    }

    /// Find root with path compression. Every call compresses.
    pub fn find(&mut self, var: TypeVar) -> TypeVar {
        let mut root = var.raw_index();
        while self.parents[root as usize] != root {
            root = self.parents[root as usize];
        }
        // Path compression
        let mut current = var.raw_index();
        while current != root {
            let parent = self.parents[current as usize];
            self.parents[current as usize] = root;
            current = parent;
        }
        TypeVar::from_raw_unchecked(root)
    }

    /// Resolve a `HirType`, chasing all `Infer` bindings.
    /// Unbound vars remain as `Infer(root)`.
    /// Depth limit counts ONLY Infer chain steps, not structural recursion.
    pub fn resolve(&mut self, ty: &HirType, interner: &Interner) -> HirType {
        self.resolve_infer_depth(ty, interner, 0)
    }

    fn resolve_infer_depth(
        &mut self,
        ty: &HirType,
        interner: &Interner,
        infer_depth: u32,
    ) -> HirType {
        match ty {
            HirType::Infer(var) => {
                if infer_depth > MAX_INFER_DEPTH {
                    tracing::error!(
                        "resolve exceeded max Infer depth {} on type: {:?}",
                        MAX_INFER_DEPTH, ty
                    );
                    return HirType::Error;
                }
                let root = self.find(*var);
                match self.bindings.get(&root.raw_index()) {
                    Some(bound) => self.resolve_infer_depth(bound, interner, infer_depth + 1),
                    None => HirType::Infer(root),
                }
            }
            HirType::Generic(sym, args) => HirType::Generic(
                *sym,
                args.iter().map(|a| self.resolve_infer_depth(a, interner, infer_depth)).collect(),
            ),
            HirType::Tuple(elems) => {
                HirType::Tuple(elems.iter().map(|e| self.resolve_infer_depth(e, interner, infer_depth)).collect())
            }
            HirType::RawPtr(inner) => {
                HirType::RawPtr(Box::new(self.resolve_infer_depth(inner, interner, infer_depth)))
            }
            HirType::Option(inner) => {
                HirType::Option(Box::new(self.resolve_infer_depth(inner, interner, infer_depth)))
            }
            HirType::Result(ok, err) => HirType::Result(
                Box::new(self.resolve_infer_depth(ok, interner, infer_depth)),
                Box::new(self.resolve_infer_depth(err, interner, infer_depth)),
            ),
            HirType::Func(params, ret) => HirType::Func(
                params.iter().map(|p| self.resolve_infer_depth(p, interner, infer_depth)).collect(),
                Box::new(self.resolve_infer_depth(ret, interner, infer_depth)),
            ),
            _ => ty.clone(),
        }
    }

    /// Unify two types. Returns `true` on success, `false` on failure.
    /// Errors are reported ONLY through the `emit_err` callback.
    pub fn unify(
        &mut self,
        a: &HirType,
        b: &HirType,
        span: SpanBounds,
        emit_err: &mut dyn FnMut(TypeError),
        interner: &Interner,
    ) -> bool {
        let a_resolved = self.resolve(a, interner);
        let b_resolved = self.resolve(b, interner);

        if a_resolved == b_resolved {
            return true;
        }

        // Error recovery: Error unifies with anything
        if matches!(a_resolved, HirType::Error) || matches!(b_resolved, HirType::Error) {
            return true;
        }

        // Two Infer vars: union them
        if let (HirType::Infer(var_a), HirType::Infer(var_b)) = (&a_resolved, &b_resolved) {
            self.union(*var_a, *var_b);
            return true;
        }

        // Infer var on one side: bind (with occurs check)
        if let HirType::Infer(var) = &a_resolved {
            if self.occurs(*var, &b_resolved, interner) {
                let origin = self.spans.get(var.raw_index() as usize).cloned().unwrap_or(span);
                emit_err(TypeError::InfiniteType { span: origin });
                return false;
            }
            self.bind(*var, &b_resolved);
            return true;
        }

        if let HirType::Infer(var) = &b_resolved {
            if self.occurs(*var, &a_resolved, interner) {
                let origin = self.spans.get(var.raw_index() as usize).cloned().unwrap_or(span);
                emit_err(TypeError::InfiniteType { span: origin });
                return false;
            }
            self.bind(*var, &a_resolved);
            return true;
        }

        self.unify_structural(&a_resolved, &b_resolved, span, emit_err, interner)
    }

    fn bind(&mut self, var: TypeVar, ty: &HirType) {
        let root = self.find(var);
        self.bindings.insert(root.raw_index(), ty.clone());
    }

    fn union(&mut self, a: TypeVar, b: TypeVar) {
        let root_a = self.find(a);
        let root_b = self.find(b);
        if root_a == root_b { return; }

        let (new_root, old_root) = match self.ranks[root_a.raw_index() as usize]
            .cmp(&self.ranks[root_b.raw_index() as usize])
        {
            std::cmp::Ordering::Less => (root_b, root_a),
            std::cmp::Ordering::Greater => (root_a, root_b),
            std::cmp::Ordering::Equal => {
                self.ranks[root_a.raw_index() as usize] += 1;
                (root_a, root_b)
            }
        };

        self.parents[old_root.raw_index() as usize] = new_root.raw_index();
        if let Some(bound) = self.bindings.remove(&old_root.raw_index()) {
            self.bindings.insert(new_root.raw_index(), bound);
        }
    }

    /// Occurs check: does `var` appear in `ty`?
    fn occurs(&mut self, var: TypeVar, ty: &HirType, interner: &Interner) -> bool {
        match ty {
            HirType::Infer(v) => {
                let root_var = self.find(var);
                let root_v = self.find(*v);
                root_var == root_v
            }
            HirType::Generic(_, args) => args.iter().any(|a| self.occurs(var, a, interner)),
            HirType::Tuple(elems) => elems.iter().any(|e| self.occurs(var, e, interner)),
            HirType::RawPtr(inner) | HirType::Option(inner) => self.occurs(var, inner, interner),
            HirType::Result(ok, err) => {
                self.occurs(var, ok, interner) || self.occurs(var, err, interner)
            }
            HirType::Func(params, ret) => {
                params.iter().any(|p| self.occurs(var, p, interner))
                    || self.occurs(var, ret, interner)
            }
            _ => false,
        }
    }

    fn unify_structural(
        &mut self,
        a: &HirType,
        b: &HirType,
        span: SpanBounds,
        emit_err: &mut dyn FnMut(TypeError),
        interner: &Interner,
    ) -> bool {
        match (a, b) {
            (HirType::Int, HirType::Int)
            | (HirType::Bool, HirType::Bool)
            | (HirType::Float, HirType::Float)
            | (HirType::Str, HirType::Str)
            | (HirType::Unit, HirType::Unit)
            | (HirType::Never, HirType::Never) => true,

            (HirType::Named(s1), HirType::Named(s2)) if s1 == s2 => true,

            // Param unification: same symbol → OK. Different symbols → mismatch.
            (HirType::Param(s1), HirType::Param(s2)) if s1 == s2 => true,

            // Generic with same base and same arity → unify args recursively
            (HirType::Generic(s1, a1), HirType::Generic(s2, a2))
                if s1 == s2 && a1.len() == a2.len() =>
            {
                let a1c = a1.clone();
                let a2c = a2.clone();
                let mut ok = true;
                for (arg_a, arg_b) in a1c.iter().zip(a2c.iter()) {
                    if !self.unify(arg_a, arg_b, span.clone(), emit_err, interner) {
                        ok = false;
                    }
                }
                ok
            }

            (HirType::RawPtr(i1), HirType::RawPtr(i2)) => {
                self.unify(i1, i2, span, emit_err, interner)
            }
            (HirType::Option(i1), HirType::Option(i2)) => {
                self.unify(i1, i2, span, emit_err, interner)
            }
            (HirType::Result(ok1, err1), HirType::Result(ok2, err2)) => {
                let ok_result = self.unify(ok1, ok2, span.clone(), emit_err, interner);
                let err_result = self.unify(err1, err2, span, emit_err, interner);
                ok_result && err_result
            }
            (HirType::Func(p1, r1), HirType::Func(p2, r2)) if p1.len() == p2.len() => {
                let p1c = p1.clone();
                let p2c = p2.clone();
                let mut ok = true;
                for (pa, pb) in p1c.iter().zip(p2c.iter()) {
                    if !self.unify(pa, pb, span.clone(), emit_err, interner) {
                        ok = false;
                    }
                }
                if !self.unify(r1, r2, span, emit_err, interner) {
                    ok = false;
                }
                ok
            }
            (HirType::Tuple(e1), HirType::Tuple(e2)) if e1.len() == e2.len() => {
                let e1c = e1.clone();
                let e2c = e2.clone();
                let mut ok = true;
                for (ea, eb) in e1c.iter().zip(e2c.iter()) {
                    if !self.unify(ea, eb, span.clone(), emit_err, interner) {
                        ok = false;
                    }
                }
                ok
            }

            // Never unifies with anything (subsumption)
            (HirType::Never, _) | (_, HirType::Never) => true,

            _ => {
                // Use distinct spans for expected/found to avoid confusing diagnostics
                let found_span = span.clone();
                let expected_span = self.find_infer_origin(a).unwrap_or(span);

                emit_err(TypeError::mismatched(
                    a.clone(), b.clone(),
                    expected_span, found_span,
                    &crate::errors::TypeFormatter::new(interner),
                ));
                false
            }
        }
    }

    /// Try to find the span where an expected type originated (e.g., from an Infer var).
    fn find_infer_origin(&self, ty: &HirType) -> Option<SpanBounds> {
        match ty {
            HirType::Infer(var) => self.var_span(*var).cloned(),
            HirType::Generic(_, args) => args.iter().find_map(|a| self.find_infer_origin(a)),
            HirType::Func(params, ret) => {
                params.iter().find_map(|p| self.find_infer_origin(p))
                    .or_else(|| self.find_infer_origin(ret))
            }
            _ => None,
        }
    }

    pub fn var_count(&self) -> usize { self.parents.len() }
}

/// Extract type parameter bindings by matching a schema against a concrete type.
/// Logs a warning if inconsistent matches are found (same param bound to different types).
pub fn extract_type_substitutions(
    schema: &HirType,
    concrete: &HirType,
    type_params: &[Symbol],
    sub: &mut HashMap<Symbol, HirType>,
) {
    match (schema, concrete) {
        (HirType::Param(sym), _) if type_params.contains(sym) => {
            if let Some(existing) = sub.get(sym) {
                if existing != concrete {
                    tracing::warn!(
                        "Inconsistent type substitution for {:?}: existing {:?}, new {:?}",
                        sym, existing, concrete
                    );
                }
                // Keep the first binding (standard approach for error recovery)
            } else {
                sub.insert(*sym, concrete.clone());
            }
        }
        (HirType::Generic(sym_a, args_a), HirType::Generic(sym_b, args_b))
            if sym_a == sym_b && args_a.len() == args_b.len() =>
        {
            for (a, b) in args_a.iter().zip(args_b.iter()) {
                extract_type_substitutions(a, b, type_params, sub);
            }
        }
        (HirType::Named(a), HirType::Named(b)) if a == b => {}
        (HirType::RawPtr(inner_s), HirType::RawPtr(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }
        (HirType::Option(inner_s), HirType::Option(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }
        (HirType::Result(ok_s, err_s), HirType::Result(ok_c, err_c)) => {
            extract_type_substitutions(ok_s, ok_c, type_params, sub);
            extract_type_substitutions(err_s, err_c, type_params, sub);
        }
        (HirType::Tuple(elems_s), HirType::Tuple(elems_c)) if elems_s.len() == elems_c.len() => {
            for (s, c) in elems_s.iter().zip(elems_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
        }
        (HirType::Func(params_s, ret_s), HirType::Func(params_c, ret_c))
            if params_s.len() == params_c.len() =>
        {
            for (s, c) in params_s.iter().zip(params_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
            extract_type_substitutions(ret_s, ret_c, type_params, sub);
        }
        _ => {}
    }
}

//! Scoped type environment with O(1) push/pop/lookup.
//!
//! Uses a name→stack approach: each scope records which names it added.
//! On pop, those names' top binding is popped. Shadowing works naturally.
//!
//! LIMITATION: Only tracks named variable bindings. Field and pointer
//! mutations are not reflected in the environment; their types are
//! determined by struct/pointer definitions.
//!
//! Tracing uses lazy evaluation (`tracing::trace!(name = ?name, ...)`)
//! so format strings are NOT allocated when tracing is disabled.

use glyim_hir::types::HirType;
use glyim_interner::Symbol;
use std::collections::HashMap;

#[derive(Clone, Debug)]
struct Binding {
    ty: HirType,
    mutable: bool,
}

/// Scoped type environment for name → type mappings.
///
/// Tracks named variable bindings only. Field and pointer mutations
/// are not reflected here; their types come from struct definitions
/// and pointer inner types respectively.
pub struct TypeEnv {
    /// Stack of scopes. Each scope stores the names that were declared in it.
    scopes: Vec<Vec<Symbol>>,
    /// Name → stack of bindings (innermost on top).
    bindings: HashMap<Symbol, Vec<Binding>>,
}

impl Default for TypeEnv {
    fn default() -> Self { Self::new() }
}

impl TypeEnv {
    pub fn new() -> Self {
        Self {
            scopes: vec![vec![]], // Root scope (depth 0)
            bindings: HashMap::new(),
        }
    }

    /// Push a new scope.
    pub fn push_scope(&mut self) {
        tracing::trace!(old_depth = self.scopes.len() - 1, new_depth = self.scopes.len(), "TypeEnv::push_scope");
        self.scopes.push(vec![]);
    }

    /// Pop the innermost scope, removing all its bindings.
    ///
    /// # Panics
    /// Panics if attempted on the root scope.
    pub fn pop_scope(&mut self) {
        if self.scopes.len() <= 1 {
            panic!("TypeEnv: attempted to pop the root scope");
        }
        let popped_names = self.scopes.pop().unwrap();
        for name in popped_names {
            if let Some(stack) = self.bindings.get_mut(&name) {
                stack.pop();
                if stack.is_empty() {
                    self.bindings.remove(&name);
                }
            }
        }
        tracing::trace!(depth = self.scopes.len() - 1, "TypeEnv::pop_scope");
    }

    /// Insert a NEW binding in the current (innermost) scope.
    /// If the name already exists in any scope, it is shadowed (not overwritten).
    pub fn insert(&mut self, name: Symbol, ty: HirType, mutable: bool) {
        tracing::trace!(name = ?name, ?ty, ?mutable, "TypeEnv::insert");
        self.scopes.last_mut().unwrap().push(name);
        self.bindings.entry(name).or_default().push(Binding { ty, mutable });
    }

    /// Update the type of an EXISTING binding in-place.
    /// Used for assignment, where we want to update the current binding
    /// rather than push a shadow (which would be reverted on scope pop).
    ///
    /// Returns `true` if the binding existed and was updated, `false` if
    /// the name was not found in any scope.
    pub fn update(&mut self, name: Symbol, new_ty: HirType) -> bool {
        if let Some(stack) = self.bindings.get_mut(&name) {
            if let Some(top) = stack.last_mut() {
                tracing::trace!(name = ?name, new_ty = ?new_ty, "TypeEnv::update");
                top.ty = new_ty;
                return true;
            }
        }
        false
    }

    /// Look up the innermost binding for a name.
    pub fn lookup(&self, name: Symbol) -> Option<&HirType> {
        self.bindings.get(&name).and_then(|stack| stack.last().map(|b| &b.ty))
    }

    /// Check if the innermost binding for a name is mutable.
    pub fn is_mutable(&self, name: Symbol) -> bool {
        self.bindings.get(&name).and_then(|stack| stack.last().map(|b| b.mutable)).unwrap_or(false)
    }

    /// Reset to initial state.
    pub fn clear(&mut self) {
        self.scopes = vec![vec![]];
        self.bindings.clear();
    }

    pub fn depth(&self) -> usize {
        self.scopes.len().saturating_sub(1)
    }
}

//! Generic type parameter solving.
//!
//! Uses the CALLER'S UnificationTable, enabling constraint propagation
//! within a function body.
//!
//! CONSTRAINT SOLVING IS DESTRUCTIVE: once types are unified, there
//! is no rollback. Each function gets a fresh UnificationTable, so
//! failed solving means error recovery (best-effort), not rollback.
//!
//! PRECONDITION: The caller must ensure that `table` is not dropped or
//! reset while the returned `SolveResult` substitution map is in use,
//! as the types reference inference variables resolved within the table.

use crate::errors::{SpanBounds, TypeError};
use crate::unify::UnificationTable;
use glyim_hir::index::FnInfo;
use glyim_hir::types::{substitute_type_with, HirType, TypeVar};
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

pub struct SolveResult {
    /// Type parameter symbol → concrete HirType.
    pub subst: HashMap<Symbol, HirType>,
    /// Concrete type args in the order of fn_info.type_params.
    pub concrete_args: Vec<HirType>,
    /// Whether ALL type params resolved to concrete (non-Param, non-Infer) types.
    pub fully_resolved: bool,
    /// Whether any unification errors occurred.
    pub had_errors: bool,
}

/// Solve generic type params for a function call.
///
/// 1. Create fresh TypeVars in `table` for each type param
/// 2. Unify formal params with actual arg types (forward)
/// 3. If expected return provided, unify return type (backward)
/// 4. Resolve each TypeVar to extract concrete types
pub fn solve_generic_params(
    table: &mut UnificationTable,
    fn_info: &FnInfo,
    arg_types: &[HirType],
    expected_return: Option<&HirType>,
    interner: &Interner,
    call_span: SpanBounds,
    emit_err: &mut dyn FnMut(TypeError),
) -> SolveResult {
    let mut param_vars: HashMap<Symbol, TypeVar> = HashMap::new();
    let mut had_errors = false;

    // Step 1: Fresh TypeVars in the caller's table
    for tp in &fn_info.type_params {
        let var = table.fresh_var(call_span.clone());
        param_vars.insert(*tp, var);
    }

    // Step 2: Forward — unify formal params with actual args
    for (formal, actual) in fn_info.params.iter().zip(arg_types.iter()) {
        let formal_ty = substitute_type_with(&formal.1, &mut |sym| {
            param_vars.get(sym).map(|&var| HirType::Infer(var))
        });
        tracing::debug!(
            "solve: unify formal {} with actual {}",
            crate::errors::format_type_for_error(&formal_ty, interner),
            crate::errors::format_type_for_error(actual, interner),
        );
        if !table.unify(&formal_ty, actual, call_span.clone(), emit_err, interner) {
            had_errors = true;
        }
    }

    // Step 3: Backward — unify return type with expected
    if let Some(expected) = expected_return {
        if let Some(ret) = &fn_info.ret {
            let ret_ty = substitute_type_with(ret, &mut |sym| {
                param_vars.get(sym).map(|&var| HirType::Infer(var))
            });
            if !table.unify(&ret_ty, expected, call_span.clone(), emit_err, interner) {
                had_errors = true;
            }
        }
    }

    // Step 4: Extract results
    let mut subst = HashMap::new();
    let mut concrete_args = Vec::new();
    let mut fully_resolved = true;

    for tp in &fn_info.type_params {
        let var = param_vars[tp];
        let resolved = table.resolve(&HirType::Infer(var), interner);

        let is_unresolved = matches!(resolved, HirType::Infer(_))
            || matches!(&resolved, HirType::Param(s) if fn_info.type_params.contains(s));

        if is_unresolved || resolved == HirType::Error {
            fully_resolved = false;
            subst.insert(*tp, HirType::Param(*tp));
        } else {
            subst.insert(*tp, resolved.clone());
        }
        concrete_args.push(resolved);
    }

    SolveResult { subst, concrete_args, fully_resolved, had_errors }
}

```rust
//! Resolve type variables, producing output for downstream phases.
//!
//! VALIDATION BOUNDARY between type checking and downstream phases.
//!
//! KEY DESIGN: For generic functions, unbound Infer vars that were
//! created as type parameter stand-ins are replaced with `Param(Symbol)`,
//! NOT `Error`. This preserves the template structure so the monomorphizer
//! can substitute `Param(T) → concrete_type` when specializing.
//!
//! Only Infer vars that are NOT type parameter stand-ins become `Error`.
//!
//! Uses `HashMap<TypeVar, Symbol>` as the map key (NOT raw u32).
//!
//! `InferLeakedPastFreeze` is only emitted when `verbose_internal_errors`
//! is true — it's a compiler-developer diagnostic, not a user-facing one.
//!
//! NOTE: This operation mutates the UnificationTable via path compression
//! during `resolve`, even though it is logically a "read" operation.
//! The table remains internally consistent, but callers should not assume
//! the table is unchanged after freeze.

use crate::errors::{InferKind, SpanBounds, TypeError};
use crate::unify::UnificationTable;
use glyim_hir::types::{ExprId, HirType, TypeVar};
use glyim_interner::Symbol;
use std::collections::HashMap;

/// Maps TypeVar → the Symbol of the type parameter it stands for.
/// Populated by `check_fn` when creating fresh Infer vars for type params.
/// Uses TypeVar as key (NOT u32) to enforce the construction invariant.
pub type TypeParamMap = HashMap<TypeVar, Symbol>;

/// Resolve a single HirType through the unification table.
///
/// Unbound `Infer` vars that correspond to type parameters (via `type_param_map`)
/// become `Param(Symbol)`. Other unbound `Infer` vars become `Error` with diagnostics.
pub fn resolve_type(
    ty: &HirType,
    table: &mut UnificationTable,
    interner: &glyim_interner::Interner,
    errors: &mut Vec<TypeError>,
    type_param_map: &TypeParamMap,
    verbose_internal_errors: bool,
) -> HirType {
    let resolved = table.resolve(ty, interner);
    replace_infer(resolved, table, errors, type_param_map, verbose_internal_errors)
}

fn replace_infer(
    ty: HirType,
    table: &mut UnificationTable,
    errors: &mut Vec<TypeError>,
    type_param_map: &TypeParamMap,
    verbose_internal_errors: bool,
) -> HirType {
    match ty {
        HirType::Infer(var) => {
            // Use TypeVar as key, not raw u32
            if let Some(&sym) = type_param_map.get(&var) {
                tracing::debug!(
                    "freeze: Infer(?{}) → Param({:?}) — type param stand-in",
                    var.raw_index(), sym
                );
                return HirType::Param(sym);
            }

            // Not a type param stand-in — genuine inference failure
            let span = table.var_span(var)
                .cloned()
                .unwrap_or(SpanBounds::new(0, 0));

            // Only emit the internal compiler bug warning in verbose mode
            if verbose_internal_errors {
                errors.push(TypeError::InferLeakedPastFreeze {
                    type_var: var,
                    span: span.clone(),
                });
            }

            // Always emit the user-facing error
            errors.push(TypeError::CannotInferType {
                kind: InferKind::Variable,
                type_var: Some(var),
                span,
            });
            HirType::Error
        }
        HirType::Generic(sym, args) => {
            let new_args: Vec<HirType> = args.into_iter()
                .map(|a| replace_infer(a, table, errors, type_param_map, verbose_internal_errors))
                .collect();
            HirType::Generic(sym, new_args)
        }
        HirType::Tuple(elems) => {
            HirType::Tuple(elems.into_iter().map(|e| replace_infer(e, table, errors, type_param_map, verbose_internal_errors)).collect())
        }
        HirType::RawPtr(inner) => {
            HirType::RawPtr(Box::new(replace_infer(*inner, table, errors, type_param_map, verbose_internal_errors)))
        }
        HirType::Option(inner) => {
            HirType::Option(Box::new(replace_infer(*inner, table, errors, type_param_map, verbose_internal_errors)))
        }
        HirType::Result(ok, err) => HirType::Result(
            Box::new(replace_infer(*ok, table, errors, type_param_map, verbose_internal_errors)),
            Box::new(replace_infer(*err, table, errors, type_param_map, verbose_internal_errors)),
        ),
        HirType::Func(params, ret) => HirType::Func(
            params.into_iter().map(|p| replace_infer(p, table, errors, type_param_map, verbose_internal_errors)).collect(),
            Box::new(replace_infer(*ret, table, errors, type_param_map, verbose_internal_errors)),
        ),
        // Leaf types — no Infer possible
        HirType::Param(_)
        | HirType::Named(_)
        | HirType::Int
        | HirType::Bool
        | HirType::Float
        | HirType::Str
        | HirType::Unit
        | HirType::Never
        | HirType::Error
        | HirType::Opaque(_) => ty,
    }
}

/// Freeze an expression type map for a single function.
/// Called inside `check_fn` before the UnificationTable is discarded.
///
/// Returns `HashMap<ExprId, HirType>` — no sparse Vec, no Option wrapping.
/// Every expression that was type-checked gets an entry.
///
/// `call_type_args` entries with some Error args are KEPT (not discarded)
/// so the monomorphizer can do best-effort partial specialization.
pub fn freeze_function_types(
    table: &mut UnificationTable,
    expr_types: &HashMap<ExprId, HirType>,
    call_type_args: &HashMap<ExprId, Vec<HirType>>,
    interner: &glyim_interner::Interner,
    errors: &mut Vec<TypeError>,
    type_param_map: &TypeParamMap,
    verbose_internal_errors: bool,
) -> FreezeOutput {
    let mut frozen_expr_types = HashMap::with_capacity(expr_types.len());
    for (&id, ty) in expr_types {
        let frozen = resolve_type(ty, table, interner, errors, type_param_map, verbose_internal_errors);
        tracing::debug!(
            "freeze: expr {:?} → {}",
            id,
            crate::errors::format_type_for_error(&frozen, interner),
        );
        frozen_expr_types.insert(id, frozen);
    }

    let mut frozen_call_args = HashMap::with_capacity(call_type_args.len());
    for (&id, args) in call_type_args {
        let frozen_args: Vec<HirType> = args.iter()
            .map(|a| resolve_type(a, table, interner, errors, type_param_map, verbose_internal_errors))
            .collect();
        // Keep ALL entries, even those with some Error args.
        frozen_call_args.insert(id, frozen_args);
    }

    FreezeOutput {
        expr_types: frozen_expr_types,
        call_type_args: frozen_call_args,
    }
}

pub struct FreezeOutput {
    pub expr_types: HashMap<ExprId, HirType>,
    pub call_type_args: HashMap<ExprId, Vec<HirType>>,
}

Key design decisions:
- **`TypeCheckContext`** holds only the fields that submodules need. `frozen_*` and `generic_expr_ids` stay on the outer `TypeChecker`.
- **`Option<HirIndex>`** eliminates the stub-index-with-throwaway-Interner footgun.
- **`HashSet<ExprId> generic_expr_ids`** tracks which ExprIds belong to generic functions, enabling `validate_mono_input` to permit `Param` in generic output.
- **No `formatter` field** on any struct. `TypeFormatter` created per-use from context.
- `check_fn` builds a `TypeParamMap` (with `TypeVar` keys) and passes it to freeze.
- `HashMap<ExprId, HirType>` for all type maps — no sparse Vec.
- Method name construction uses `glyim_hir::naming::method_base_name`.
- `KnownSymbols` used for Option/Result/Range.
- `resolve_receiver_type` returns `Result` with diagnostic info.
- `verbose_internal_errors` field on TypeChecker, defaults to false.
- **Assignment fix**: Checks `lookup` before `is_mutable`, uses `env.update`.
- **`infer_match` fix**: Uses `match` instead of `if/else` for expected type, no double `fresh_var`.
- **`TypeCheckOutput` moves both interner and known** instead of cloning known.

/// Error when resolve_receiver_type fails — carries diagnostic info.
#[derive(Debug, Clone)]
pub enum ResolveReceiverError {
    NotANamedType(HirType),
}

impl std::fmt::Display for ResolveReceiverError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            ResolveReceiverError::NotANamedType(ty) => {
                write!(f, "receiver is not a named type (found {:?})", ty)
            }
        }
    }
}

/// Limited view of TypeChecker state that submodules can access.
///
/// This struct intentionally does NOT contain `frozen_expr_types`,
/// `frozen_call_type_args`, or `generic_expr_ids` — those are managed
/// by the outer TypeChecker and should not be modified by submodules.
pub struct TypeCheckContext<'a> {
    pub interner: &'a mut Interner,
    pub known: &'a KnownSymbols,
    pub index: &'a HirIndex,
    pub env: &'a mut TypeEnv,
    pub expr_types: &'a mut HashMap<ExprId, HirType>,
    pub call_type_args: &'a mut HashMap<ExprId, Vec<HirType>>,
    pub errors: &'a mut Vec<TypeError>,
}

impl<'a> TypeCheckContext<'a> {
    /// Create a TypeFormatter referencing our interner. Created per-use
    /// to ensure lifetime safety. NO formatter field anywhere.
    pub fn formatter(&self) -> TypeFormatter<'_> {
        TypeFormatter::new(self.interner)
    }

    pub fn unify_types(
        &mut self,
        table: &mut UnificationTable,
        expected: &HirType,
        found: &HirType,
        span: SpanBounds,
    ) -> bool {
        table.unify(expected, found, span, &mut |e| self.errors.push(e), self.interner)
    }

    /// If a `Named` type refers to a generic struct/enum, promote it to `Generic`.
    pub fn promote_named_if_generic(
        &mut self,
        table: &mut UnificationTable,
        ty: &HirType,
        span: SpanBounds,
    ) -> HirType {
        match ty {
            HirType::Named(sym) => {
                if let Some(count) = self.index.type_param_count(*sym) {
                    let vars: Vec<HirType> = (0..count)
                        .map(|_| HirType::Infer(table.fresh_var(span.clone())))
                        .collect();
                    HirType::Generic(*sym, vars)
                } else {
                    ty.clone()
                }
            }
            HirType::Generic(sym, args) => {
                let promoted_args: Vec<HirType> = args.iter()
                    .map(|a| self.promote_named_if_generic(table, a, span.clone()))
                    .collect();
                HirType::Generic(*sym, promoted_args)
            }
            other => other.clone(),
        }
    }

    pub fn record_call_type_args(&mut self, id: ExprId, args: Vec<HirType>) {
        if args.is_empty() { return; }
        self.call_type_args.insert(id, args);
    }

    pub fn resolve_receiver_type(
        &self,
        recv_ty: &HirType,
    ) -> Result<(Symbol, Vec<HirType>), ResolveReceiverError> {
        match recv_ty {
            HirType::Named(name) => Ok((*name, vec![])),
            HirType::Generic(name, args) => Ok((*name, args.clone())),
            HirType::RawPtr(inner) => self.resolve_receiver_type(inner),
            _ => Err(ResolveReceiverError::NotANamedType(recv_ty.clone())),
        }
    }
}

pub struct TypeChecker {
    pub interner: Interner,
    pub known: KnownSymbols,
    /// Option to avoid stub-index footgun. Set during check().
    index: Option<HirIndex>,
    pub env: TypeEnv,
    /// Per-function expression types (cleared per function)
    expr_types: HashMap<ExprId, HirType>,
    /// Per-function call type args (cleared per function)
    call_type_args: HashMap<ExprId, Vec<HirType>>,
    pub errors: Vec<TypeError>,
    /// Accumulated frozen results from all functions.
    frozen_expr_types: HashMap<ExprId, HirType>,
    frozen_call_type_args: HashMap<ExprId, Vec<HirType>>,
    /// ExprIds belonging to generic functions (Param types allowed).
    generic_expr_ids: HashSet<ExprId>,
    /// Whether to emit internal compiler bug warnings (InferLeakedPastFreeze).
    pub verbose_internal_errors: bool,
    /// Observability metrics.
    pub metrics: TypeCheckMetrics,
}

impl TypeChecker {
    pub fn new(interner: Interner, known: KnownSymbols) -> Self {
        Self {
            interner,
            known,
            index: None, // No stub! Must call check() first.
            env: TypeEnv::new(),
            expr_types: HashMap::new(),
            call_type_args: HashMap::new(),
            errors: Vec::new(),
            frozen_expr_types: HashMap::new(),
            frozen_call_type_args: HashMap::new(),
            generic_expr_ids: HashSet::new(),
            verbose_internal_errors: false,
            metrics: TypeCheckMetrics::default(),
        }
    }

    /// Get the HirIndex. Panics with a clear message if called before check().
    pub fn index(&self) -> &HirIndex {
        self.index.as_ref().expect("TypeChecker::index() called before check()")
    }

    pub fn check(&mut self, hir: &Hir) -> Result<TypeCheckOutput, Vec<TypeError>> {
        let built_index = HirIndex::build(hir, &mut self.interner);
        self.index = Some(built_index);

        for item in &hir.items {
            match item {
                HirItem::Fn(f) => self.check_fn(f),
                HirItem::Impl(imp) => {
                    for method in &imp.methods {
                        self.check_fn(method);
                    }
                }
                _ => {}
            }
        }

        if self.errors.is_empty() {
            // Take the index out of Option to move it into the output
            let index = self.index.take().expect("index was set at start of check()");
            Ok(TypeCheckOutput {
                expr_types: std::mem::take(&mut self.frozen_expr_types),
                call_type_args: std::mem::take(&mut self.frozen_call_type_args),
                generic_expr_ids: std::mem::take(&mut self.generic_expr_ids),
                index,
                // Move both interner and known — no cloning
                interner: std::mem::take(&mut self.interner),
                known: std::mem::replace(&mut self.known, KnownSymbols::intern_all(&mut Interner::new())),
            })
        } else {
            Err(std::mem::take(&mut self.errors))
        }
    }

    /// Type-check a single function.
    ///
    /// For generic functions:
    /// 1. Create fresh Infer vars for each type parameter
    /// 2. Record TypeVar→Symbol mapping in type_param_map
    /// 3. Substitute Param(T)→Infer(var) in param/return types
    /// 4. Type-check the body
    /// 5. Record all ExprIds in this function as generic if the function has type params
    /// 6. Freeze: unbound Infer vars that map to type params become Param(Symbol).
    fn check_fn(&mut self, f: &HirFn) {
        self.env.clear();
        let mut table = UnificationTable::new();
        self.expr_types.clear();
        self.call_type_args.clear();
        self.metrics.functions_checked += 1;

        let is_generic = !f.type_params.is_empty();
        if is_generic {
            self.metrics.generic_functions += 1;
        }

        tracing::debug!("check_fn: {} (generic: {})",
            self.interner.resolve(f.name), is_generic);

        let mut type_param_sub: HashMap<Symbol, HirType> = HashMap::new();
        let mut type_param_map: TypeParamMap = HashMap::new();

        for tp in &f.type_params {
            let span = SpanBounds::from_span(f.body.get_span());
            let var = table.fresh_var(span);
            self.metrics.type_vars_created += 1;
            type_param_map.insert(var, *tp);
            type_param_sub.insert(*tp, HirType::Infer(var));
        }

        self.env.push_scope();
        for (i, (sym, ty)) in f.params.iter().enumerate() {
            let mutable = f.param_mutability.get(i).copied().unwrap_or(false);
            let concrete_param_ty = glyim_hir::types::substitute_type(ty, &type_param_sub);
            self.env.insert(*sym, concrete_param_ty, mutable);
        }

        let concrete_ret = f.ret.as_ref()
            .map(|r| glyim_hir::types::substitute_type(r, &type_param_sub));

        // Build the TypeCheckContext for this function
        let mut ctx = TypeCheckContext {
            interner: &mut self.interner,
            known: &self.known,
            index: self.index.as_ref().expect("index set before check_fn"),
            env: &mut self.env,
            expr_types: &mut self.expr_types,
            call_type_args: &mut self.call_type_args,
            errors: &mut self.errors,
        };

        let body_ty = infer_expr_with_table(&mut ctx, &mut table, &f.body, concrete_ret.as_ref());

        if let Some(expected) = &concrete_ret {
            let span = SpanBounds::from_span(f.body.get_span());
            ctx.unify_types(&mut table, expected, &body_ty, span);
        }

        self.env.pop_scope();

        // Record all ExprIds from this function as generic if it has type params.
        // This allows validate_mono_input to permit Param types in these expressions.
        if is_generic {
            for &id in self.expr_types.keys() {
                self.generic_expr_ids.insert(id);
            }
        }

        let FreezeOutput { expr_types, call_type_args } = freeze::freeze_function_types(
            &mut table,
            &self.expr_types,
            &self.call_type_args,
            &self.interner,
            &mut self.errors,
            &type_param_map,
            self.verbose_internal_errors,
        );

        self.frozen_expr_types.extend(expr_types);
        self.frozen_call_type_args.extend(call_type_args);
        self.metrics.errors_emitted += self.errors.len();
    }
}

/// Top-level expression inference. Creates the TypeCheckContext and delegates.
/// This is a free function (not a method) to avoid borrow-checker conflicts
/// with the TypeChecker's multiple &mut fields.
fn infer_expr_with_table(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    expr: &glyim_hir::HirExpr,
    expected: Option<&HirType>,
) -> HirType {
    let ty = match expr {
        glyim_hir::HirExpr::IntLit { .. } => HirType::Int,
        glyim_hir::HirExpr::FloatLit { .. } => HirType::Float,
        glyim_hir::HirExpr::BoolLit { .. } => HirType::Bool,
        glyim_hir::HirExpr::StrLit { .. } => HirType::Str,
        glyim_hir::HirExpr::UnitLit { .. } => HirType::Unit,

        glyim_hir::HirExpr::AddrOf { target, span, .. } => {
            let inner_ty = infer_expr_with_table(ctx, table, target, None);
            HirType::RawPtr(Box::new(inner_ty))
        }

        glyim_hir::HirExpr::Ident { name, span, .. } => {
            match ctx.env.lookup(*name).cloned() {
                Some(ty) => ty,
                None => {
                    let resolved = ctx.interner.resolve(*name).to_string();
                    ctx.errors.push(TypeError::UnresolvedName {
                        name: resolved,
                        span: SpanBounds::from_span(*span),
                        suggestions: vec![],
                    });
                    HirType::Error
                }
            }
        }

        glyim_hir::HirExpr::Binary { op, lhs, rhs, span, .. } => {
            let lhs_ty = infer_expr_with_table(ctx, table, lhs, None);
            let rhs_ty = infer_expr_with_table(ctx, table, rhs, None);
            operators::check_binary_op(ctx, table, *op, &lhs_ty, &rhs_ty, *span)
        }

        glyim_hir::HirExpr::Unary { op, operand, span, .. } => {
            let inner = infer_expr_with_table(ctx, table, operand, None);
            operators::check_unary_op(ctx, table, *op, &inner, *span)
        }

        glyim_hir::HirExpr::Block { stmts, span, .. } => {
            control_flow::infer_block(ctx, table, stmts, expected, *span)
        }

        glyim_hir::HirExpr::If { condition, then_branch, else_branch, span, .. } => {
            control_flow::infer_if(ctx, table, condition, then_branch, else_branch.as_deref(), expected, *span)
        }

        glyim_hir::HirExpr::Call { id, callee, args, span, .. } => {
            calls::infer_call(ctx, table, *id, *callee, args, expected, *span)
        }

        glyim_hir::HirExpr::MethodCall { id, receiver, method_name, args, span, .. } => {
            calls::infer_method_call(ctx, table, *id, receiver, *method_name, args, expected, *span)
        }

        glyim_hir::HirExpr::StructLit { struct_name, fields, span, .. } => {
            literals::infer_struct_lit(ctx, table, *struct_name, fields, expected, expr.get_id(), *span)
        }

        glyim_hir::HirExpr::EnumVariant { enum_name, variant_name, args, span, .. } => {
            literals::infer_enum_variant(ctx, table, *enum_name, *variant_name, args, *span)
        }

        glyim_hir::HirExpr::FieldAccess { object, field, span, .. } => {
            literals::infer_field_access(ctx, table, object, *field, *span)
        }

        glyim_hir::HirExpr::As { expr: inner, target_type, span, .. } => {
            let from_ty = infer_expr_with_table(ctx, table, inner, None);
            if !operators::is_valid_cast(&from_ty, target_type) {
                ctx.errors.push(TypeError::mismatched(
                    target_type.clone(),
                    from_ty.clone(),
                    SpanBounds::from_span(*span),
                    SpanBounds::from_span(inner.get_span()),
                    &ctx.formatter(),
                ));
            }
            target_type.clone()
        }

        glyim_hir::HirExpr::Deref { expr: inner, span, .. } => {
            let inner_ty = infer_expr_with_table(ctx, table, inner, None);
            match &inner_ty {
                HirType::RawPtr(t) => *t.clone(),
                _ => {
                    ctx.errors.push(TypeError::DerefNonPointer {
                        found: ctx.formatter().format(&inner_ty),
                        span: SpanBounds::from_span(*span),
                    });
                    HirType::Error
                }
            }
        }

        glyim_hir::HirExpr::Match { scrutinee, arms, span, .. } => {
            control_flow::infer_match(ctx, table, scrutinee, arms, expected, *span)
        }

        glyim_hir::HirExpr::While { condition, body, span, .. } => {
            infer_expr_with_table(ctx, table, condition, None);
            infer_expr_with_table(ctx, table, body, None);
            HirType::Unit
        }

        glyim_hir::HirExpr::ForIn { pattern, iter, body, span, .. } => {
            control_flow::infer_forin(ctx, table, pattern, iter, body, *span)
        }

        glyim_hir::HirExpr::Return { value, span, .. } => {
            if let Some(v) = value { infer_expr_with_table(ctx, table, v, expected); }
            HirType::Never
        }

        glyim_hir::HirExpr::Println { arg, .. } => {
            infer_expr_with_table(ctx, table, arg, None);
            HirType::Unit
        }

        glyim_hir::HirExpr::Assert { condition, message, .. } => {
            infer_expr_with_table(ctx, table, condition, None);
            if let Some(m) = message { infer_expr_with_table(ctx, table, m, None); }
            HirType::Unit
        }

        glyim_hir::HirExpr::SizeOf { id, target_type, .. } => {
            ctx.record_call_type_args(*id, vec![target_type.clone()]);
            HirType::Int
        }

        glyim_hir::HirExpr::TupleLit { elements, .. } => {
            let types: Vec<HirType> = elements.iter()
                .map(|e| infer_expr_with_table(ctx, table, e, None))
                .collect();
            HirType::Tuple(types)
        }
    };

    ctx.expr_types.insert(expr.get_id(), ty.clone());
    ty
}

fn infer_stmt_with_table(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    stmt: &glyim_hir::HirStmt,
    expected: Option<&HirType>,
) -> Option<HirType> {
    match stmt {
        glyim_hir::HirStmt::Let { name, mutable, value, span } => {
            let ty = infer_expr_with_table(ctx, table, value, None);
            ctx.env.insert(*name, ty.clone(), *mutable);
            None
        }
        glyim_hir::HirStmt::LetPat { pattern, mutable, value, ty: annotation, span } => {
            let inferred = infer_expr_with_table(ctx, table, value, annotation.as_ref());
            let ty = if let Some(annotated) = annotation {
                let span_b = SpanBounds::from_span(*span);
                ctx.unify_types(table, annotated, &inferred, span_b);
                annotated.clone()
            } else {
                inferred
            };
            patterns::bind_pattern(ctx, pattern, &ty, *mutable);
            None
        }
        glyim_hir::HirStmt::Assign { target, value, span } => {
            // FIX: Check existence BEFORE mutability to emit correct error
            if ctx.env.lookup(*target).is_none() {
                ctx.errors.push(TypeError::UnresolvedName {
                    name: ctx.interner.resolve(*target).to_string(),
                    span: SpanBounds::from_span(*span),
                    suggestions: vec![],
                });
                infer_expr_with_table(ctx, table, value, None);
                return None;
            }
            if !ctx.env.is_mutable(*target) {
                ctx.errors.push(TypeError::AssignToImmutable {
                    name: ctx.interner.resolve(*target).to_string(),
                    span: SpanBounds::from_span(*span),
                });
                infer_expr_with_table(ctx, table, value, None);
                return None;
            }
            let val_ty = infer_expr_with_table(ctx, table, value, None);
            if let Some(existing) = ctx.env.lookup(*target).cloned() {
                let span_b = SpanBounds::from_span(*span);
                ctx.unify_types(table, &existing, &val_ty, span_b);
            }
            // FIX: Use update instead of insert for in-place mutation
            ctx.env.update(*target, val_ty.clone());
            Some(val_ty)
        }
        glyim_hir::HirStmt::AssignField { object, field, value, span } => {
            let obj_ty = infer_expr_with_table(ctx, table, object, None);
            let val_ty = infer_expr_with_table(ctx, table, value, None);
            if let Some(field_ty) = literals::field_type_of(ctx, &obj_ty, *field) {
                let span_b = SpanBounds::from_span(*span);
                ctx.unify_types(table, &field_ty, &val_ty, span_b);
            }
            None
        }
        glyim_hir::HirStmt::AssignDeref { target, value, span } => {
            let ptr_ty = infer_expr_with_table(ctx, table, target, None);
            let val_ty = infer_expr_with_table(ctx, table, value, None);
            if let HirType::RawPtr(inner) = &ptr_ty {
                let span_b = SpanBounds::from_span(*span);
                ctx.unify_types(table, inner, &val_ty, span_b);
            } else if ptr_ty != HirType::Error {
                ctx.errors.push(TypeError::AssignThroughNonPointer {
                    found: ctx.formatter().format(&ptr_ty),
                    span: SpanBounds::from_span(*span),
                });
            }
            Some(val_ty)
        }
        glyim_hir::HirStmt::Expr(e) => Some(infer_expr_with_table(ctx, table, e, expected)),
    }
}

//! Binary and unary operator type checking and cast validation.

use crate::errors::{SpanBounds, TypeError};
use crate::typeck::TypeCheckContext;
use crate::unify::UnificationTable;
use glyim_hir::types::{HirType, HirUnOp};

pub fn check_binary_op(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    op: glyim_hir::HirBinOp,
    lhs_ty: &HirType,
    rhs_ty: &HirType,
    span: glyim_diag::Span,
) -> HirType {
    let span_b = SpanBounds::from_span(span);
    match op {
        glyim_hir::HirBinOp::Eq
        | glyim_hir::HirBinOp::Neq
        | glyim_hir::HirBinOp::Lt
        | glyim_hir::HirBinOp::Gt
        | glyim_hir::HirBinOp::Lte
        | glyim_hir::HirBinOp::Gte => {
            ctx.unify_types(table, lhs_ty, rhs_ty, span_b);
            HirType::Bool
        }
        glyim_hir::HirBinOp::Add
        | glyim_hir::HirBinOp::Sub
        | glyim_hir::HirBinOp::Mul
        | glyim_hir::HirBinOp::Div
        | glyim_hir::HirBinOp::Mod => {
            ctx.unify_types(table, lhs_ty, rhs_ty, span_b.clone());
            let resolved = table.resolve(lhs_ty, ctx.interner);
            if !resolved.has_infer() && !resolved.has_param()
                && resolved != HirType::Int && resolved != HirType::Float
                && resolved != HirType::Error
            {
                ctx.errors.push(TypeError::InvalidBinaryOp {
                    op: format!("{:?}", op),
                    expected: "numeric".into(),
                    found: ctx.formatter().format(&resolved),
                    span: span_b,
                });
            }
            resolved
        }
        glyim_hir::HirBinOp::And | glyim_hir::HirBinOp::Or => {
            ctx.unify_types(table, &HirType::Bool, lhs_ty, span_b.clone());
            ctx.unify_types(table, &HirType::Bool, rhs_ty, span_b);
            HirType::Bool
        }
        _ => lhs_ty.clone(),
    }
}

pub fn check_unary_op(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    op: HirUnOp,
    operand_ty: &HirType,
    span: glyim_diag::Span,
) -> HirType {
    let span_b = SpanBounds::from_span(span);
    match op {
        HirUnOp::Not => {
            ctx.unify_types(table, &HirType::Bool, operand_ty, span_b.clone());
            let resolved = table.resolve(operand_ty, ctx.interner);
            if !resolved.has_infer() && !resolved.has_param()
                && resolved != HirType::Bool && resolved != HirType::Error
            {
                ctx.errors.push(TypeError::InvalidUnaryOp {
                    op: "!".into(),
                    expected: "bool".into(),
                    found: ctx.formatter().format(&resolved),
                    span: span_b,
                });
            }
            HirType::Bool
        }
        HirUnOp::Neg => {
            let resolved = table.resolve(operand_ty, ctx.interner);
            if !resolved.has_infer() && !resolved.has_param()
                && resolved != HirType::Int && resolved != HirType::Float
                && resolved != HirType::Error
            {
                ctx.errors.push(TypeError::InvalidUnaryOp {
                    op: "-".into(),
                    expected: "numeric".into(),
                    found: ctx.formatter().format(&resolved),
                    span: span_b,
                });
            }
            resolved
        }
    }
}

pub fn is_valid_cast(from: &HirType, to: &HirType) -> bool {
    if from.has_param() || to.has_param() { return true; }
    if from.has_infer() || to.has_infer() { return true; }
    match (from, to) {
        (HirType::Int, HirType::Float) | (HirType::Float, HirType::Int) => true,
        (HirType::Int, HirType::Int) | (HirType::Float, HirType::Float) => true,
        (_, HirType::RawPtr(_)) | (HirType::RawPtr(_), _) => true,
        (a, b) if a == b => true,
        _ => false,
    }
}

//! Function call and method call inference.

use crate::errors::{SpanBounds, TypeError};
use crate::solve;
use crate::typeck::{TypeCheckContext, ResolveReceiverError};
use crate::unify::UnificationTable;
use glyim_hir::index::FnInfo;
use glyim_hir::naming::method_base_name;
use glyim_hir::types::{ExprId, HirType};
use glyim_interner::Symbol;

pub fn infer_call(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    call_id: ExprId,
    callee: Symbol,
    args: &[glyim_hir::HirExpr],
    expected: Option<&HirType>,
    span: glyim_diag::Span,
) -> HirType {
    let arg_types: Vec<HirType> = args.iter()
        .map(|a| super::infer_expr_with_table(ctx, table, a, None))
        .collect();

    if let Some(ext_fn) = ctx.index.extern_fns.get(&callee).cloned() {
        if arg_types.len() != ext_fn.params.len() {
            ctx.errors.push(TypeError::ArgumentCountMismatch {
                expected: ext_fn.params.len(),
                actual: arg_types.len(),
                span: SpanBounds::from_span(span),
            });
        } else {
            for (i, (_, param_ty)) in ext_fn.params.iter().enumerate() {
                if i < arg_types.len() && arg_types[i] != HirType::Error {
                    ctx.unify_types(table, param_ty, &arg_types[i], SpanBounds::from_span(span));
                }
            }
        }
        return ext_fn.ret.clone();
    }

    let fn_info = match ctx.index.find_fn(callee) {
        Some(info) => info.clone(),
        None => {
            ctx.errors.push(TypeError::UnknownFunction {
                name: ctx.interner.resolve(callee).to_string(),
                span: SpanBounds::from_span(span),
            });
            return HirType::Error;
        }
    };

    if arg_types.len() != fn_info.params.len() {
        ctx.errors.push(TypeError::ArgumentCountMismatch {
            expected: fn_info.params.len(),
            actual: arg_types.len(),
            span: SpanBounds::from_span(span),
        });
    }

    if fn_info.is_generic {
        return infer_generic_call(ctx, table, call_id, callee, &fn_info, &arg_types, expected, span);
    }

    for ((_, param_ty), arg_ty) in fn_info.params.iter().zip(arg_types.iter()) {
        if *arg_ty == HirType::Error || *param_ty == HirType::Never { continue; }
        ctx.unify_types(table, param_ty, arg_ty, SpanBounds::from_span(span));
    }

    fn_info.ret.clone().unwrap_or(HirType::Unit)
}

fn infer_generic_call(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    call_id: ExprId,
    callee: Symbol,
    fn_info: &FnInfo,
    arg_types: &[HirType],
    expected: Option<&HirType>,
    span: glyim_diag::Span,
) -> HirType {
    let call_span = SpanBounds::from_span(span);

    let result = solve::solve_generic_params(
        table, fn_info, arg_types, expected, ctx.interner, call_span,
        &mut |e| ctx.errors.push(e),
    );

    ctx.record_call_type_args(call_id, result.concrete_args.clone());

    let ret = fn_info.ret.clone().unwrap_or(HirType::Unit);
    let concrete_ret = glyim_hir::types::substitute_type(&ret, &result.subst);

    if !result.fully_resolved {
        ctx.errors.push(TypeError::CannotInferGenericArgs {
            name: ctx.interner.resolve(callee).to_string(),
            span: SpanBounds::from_span(span),
        });
    }

    concrete_ret
}

pub fn infer_method_call(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    call_id: ExprId,
    receiver: &glyim_hir::HirExpr,
    method_name: Symbol,
    args: &[glyim_hir::HirExpr],
    expected: Option<&HirType>,
    span: glyim_diag::Span,
) -> HirType {
    let recv_ty = super::infer_expr_with_table(ctx, table, receiver, None);
    let arg_types: Vec<HirType> = args.iter()
        .map(|a| super::infer_expr_with_table(ctx, table, a, None))
        .collect();

    let (type_name, type_args) = match ctx.resolve_receiver_type(&recv_ty) {
        Ok(result) => result,
        Err(_e) => {
            let candidates: Vec<String> = ctx.index.impl_methods.values()
                .flatten()
                .map(|s| ctx.interner.resolve(*s).to_string())
                .collect();
            ctx.errors.push(TypeError::UnresolvedMethod {
                method_name: ctx.interner.resolve(method_name).to_string(),
                receiver_type: ctx.formatter().format(&recv_ty),
                span: SpanBounds::from_span(span),
                candidates,
            });
            return HirType::Error;
        }
    };

    let mangled_sym = method_base_name(ctx.interner, type_name, method_name);

    let fn_info = match ctx.index.find_fn(mangled_sym) {
        Some(info) => info.clone(),
        None => {
            let candidates: Vec<String> = ctx.index.impl_methods
                .get(&type_name)
                .map(|methods| methods.iter().map(|m| ctx.interner.resolve(*m).to_string()).collect())
                .unwrap_or_default();
            ctx.errors.push(TypeError::UnresolvedMethod {
                method_name: ctx.interner.resolve(method_name).to_string(),
                receiver_type: ctx.formatter().format(&recv_ty),
                span: SpanBounds::from_span(span),
                candidates,
            });
            return HirType::Error;
        }
    };

    if fn_info.is_generic {
        let mut all_arg_types = vec![recv_ty.clone()];
        all_arg_types.extend(arg_types);

        let result = solve::solve_generic_params(
            table, &fn_info, &all_arg_types, expected,
            ctx.interner, SpanBounds::from_span(span),
            &mut |e| ctx.errors.push(e),
        );

        ctx.record_call_type_args(call_id, result.concrete_args);

        let ret = fn_info.ret.clone().unwrap_or(HirType::Unit);
        let concrete_ret = glyim_hir::types::substitute_type(&ret, &result.subst);

        if !result.fully_resolved {
            ctx.errors.push(TypeError::CannotInferGenericArgs {
                name: format!("{}.{}",
                    ctx.interner.resolve(type_name),
                    ctx.interner.resolve(method_name)),
                span: SpanBounds::from_span(span),
            });
        }

        concrete_ret
    } else {
        if !type_args.is_empty() {
            ctx.record_call_type_args(call_id, type_args);
        }
        fn_info.ret.clone().unwrap_or(HirType::Unit)
    }
}

//! Struct, enum, and field access type checking.

use crate::errors::{SpanBounds, TypeError};
use crate::typeck::{TypeCheckContext, infer_expr_with_table};
use crate::unify::UnificationTable;
use glyim_hir::types::{ExprId, HirType};
use glyim_interner::Symbol;

/// Get the type of a field on an object, if it exists.
pub fn field_type_of(ctx: &TypeCheckContext, obj_ty: &HirType, field: Symbol) -> Option<HirType> {
    match obj_ty {
        HirType::Named(name) | HirType::Generic(name, _) => {
            ctx.index.find_struct(*name).and_then(|s| {
                s.field_map.get(&field).map(|&i| s.fields[i].ty.clone())
            })
        }
        _ => None,
    }

pub fn infer_struct_lit(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    struct_name: Symbol,
    fields: &[(Symbol, glyim_hir::HirExpr)],
    expected: Option<&HirType>,
    id: ExprId,
    span: glyim_diag::Span,
) -> HirType {
    let struct_info = match ctx.index.find_struct(struct_name) {
        Some(info) => info.clone(),
        None => {
            ctx.errors.push(TypeError::UnknownType {
                name: struct_name,
                span: SpanBounds::from_span(span),
            });
            return HirType::Error;
        }
    };

    let result_ty = if struct_info.type_params.is_empty() {
        HirType::Named(struct_name)
    } else {
        let ty = ctx.promote_named_if_generic(table, &HirType::Named(struct_name), SpanBounds::from_span(span));
        if let Some(exp) = expected {
            ctx.unify_types(table, &ty, exp, SpanBounds::from_span(span));
        }
        ty
    };

    for (field_name, field_expr) in fields {
        let field_ty = match field_type_of(ctx, &result_ty, *field_name) {
            Some(ty) => ty,
            None => {
                ctx.errors.push(TypeError::UnknownField {
                    struct_name: ctx.interner.resolve(struct_name).to_string(),
                    field: ctx.interner.resolve(*field_name).to_string(),
                    span: SpanBounds::from_span(field_expr.get_span()),
                });
                continue;
            }
        };
        let expr_ty = infer_expr_with_table(ctx, table, field_expr, Some(&field_ty));
        ctx.unify_types(table, &field_ty, &expr_ty, SpanBounds::from_span(field_expr.get_span()));
    }

    // Check for missing fields
    let provided_names: Vec<Symbol> = fields.iter().map(|(s, _)| *s).collect();
    for fi in &struct_info.fields {
        if !provided_names.contains(&fi.name) {
            ctx.errors.push(TypeError::MissingField {
                struct_name: ctx.interner.resolve(struct_name).to_string(),
                field: ctx.interner.resolve(fi.name).to_string(),
                span: SpanBounds::from_span(span),
            });
        }
    }

    ctx.expr_types.insert(id, result_ty.clone());
    result_ty
}

pub fn infer_enum_variant(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    enum_name: Symbol,
    variant_name: Symbol,
    args: &[glyim_hir::HirExpr],
    span: glyim_diag::Span,
) -> HirType {
    let enum_info = match ctx.index.find_enum(enum_name) {
        Some(info) => info.clone(),
        None => {
            ctx.errors.push(TypeError::UnknownType { name: enum_name, span: SpanBounds::from_span(span) });
            return HirType::Error;
        }
    };

    let variant_idx = match enum_info.variant_map.get(&variant_name) {
        Some(&idx) => idx,
        None => {
            ctx.errors.push(TypeError::UnknownVariant {
                enum_name: ctx.interner.resolve(enum_name).to_string(),
                variant_name: ctx.interner.resolve(variant_name).to_string(),
                span: SpanBounds::from_span(span),
            });
            return HirType::Error;
        }
    };

    let variant = &enum_info.variants[variant_idx];
    if args.len() != variant.fields.len() {
        ctx.errors.push(TypeError::ArgumentCountMismatch {
            expected: variant.fields.len(),
            actual: args.len(),
            span: SpanBounds::from_span(span),
        });
    }

    let result_ty = if enum_info.type_params.is_empty() {
        HirType::Named(enum_name)
    } else {
        ctx.promote_named_if_generic(table, &HirType::Named(enum_name), SpanBounds::from_span(span))
    };

    for (i, arg) in args.iter().enumerate() {
        if i < variant.fields.len() {
            let arg_ty = infer_expr_with_table(ctx, table, arg, None);
            ctx.unify_types(table, &variant.fields[i].ty, &arg_ty, SpanBounds::from_span(arg.get_span()));
        }
    }

    result_ty
}

pub fn infer_field_access(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    object: &glyim_hir::HirExpr,
    field: Symbol,
    span: glyim_diag::Span,
) -> HirType {
    let obj_ty = infer_expr_with_table(ctx, table, object, None);
    match field_type_of(ctx, &obj_ty, field) {
        Some(ty) => ty,
        None => {
            let struct_name = match &obj_ty {
                HirType::Named(n) | HirType::Generic(n, _) => ctx.interner.resolve(*n).to_string(),
                _ => format!("{:?}", obj_ty),
            };
            ctx.errors.push(TypeError::UnknownField {
                struct_name,
                field: ctx.interner.resolve(field).to_string(),
                span: SpanBounds::from_span(span),
            });
            HirType::Error
        }
    }
}

//! Pattern binding for let statements and match arms.

use crate::typeck::TypeCheckContext;
use glyim_hir::types::{HirPattern, HirType};
use glyim_interner::Symbol;

pub fn bind_pattern(ctx: &mut TypeCheckContext, pat: &HirPattern, ty: &HirType, mutable: bool) {
    match pat {
        HirPattern::Var(sym) => {
            ctx.env.insert(*sym, ty.clone(), mutable);
        }
        HirPattern::Wild => {}
        HirPattern::Struct { name, bindings, .. } => {
            if let Some(struct_info) = ctx.index.find_struct(*name) {
                for (field_sym, sub_pat) in bindings {
                    if let Some(&idx) = struct_info.field_map.get(field_sym) {
                        bind_pattern(ctx, sub_pat, &struct_info.fields[idx].ty, mutable);
                    }
                }
            }
        }
        HirPattern::Tuple { elements, .. } => {
            if let HirType::Tuple(elem_types) = ty {
                for (sub_pat, elem_ty) in elements.iter().zip(elem_types.iter()) {
                    bind_pattern(ctx, sub_pat, elem_ty, mutable);
                }
            }
        }
        HirPattern::EnumVariant { enum_name, variant_name, bindings, .. } => {
            if let Some(enum_info) = ctx.index.find_enum(*enum_name) {
                if let Some(&idx) = enum_info.variant_map.get(variant_name) {
                    let variant = &enum_info.variants[idx];
                    for (i, sub_pat) in bindings.iter().enumerate() {
                        if i < variant.fields.len() {
                            bind_pattern(ctx, sub_pat, &variant.fields[i].ty, mutable);
                        }
                    }
                }
            }
        }
        HirPattern::OptionSome(inner) => {
            if let HirType::Option(inner_ty) = ty {
                bind_pattern(ctx, inner, inner_ty, mutable);
            }
        }
        HirPattern::OptionNone => {}
        HirPattern::ResultOk(inner) => {
            if let HirType::Result(ok_ty, _) = ty {
                bind_pattern(ctx, inner, ok_ty, mutable);
            }
        }
        HirPattern::ResultErr(inner) => {
            if let HirType::Result(_, err_ty) = ty {
                bind_pattern(ctx, inner, err_ty, mutable);
            }
        }
    }
}

            missing.push(ctx.interner.resolve(variant.name).to_string());
        }
    }

//! Control flow type checking: blocks, if, match, for-in.

use crate::errors::{InferKind, SpanBounds, TypeError};

    ctx.env.push_scope();
    let mut last_ty = HirType::Unit;
    for (i, stmt) in stmts.iter().enumerate() {
        let is_last = i == stmts.len() - 1;
        let exp = if is_last { expected } else { None };
        if let Some(ty) = infer_stmt_with_table(ctx, table, stmt, exp) {
            last_ty = ty;
        }
    }
    ctx.env.pop_scope();
    last_ty
}

pub fn infer_if(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    condition: &glyim_hir::HirExpr,
    then_branch: &glyim_hir::HirExpr,
    else_branch: Option<&glyim_hir::HirExpr>,
    expected: Option<&HirType>,
    span: glyim_diag::Span,
) -> HirType {
    let cond_ty = infer_expr_with_table(ctx, table, condition, None);
    ctx.unify_types(table, &HirType::Bool, &cond_ty, SpanBounds::from_span(condition.get_span()));

    let then_ty = infer_expr_with_table(ctx, table, then_branch, expected);

    let else_ty = match else_branch {
        Some(else_expr) => infer_expr_with_table(ctx, table, else_expr, expected),
        None => HirType::Unit,
    };

    if let Some(exp) = expected {
        ctx.unify_types(table, exp, &then_ty, SpanBounds::from_span(then_branch.get_span()));
        if let Some(else_expr) = else_branch {
            ctx.unify_types(table, exp, &else_ty, SpanBounds::from_span(else_expr.get_span()));
        }
    } else {
        ctx.unify_types(table, &then_ty, &else_ty, SpanBounds::from_span(span));
    }

    then_ty
}

pub fn infer_match(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    scrutinee: &glyim_hir::HirExpr,
    arms: &[glyim_hir::MatchArm],
    expected: Option<&HirType>,
    span: glyim_diag::Span,
) -> HirType {
    let scrutinee_ty = infer_expr_with_table(ctx, table, scrutinee, None);

    let mut matched_variants = Vec::new();

    // FIX: Use match instead of if/else — no double fresh_var
    let result_ty = match expected {
        Some(exp) => exp.clone(),
        None => HirType::Infer(table.fresh_var(SpanBounds::from_span(span))),
    };

    for arm in arms {
        ctx.env.push_scope();

        // Extract variant name for exhaustiveness checking
        if let glyim_hir::types::HirPattern::EnumVariant { variant_name, .. } = &arm.pattern {
            matched_variants.push(*variant_name);
        }

        patterns::bind_pattern(ctx, &arm.pattern, &scrutinee_ty, false);

        if let Some(guard) = &arm.guard {
            let guard_ty = infer_expr_with_table(ctx, table, guard, None);
            ctx.unify_types(table, &HirType::Bool, &guard_ty, SpanBounds::from_span(guard.get_span()));
        }

        let arm_ty = infer_expr_with_table(ctx, table, &arm.body, expected);
        ctx.unify_types(table, &result_ty, &arm_ty, SpanBounds::from_span(arm.body.get_span()));

        ctx.env.pop_scope();
    }

    // Check exhaustiveness for enum matches
    let missing = exhaustiveness::check_exhaustiveness(ctx, &scrutinee_ty, &matched_variants);
    if !missing.is_empty() {
        ctx.errors.push(TypeError::NonExhaustiveMatch {
            missing,
            span: SpanBounds::from_span(span),
        });
    }

    result_ty
}

pub fn infer_forin(
    ctx: &mut TypeCheckContext,
    table: &mut UnificationTable,
    pattern: &glyim_hir::types::HirPattern,
    iter: &glyim_hir::HirExpr,
    body: &glyim_hir::HirExpr,
    span: glyim_diag::Span,
) -> HirType {
    let iter_ty = infer_expr_with_table(ctx, table, iter, None);

    // Try to extract element type from iterator
    let element_ty = match &iter_ty {
        HirType::Generic(sym, args) if *sym == ctx.known.range && args.len() == 1 => {
            args[0].clone()
        }
        HirType::Generic(sym, args) if *sym == ctx.known.range_inclusive && args.len() == 1 => {
            args[0].clone()
        }
        HirType::Generic(sym, args) if *sym == ctx.known.vec && args.len() == 1 => {
            args[0].clone()
        }
        _ => {
            ctx.errors.push(TypeError::CannotInferType {
                kind: InferKind::ForInIterator,
                type_var: None,
                span: SpanBounds::from_span(span),
            });
            HirType::Error
        }
    };

    ctx.env.push_scope();
    patterns::bind_pattern(ctx, pattern, &element_ty, true);
    infer_expr_with_table(ctx, table, body, None);
    ctx.env.pop_scope();

    HirType::Unit
}

```rust
//! Validation boundary between type checker output and monomorphizer input.
//!
//! CALLED IN THE PIPELINE before monomorphization.
//! This is not just dead code — the compiler driver enforces this boundary.
//!
//! KEY DESIGN: Param types are VALID in generic function template output.
//! A generic function like `fn id<T>(x: T) -> T { x }` will produce
//! `Param(T)` in its expr_types after freeze — this is correct, because
//! the monomorphizer needs these templates to create specializations.
//!
//! Only Param types that appear in NON-GENERIC function output are errors.
//! We distinguish between the two using `generic_expr_ids`, a set of
//! ExprIds that belong to generic functions.

use glyim_hir::types::{ExprId, HirType};
use std::collections::{HashSet, HashMap};

#[derive(Debug)]
pub struct ValidationError {
    pub kind: ValidationErrorKind,
    pub expr_id: Option<ExprId>,
    pub ty: HirType,
}

#[derive(Debug)]
pub enum ValidationErrorKind {
    /// An Infer type leaked past type checking — always a compiler bug.
    InferInOutput,
    /// A Param type appeared in a non-generic function's output — compiler bug.
    /// (Param types in generic function output are expected and correct.)
    ParamInNonGenericOutput,
}

/// Validate that type checker output is safe for monomorphization.
///
/// # Arguments
/// * `expr_types` — Type checker's expression type map
/// * `call_type_args` — Type checker's call type arguments
/// * `generic_expr_ids` — ExprIds belonging to generic functions.
///   Param types are permitted ONLY for these ExprIds.
///
/// # Returns
/// `Ok(())` if all invariants hold, `Err(list)` otherwise.
pub fn validate_mono_input(
    expr_types: &HashMap<ExprId, HirType>,
    call_type_args: &HashMap<ExprId, Vec<HirType>>,
    generic_expr_ids: &HashSet<ExprId>,
) -> Result<(), Vec<ValidationError>> {
    let mut errors = Vec::new();

    for (&id, ty) in expr_types {
        // Infer types are NEVER valid in output — always a compiler bug
        if ty.has_infer() {
            errors.push(ValidationError {
                kind: ValidationErrorKind::InferInOutput,
                expr_id: Some(id),
                ty: ty.clone(),
            });
        }

        // Param types are only valid inside generic function templates.
        // If an ExprId is NOT in generic_expr_ids, Param is a leak.
        if ty.has_param() && !generic_expr_ids.contains(&id) {
            errors.push(ValidationError {
                kind: ValidationErrorKind::ParamInNonGenericOutput,
                expr_id: Some(id),
                ty: ty.clone(),
            });
        }
    }

    for (&id, args) in call_type_args {
        for arg in args {
            if arg.has_infer() {
                errors.push(ValidationError {
                    kind: ValidationErrorKind::InferInOutput,
                    expr_id: Some(id),
                    ty: arg.clone(),
                });
            }
            // Call type args should always be concrete (even for generic calls,
            // the args represent the inferred substitutions, not the templates)
            if arg.has_param() {
                errors.push(ValidationError {
                    kind: ValidationErrorKind::ParamInNonGenericOutput,
                    expr_id: Some(id),
                    ty: arg.clone(),
                });
            }
        }
    }

    if errors.is_empty() {
        Ok(())
    } else {
        Err(errors)
    }
}
```

//! Mangle table: caches mangling results using HirType-based keys.
//!
//! TRADEOFF: Clones Vec<HirType> on cache miss to construct the key.
//! This is acceptable because the table typically holds tens to hundreds
//! of entries, and each insertion happens once per unique (base, args)
//! combination. Cache hits return a Symbol without cloning.

use crate::monomorphize::mangling;
use crate::types::HirType;
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

/// Key for the mangle table. Uses HirType directly instead of formatted strings.
/// HirType derives Hash and Eq, so this is safe and avoids redundant formatting.
#[derive(Debug, Clone, PartialEq, Eq, Hash)]
struct MangleKey {
    base: Symbol,
    args: Vec<HirType>,
}

/// Cache: MangleKey → mangled_symbol.
#[derive(Debug, Default)]
pub struct MangleTable {
    map: HashMap<MangleKey, Symbol>,
}

impl MangleTable {
    pub fn new() -> Self { Self::default() }

    /// Mangle a name with type arguments. Returns the cached symbol if
    /// this (base, args) combination has been seen before.
    ///
    /// Clones `args` on cache miss to construct the lookup key.
    /// Cache hits return immediately without cloning.
    pub fn mangle(
        &mut self,
        base: Symbol,
        args: &[HirType],
        interner: &mut Interner,
    ) -> Symbol {
        if args.is_empty() { return base; }
        // Check cache without cloning first
        let temp_key = MangleKey { base, args: args.to_vec() };
        if let Some(&mangled) = self.map.get(&temp_key) {
            return mangled;
        }
        let mangled = mangling::mangle_name(interner, base, args);
        self.map.insert(temp_key, mangled);
        mangled
    }

    /// Mangle a function name with type arguments.
    pub fn mangle_fn(
        &mut self,
        base: Symbol,
        args: &[HirType],
        interner: &mut Interner,
    ) -> Symbol {
        self.mangle(base, args, interner)
    }

    pub fn len(&self) -> usize { self.map.len() }
    pub fn is_empty(&self) -> bool { self.map.is_empty() }
}
```

//! BFS work queue for monomorphization.
//!
//! Uses struct-based dedup keys (no string allocation).
//! HirType derives Hash+Eq, so WorkItem can be used directly
//! in a HashSet for O(1) dedup.

use crate::types::HirType;
use glyim_interner::Symbol;
use std::collections::{HashSet, VecDeque};

#[derive(Clone, Debug, PartialEq, Eq, Hash)]
pub enum ItemKind {
    FnSpecialize,
    FnPassthrough,
    StructSpecialize,
    StructPassthrough,
    EnumSpecialize,
    EnumPassthrough,
}

#[derive(Clone, Debug, PartialEq, Eq, Hash)]
pub struct WorkItem {
    pub kind: ItemKind,
    pub def_id: Symbol,
    pub type_args: Vec<HirType>,
}

impl WorkItem {
    pub fn fn_specialize(name: Symbol, args: Vec<HirType>) -> Self {
        Self { kind: ItemKind::FnSpecialize, def_id: name, type_args: args }
    }
    pub fn fn_passthrough(name: Symbol) -> Self {
        Self { kind: ItemKind::FnPassthrough, def_id: name, type_args: vec![] }
    }
    pub fn struct_specialize(name: Symbol, args: Vec<HirType>) -> Self {
        Self { kind: ItemKind::StructSpecialize, def_id: name, type_args: args }
    }
    pub fn struct_passthrough(name: Symbol) -> Self {
        Self { kind: ItemKind::StructPassthrough, def_id: name, type_args: vec![] }
    }
    pub fn enum_specialize(name: Symbol, args: Vec<HirType>) -> Self {
        Self { kind: ItemKind::EnumSpecialize, def_id: name, type_args: args }
    }
    pub fn enum_passthrough(name: Symbol) -> Self {
        Self { kind: ItemKind::EnumPassthrough, def_id: name, type_args: vec![] }
    }
}

/// Work queue with struct-based deduplication.
/// No string allocation on push — WorkItem derives Hash+Eq directly.
pub struct WorkQueue {
    queue: VecDeque<WorkItem>,
    seen: HashSet<WorkItem>,
}

impl WorkQueue {
    pub fn new() -> Self {
        Self {
            queue: VecDeque::new(),
            seen: HashSet::new(),
        }
    }

    /// Push an item onto the queue. Deduplicates using struct-based Hash+Eq.
    /// No string allocation.
    pub fn push(&mut self, item: WorkItem) {
        if self.seen.insert(item.clone()) {
            tracing::trace!(?item, "WorkQueue::push");
            self.queue.push_back(item);
        }
    }

    pub fn pop(&mut self) -> Option<WorkItem> {
        self.queue.pop_front()
    }

    pub fn extend(&mut self, items: Vec<WorkItem>) {
        for item in items {
            self.push(item);
        }
    }

    pub fn is_empty(&self) -> bool {
        self.queue.is_empty()
    }

    pub fn len(&self) -> usize { self.queue.len() }
}

impl Default for WorkQueue {
    fn default() -> Self { Self::new() }
}

//! Concretization: replace type parameters with concrete types,
//! mangle type names using `$`, and record structural type metadata.

use crate::index::HirIndex;
use crate::monomorphize::mangle_table::MangleTable;
use crate::monomorphize::metadata::TypeStructure;

use crate::symbols::KnownSymbols;

use std::collections::HashMap;

/// Substitute type params and then concretize.
pub fn substitute_and_concretize(
    ty: &HirType,
    sub: &HashMap<Symbol, HirType>,
    interner: &mut Interner,
    mangle_table: &mut MangleTable,
    metadata: &mut crate::monomorphize::metadata::TypeMetadata,
    known: &KnownSymbols,
) -> HirType {

    concretize_type(substituted, interner, mangle_table, metadata, known)
}

/// Concretize a type: mangle Generic type names, record metadata for
/// structural types (Option/Result/Tuple), and check for unresolved params.
///
/// # Panics
/// `debug_assert!(false)` on Param or Infer in debug builds.
/// Returns `HirType::Error` with `tracing::error!` in release builds.
pub fn concretize_type(
    ty: HirType,
    interner: &mut Interner,
    mangle_table: &mut MangleTable,
    metadata: &mut crate::monomorphize::metadata::TypeMetadata,
    known: &KnownSymbols,
) -> HirType {
    match ty {
        HirType::Generic(sym, args) => {
            let concrete_args: Vec<HirType> = args.into_iter()
                .map(|a| concretize_type(a, interner, mangle_table, metadata, known))
                .collect();

            if concrete_args.iter().any(|a| a.has_param() || a.has_infer()) {
                return HirType::Generic(sym, concrete_args);
            }

            let mangled = mangle_table.mangle(sym, &concrete_args, interner);

            metadata.record(mangled, TypeStructure::Generic {
                base: sym,
                args: concrete_args.clone(),
            });

            HirType::Named(mangled)
        }
        HirType::Option(inner) => {
            let concrete_inner = concretize_type(*inner, interner, mangle_table, metadata, known);
            if concrete_inner.has_param() || concrete_inner.has_infer() {
                return HirType::Option(Box::new(concrete_inner));
            }
            let mangled = mangle_table.mangle(
                known.option, &[concrete_inner.clone()], interner,
            );

            metadata.record(mangled, TypeStructure::Option { inner: concrete_inner.clone() });

            HirType::Named(mangled)
        }
        HirType::Result(ok, err) => {
            let concrete_ok = concretize_type(*ok, interner, mangle_table, metadata, known);
            let concrete_err = concretize_type(*err, interner, mangle_table, metadata, known);
            if concrete_ok.has_param() || concrete_ok.has_infer()
                || concrete_err.has_param() || concrete_err.has_infer()
            {
                return HirType::Result(Box::new(concrete_ok), Box::new(concrete_err));
            }
            let mangled = mangle_table.mangle(
                known.result,
                &[concrete_ok.clone(), concrete_err.clone()],
                interner,
            );

            metadata.record(mangled, TypeStructure::Result {
                ok: concrete_ok.clone(),
                err: concrete_err.clone(),
            });

            HirType::Named(mangled)
        }
        HirType::Tuple(elems) => {
            let concrete_elems: Vec<HirType> = elems.into_iter()
                .map(|e| concretize_type(e, interner, mangle_table, metadata, known))
                .collect();
            HirType::Tuple(concrete_elems)
        }
        HirType::RawPtr(inner) => {
            HirType::RawPtr(Box::new(concretize_type(*inner, interner, mangle_table, metadata, known)))
        }
        HirType::Func(params, ret) => {
            HirType::Func(
                params.into_iter().map(|p| concretize_type(p, interner, mangle_table, metadata, known)).collect(),
                Box::new(concretize_type(*ret, interner, mangle_table, metadata, known)),
            )
        }
        HirType::Param(sym) => {
            debug_assert!(false, "FATAL: Unresolved Param({:?}) in concretize_type", sym);
            tracing::error!(
                "FATAL: Unresolved Param({:?}) in concretize_type. This should have been substituted.",
                sym
            );
            HirType::Error
        }
        HirType::Infer(var) => {
            debug_assert!(false, "FATAL: Infer(?{}) in concretize_type", var.raw_index());
            tracing::error!(
                "FATAL: Infer(?{}) reached concretize_type — this is a compiler bug.",
                var.raw_index()
            );
            HirType::Error
        }
        other => other,
    }
}

/// Check if a type contains an unresolved type parameter or Infer var.
pub fn has_unresolved(ty: &HirType) -> bool {
    ty.has_param() || ty.has_infer()
}

/// Build a substitution map from type params to concrete args.
pub fn build_subst(type_params: &[Symbol], type_args: &[HirType]) -> HashMap<Symbol, HirType> {
    type_params.iter().zip(type_args.iter()).map(|(p, a)| (*p, a.clone())).collect()
}

/// Discover work items from a type.
pub fn discover_from_type(
    ty: &HirType,
    interner: &Interner,
    index: &HirIndex,
    hir: &crate::Hir,
    known: &KnownSymbols,
) -> Vec<WorkItem> {
    let mut items = Vec::new();
    discover_from_type_into(ty, interner, index, hir, known, &mut items);
    items
}

fn discover_from_type_into(
    ty: &HirType,
    interner: &Interner,
    index: &HirIndex,
    hir: &crate::Hir,
    known: &KnownSymbols,
    items: &mut Vec<WorkItem>,
) {
    match ty {
        HirType::Generic(sym, args) => {
            if args.iter().any(|a| has_unresolved(a)) {
                for a in args { discover_from_type_into(a, interner, index, hir, known, items); }
                return;
            }
            if index.find_struct_def(hir, *sym).is_some() {
                items.push(WorkItem::struct_specialize(*sym, args.clone()));
            } else if index.find_enum_def(hir, *sym).is_some() {
                items.push(WorkItem::enum_specialize(*sym, args.clone()));
            }
            for a in args { discover_from_type_into(a, interner, index, hir, known, items); }
        }
        HirType::Option(inner) => {
            if !has_unresolved(inner) {
                if index.find_enum_def(hir, known.option).is_some() {
                    items.push(WorkItem::enum_specialize(known.option, vec![inner.as_ref().clone()]));
                }
            }
            discover_from_type_into(inner, interner, index, hir, known, items);
        }
        HirType::Result(ok, err) => {
            if !has_unresolved(ok) && !has_unresolved(err) {
                if index.find_enum_def(hir, known.result).is_some() {
                    items.push(WorkItem::enum_specialize(
                        known.result,
                        vec![ok.as_ref().clone(), err.as_ref().clone()],
                    ));
                }
            }
            discover_from_type_into(ok, interner, index, hir, known, items);
            discover_from_type_into(err, interner, index, hir, known, items);
        }
        HirType::RawPtr(inner) => discover_from_type_into(inner, interner, index, hir, known, items),
        HirType::Tuple(elems) => {
            for e in elems { discover_from_type_into(e, interner, index, hir, known, items); }
        }
        HirType::Func(params, ret) => {
            for p in params { discover_from_type_into(p, interner, index, hir, known, items); }
            discover_from_type_into(ret, interner, index, hir, known, items);
        }
        _ => {}
    }
}

//! Pattern substitution for monomorphization.

use crate::monomorphize::concretize;
use crate::monomorphize::mangle_table::MangleTable;
use crate::monomorphize::metadata::TypeMetadata;
use crate::symbols::KnownSymbols;
use crate::types::{HirPattern, HirType};
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

/// Substitute a pattern, concretizing types in struct/enum patterns.
/// Passes concrete state directly instead of trait-object closures.
pub fn substitute_pattern(
    pat: &HirPattern,
    interner: &mut Interner,
    mangle_table: &mut MangleTable,
    metadata: &mut TypeMetadata,
    known: &KnownSymbols,
    sub: &HashMap<Symbol, HirType>,
) -> HirPattern {
    match pat {
        HirPattern::Var(sym) => HirPattern::Var(*sym),
        HirPattern::Wild => HirPattern::Wild,
        HirPattern::Struct { name, bindings, span } => {
            let concrete_name = concretize::substitute_and_concretize(
                &HirType::Named(*name), sub, interner, mangle_table, metadata, known,
            );
            let new_name = match &concrete_name {
                HirType::Named(mangled) => *mangled,
                HirType::Generic(base, args) => mangle_table.mangle(*base, args, interner),
                _ => *name,
            };
            let new_bindings: Vec<(Symbol, HirPattern)> = bindings.iter()
                .map(|(sym, sub_pat)| {
                    (*sym, substitute_pattern(sub_pat, interner, mangle_table, metadata, known, sub))
                })
                .collect();
            HirPattern::Struct { name: new_name, bindings: new_bindings, span: *span }
        }
        HirPattern::Tuple { elements, span } => {
            let new_elements: Vec<HirPattern> = elements.iter()
                .map(|e| substitute_pattern(e, interner, mangle_table, metadata, known, sub))
                .collect();
            HirPattern::Tuple { elements: new_elements, span: *span }
        }
        HirPattern::OptionSome(inner) => {
            HirPattern::OptionSome(Box::new(substitute_pattern(inner, interner, mangle_table, metadata, known, sub)))
        }
        HirPattern::OptionNone => HirPattern::OptionNone,
        HirPattern::ResultOk(inner) => {
            HirPattern::ResultOk(Box::new(substitute_pattern(inner, interner, mangle_table, metadata, known, sub)))
        }
        HirPattern::ResultErr(inner) => {
            HirPattern::ResultErr(Box::new(substitute_pattern(inner, interner, mangle_table, metadata, known, sub)))
        }
        HirPattern::EnumVariant { enum_name, variant_name, bindings, span } => {
            let concrete_name = concretize::substitute_and_concretize(
                &HirType::Named(*enum_name), sub, interner, mangle_table, metadata, known,
            );
            let new_enum_name = match &concrete_name {
                HirType::Named(mangled) => *mangled,
                HirType::Generic(base, args) => mangle_table.mangle(*base, args, interner),
                _ => *enum_name,
            };
            let new_bindings: Vec<HirPattern> = bindings.iter()
                .map(|b| substitute_pattern(b, interner, mangle_table, metadata, known, sub))
                .collect();
            HirPattern::EnumVariant {
                enum_name: new_enum_name,
                variant_name: *variant_name,
                bindings: new_bindings,
                span: *span,
            }
        }
    }
}

```rust
//! Expression substitution: apply type substitution, assign fresh IDs.

use crate::monomorphize::concretize;
use crate::monomorphize::mangle_table::MangleTable;
use crate::monomorphize::mangling;
use crate::monomorphize::metadata::TypeMetadata;
use crate::monomorphize::pattern;
use crate::node::{HirExpr, HirStmt, MatchArm};
use crate::symbols::KnownSymbols;
use crate::types::{ExprId, HirType};
use glyim_interner::{Interner, Symbol};
use std::borrow::Cow;
use std::collections::HashMap;

/// Error when type lookup fails during substitution.
/// This is a HARD error — missing ExprId means the type checker
/// produced incomplete output, which is a compiler bug.
#[derive(Debug, Clone)]
pub struct SubstError {
    pub expr_id: ExprId,
    /// Uses Cow<'static, str> to avoid per-failure heap allocation.
    /// Known error reasons use &'static str; only truly dynamic messages
    /// allocate.
    pub reason: Cow<'static, str>,
}

pub struct SubstWalker<'a> {
    interner: &'a mut Interner,
    mangle_table: &'a mut MangleTable,
    metadata: &'a mut TypeMetadata,
    known: &'a KnownSymbols,
    input_expr_types: &'a HashMap<ExprId, HirType>,
    call_type_args: &'a HashMap<ExprId, Vec<HirType>>,
    next_id: u32,
    output_expr_types: HashMap<ExprId, HirType>,
    id_map: HashMap<ExprId, ExprId>,
    /// Set to true if any SubstError occurs. Poisons the function output.
    has_errors: bool,
}

impl<'a> SubstWalker<'a> {
    pub fn new(
        start_id: u32,
        interner: &'a mut Interner,
        mangle_table: &'a mut MangleTable,
        metadata: &'a mut TypeMetadata,
        known: &'a KnownSymbols,
        input_expr_types: &'a HashMap<ExprId, HirType>,
        call_type_args: &'a HashMap<ExprId, Vec<HirType>>,
    ) -> Self {
        Self {
            interner,
            mangle_table,
            metadata,
            known,
            input_expr_types,
            call_type_args,
            next_id: start_id,
            output_expr_types: HashMap::with_capacity(input_expr_types.len()),
            id_map: HashMap::with_capacity(input_expr_types.len()),
            has_errors: false,
        }
    }

    pub fn next_expr_id(&self) -> u32 { self.next_id }
    pub fn into_expr_types(self) -> HashMap<ExprId, HirType> { self.output_expr_types }
    pub fn into_id_map(self) -> HashMap<ExprId, ExprId> { self.id_map }
    pub fn has_errors(&self) -> bool { self.has_errors }

    fn fresh_id(&mut self, original_id: ExprId) -> ExprId {
        let id = ExprId::new(self.next_id);
        self.next_id += 1;
        self.id_map.insert(original_id, id);
        id
    }

    fn store_type(&mut self, id: ExprId, ty: HirType) {
        self.output_expr_types.insert(id, ty);
    }

    /// Look up and concretize the type for an expression.
    /// Returns Result — missing ExprId is a HARD error that poisons output.
    fn concrete_type_for(&mut self, original_id: ExprId, sub: &HashMap<Symbol, HirType>) -> Result<HirType, SubstError> {
        let orig_ty = match self.input_expr_types.get(&original_id) {
            Some(ty) => ty.clone(),
            None => {
                tracing::error!(
                    "concrete_type_for: ExprId {:?} not found in input_expr_types (len {})",
                    original_id, self.input_expr_types.len()
                );
                self.has_errors = true;
                return Err(SubstError {
                    expr_id: original_id,
                    reason: Cow::Owned(format!("ExprId {:?} not in type map", original_id)),
                });
            }
        };
        let subbed = crate::types::substitute_type(&orig_ty, sub);
        Ok(concretize::concretize_type(subbed, self.interner, self.mangle_table, self.metadata, self.known))
    }

    pub fn substitute_expr(&mut self, expr: &HirExpr, sub: &HashMap<Symbol, HirType>) -> HirExpr {
        let original_id = expr.get_id();
        let new_id = self.fresh_id(original_id);
        let concrete_ty = match self.concrete_type_for(original_id, sub) {
            Ok(ty) => ty,
            Err(_) => HirType::Error,
        };
        self.store_type(new_id, concrete_ty.clone());
        let span = expr.get_span();

        match expr {
            HirExpr::IntLit { value, .. } => HirExpr::IntLit { id: new_id, value: *value, span },
            HirExpr::FloatLit { value, .. } => HirExpr::FloatLit { id: new_id, value: *value, span },
            HirExpr::BoolLit { value, .. } => HirExpr::BoolLit { id: new_id, value: *value, span },
            HirExpr::StrLit { value, .. } => HirExpr::StrLit { id: new_id, value: value.clone(), span },
            HirExpr::UnitLit { .. } => HirExpr::UnitLit { id: new_id, span },
            HirExpr::Ident { name, .. } => HirExpr::Ident { id: new_id, name: *name, span },
            HirExpr::AddrOf { target, .. } => {
                let new_target = Box::new(self.substitute_expr(target, sub));
                HirExpr::AddrOf { id: new_id, target: new_target, span }
            }
            HirExpr::Binary { op, lhs, rhs, .. } => HirExpr::Binary {
                id: new_id, op: *op,
                lhs: Box::new(self.substitute_expr(lhs, sub)),
                rhs: Box::new(self.substitute_expr(rhs, sub)),
                span,
            },
            HirExpr::Unary { op, operand, .. } => HirExpr::Unary {
                id: new_id, op: *op,
                operand: Box::new(self.substitute_expr(operand, sub)),
                span,
            },
            HirExpr::Block { stmts, .. } => HirExpr::Block {
                id: new_id,
                stmts: stmts.iter().map(|s| self.substitute_stmt(s, sub)).collect(),
                span,
            },
            HirExpr::If { condition, then_branch, else_branch, .. } => HirExpr::If {
                id: new_id,
                condition: Box::new(self.substitute_expr(condition, sub)),
                then_branch: Box::new(self.substitute_expr(then_branch, sub)),
                else_branch: else_branch.as_ref().map(|e| Box::new(self.substitute_expr(e, sub))),
                span,
            },
            HirExpr::Call { callee, args, .. } => {
                let new_callee = self.resolve_callee(*callee, original_id, sub);
                let new_args = args.iter().map(|a| self.substitute_expr(a, sub)).collect();
                HirExpr::Call { id: new_id, callee: new_callee, args: new_args, span }
            }
            // FIX: MethodCall is NOT transformed to Call here.
            // The transformation is deferred to codegen to avoid breaking
            // the call_type_args contract (method type args don't include self,
            // but a transformed Call would have self as a positional arg).
            HirExpr::MethodCall { receiver, method_name, args, .. } => {
                let new_receiver = Box::new(self.substitute_expr(receiver, sub));
                let new_args = args.iter().map(|a| self.substitute_expr(a, sub)).collect();
                // Mangle the method name for the specialized type
                let new_method_name = self.resolve_method_name(receiver.get_id(), *method_name, original_id, sub);
                HirExpr::MethodCall {
                    id: new_id, receiver: new_receiver,
                    method_name: new_method_name,
                    args: new_args, span,
                }
            }
            HirExpr::StructLit { struct_name, fields, .. } => {
                let new_name = self.concretize_name(*struct_name, &concrete_ty);
                let new_fields = fields.iter()
                    .map(|(s, e)| (*s, self.substitute_expr(e, sub)))
                    .collect();
                HirExpr::StructLit { id: new_id, struct_name: new_name, fields: new_fields, span }
            }
            HirExpr::EnumVariant { enum_name, variant_name, args, .. } => {
                let new_name = self.concretize_name(*enum_name, &concrete_ty);
                let new_args = args.iter().map(|a| self.substitute_expr(a, sub)).collect();
                HirExpr::EnumVariant {
                    id: new_id, enum_name: new_name, variant_name: *variant_name,
                    args: new_args, span,
                }
            }
            HirExpr::FieldAccess { object, field, .. } => HirExpr::FieldAccess {
                id: new_id,
                object: Box::new(self.substitute_expr(object, sub)),
                field: *field, span,
            },
            HirExpr::As { expr: inner, target_type, .. } => {
                let new_target = concretize::substitute_and_concretize(
                    target_type, sub, self.interner, self.mangle_table, self.metadata, self.known,
                );
                HirExpr::As {
                    id: new_id,
                    expr: Box::new(self.substitute_expr(inner, sub)),
                    target_type: new_target, span,
                }
            }
            HirExpr::SizeOf { target_type, .. } => {
                let new_target = concretize::substitute_and_concretize(
                    target_type, sub, self.interner, self.mangle_table, self.metadata, self.known,
                );
                HirExpr::SizeOf { id: new_id, target_type: new_target, span }
            }
            HirExpr::Deref { expr: inner, .. } => HirExpr::Deref {
                id: new_id,
                expr: Box::new(self.substitute_expr(inner, sub)),
                span,
            },
            HirExpr::Match { scrutinee, arms, .. } => {
                let new_scrutinee = Box::new(self.substitute_expr(scrutinee, sub));
                let new_arms = arms.iter().map(|arm| MatchArm {
                    pattern: pattern::substitute_pattern(
                        &arm.pattern,
                        self.interner, self.mangle_table, self.metadata, self.known, sub,
                    ),
                    guard: arm.guard.as_ref().map(|g| self.substitute_expr(g, sub)),
                    body: self.substitute_expr(&arm.body, sub),
                }).collect();
                HirExpr::Match { id: new_id, scrutinee: new_scrutinee, arms: new_arms, span }
            }
            HirExpr::While { condition, body, .. } => HirExpr::While {
                id: new_id,
                condition: Box::new(self.substitute_expr(condition, sub)),
                body: Box::new(self.substitute_expr(body, sub)),
                span,
            },
            HirExpr::ForIn { pattern: pat, iter, body, .. } => {
                let new_pattern = pattern::substitute_pattern(
                    pat, self.interner, self.mangle_table, self.metadata, self.known, sub,
                );
                let new_iter = Box::new(self.substitute_expr(iter, sub));
                let new_body = Box::new(self.substitute_expr(body, sub));
                HirExpr::ForIn {
                    id: new_id,
                    pattern: new_pattern,
                    iter: new_iter,
                    body: new_body,
                    span,
                }
            }
            HirExpr::Return { value, .. } => HirExpr::Return {
                id: new_id,
                value: value.as_ref().map(|v| Box::new(self.substitute_expr(v, sub))),
                span,
            }
            HirExpr::Println { arg, .. } => HirExpr::Println {
                id: new_id,
                arg: Box::new(self.substitute_expr(arg, sub)),
                span,
            }
            HirExpr::Assert { condition, message, .. } => HirExpr::Assert {
                id: new_id,
                condition: Box::new(self.substitute_expr(condition, sub)),
                message: message.as_ref().map(|m| Box::new(self.substitute_expr(m, sub))),
                span,
            }
            HirExpr::TupleLit { elements, .. } => HirExpr::TupleLit {
                id: new_id,
                elements: elements.iter().map(|e| self.substitute_expr(e, sub)).collect(),
                span,
            },
        }
    }

    pub fn substitute_stmt(&mut self, stmt: &HirStmt, sub: &HashMap<Symbol, HirType>) -> HirStmt {
        match stmt {
            HirStmt::Let { name, mutable, value, span } => HirStmt::Let {
                name: *name, mutable: *mutable,
                value: self.substitute_expr(value, sub), span: *span,
            }
            HirStmt::LetPat { pattern: pat, mutable, value, ty, span } => {
                let new_ty = ty.as_ref().map(|t| {
                    concretize::substitute_and_concretize(
                        t, sub, self.interner, self.mangle_table, self.metadata, self.known,
                    )
                });
                let new_pattern = pattern::substitute_pattern(
                    pat, self.interner, self.mangle_table, self.metadata, self.known, sub,
                );
                HirStmt::LetPat {
                    pattern: new_pattern, mutable: *mutable,
                    value: self.substitute_expr(value, sub),
                    ty: new_ty, span: *span,
                }
            }
            HirStmt::Assign { target, value, span } => HirStmt::Assign {
                target: *target, value: self.substitute_expr(value, sub), span: *span,
            }
            HirStmt::AssignField { object, field, value, span } => HirStmt::AssignField {
                object: Box::new(self.substitute_expr(object, sub)),
                field: *field,
                value: self.substitute_expr(value, sub),
                span: *span,
            }
            HirStmt::AssignDeref { target, value, span } => HirStmt::AssignDeref {
                target: Box::new(self.substitute_expr(target, sub)),
                value: self.substitute_expr(value, sub),
                span: *span,
            }
            HirStmt::Expr(e) => HirStmt::Expr(self.substitute_expr(e, sub)),
        }
    }

    fn resolve_callee(&mut self, callee: Symbol, original_id: ExprId, sub: &HashMap<Symbol, HirType>) -> Symbol {
        if let Some(raw_args) = self.call_type_args.get(&original_id) {
            let concrete_args: Vec<HirType> = raw_args.iter()
                .map(|t| concretize::substitute_and_concretize(t, sub, self.interner, self.mangle_table, self.metadata, self.known))
                .collect();
            if !concrete_args.is_empty() {
                let base = mangling::demangle_base(self.interner, callee);
                return self.mangle_table.mangle_fn(base, &concrete_args, self.interner);
            }
        }
        callee
    }

    /// Resolve the method name for a MethodCall, mangling for the specialized type.
    /// Does NOT transform MethodCall into Call — that's deferred to codegen.
    fn resolve_method_name(&mut self, receiver_original_id: ExprId, method_name: Symbol, call_original_id: ExprId, sub: &HashMap<Symbol, HirType>) -> Symbol {
        let recv_ty = self.input_expr_types.get(&receiver_original_id).cloned();
        let inner_ty = recv_ty.as_ref().and_then(|ty| {
            match ty {
                HirType::RawPtr(i) => Some(i.as_ref()),
                other => Some(other),
            }
        });

        if let Some(inner) = inner_ty {
            let type_name = match inner {
                HirType::Named(n) => *n,
                HirType::Generic(n, _) => *n,
                _ => {
                    // Can't mangle — keep original method name
                    return method_name;
                }
            };
            let mangled = mangling::method_base_name(self.interner, type_name, method_name);

            // If the method is generic, mangle further with type args
            if let Some(raw_args) = self.call_type_args.get(&call_original_id) {
                let concrete_args: Vec<HirType> = raw_args.iter()
                    .map(|t| concretize::substitute_and_concretize(t, sub, self.interner, self.mangle_table, self.metadata, self.known))
                    .collect();
                if !concrete_args.is_empty() {
                    return self.mangle_table.mangle_fn(mangled, &concrete_args, self.interner);
                }
            }

            // For non-generic methods on generic types, mangle with type args from receiver
            if let HirType::Generic(_, type_args) = inner {
                let concrete_args: Vec<HirType> = type_args.iter()
                    .map(|a| concretize::substitute_and_concretize(a, sub, self.interner, self.mangle_table, self.metadata, self.known))
                    .collect();
                if !concrete_args.is_empty() {
                    return self.mangle_table.mangle_fn(mangled, &concrete_args, self.interner);
                }
            }

            return mangled;
        }

        method_name
    }

    fn concretize_name(&mut self, original: Symbol, concrete_ty: &HirType) -> Symbol {
        match concrete_ty {
            HirType::Named(mangled) => *mangled,
            HirType::Generic(base, args) => self.mangle_table.mangle(*base, args, self.interner),
            _ => original,
        }
    }
}

//! Discovery: given an expression body, return work items.
//!
//! Every HirExpr variant with sub-expressions is explicitly handled.
//! No catch-all — generic calls inside While/ForIn/Return/etc. are
//! always discovered.

use crate::index::HirIndex;
use crate::monomorphize::concretize;
use crate::monomorphize::mangle_table::MangleTable;
use crate::monomorphize::mangling;
use crate::monomorphize::metadata::TypeMetadata;
use crate::monomorphize::queue::WorkItem;
use crate::symbols::KnownSymbols;
use crate::types::{ExprId, HirType};
use crate::HirExpr;

use std::collections::HashMap;

pub fn from_body(
    expr: &crate::HirExpr,
    interner: &mut Interner,
    index: &HirIndex,
    mangle_table: &mut MangleTable,
    metadata: &mut TypeMetadata,
    known: &KnownSymbols,
    hir: &crate::Hir,
    expr_types: &HashMap<ExprId, HirType>,
    remapped_call_type_args: &HashMap<ExprId, Vec<HirType>>,
    sub: &HashMap<Symbol, HirType>,
) -> Vec<WorkItem> {

}

fn walk_expr(
    expr: &crate::HirExpr,
    interner: &mut Interner,
    index: &HirIndex,
    mangle_table: &mut MangleTable,
    metadata: &mut TypeMetadata,
    known: &KnownSymbols,
    hir: &crate::Hir,
    expr_types: &HashMap<ExprId, HirType>,
    remapped_call_type_args: &HashMap<ExprId, Vec<HirType>>,
    sub: &HashMap<Symbol, HirType>,
    items: &mut Vec<WorkItem>,
) {
    match expr {
        HirExpr::Call { id, callee, args, .. } => {
            discover_call(*id, *callee, interner, index, hir, remapped_call_type_args, items);
            for a in args {
                walk_expr(a, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::MethodCall { id, receiver, method_name, args, .. } => {
            discover_method_call(
                *id, receiver, *method_name,
                interner, index, hir, expr_types, remapped_call_type_args, known, items,
            );
            walk_expr(receiver, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            for a in args {
                walk_expr(a, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::StructLit { id, fields, .. } => {
            if let Some(ty) = expr_types.get(id) {
                let type_discoveries = concretize::discover_from_type(ty, interner, index, hir, known);
                items.extend(type_discoveries);
            }
            for (_, val) in fields {
                walk_expr(val, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::EnumVariant { id, args, .. } => {
            if let Some(ty) = expr_types.get(id) {
                let type_discoveries = concretize::discover_from_type(ty, interner, index, hir, known);
                items.extend(type_discoveries);
            }
            for a in args {
                walk_expr(a, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::Block { stmts, .. } => {
            for s in stmts {
                walk_stmt(s, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::If { condition, then_branch, else_branch, .. } => {
            walk_expr(condition, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(then_branch, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            if let Some(e) = else_branch {
                walk_expr(e, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::Match { scrutinee, arms, .. } => {
            walk_expr(scrutinee, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            for arm in arms {
                if let Some(g) = &arm.guard {
                    walk_expr(g, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
                }
                walk_expr(&arm.body, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        // FIX: Binary and Unary handled as SEPARATE match arms (no or-pattern bug)
        HirExpr::Binary { lhs, rhs, .. } => {
            walk_expr(lhs, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(rhs, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::Unary { operand, .. } => {
            walk_expr(operand, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::While { condition, body, .. } => {
            walk_expr(condition, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(body, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::ForIn { iter, body, .. } => {
            walk_expr(iter, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(body, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::Return { value, .. } => {
            if let Some(v) = value {
                walk_expr(v, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::Println { arg, .. } => {
            walk_expr(arg, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::Assert { condition, message, .. } => {
            walk_expr(condition, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            if let Some(m) = message {
                walk_expr(m, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        HirExpr::Deref { expr: inner, .. } => {
            walk_expr(inner, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::AddrOf { target, .. } => {
            walk_expr(target, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::FieldAccess { object, .. } => {
            walk_expr(object, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::As { expr: inner, .. } => {
            walk_expr(inner, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        HirExpr::TupleLit { elements, .. } => {
            for e in elements {
                walk_expr(e, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            }
        }
        // Leaf expressions — no sub-expressions to walk
        HirExpr::IntLit { .. }
        | HirExpr::FloatLit { .. }
        | HirExpr::BoolLit { .. }
        | HirExpr::StrLit { .. }
        | HirExpr::UnitLit { .. }
        | HirExpr::Ident { .. }
        | HirExpr::SizeOf { .. } => {}
    }
}

fn walk_stmt(
    stmt: &crate::HirStmt,
    interner: &mut Interner,
    index: &HirIndex,
    mangle_table: &mut MangleTable,
    metadata: &mut TypeMetadata,
    known: &KnownSymbols,
    hir: &crate::Hir,
    expr_types: &HashMap<ExprId, HirType>,
    remapped_call_type_args: &HashMap<ExprId, Vec<HirType>>,
    sub: &HashMap<Symbol, HirType>,
    items: &mut Vec<WorkItem>,
) {
    match stmt {
        crate::HirStmt::Expr(e) => {
            walk_expr(e, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        crate::HirStmt::Let { value, .. } => {
            walk_expr(value, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }

        }
        crate::HirStmt::Assign { value, .. } => {
            walk_expr(value, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }

            walk_expr(object, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(value, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
        crate::HirStmt::AssignDeref { target, value, .. } => {
            walk_expr(target, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
            walk_expr(value, interner, index, mangle_table, metadata, known, hir, expr_types, remapped_call_type_args, sub, items);
        }
    }
}

fn discover_call(
    new_id: ExprId,
    callee: Symbol,
    interner: &mut Interner,
    index: &HirIndex,
    hir: &crate::Hir,
    remapped_call_type_args: &HashMap<ExprId, Vec<HirType>>,
    items: &mut Vec<WorkItem>,
) {
    let raw_type_args = match remapped_call_type_args.get(&new_id) {
        Some(args) => args.clone(),
        None => return,
    };

    if raw_type_args.is_empty() { return; }
    if raw_type_args.iter().any(|t| concretize::has_unresolved(t)) {
        return;
    }

    let base_callee = mangling::demangle_base(interner, callee);
    if index.is_generic_fn(base_callee) {
        items.push(WorkItem::fn_specialize(base_callee, raw_type_args));
    } else if index.find_fn_def(hir, base_callee).is_some() {
        items.push(WorkItem::fn_passthrough(base_callee));
    }
}

/// FIX: Receives `&KnownSymbols` through the walk chain instead of
/// creating a stub with wrong Interner.
fn discover_method_call(
    new_id: ExprId,
    receiver: &crate::HirExpr,
    method_name: Symbol,
    interner: &mut Interner,
    index: &HirIndex,
    hir: &crate::Hir,
    expr_types: &HashMap<ExprId, HirType>,
    remapped_call_type_args: &HashMap<ExprId, Vec<HirType>>,
    known: &KnownSymbols,
    items: &mut Vec<WorkItem>,
) {
    let recv_ty = expr_types.get(&receiver.get_id())
        .cloned()
        .unwrap_or(HirType::Error);
    let inner_ty = match &recv_ty {
        HirType::RawPtr(i) => i.as_ref().clone(),
        other => other.clone(),
    };
    let type_name = match &inner_ty {
        HirType::Named(n) => *n,
        HirType::Generic(n, _) => *n,
        _ => return,
    };
    let mangled = mangling::method_base_name(interner, type_name, method_name);

    if let Some(raw_args) = remapped_call_type_args.get(&new_id) {
        if !raw_args.is_empty() && raw_args.iter().all(|t| !concretize::has_unresolved(t)) {
            if index.is_generic_fn(mangled) {
                items.push(WorkItem::fn_specialize(mangled, raw_args.clone()));
            } else if index.find_fn_def(hir, mangled).is_some() {
                items.push(WorkItem::fn_passthrough(mangled));
            }
            return;
        }
    }

    // Use the passed-in known (not a stub with wrong Interner!)
    let type_discoveries = concretize::discover_from_type(&recv_ty, interner, index, hir, known);
    items.extend(type_discoveries);

    if !index.is_generic_fn(mangled) {
        if index.find_fn_def(hir, mangled).is_some() {
            items.push(WorkItem::fn_passthrough(mangled));
        }
        return;
    }

    if let HirType::Generic(_, type_args) = &inner_ty {
        if !type_args.is_empty() && type_args.iter().all(|t| !concretize::has_unresolved(t)) {
            items.push(WorkItem::fn_specialize(mangled, type_args.clone()));
        }
    }
}

```rust
//! BFS monomorphization driver.
//!
//! ARCHITECTURE:
//! - MonoOutput: the output being built (items, expr_types, etc.)
//! - MonoState: the working state (mangle table, metadata, queue, index)
//! - MonoInput: read-only inputs (interner, known, input type maps)
//!
//! The driver owns all state directly (no MonoContext god-borrow).
//! Sub-module functions receive individual &mut refs via partial borrows.

use crate::index::HirIndex;
use crate::monomorphize::concretize;
use crate::monomorphize::discover;
use crate::monomorphize::mangle_table::MangleTable;
use crate::monomorphize::mangling;
use crate::monomorphize::metadata::TypeMetadata;
use crate::monomorphize::queue::{ItemKind, WorkItem, WorkQueue};
use crate::monomorphize::specialize;
use crate::monomorphize::subst::SubstWalker;
use crate::symbols::KnownSymbols;
use crate::types::HirType;
use crate::Hir;
use crate::HirItem;
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

pub struct MonoResult {
    pub hir: crate::Hir,
    pub expr_types: HashMap<crate::types::ExprId, HirType>,
    pub metadata: TypeMetadata,
    pub metrics: MonoMetrics,
}

#[derive(Debug, Default)]
pub struct MonoMetrics {
    pub fn_specializations: usize,
    pub fn_passthroughs: usize,
    pub struct_specializations: usize,
    pub struct_passthroughs: usize,
    pub enum_specializations: usize,
    pub enum_passthroughs: usize,
    pub max_queue_depth: usize,
    pub metadata_entries: usize,
    pub skipped_unresolved: usize,
    pub errors: usize,
    /// Specific failed item names for debuggability.
    pub failed_items: Vec<String>,
}

/// Output being built by the driver.
struct MonoOutput {
    items: Vec<HirItem>,
    expr_types: HashMap<crate::types::ExprId, HirType>,
    id_map: HashMap<crate::types::ExprId, crate::types::ExprId>,
    remapped_call_type_args: HashMap<crate::types::ExprId, Vec<HirType>>,
    next_expr_id: u32,
}

/// Working state of the driver.
struct MonoState {
    mangle: MangleTable,
    metadata: TypeMetadata,
    queue: WorkQueue,
    index: HirIndex,
    metrics: MonoMetrics,
}

/// Read-only inputs to the driver.
struct MonoInput<'a> {
    interner: &'a mut Interner,
    known: &'a KnownSymbols,
    input_expr_types: &'a HashMap<crate::types::ExprId, HirType>,
    input_call_type_args: &'a HashMap<crate::types::ExprId, Vec<HirType>>,
}

pub struct MonoDriver<'a> {
    output: MonoOutput,
    state: MonoState,
    input: MonoInput<'a>,
}

impl<'a> MonoDriver<'a> {
    pub fn new(
        interner: &'a mut Interner,
        known: &'a KnownSymbols,
        input_expr_types: &'a HashMap<crate::types::ExprId, HirType>,
        input_call_type_args: &'a HashMap<crate::types::ExprId, Vec<HirType>>,
        hir: &Hir,
    ) -> Self {
        Self {
            output: MonoOutput {
                items: Vec::new(),
                expr_types: HashMap::new(),
                id_map: HashMap::new(),
                remapped_call_type_args: HashMap::new(),
                next_expr_id: 0,
            },
            state: MonoState {
                mangle: MangleTable::new(),
                metadata: TypeMetadata::new(),
                queue: WorkQueue::new(),
                index: HirIndex::build(hir, interner),
                metrics: MonoMetrics::default(),
            },
            input: MonoInput {
                interner,
                known,
                input_expr_types,
                input_call_type_args,
            },
        }
    }

    pub fn run(&mut self, hir: &Hir, entry_points: &[Symbol]) {
        self.seed_queue(entry_points);

        while let Some(item) = self.state.queue.pop() {
            self.state.metrics.max_queue_depth = self.state.metrics.max_queue_depth.max(self.state.queue.len());

            tracing::debug!(
                kind = ?item.kind,
                name = %self.input.interner.resolve(item.def_id),
                args = ?item.type_args.iter().map(|t| mangling::type_to_short_string(t, self.input.interner)).collect::<Vec<_>>(),
                "Processing mono item"
            );

            match item.kind {
                ItemKind::FnSpecialize => {
                    self.state.metrics.fn_specializations += 1;
                    self.process_fn_specialize(hir, item.def_id, item.type_args)
                }
                ItemKind::FnPassthrough => {
                    self.state.metrics.fn_passthroughs += 1;
                    self.process_fn_passthrough(hir, item.def_id)
                }
                ItemKind::StructSpecialize => {
                    self.state.metrics.struct_specializations += 1;
                    self.process_struct_specialize(hir, item.def_id, item.type_args)
                }
                ItemKind::StructPassthrough => {
                    self.state.metrics.struct_passthroughs += 1;
                    self.process_struct_passthrough(hir, item.def_id)
                }
                ItemKind::EnumSpecialize => {
                    self.state.metrics.enum_specializations += 1;
                    self.process_enum_specialize(hir, item.def_id, item.type_args)
                }
                ItemKind::EnumPassthrough => {
                    self.state.metrics.enum_passthroughs += 1;
                    self.process_enum_passthrough(hir, item.def_id)
                }
            }
        }

        // Pass through extern functions
        for item in &hir.items {
            if let HirItem::Extern(ext) = item {
                self.output.items.push(HirItem::Extern(ext.clone()));
            }
        }
    }

    fn seed_queue(&mut self, entry_points: &[Symbol]) {
        for &ep in entry_points {
            self.state.queue.push(WorkItem::fn_passthrough(ep));
        }
    }

    // ── Function processing ────────────────────────────────────

    fn process_fn_specialize(&mut self, hir: &Hir, name: Symbol, type_args: Vec<HirType>) {
        let generic_fn = match self.state.index.find_fn_def(hir, name) {
            Some(f) => f,
            None => {
                tracing::warn!(name = %self.input.interner.resolve(name), "FnSpecialize: function not found, skipping");
                return;
            }
        };
        if type_args.len() < generic_fn.type_params.len() {
            tracing::warn!(name = %self.input.interner.resolve(name), "FnSpecialize: insufficient type args, skipping");
            self.state.metrics.skipped_unresolved += 1;
            return;
        }
        if type_args.iter().any(|t| concretize::has_unresolved(t)) {
            tracing::warn!(name = %self.input.interner.resolve(name), "FnSpecialize: unresolved type args, skipping");
            self.state.metrics.skipped_unresolved += 1;
            return;
        }

        let sub = concretize::build_subst(&generic_fn.type_params, &type_args);
        let mangled_name = self.state.mangle.mangle_fn(name, &type_args, self.input.interner);

        // Discover from param and return types
        for (_, ty) in &generic_fn.params {
            let sub_ty = crate::types::substitute_type(ty, &sub);
            let discoveries = concretize::discover_from_type(&sub_ty, self.input.interner, &self.state.index, hir, self.input.known);
            self.state.queue.extend(discoveries);
        }
        if let Some(ret) = &generic_fn.ret {
            let sub_ty = crate::types::substitute_type(ret, &sub);
            let discoveries = concretize::discover_from_type(&sub_ty, self.input.interner, &self.state.index, hir, self.input.known);
            self.state.queue.extend(discoveries);
        }

        self.process_fn_body(hir, generic_fn, Some((&sub, mangled_name)));
    }

    fn process_fn_passthrough(&mut self, hir: &Hir, name: Symbol) {
        let fn_def = match self.state.index.find_fn_def(hir, name) {
            Some(f) => f,
            None => {
                tracing::warn!(name = %self.input.interner.resolve(name), "FnPassthrough: function not found, skipping");
                return;
            }
        };
        self.process_fn_body(hir, fn_def, None);
    }

    fn process_fn_body(
        &mut self,
        hir: &Hir,
        fn_def: &crate::HirFn,
        specialize: Option<(&HashMap<Symbol, HirType>, Symbol)>,
    ) {
        let sub = specialize.map(|(s, _)| s.clone()).unwrap_or_default();
        let mangled_name = specialize.map(|(_, m)| m).unwrap_or(fn_def.name);

        let start_id = self.output.next_expr_id;

        // Extract fields for partial borrows
        let interner = &mut *self.input.interner;
        let mangle_table = &mut self.state.mangle;
        let metadata = &mut self.state.metadata;
        let known = self.input.known;
        let input_expr_types = self.input.input_expr_types;
        let input_call_type_args = self.input.input_call_type_args;

        let mut walker = SubstWalker::new(
            start_id, interner, mangle_table, metadata, known,
            input_expr_types, input_call_type_args,
        );
        let new_body = walker.substitute_expr(&fn_def.body, &sub);
        self.output.next_expr_id = walker.next_expr_id();
        let local_expr_types = walker.into_expr_types();
        let local_id_map = walker.into_id_map();
        let walker_had_errors = walker.has_errors();

        // If SubstWalker failed, do NOT emit garbage to output
        if walker_had_errors {
            self.state.metrics.errors += 1;
            self.state.metrics.failed_items.push(self.input.interner.resolve(mangled_name).to_string());
            return;
        }

        self.output.expr_types.extend(local_expr_types);

        // CONCRETIZE remapped_call_type_args
        for (&old_id, &new_id) in &local_id_map {
            self.output.id_map.insert(old_id, new_id);
            if let Some(raw_args) = self.input.input_call_type_args.get(&old_id) {
                let concretized_args: Vec<HirType> = raw_args.iter()
                    .map(|t| concretize::substitute_and_concretize(
                        t, &sub, self.input.interner, &mut self.state.mangle, &mut self.state.metadata, self.input.known,
                    ))
                    .collect();
                self.output.remapped_call_type_args.insert(new_id, concretized_args);
            }
        }

        // Discover from the substituted body
        let discoveries = discover::from_body(
            &new_body, self.input.interner, &self.state.index, &mut self.state.mangle, &mut self.state.metadata, self.input.known, hir,
            &self.output.expr_types, &self.output.remapped_call_type_args, &sub,
        );
        self.state.queue.extend(discoveries);

        // Build concrete params without cloning the entire HirFn body
        let concrete_params: Vec<(Symbol, HirType)> = fn_def.params.iter()
            .map(|(sym, ty)| {
                let ct = concretize::substitute_and_concretize(
                    ty, &sub, self.input.interner, &mut self.state.mangle, &mut self.state.metadata, self.input.known,
                );
                (*sym, ct)
            })
            .collect();

        let concrete_ret = fn_def.ret.as_ref().map(|ty| {
            concretize::substitute_and_concretize(
                ty, &sub, self.input.interner, &mut self.state.mangle, &mut self.state.metadata, self.input.known,
            )
        });

        // Clone only metadata, substitute body directly
        let concrete_fn = crate::HirFn {
            name: mangled_name,
            type_params: vec![],
            params: concrete_params,
            param_mutability: fn_def.param_mutability.clone(),
            ret: concrete_ret,
            body: new_body,
            span: fn_def.span,
        };

        self.output.items.push(HirItem::Fn(concrete_fn));
    }

    // ── Struct processing ──────────────────────────────────────

    fn process_struct_specialize(&mut self, hir: &Hir, name: Symbol, type_args: Vec<HirType>) {
        let generic = match self.state.index.find_struct_def(hir, name) {
            Some(s) => s,
            None => {
                tracing::warn!(name = %self.input.interner.resolve(name), "StructSpecialize: struct not found, skipping");
                return;
            }
        };
        if type_args.iter().any(|t| concretize::has_unresolved(t)) {
            self.state.metrics.skipped_unresolved += 1;
            return;
        }

        let sub = concretize::build_subst(&generic.type_params, &type_args);
        let concrete = specialize::specialize_struct(
            &generic, &type_args, &sub,
            self.input.interner, &mut self.state.mangle, &mut self.state.metadata, self.input.known,
        );

        for field in &generic.fields {
            let sub_ty = crate::types::substitute_type(&field.ty, &sub);
            let discoveries = concretize::discover_from_type(&sub_ty, self.input.interner, &self.state.index, hir, self.input.known);
            self.state.queue.extend(discoveries);
        }

        self.output.items.push(HirItem::Struct(concrete));
    }

    fn process_struct_passthrough(&mut self, hir: &Hir, name: Symbol) {
        if let Some(s) = self.state.index.find_struct_def(hir, name).cloned() {
            self.output.items.push(HirItem::Struct(s));
        } else {
            tracing::warn!(name = %self.input.interner.resolve(name), "StructPassthrough: struct not found, skipping");
        }
    }

    // ── Enum processing ────────────────────────────────────────

    fn process_enum_specialize(&mut self, hir: &Hir, name: Symbol, type_args: Vec<HirType>) {
        let generic = match self.state.index.find_enum_def(hir, name) {
            Some(e) => e,
            None => {
                tracing::warn!(name = %self.input.interner.resolve(name), "EnumSpecialize: enum not found, skipping");
                return;
            }
        };
        if type_args.iter().any(|t| concretize::has_unresolved(t)) {
            self.state.metrics.skipped_unresolved += 1;
            return;
        }

        let sub = concretize::build_subst(&generic.type_params, &type_args);
        let concrete = specialize::specialize_enum(
            &generic, &type_args, &sub,
            self.input.interner, &mut self.state.mangle, &mut self.state.metadata, self.input.known,
        );

        for variant in &generic.variants {
            for field in &variant.fields {
                let sub_ty = crate::types::substitute_type(&field.ty, &sub);
                let discoveries = concretize::discover_from_type(&sub_ty, self.input.interner, &self.state.index, hir, self.input.known);
                self.state.queue.extend(discoveries);
            }
        }

        self.output.items.push(HirItem::Enum(concrete));
    }

    fn process_enum_passthrough(&mut self, hir: &Hir, name: Symbol) {
        if let Some(e) = self.state.index.find_enum_def(hir, name).cloned() {
            self.output.items.push(HirItem::Enum(e));
        } else {
            tracing::warn!(name = %self.input.interner.resolve(name), "EnumPassthrough: enum not found, skipping");
        }
    }

    pub fn into_result(self) -> MonoResult {
        let metadata_entries = self.state.metadata.len();
        MonoResult {
            hir: crate::Hir { items: self.output.items },
            expr_types: self.output.expr_types,
            metadata: self.state.metadata,
            metrics: MonoMetrics {
                metadata_entries,
                ..self.state.metrics
            },
        }
    }
}
