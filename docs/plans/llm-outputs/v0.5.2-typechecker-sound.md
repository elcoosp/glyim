# Step 1: Typechecker P0 Fixes — Implementation Plan

> **For agentic workers:** Use `superpowers:executing-plans` to implement. Steps use checkbox syntax for tracking.

**Goal:** Make the typechecker produce data that any monomorphizer can use correctly — no silent `Int` fallbacks, and `call_type_args` populated for generic method calls.

**Principle:** Every `HirType::Int` that appears as a fallback when inference fails becomes `HirType::Error` plus a reported error. This makes failures visible instead of silently producing wrong types.

---

## Audit: Find All `Int` Fallbacks

Before writing any code, we need a complete inventory of every place the typechecker defaults to `Int`. Run this in the typechecker crate:

```bash
grep -n "HirType::Int" crates/glyim-typeck/src/typechecker.rs
```

Typical locations (based on the patterns in the codebase):

1. End of `infer_expr` — catch-all fallback
2. `EnumVariant` arm — when enum is unknown
3. `StructLit` arm — when struct is unknown or generic
4. `MethodCall` arm — when method can't be resolved
5. `Call` arm — when callee is unknown
6. `As` arm — when cast target is unknown
7. `FieldAccess` arm — when object type is unknown
8. `Ident` arm — when identifier is unknown
9. `Return` arm — when return value type is unknown
10. Parameter/return type defaults in `check_fn`

Each needs individual treatment — some should produce `Error`, some should produce a more specific type.

## Task 2: Add `extract_type_substitutions` Helper

**Files:** Create or modify a helpers module in `crates/glyim-typeck/src/`

- [ ] **Step 1: Write the helper function**

This function performs positional pattern-matching between a type schema (which may contain type parameters) and a concrete type, extracting bindings for the type parameters.

```rust
// In crates/glyim-typeck/src/unify.rs (new file)

//! Simple type unification for extracting type parameter bindings.
//!
//! This is NOT a full Hindley-Milner unification engine. It performs
//! positional pattern-matching between a schema type and a concrete type,
//! extracting bindings for known type parameters.
//!
//! Example: schema = `Vec<T>`, concrete = `Vec<Int>` → `{ T => Int }`

use glyim_hir::types::HirType;
use glyim_interner::Symbol;
use std::collections::HashMap;

/// Extract type parameter bindings by matching a schema against a concrete type.
///
/// `schema` is the type from the generic definition (e.g., `T`, `Vec<T>`, `(T, U)`).
/// `concrete` is the actual type at the call site (e.g., `Int`, `Vec<Int>`, `(Int, Bool)`).
/// `type_params` is the set of symbols that are type parameters (not type names).
///
/// Results are accumulated into `sub`. If a parameter is already bound and the
/// new binding conflicts, the conflict is silently ignored (the first binding wins).
/// Callers should check for conflicts if needed.
pub fn extract_type_substitutions(
    schema: &HirType,
    concrete: &HirType,
    type_params: &[Symbol],
    sub: &mut HashMap<Symbol, HirType>,
) {
    match (schema, concrete) {
        // Type parameter matches anything — record the binding
        (HirType::Named(sym), _) if type_params.contains(sym) => {
            // Only insert if not already bound (first binding wins)
            sub.entry(*sym).or_insert_with(|| concrete.clone());
        }

        // Generic types: match base name and recurse on args
        (HirType::Generic(sym_a, args_a), HirType::Generic(sym_b, args_b))
            if sym_a == sym_b && args_a.len() == args_b.len() =>
        {
            for (a, b) in args_a.iter().zip(args_b.iter()) {
                extract_type_substitutions(a, b, type_params, sub);
            }
        }

        // Named matching Named (not a type param) — must match exactly, no bindings
        (HirType::Named(a), HirType::Named(b)) if a == b => {}

        // RawPtr
        (HirType::RawPtr(inner_s), HirType::RawPtr(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }

        // Option
        (HirType::Option(inner_s), HirType::Option(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }

        // Result
        (HirType::Result(ok_s, err_s), HirType::Result(ok_c, err_c)) => {
            extract_type_substitutions(ok_s, ok_c, type_params, sub);
            extract_type_substitutions(err_s, err_c, type_params, sub);
        }

        // Tuple
        (HirType::Tuple(elems_s), HirType::Tuple(elems_c))
            if elems_s.len() == elems_c.len() =>
        {
            for (s, c) in elems_s.iter().zip(elems_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
        }

        // Function types
        (HirType::Func(params_s, ret_s), HirType::Func(params_c, ret_c))
            if params_s.len() == params_c.len() =>
        {
            for (s, c) in params_s.iter().zip(params_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
            extract_type_substitutions(ret_s, ret_c, type_params, sub);
        }

        // A type parameter schema can match a concrete Generic
        // e.g., schema = `Named("T")` matches `Generic("Vec", [Int])` if T is a type param
        (HirType::Named(sym), HirType::Generic(_, _)) if type_params.contains(sym) => {
            sub.entry(*sym).or_insert_with(|| concrete.clone());
        }

        // Mismatch or unsupported — skip (caller may report an error)
        _ => {}
    }
}

/// Build a substitution map by unifying function parameters with argument types.
///
/// Returns `Ok(sub)` if unification succeeded, `Err` if conflicts were found.
pub fn unify_fn_call(
    fn_params: &[(Symbol, HirType)],
    arg_types: &[HirType],
    type_params: &[Symbol],
) -> Result<HashMap<Symbol, HirType>, UnifyError> {
    let mut sub = HashMap::new();

    for ((_, param_ty), arg_ty) in fn_params.iter().zip(arg_types.iter()) {
        extract_type_substitutions(param_ty, arg_ty, type_params, &mut sub);
    }

    // Check for conflicting bindings
    // (extract_type_substitutions uses first-binding-wins, but we want to detect conflicts)
    let mut conflict_sub = HashMap::new();
    for ((_, param_ty), arg_ty) in fn_params.iter().zip(arg_types.iter()) {
        check_for_conflicts(param_ty, arg_ty, type_params, &sub, &mut conflict_sub)?;
    }

    Ok(sub)
}

/// Check for conflicting type parameter bindings.
fn check_for_conflicts(
    schema: &HirType,
    concrete: &HirType,
    type_params: &[Symbol],
    sub: &HashMap<Symbol, HirType>,
    _conflict_sub: &mut HashMap<Symbol, (HirType, HirType)>,
) -> Result<(), UnifyError> {
    match (schema, concrete) {
        (HirType::Named(sym), _) if type_params.contains(sym) => {
            if let Some(existing) = sub.get(sym) {
                // Check structural equality (simplified)
                if !types_structurally_equal(existing, concrete) {
                    return Err(UnifyError::Conflict {
                        param: *sym,
                        existing: existing.clone(),
                        new: concrete.clone(),
                    });
                }
            }
            Ok(())
        }
        (HirType::Generic(sym_a, args_a), HirType::Generic(sym_b, args_b))
            if sym_a == sym_b && args_a.len() == args_b.len() =>
        {
            for (a, b) in args_a.iter().zip(args_b.iter()) {
                check_for_conflicts(a, b, type_params, sub, _conflict_sub)?;
            }
            Ok(())
        }
        _ => Ok(()),
    }
}

/// Simplified structural equality check for types.
fn types_structurally_equal(a: &HirType, b: &HirType) -> bool {
    match (a, b) {
        (HirType::Int, HirType::Int) => true,
        (HirType::Bool, HirType::Bool) => true,
        (HirType::Float, HirType::Float) => true,
        (HirType::Str, HirType::Str) => true,
        (HirType::Unit, HirType::Unit) => true,
        (HirType::Named(a), HirType::Named(b)) => a == b,
        (HirType::Generic(sa, aa), HirType::Generic(sb, ab)) => {
            sa == sb
                && aa.len() == ab.len()
                && aa.iter().zip(ab.iter()).all(|(a, b)| types_structurally_equal(a, b))
        }
        _ => false,
    }
}

/// Errors from unification.
#[derive(Debug, Clone)]
pub enum UnifyError {
    /// A type parameter has conflicting bindings.
    Conflict {
        param: Symbol,
        existing: HirType,
        new: HirType,
    },
}
```

- [ ] **Step 1: Locate the `MethodCall` arm in `infer_expr`**

- [ ] **Step 2: Add `call_type_args` population**

The structure of the fix:

```rust
HirExpr::MethodCall { id, receiver, method_name, args, .. } => {
    // 1. Type-check receiver and arguments
    let recv_ty = self.infer_expr(receiver);
    for a in args { self.check_expr(a); }
    
    let arg_types: Vec<HirType> = args.iter()
        .map(|a| self.expr_types[a.get_id().as_usize()].clone())
        .collect();

    // 2. Resolve the receiver's base type
    let (type_name, type_args) = match &recv_ty {
        HirType::Named(name) => (*name, vec![]),
        HirType::Generic(name, args) => (*name, args.clone()),
        HirType::RawPtr(inner) => match inner.as_ref() {
            HirType::Named(name) => (*name, vec![]),
            HirType::Generic(name, args) => (*name, args.clone()),
            _ => {
                self.errors.push(TypeError::UnresolvedMethod {
                    method_name: self.interner.resolve(*method_name).into(),
                    receiver_type: format!("{:?}", recv_ty),
                    span: (expr.get_span().start, expr.get_span().end),
                });
                return HirType::Error;
            }
        },
        _ => {
            self.errors.push(TypeError::UnresolvedMethod {
                method_name: self.interner.resolve(*method_name).into(),
                receiver_type: format!("{:?}", recv_ty),
                span: (expr.get_span().start, expr.get_span().end),
            });
            return HirType::Error;
        }
    };

    // 3. Build the mangled method name (TypeName_method)
    let mangled = self.interner.intern(&format!(
        "{}_{}",
        self.interner.resolve(type_name),
        self.interner.resolve(*method_name),
    ));

    // 4. Look up the method
    let fn_def = self.find_fn(mangled);  // or however the typechecker looks up functions
    
    match fn_def {
        Some(fn_def) if !fn_def.type_params.is_empty() => {
            // Generic method — unify to get type args
            let all_arg_types: Vec<HirType> = std::iter::once(recv_ty.clone())
                .chain(arg_types.iter().cloned())
                .collect();
            
            let sub = crate::unify::unify_fn_call(
                &fn_def.params,
                &all_arg_types,
                &fn_def.type_params,
            );
            
            match sub {
                Ok(sub) => {
                    let concrete_args: Vec<HirType> = fn_def.type_params.iter()
                        .map(|tp| sub.get(tp).cloned().unwrap_or(HirType::Error))
                        .collect();
                    
                    if concrete_args.iter().any(|t| matches!(t, HirType::Error)) {
                        self.errors.push(TypeError::CannotInferGenericArgs {
                            name: format!(
                                "{}.{}",
                                self.interner.resolve(type_name),
                                self.interner.resolve(*method_name)
                            ),
                            span: (expr.get_span().start, expr.get_span().end),
                        });
                    } else {
                        self.call_type_args.insert(*id, concrete_args);
                    }
                    
                    // Compute return type
                    let ret = fn_def.ret.clone().unwrap_or(HirType::Unit);
                    crate::types::substitute_type(&ret, &sub)
                }
                Err(_) => {
                    self.errors.push(TypeError::CannotInferGenericArgs {
                        name: format!(
                            "{}.{}",
                            self.interner.resolve(type_name),
                            self.interner.resolve(*method_name)
                        ),
                        span: (expr.get_span().start, expr.get_span().end),
                    });
                    HirType::Error
                }
            }
        }
        Some(fn_def) => {
            // Non-generic method — just use its return type
            fn_def.ret.clone().unwrap_or(HirType::Unit)
        }
        None => {
            // Method not found — could be extern-backed or an error
            // Check extern_methods before reporting error
            if self.extern_methods.contains_key(method_name) {
                // Extern-backed method — can't infer type, return receiver type as fallback
                recv_ty
            } else {
                self.errors.push(TypeError::UnresolvedMethod {
                    method_name: self.interner.resolve(*method_name).into(),
                    receiver_type: self.interner.resolve(type_name).into(),
                    span: (expr.get_span().start, expr.get_span().end),
                });
                HirType::Error
            }
        }
    }
}
```
