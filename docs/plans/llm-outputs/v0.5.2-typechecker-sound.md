## Architecture

**Core principle:** A single `HirType` with `Infer(TypeVar)` for inference. One `UnificationTable` per function (shared across all unification within that function). The `freeze` boundary ensures no `Infer` variants leak past type checking. The monomorphizer uses a `SubstWalker` that maintains an old→new `ExprId` map so `call_type_args` can be correctly remapped.

**Key decisions:**

1. **`Infer(TypeVar)` lives in `HirType`** but the `freeze` module is the validation gateway — downstream code receives `Vec<HirType>` with a debug-asserted invariant that no `Infer` remains.
2. **One `UnificationTable` per function body.** `solve_generic_params` borrows the caller's table instead of creating its own. This enables constraint propagation within a function.
3. **Block expressions introduce scopes.** `TypeEnv` uses a scope-stack (`Vec<Vec<...>>`) for O(1) push/pop.
4. **`has_type_param` uses the Registry**, not a heuristic. The TypeChecker tracks `current_type_params: HashSet<Symbol>`.
5. **ForIn is NOT desugared in the monomorphizer.** The type checker records iterator type info; the monomorphizer's discover generates work items from the iterable type; SubstWalker passes ForIn through. Desugaring is deferred to codegen.
6. **SubstWalker maintains `id_map: HashMap<ExprId, ExprId>`** (old→new). The driver remaps `call_type_args` before calling discover on substituted bodies.
7. **Mangling uses `__` separator.** The lexer/parser rejects user identifiers containing `__`, reserving it for the implementation (same as C++).
8. **BFS from entry points only**, not all non-generic items.
9. **`merge_mono_types` is deleted.** The monomorphizer must produce fully concrete output.
10. **Dead modules deleted:** `biabduction.rs`, `zippering.rs`, `context.rs`.

```rust
        match self {
            HirType::Infer(_) => true,
            HirType::Generic(_, args) => args.iter().any(|a| a.has_infer()),
```

```rust
//! Unification table operating directly on `HirType`.
//!
//! Uses union-find with path compression on every `find` call.
//! `Infer(TypeVar)` is the only representation of type variables.

use crate::errors::{AutoFix, SpanBounds, TypeError};
use glyim_hir::types::{HirType, TypeVar};
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

/// Union-find–based unification table.
pub struct UnificationTable {
    /// Parent pointers. parents[i] == i means root.
    parents: Vec<u32>,
    /// Rank for union-by-rank.
    ranks: Vec<u8>,
    /// Span where each TypeVar was created (for diagnostics).
    spans: Vec<SpanBounds>,
    /// Bindings: root TypeVar index → concrete HirType.
    bindings: HashMap<u32, HirType>,
}

impl Default for UnificationTable {
    fn default() -> Self { Self::new() }
}

impl UnificationTable {
    pub fn new() -> Self {
        Self {
            parents: Vec::new(),
            ranks: Vec::new(),
            spans: Vec::new(),
            bindings: HashMap::new(),
        }
    }

    /// Create a fresh type variable associated with a source span.
    pub fn fresh_var(&mut self, span: SpanBounds) -> TypeVar {
        let idx = self.parents.len() as u32;
        self.parents.push(idx);
        self.ranks.push(0);
        self.spans.push(span);
        TypeVar::new(idx)
    }

    /// Find root with path compression. Every call compresses.
    pub fn find(&mut self, var: TypeVar) -> TypeVar {
        let mut root = var.0;
        while self.parents[root as usize] != root {
            root = self.parents[root as usize];
        }
        // Path compression: point every node on the path directly to root.
        let mut current = var.0;
        while current != root {
            let parent = self.parents[current as usize];
            self.parents[current as usize] = root;
            current = parent;
        }
        TypeVar::new(root)
    }

    /// Read-only find (no compression). Use only when &self is required.
    pub fn find_root(&self, var: TypeVar) -> TypeVar {
        let mut current = var.0;
        while self.parents[current as usize] != current {
            current = self.parents[current as usize];
        }
        TypeVar::new(current)
    }

    /// Resolve a `HirType`, chasing all `Infer` bindings.
    /// Unbound vars remain as `Infer(root)`.
    pub fn resolve(&mut self, ty: &HirType, interner: &Interner) -> HirType {
        match ty {
            HirType::Infer(var) => {
                let root = self.find(*var);
                match self.bindings.get(&root.0) {
                    Some(bound) => self.resolve(bound, interner),
                    None => HirType::Infer(root),
                }
            }
            HirType::Generic(sym, args) => HirType::Generic(
                *sym,
                args.iter().map(|a| self.resolve(a, interner)).collect(),
            ),
            HirType::Tuple(elems) => {
                HirType::Tuple(elems.iter().map(|e| self.resolve(e, interner)).collect())
            }
            HirType::RawPtr(inner) => {
                HirType::RawPtr(Box::new(self.resolve(inner, interner)))
            }
            HirType::Option(inner) => {
                HirType::Option(Box::new(self.resolve(inner, interner)))
            }
            HirType::Result(ok, err) => HirType::Result(
                Box::new(self.resolve(ok, interner)),
                Box::new(self.resolve(err, interner)),
            ),
            HirType::Func(params, ret) => HirType::Func(
                params.iter().map(|p| self.resolve(p, interner)).collect(),
                Box::new(self.resolve(ret, interner)),
            ),
            _ => ty.clone(),
        }
    }

    /// Unify two types. Returns `Ok(())` on success, `Err(())` on failure.
    /// Errors are pushed into `emit_err`.
    pub fn unify(
        &mut self,
        a: &HirType,
        b: &HirType,
        span: SpanBounds,
        emit_err: &mut dyn FnMut(TypeError),
        interner: &Interner,
    ) -> Result<(), ()> {
        let a_resolved = self.resolve(a, interner);
        let b_resolved = self.resolve(b, interner);

        if a_resolved == b_resolved {
            return Ok(());
        }

        // Error recovery: Error unifies with anything
        if matches!(a_resolved, HirType::Error) || matches!(b_resolved, HirType::Error) {
            return Ok(());
        }

        // Two Infer vars: union them
        if let (HirType::Infer(var_a), HirType::Infer(var_b)) = (&a_resolved, &b_resolved) {
            self.union(*var_a, *var_b);
            return Ok(());
        }

        // Infer var on one side: bind (with occurs check)
        if let HirType::Infer(var) = &a_resolved {
            if self.occurs(*var, &b_resolved, interner) {
                let origin = self.spans.get(var.0 as usize).cloned().unwrap_or(span);
                emit_err(TypeError::InfiniteType { span: origin });
                return Err(());
            }
            self.bind(*var, &b_resolved);
            return Ok(());
        }

        if let HirType::Infer(var) = &b_resolved {
            if self.occurs(*var, &a_resolved, interner) {
                let origin = self.spans.get(var.0 as usize).cloned().unwrap_or(span);
                emit_err(TypeError::InfiniteType { span: origin });
                return Err(());
            }
            self.bind(*var, &a_resolved);
            return Ok(());
        }

        self.unify_structural(&a_resolved, &b_resolved, span, emit_err, interner)
    }

    /// Bind a type variable (must be root) to a type.
    fn bind(&mut self, var: TypeVar, ty: &HirType) {
        let root = self.find(var);
        self.bindings.insert(root.0, ty.clone());
    }

    /// Union two type variables by rank, migrating bindings.
    fn union(&mut self, a: TypeVar, b: TypeVar) {
        let root_a = self.find(a);
        let root_b = self.find(b);
        if root_a == root_b { return; }

        let (new_root, old_root) = match self.ranks[root_a.0 as usize].cmp(&self.ranks[root_b.0 as usize]) {
            std::cmp::Ordering::Less => (root_b, root_a),
            std::cmp::Ordering::Greater => (root_a, root_b),
            std::cmp::Ordering::Equal => {
                self.ranks[root_a.0 as usize] += 1;
                (root_a, root_b)
            }
        };

        self.parents[old_root.0 as usize] = new_root.0;
        if let Some(bound) = self.bindings.remove(&old_root.0) {
            self.bindings.insert(new_root.0, bound);
        }
    }

    /// Occurs check: does `var` appear in `ty`?
    fn occurs(&mut self, var: TypeVar, ty: &HirType, interner: &Interner) -> bool {
        match ty {
            HirType::Infer(v) => {
                let root_var = self.find(var);
                let root_v = self.find(*v);
                root_var == root_v
            }
            HirType::Generic(_, args) => args.iter().any(|a| self.occurs(var, a, interner)),
            HirType::Tuple(elems) => elems.iter().any(|e| self.occurs(var, e, interner)),
            HirType::RawPtr(inner) => self.occurs(var, inner, interner),
            HirType::Option(inner) => self.occurs(var, inner, interner),
            HirType::Result(ok, err) => {
                self.occurs(var, ok, interner) || self.occurs(var, err, interner)
            }
            HirType::Func(params, ret) => {
                params.iter().any(|p| self.occurs(var, p, interner))
                    || self.occurs(var, ret, interner)
            }
            _ => false,
        }
    }

    fn unify_structural(
        &mut self,
        a: &HirType,
        b: &HirType,
        span: SpanBounds,
        emit_err: &mut dyn FnMut(TypeError),
        interner: &Interner,
    ) -> Result<(), ()> {
        match (a, b) {
            (HirType::Int, HirType::Int)
            | (HirType::Bool, HirType::Bool)
            | (HirType::Float, HirType::Float)
            | (HirType::Str, HirType::Str)
            | (HirType::Unit, HirType::Unit)
            | (HirType::Never, HirType::Never) => Ok(()),

            (HirType::Named(s1), HirType::Named(s2)) if s1 == s2 => Ok(()),

            // Named("Vec") unifies with Generic("Vec", args) — structural equivalence
            (HirType::Named(s1), HirType::Generic(s2, args))
            | (HirType::Generic(s2, args), HirType::Named(s1))
                if s1 == s2 =>
            {
                // The Generic side may have type params; the Named side is monomorphized.
                // Unify by treating Named as Generic with those args.
                for arg in args {
                    // If the arg is a type parameter (single uppercase), skip it
                    // Otherwise, unify it with itself (trivially OK for concrete types)
                    if arg.has_infer() {
                        // Can't unify a concrete Named with an unresolved arg
                        emit_err(TypeError::MismatchedTypes {
                            expected: format_type_for_error(a, interner),
                            found: format_type_for_error(b, interner),
                            expected_span: span.clone(),
                            found_span: span,
                            expr_id: None,
                            diff_path: None,
                            autofix: None,
                        });
                        return Err(());
                    }
                }
                Ok(())
            }

            (HirType::Generic(s1, a1), HirType::Generic(s2, a2))
                if s1 == s2 && a1.len() == a2.len() =>
            {
                let a1c = a1.clone();
                let a2c = a2.clone();
                for (arg_a, arg_b) in a1c.iter().zip(a2c.iter()) {
                    self.unify(arg_a, arg_b, span.clone(), emit_err, interner)?;
                }
                Ok(())
            }

            (HirType::RawPtr(i1), HirType::RawPtr(i2)) => {
                self.unify(i1, i2, span, emit_err, interner)
            }
            (HirType::Option(i1), HirType::Option(i2)) => {
                self.unify(i1, i2, span, emit_err, interner)
            }
            (HirType::Result(ok1, err1), HirType::Result(ok2, err2)) => {
                self.unify(ok1, ok2, span.clone(), emit_err, interner)?;
                self.unify(err1, err2, span, emit_err, interner)
            }
            (HirType::Func(p1, r1), HirType::Func(p2, r2)) if p1.len() == p2.len() => {
                let p1c = p1.clone();
                let p2c = p2.clone();
                for (pa, pb) in p1c.iter().zip(p2c.iter()) {
                    self.unify(pa, pb, span.clone(), emit_err, interner)?;
                }
                self.unify(r1, r2, span, emit_err, interner)
            }
            (HirType::Tuple(e1), HirType::Tuple(e2)) if e1.len() == e2.len() => {
                let e1c = e1.clone();
                let e2c = e2.clone();
                for (ea, eb) in e1c.iter().zip(e2c.iter()) {
                    self.unify(ea, eb, span.clone(), emit_err, interner)?;
                }
                Ok(())
            }

            // Never unifies with anything (subsumption)
            (HirType::Never, _) | (_, HirType::Never) => Ok(()),

            _ => {
                emit_err(TypeError::MismatchedTypes {
                    expected: format_type_for_error(a, interner),
                    found: format_type_for_error(b, interner),
                    expected_span: span.clone(),
                    found_span: span,
                    expr_id: None,
                    diff_path: None,
                    autofix: None,
                });
                Err(())
            }
        }
    }

    pub fn var_count(&self) -> usize { self.parents.len() }
}

/// Format a HirType for human-readable error messages.
pub fn format_type_for_error(ty: &HirType, interner: &Interner) -> String {
    match ty {
        HirType::Int => "i64".into(),
        HirType::Bool => "bool".into(),
        HirType::Float => "f64".into(),
        HirType::Str => "Str".into(),
        HirType::Unit => "()".into(),
        HirType::Never => "!".into(),
        HirType::Error => "<error>".into(),
        HirType::Infer(_) => "_".into(),
        HirType::Named(sym) => interner.resolve(*sym).to_string(),
        HirType::Generic(sym, args) => {
            let a = args.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("{}<{}>", interner.resolve(*sym), a)
        }
        HirType::RawPtr(inner) => format!("*{}", format_type_for_error(inner, interner)),
        HirType::Option(inner) => format!("Option<{}>", format_type_for_error(inner, interner)),
        HirType::Result(ok, err) => format!(
            "Result<{}, {}>",
            format_type_for_error(ok, interner),
            format_type_for_error(err, interner)
        ),
        HirType::Func(params, ret) => {
            let p = params.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("fn({}) -> {}", p, format_type_for_error(ret, interner))
        }
        HirType::Tuple(elems) => {
            let e = elems.iter()
                .map(|a| format_type_for_error(a, interner))
                .collect::<Vec<_>>()
                .join(", ");
            format!("({})", e)
        }
        HirType::Opaque(sym) => format!("@{}", interner.resolve(*sym)),
    }
}

/// Extract type parameter bindings by matching a schema against a concrete type.
pub fn extract_type_substitutions(
    schema: &HirType,
    concrete: &HirType,
    type_params: &[Symbol],
    sub: &mut HashMap<Symbol, HirType>,
) {
    match (schema, concrete) {
        (HirType::Named(sym), _) if type_params.contains(sym) => {
            sub.entry(*sym).or_insert_with(|| concrete.clone());
        }
        (HirType::Generic(sym_a, args_a), HirType::Generic(sym_b, args_b))
            if sym_a == sym_b && args_a.len() == args_b.len() =>
        {
            for (a, b) in args_a.iter().zip(args_b.iter()) {
                extract_type_substitutions(a, b, type_params, sub);
            }
        }
        (HirType::Named(a), HirType::Named(b)) if a == b => {}
        (HirType::Named(a), HirType::Generic(b, _)) if a == b => {} // Named ↔ Generic equivalence
        (HirType::Generic(a, _), HirType::Named(b)) if a == b => {}
        (HirType::RawPtr(inner_s), HirType::RawPtr(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }
        (HirType::Option(inner_s), HirType::Option(inner_c)) => {
            extract_type_substitutions(inner_s, inner_c, type_params, sub);
        }
        (HirType::Result(ok_s, err_s), HirType::Result(ok_c, err_c)) => {
            extract_type_substitutions(ok_s, ok_c, type_params, sub);
            extract_type_substitutions(err_s, err_c, type_params, sub);
        }
        (HirType::Tuple(elems_s), HirType::Tuple(elems_c)) if elems_s.len() == elems_c.len() => {
            for (s, c) in elems_s.iter().zip(elems_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
        }
        (HirType::Func(params_s, ret_s), HirType::Func(params_c, ret_c))
            if params_s.len() == params_c.len() =>
        {
            for (s, c) in params_s.iter().zip(params_c.iter()) {
                extract_type_substitutions(s, c, type_params, sub);
            }
            extract_type_substitutions(ret_s, ret_c, type_params, sub);
        }
        _ => {}
    }
}
```

```rust
//! Scoped type environment with O(1) push/pop.
//!
//! Uses a scope-stack: each scope is a list of (name, binding) pairs.
//! Pushing a scope adds an empty list; popping removes all pairs in that list.
//! Lookup scans from the innermost scope outward.

use glyim_hir::types::HirType;
use glyim_interner::Symbol;
use std::collections::HashMap;

#[derive(Clone, Debug)]
struct Binding {
    ty: HirType,
    mutable: bool,
}

/// A single scope's worth of bindings, in insertion order.
type Scope = Vec<(Symbol, Binding)>;

pub struct TypeEnv {
    /// All scopes, innermost last.
    scopes: Vec<Scope>,
    /// Cache: name → (scope_depth, binding) for the innermost visible binding.
    /// Invalidated on pop_scope.
    cache: HashMap<Symbol, (usize, Binding)>,
}

impl Default for TypeEnv {
    fn default() -> Self { Self::new() }
}

impl TypeEnv {
    pub fn new() -> Self {
        Self {
            scopes: vec![vec![]], // Start with one scope (depth 0)
            cache: HashMap::new(),
        }
    }

    /// Push a new scope. Block expressions, match arms, and function bodies
    /// all introduce scopes.
    pub fn push_scope(&mut self) {
        self.scopes.push(vec![]);
    }

    /// Pop the innermost scope, removing all its bindings.
    /// O(k) where k is the number of bindings in the popped scope.
    pub fn pop_scope(&mut self) {
        if self.scopes.len() <= 1 {
            panic!("TypeEnv: attempted to pop the root scope");
        }
        let popped = self.scopes.pop().unwrap();
        // Invalidate cache entries from the popped scope
        for (name, _) in popped {
            self.cache.remove(&name);
        }
    }

    /// Insert a binding in the current (innermost) scope.
    pub fn insert(&mut self, name: Symbol, ty: HirType, mutable: bool) {
        let binding = Binding { ty, mutable };
        self.scopes.last_mut().unwrap().push((name, Binding { ty: binding.ty.clone(), mutable: binding.mutable }));
        self.cache.insert(name, (self.scopes.len() - 1, binding));
    }

    /// Look up the innermost binding for a name.
    pub fn lookup(&self, name: Symbol) -> Option<&HirType> {
        self.cache.get(&name).map(|(_, b)| &b.ty)
    }

    /// Check if the innermost binding for a name is mutable.
    pub fn is_mutable(&self, name: Symbol) -> bool {
        self.cache.get(&name).map_or(false, |(_, b)| b.mutable)
    }

    /// Reset to initial state.
    pub fn clear(&mut self) {
        self.scopes = vec![vec![]];
        self.cache.clear();
    }

    pub fn depth(&self) -> usize {
        self.scopes.len().saturating_sub(1)
    }
}
```

```rust
//! Pre-built index for O(1) lookups during type checking.
//!
//! Stores SIGNATURES ONLY — no function bodies. The type checker
//! iterates the HIR directly for bodies.

use glyim_hir::{EnumDef, ExternFn, HirFn, HirImplDef, HirItem, StructDef, types::HirType};
use glyim_interner::Symbol;
use std::collections::HashMap;

#[derive(Clone, Debug)]
pub struct StructInfo {
    pub name: Symbol,
    pub fields: Vec<glyim_hir::StructField>,
    pub field_map: HashMap<Symbol, usize>,
    pub type_params: Vec<Symbol>,
}

#[derive(Clone, Debug)]
pub struct EnumInfo {
    pub name: Symbol,
    pub variants: Vec<glyim_hir::HirVariant>,
    pub variant_map: HashMap<Symbol, usize>,
    pub type_params: Vec<Symbol>,
}

#[derive(Clone, Debug)]
pub struct FnInfo {
    pub name: Symbol,
    pub type_params: Vec<Symbol>,
    pub params: Vec<(Symbol, HirType)>,
    pub ret: Option<HirType>,
    pub is_generic: bool,
}

pub struct Registry {
    pub structs: HashMap<Symbol, StructInfo>,
    pub enums: HashMap<Symbol, EnumInfo>,
    pub fns: HashMap<Symbol, FnInfo>,
    pub extern_fns: HashMap<Symbol, ExternFn>,
    /// target type → list of method names defined in impls
    pub impl_methods: HashMap<Symbol, Vec<Symbol>>,
}

impl Registry {
    pub fn build(hir: &glyim_hir::Hir) -> Self {
        let mut reg = Self {
            structs: HashMap::new(),
            enums: HashMap::new(),
            fns: HashMap::new(),
            extern_fns: HashMap::new(),
            impl_methods: HashMap::new(),
        };

        // Pass 1: register types
        for item in &hir.items {
            match item {
                HirItem::Struct(s) => reg.register_struct(s),
                HirItem::Enum(e) => reg.register_enum(e),
                HirItem::Extern(ext) => {
                    for f in &ext.functions {
                        reg.extern_fns.insert(f.name, f.clone());
                    }
                }
                _ => {}
            }
        }

        // Pass 2: register functions and impls (can reference types)
        for item in &hir.items {
            match item {
                HirItem::Fn(f) => reg.register_fn(f, false),
                HirItem::Impl(imp) => reg.register_impl(imp),
                _ => {}
            }
        }

        reg
    }

    fn register_struct(&mut self, s: &StructDef) {
        let mut field_map = HashMap::new();
        for (i, f) in s.fields.iter().enumerate() {
            field_map.insert(f.name, i);
        }
        self.structs.insert(s.name, StructInfo {
            name: s.name,
            fields: s.fields.clone(),
            field_map,
            type_params: s.type_params.clone(),
        });
    }

    fn register_enum(&mut self, e: &EnumDef) {
        let mut variant_map = HashMap::new();
        for (i, v) in e.variants.iter().enumerate() {
            variant_map.insert(v.name, i);
        }
        self.enums.insert(e.name, EnumInfo {
            name: e.name,
            variants: e.variants.clone(),
            variant_map,
            type_params: e.type_params.clone(),
        });
    }

    fn register_fn(&mut self, f: &HirFn, is_impl_on_generic: bool) {
        let is_generic = !f.type_params.is_empty() || is_impl_on_generic;
        self.fns.insert(f.name, FnInfo {
            name: f.name,
            type_params: f.type_params.clone(),
            params: f.params.clone(),
            ret: f.ret.clone(),
            is_generic,
        });
    }

    fn register_impl(&mut self, imp: &HirImplDef) {
        let target_is_generic = self.structs.get(&imp.target_name)
            .map_or(false, |s| !s.type_params.is_empty())
            || self.enums.get(&imp.target_name)
                .map_or(false, |e| !e.type_params.is_empty())
            || !imp.type_params.is_empty();

        let method_names: Vec<Symbol> = imp.methods.iter().map(|m| m.name).collect();
        for m in &imp.methods {
            self.register_fn(m, target_is_generic);
        }
        self.impl_methods.insert(imp.target_name, method_names);
    }

    pub fn find_fn(&self, name: Symbol) -> Option<&FnInfo> { self.fns.get(&name) }
    pub fn find_struct(&self, name: Symbol) -> Option<&StructInfo> { self.structs.get(&name) }
    pub fn find_enum(&self, name: Symbol) -> Option<&EnumInfo> { self.enums.get(&name) }

    /// Is the given symbol a type parameter of any currently-known generic entity?
    /// This replaces the heuristic "single uppercase letter" check.
```

```rust
//! Generic type parameter solving.
//!
//! Uses the CALLER's UnificationTable, enabling constraint propagation
//! within a function body.
//!
//! Limitation: each generic call gets its own TypeVars, so constraints
//! do NOT propagate across independent call sites. This is sufficient for
//! Glyim but would need let-polymorphism for more complex inference.

use crate::errors::{SpanBounds, TypeError};
use crate::registry::FnInfo;
use crate::unify::UnificationTable;
use glyim_hir::types::{HirType, TypeVar};
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

pub struct SolveResult {
    /// Type parameter symbol → concrete HirType.
    pub subst: HashMap<Symbol, HirType>,
    /// Concrete type args in the order of fn_info.type_params.
    pub concrete_args: Vec<HirType>,
    /// Whether ALL type params resolved to concrete (non-param) types.
    pub fully_resolved: bool,
}

/// Solve generic type params for a function call.
///
/// 1. Create fresh TypeVars in `table` for each type param
/// 2. Unify formal params with actual arg types (forward)
/// 3. If expected return provided, unify return type (backward)
/// 4. Resolve each TypeVar to extract concrete types
pub fn solve_generic_params(
    table: &mut UnificationTable,
    fn_info: &FnInfo,
    arg_types: &[HirType],
    expected_return: Option<&HirType>,
    interner: &Interner,
    call_span: SpanBounds,
) -> SolveResult {
    let mut param_vars: HashMap<Symbol, TypeVar> = HashMap::new();

    // Step 1: Fresh TypeVars in the caller's table
    for tp in &fn_info.type_params {
        let var = table.fresh_var(call_span.clone());
        param_vars.insert(*tp, var);
    }

    // Step 2: Forward — unify formal params with actual args
    for (formal, actual) in fn_info.params.iter().zip(arg_types.iter()) {
        let formal_ty = substitute_type_vars(&formal.1, &param_vars);
        let _ = table.unify(&formal_ty, actual, call_span.clone(), &mut |_| {}, interner);
    }

    // Step 3: Backward — unify return type with expected
    if let Some(expected) = expected_return {
        if let Some(ret) = &fn_info.ret {
            let ret_ty = substitute_type_vars(ret, &param_vars);
            let _ = table.unify(&ret_ty, expected, call_span.clone(), &mut |_| {}, interner);
        }
    }

    // Step 4: Extract results
    let mut subst = HashMap::new();
    let mut concrete_args = Vec::new();
    let mut fully_resolved = true;

    for tp in &fn_info.type_params {
        let var = param_vars[tp];
        let resolved = table.resolve(&HirType::Infer(var), interner);

        // Unresolved if still Infer, or still a Named that is one of the type params
        let is_unresolved = resolved.has_infer()
            || matches!(&resolved, HirType::Named(s) if fn_info.type_params.contains(s));

        if is_unresolved || resolved == HirType::Error {
            fully_resolved = false;
            subst.insert(*tp, HirType::Named(*tp));
        } else {
            subst.insert(*tp, resolved.clone());
        }
        concrete_args.push(resolved);
    }

    SolveResult { subst, concrete_args, fully_resolved }
}

/// Replace type parameter symbols with their TypeVar counterparts.
fn substitute_type_vars(ty: &HirType, vars: &HashMap<Symbol, TypeVar>) -> HirType {
    match ty {
        HirType::Named(sym) => {
            if let Some(&var) = vars.get(sym) {
                HirType::Infer(var)
            } else {
                ty.clone()
            }
        }
        HirType::Generic(sym, args) => HirType::Generic(
            *sym,
            args.iter().map(|a| substitute_type_vars(a, vars)).collect(),
        ),
        HirType::Tuple(elems) => {
            HirType::Tuple(elems.iter().map(|e| substitute_type_vars(e, vars)).collect())
        }
        HirType::RawPtr(inner) => HirType::RawPtr(Box::new(substitute_type_vars(inner, vars))),
        HirType::Option(inner) => HirType::Option(Box::new(substitute_type_vars(inner, vars))),
        HirType::Result(ok, err) => HirType::Result(
            Box::new(substitute_type_vars(ok, vars)),
            Box::new(substitute_type_vars(err, vars)),
        ),
        HirType::Func(params, ret) => HirType::Func(
            params.iter().map(|p| substitute_type_vars(p, vars)).collect(),
            Box::new(substitute_type_vars(ret, vars)),
        ),
        _ => ty.clone(),
    }
}
```

```rust
//! Resolve type variables, producing fully concrete output.
//!
//! This is the VALIDATION BOUNDARY between type checking and downstream
//! phases. No `Infer` variant may pass through.

use crate::errors::{SpanBounds, TypeError};
use crate::unify::UnificationTable;
use glyim_hir::types::{ExprId, HirType};
use glyim_interner::Interner;
use std::collections::HashMap;

/// Resolve a single HirType. Unbound `Infer` vars become `Error`
/// AND emit a `CannotInferType` diagnostic.
pub fn resolve_type(
    ty: &HirType,
    table: &mut UnificationTable,
    interner: &Interner,
    errors: &mut Vec<TypeError>,
) -> HirType {
    let resolved = table.resolve(ty, interner);
    let result = replace_infer_with_error(&resolved, errors);
    debug_assert!(!result.has_infer(),
        "freeze: Infer variant leaked past resolution — type: {:?}", result);
    result
}

fn replace_infer_with_error(ty: &HirType, errors: &mut Vec<TypeError>) -> HirType {
    match ty {
        HirType::Infer(_) => {
            errors.push(TypeError::CannotInferType {
                expr_kind: "type variable".into(),
                span: SpanBounds::new(0, 0), // Best effort; real span comes from UnificationTable
            });
            HirType::Error
        }
        HirType::Generic(sym, args) => HirType::Generic(
            *sym,
            args.iter().map(|a| replace_infer_with_error(a, errors)).collect(),
        ),
        HirType::Tuple(elems) => {
            HirType::Tuple(elems.iter().map(|e| replace_infer_with_error(e, errors)).collect())
        }
        HirType::RawPtr(inner) => HirType::RawPtr(Box::new(replace_infer_with_error(inner, errors))),
        HirType::Option(inner) => HirType::Option(Box::new(replace_infer_with_error(inner, errors))),
        HirType::Result(ok, err) => HirType::Result(
            Box::new(replace_infer_with_error(ok, errors)),
            Box::new(replace_infer_with_error(err, errors)),
        ),
        HirType::Func(params, ret) => HirType::Func(
            params.iter().map(|p| replace_infer_with_error(p, errors)).collect(),
            Box::new(replace_infer_with_error(ret, errors)),
        ),
        _ => ty.clone(),
    }
}

/// Convert an expression type map into fully concrete types.
/// Emits CannotInferType errors for unresolved variables.
pub fn resolve_expr_types(
    table: &mut UnificationTable,
    elab_map: &HashMap<ExprId, HirType>,
    interner: &Interner,
    errors: &mut Vec<TypeError>,
) -> Vec<HirType> {
    let max_id = elab_map.keys().map(|id| id.as_usize()).max().unwrap_or(0);
    let mut expr_types = vec![HirType::Error; max_id + 1];
    for (&id, ty) in elab_map {
        expr_types[id.as_usize()] = resolve_type(ty, table, interner, errors);
    }
    expr_types
}
```

pub struct TypeChecker {
    pub interner: Interner,
    registry: Registry,
    env: TypeEnv,
    table: UnificationTable,
    expr_types: HashMap<ExprId, HirType>,
    call_type_args: HashMap<ExprId, Vec<HirType>>,
    errors: Vec<TypeError>,
    /// Type parameters of the function currently being checked.
    current_type_params: Vec<Symbol>,
}

impl TypeChecker {
    pub fn new(interner: Interner) -> Self {
        Self {
            interner,
            registry: Registry {
                structs: HashMap::new(),
                enums: HashMap::new(),
                fns: HashMap::new(),
                extern_fns: HashMap::new(),
                impl_methods: HashMap::new(),
            },
            env: TypeEnv::new(),
            table: UnificationTable::new(),
            expr_types: HashMap::new(),
            call_type_args: HashMap::new(),
            errors: Vec::new(),
            current_type_params: Vec::new(),
        }
    }

    pub fn check(&mut self, hir: &Hir) -> Result<TypeCheckOutput, Vec<TypeError>> {
        self.registry = Registry::build(hir);

        for item in &hir.items {
            match item {
                HirItem::Fn(f) => self.check_fn(f),
                HirItem::Impl(imp) => {
                    for method in &imp.methods {
                        self.check_fn(method);
                    }
                }
                _ => {}
            }
        }

        // Freeze: resolve all Infer vars, emit errors for unresolved ones
        let mut frozen_types = vec![HirType::Error; self.expr_types.keys().map(|id| id.as_usize()).max().map(|m| m + 1).unwrap_or(0)];
        for (&id, ty) in &self.expr_types {
            frozen_types[id.as_usize()] = freeze::resolve_type(
                ty, &mut self.table, &self.interner, &mut self.errors,
            );
        }

        if self.errors.is_empty() {
            Ok(TypeCheckOutput {
                expr_types: frozen_types,
                call_type_args: std::mem::take(&mut self.call_type_args),
                interner: self.interner.clone(),
            })
        } else {
            Err(std::mem::take(&mut self.errors))
        }
    }

    fn check_fn(&mut self, f: &HirFn) {
        // Reset per-function state
        self.env.clear();
        self.table = UnificationTable::new();
        self.current_type_params = f.type_params.clone();

        self.env.push_scope();
        for (i, (sym, _ty)) in f.params.iter().enumerate() {
            let mutable = f.param_mutability.get(i).copied().unwrap_or(false);
            self.env.insert(*sym, f.params[i].1.clone(), mutable);
        }

        let body_ty = self.infer_expr(&f.body, f.ret.as_ref());

        // Check return type via unification
        if let Some(expected) = &f.ret {
            if !self.current_type_params.is_empty() {
                // For generic functions, don't check return — type params may be unresolved
            } else {
                let span = SpanBounds::from_span(f.body.get_span());
                self.unify_types(expected, &body_ty, span);
            }
        }

        self.env.pop_scope();
    }

    // ── Unification wrapper ──────────────────────────────────────

    /// Unify two types, emitting a MismatchedTypes error on failure.
    fn unify_types(&mut self, expected: &HirType, found: &HirType, span: SpanBounds) -> bool {
        let mut local_errors = Vec::new();
        let result = self.table.unify(expected, found, span.clone(), &mut |e| local_errors.push(e), &self.interner);
        if result.is_err() {
            self.errors.extend(local_errors);
        }
        result.is_ok()
    }

    /// Check if a type contains type parameters from the current function.
    fn has_current_type_param(&self, ty: &HirType) -> bool {
        match ty {
            HirType::Named(sym) => self.current_type_params.contains(sym),
            HirType::Generic(_, args) => args.iter().any(|a| self.has_current_type_param(a)),
            HirType::Tuple(elems) => elems.iter().any(|e| self.has_current_type_param(e)),
            HirType::RawPtr(inner) | HirType::Option(inner) => self.has_current_type_param(inner),
            HirType::Result(ok, err) => self.has_current_type_param(ok) || self.has_current_type_param(err),
            HirType::Func(params, ret) => {
                params.iter().any(|p| self.has_current_type_param(p)) || self.has_current_type_param(ret)
            }
            _ => false,
        }
    }

    // ── Core inference ──────────────────────────────────────────

    fn infer_expr(&mut self, expr: &HirExpr, expected: Option<&HirType>) -> HirType {
        let ty = match expr {
            HirExpr::IntLit { .. } => HirType::Int,
            HirExpr::FloatLit { .. } => HirType::Float,
            HirExpr::BoolLit { .. } => HirType::Bool,
            HirExpr::StrLit { .. } => HirType::Str,
            HirExpr::UnitLit { .. } => HirType::Unit,

            HirExpr::AddrOf { target, span, .. } => {
                let inner_ty = self.type_of_ident_or_expr(*target);
                HirType::RawPtr(Box::new(inner_ty))
            }

            HirExpr::Ident { name, span, .. } => {
                match self.env.lookup(*name).cloned() {
                    Some(ty) => ty,
                    None => {
                        let resolved = self.interner.resolve(*name).to_string();
                        let suggestions = glyim_diag::suggest::suggest_similar(
                            &resolved, &self.interner, 3,
                        );
                        self.errors.push(TypeError::UnresolvedName {
                            name: resolved,
                            span: SpanBounds::from_span(*span),
                            suggestions,
                        });
                        HirType::Error
                    }
                }
            }

            HirExpr::Binary { op, lhs, rhs, span, .. } => {
                let lhs_ty = self.infer_expr(lhs, None);
                let rhs_ty = self.infer_expr(rhs, None);
                match op {
                    glyim_hir::HirBinOp::Eq
                    | glyim_hir::HirBinOp::Neq
                    | glyim_hir::HirBinOp::Lt
                    | glyim_hir::HirBinOp::Gt
                    | glyim_hir::HirBinOp::Lte
                    | glyim_hir::HirBinOp::Gte => HirType::Bool,
                    _ => lhs_ty,
                }
            }

            HirExpr::Unary { op, operand, span, .. } => {
                let inner = self.infer_expr(operand, None);
                match op {
                    glyim_hir::HirUnOp::Not => HirType::Bool,
                    glyim_hir::HirUnOp::Neg => inner,
                }
            }

            HirExpr::Block { stmts, span, .. } => {
                self.env.push_scope(); // Block introduces a scope
                let mut last = HirType::Unit;
                for (i, stmt) in stmts.iter().enumerate() {
                    let exp = if i == stmts.len() - 1 { expected } else { None };
                    if let Some(t) = self.infer_stmt(stmt, exp) { last = t; }
                }
                self.env.pop_scope();
                last
            }

            HirExpr::If { condition, then_branch, else_branch, span, .. } => {
                let cond_ty = self.infer_expr(condition, None);
                let cond_span = SpanBounds::from_span(condition.get_span());
                self.unify_types(&HirType::Bool, &cond_ty, cond_span);

                let then_ty = self.infer_expr(then_branch, expected);
                if let Some(e) = else_branch {
                    let else_ty = self.infer_expr(e, expected);
                    let then_span = SpanBounds::from_span(then_branch.get_span());
                    self.unify_types(&then_ty, &else_ty, then_span);
                    then_ty
                } else {
                    let then_span = SpanBounds::from_span(then_branch.get_span());
                    self.unify_types(&HirType::Unit, &then_ty, then_span);
                    HirType::Unit
                }
            }

            HirExpr::Call { id, callee, args, span, .. } => {
                self.infer_call(*id, *callee, args, expected, *span)
            }

            HirExpr::MethodCall { id, receiver, method_name, args, span, .. } => {
                self.infer_method_call(*id, receiver, *method_name, args, expected, *span)
            }

            HirExpr::StructLit { struct_name, fields, span, .. } => {
                self.infer_struct_lit(*struct_name, fields, expected, expr.get_id(), *span)
            }

            HirExpr::EnumVariant { enum_name, variant_name, args, span, .. } => {
                self.infer_enum_variant(*enum_name, *variant_name, args, *span)
            }

            HirExpr::FieldAccess { object, field, span, .. } => {
                self.infer_field_access(object, *field, *span)
            }

            HirExpr::As { expr: inner, target_type, span, .. } => {
                let from_ty = self.infer_expr(inner, None);
                if !self.is_valid_cast(&from_ty, target_type) {
                    self.errors.push(TypeError::MismatchedTypes {
                        expected: unify::format_type_for_error(target_type, &self.interner),
                        found: unify::format_type_for_error(&from_ty, &self.interner),
                        expected_span: SpanBounds::from_span(*span),
                        found_span: SpanBounds::from_span(inner.get_span()),
                        expr_id: Some(inner.get_id()),
                        diff_path: None,
                        autofix: None,
                    });
                }
                target_type.clone()
            }

            HirExpr::Deref { expr: inner, span, .. } => {
                let inner_ty = self.infer_expr(inner, None);
                match &inner_ty {
                    HirType::RawPtr(t) => *t.clone(),
                    _ => {
                        self.errors.push(TypeError::DerefNonPointer {
                            found: inner_ty,
                            expr_id: inner.get_id(),
                            span: SpanBounds::from_span(*span),
                        });
                        HirType::Error
                    }
                }
            }

            HirExpr::Match { scrutinee, arms, span, .. } => {
                self.infer_match(scrutinee, arms, expected, *span)
            }

            HirExpr::While { condition, body, span, .. } => {
                self.infer_expr(condition, None);
                self.infer_expr(body, None);
                HirType::Unit
            }

            HirExpr::ForIn { pattern, iter, body, span, .. } => {
                self.infer_forin(pattern, iter, body, *span)
            }

            HirExpr::Return { value, span, .. } => {
                if let Some(v) = value { self.infer_expr(v, expected); }
                HirType::Never
            }

            HirExpr::Println { arg, .. } => {
                self.infer_expr(arg, None);
                HirType::Unit
            }

            HirExpr::Assert { condition, message, .. } => {
                self.infer_expr(condition, None);
                if let Some(m) = message { self.infer_expr(m, None); }
                HirType::Unit
            }

            HirExpr::SizeOf { id, target_type, .. } => {
                self.record_call_type_args(*id, vec![target_type.clone()]);
                HirType::Int
            }

            HirExpr::TupleLit { elements, .. } => {
                let types: Vec<HirType> = elements.iter()
                    .map(|e| self.infer_expr(e, None))
                    .collect();
                HirType::Tuple(types)
            }
        };

        self.set_type(expr.get_id(), &ty);
        ty
    }

    // ── Call inference ──────────────────────────────────────────

    fn infer_call(
        &mut self,
        call_id: ExprId,
        callee: Symbol,
        args: &[HirExpr],
        expected: Option<&HirType>,
        span: glyim_diag::Span,
    ) -> HirType {
        let arg_types: Vec<HirType> = args.iter()
            .map(|a| self.infer_expr(a, None))
            .collect();

        if let Some(ext_fn) = self.registry.extern_fns.get(&callee).cloned() {
            // Check arg count
            if arg_types.len() != ext_fn.params.len() {
                self.errors.push(TypeError::ArgumentCountMismatch {
                    expected: ext_fn.params.len(),
                    actual: arg_types.len(),
                    span: SpanBounds::from_span(span),
                });
            }
            return ext_fn.ret.clone();
        }

        let fn_info = match self.registry.find_fn(callee) {
            Some(info) => info.clone(),
            None => {
                let name = self.interner.resolve(callee).to_string();
                self.errors.push(TypeError::UnknownFunction {
                    name,
                    span: SpanBounds::from_span(span),
                });
                return HirType::Error;
            }
        };

        // Check arg count
        if arg_types.len() != fn_info.params.len() {
            self.errors.push(TypeError::ArgumentCountMismatch {
                expected: fn_info.params.len(),
                actual: arg_types.len(),
                span: SpanBounds::from_span(span),
            });
        }

        if fn_info.is_generic {
            return self.infer_generic_call(call_id, callee, &fn_info, &arg_types, expected, span);
        }

        // Non-generic: unify each arg type with the declared param type
        for ((_, param_ty), arg_ty) in fn_info.params.iter().zip(arg_types.iter()) {
            if arg_ty == &HirType::Error || param_ty == &HirType::Never { continue; }
            let arg_span = SpanBounds::from_span(span);
            self.unify_types(param_ty, arg_ty, arg_span);
        }

        fn_info.ret.clone().unwrap_or(HirType::Unit)
    }

    fn infer_generic_call(
        &mut self,
        call_id: ExprId,
        callee: Symbol,
        fn_info: &registry::FnInfo,
        arg_types: &[HirType],
        expected: Option<&HirType>,
        span: glyim_diag::Span,
    ) -> HirType {
        let call_span = SpanBounds::from_span(span);

        let result = solve::solve_generic_params(
            &mut self.table, fn_info, arg_types, expected, &self.interner, call_span,
        );

        self.record_call_type_args(call_id, result.concrete_args);

        let ret = fn_info.ret.clone().unwrap_or(HirType::Unit);
        let concrete_ret = glyim_hir::types::substitute_type(&ret, &result.subst);

        if !result.fully_resolved {
            self.errors.push(TypeError::CannotInferGenericArgs {
                name: self.interner.resolve(callee).to_string(),
                span: SpanBounds::from_span(span),
            });
        }

        concrete_ret
    }

    // ── Method call inference ───────────────────────────────────

    fn infer_method_call(
        &mut self,
        call_id: ExprId,
        receiver: &HirExpr,
        method_name: Symbol,
        args: &[HirExpr],
        expected: Option<&HirType>,
        span: glyim_diag::Span,
    ) -> HirType {
        let recv_ty = self.infer_expr(receiver, None);
        let arg_types: Vec<HirType> = args.iter()
            .map(|a| self.infer_expr(a, None))
            .collect();

        let (type_name, type_args) = self.resolve_receiver_type(&recv_ty);
        let mangled = self.interner.intern(&format!(
            "{}_{}",
            self.interner.resolve(type_name),
            self.interner.resolve(method_name),
        ));

        let fn_info = match self.registry.find_fn(mangled) {
            Some(info) => info.clone(),
            None => {
                self.errors.push(TypeError::UnresolvedMethod {
                    method_name: self.interner.resolve(method_name).to_string(),
                    receiver_type: unify::format_type_for_error(&recv_ty, &self.interner),
                    span: SpanBounds::from_span(span),
                });
                return HirType::Error;
            }
        };

        if fn_info.is_generic {
            let mut all_arg_types = vec![recv_ty.clone()];
            all_arg_types.extend(arg_types);

            let result = solve::solve_generic_params(
                &mut self.table, &fn_info, &all_arg_types, expected,
                &self.interner, SpanBounds::from_span(span),
            );

            self.record_call_type_args(call_id, result.concrete_args);

            let ret = fn_info.ret.clone().unwrap_or(HirType::Unit);
            let concrete_ret = glyim_hir::types::substitute_type(&ret, &result.subst);

            if !result.fully_resolved {
                self.errors.push(TypeError::CannotInferGenericArgs {
                    name: format!("{}.{}",
                        self.interner.resolve(type_name),
                        self.interner.resolve(method_name)),
                    span: SpanBounds::from_span(span),
                });
            }

            concrete_ret
        } else {
            if !type_args.is_empty() {
                self.record_call_type_args(call_id, type_args);
            }
            fn_info.ret.clone().unwrap_or(HirType::Unit)
        }
    }

    // ── Struct literal ──────────────────────────────────────────

    fn infer_struct_lit(
        &mut self,
        struct_name: Symbol,
        fields: &[(Symbol, HirExpr)],
        expected: Option<&HirType>,
        _expr_id: ExprId,
        span: glyim_diag::Span,
    ) -> HirType {
        let struct_info = self.registry.structs.get(&struct_name).cloned();

        if let Some(ref si) = struct_info {
            for (fname, fexpr) in fields {
                let exp_ty = si.fields.iter().find(|f| f.name == *fname).map(|f| f.ty.clone());
                self.infer_expr(fexpr, exp_ty.as_ref());

                if !si.field_map.contains_key(fname) {
                    self.errors.push(TypeError::UnknownField {
                        struct_name: self.interner.resolve(struct_name).to_string(),
                        field: self.interner.resolve(*fname).to_string(),
                        span: SpanBounds::from_span(span),
                    });
                }
            }

            // Check for missing fields
            let provided_names: Vec<Symbol> = fields.iter().map(|(s, _)| *s).collect();
            for field in &si.fields {
                if !provided_names.contains(&field.name) {
                    self.errors.push(TypeError::MissingField {
                        struct_name: self.interner.resolve(struct_name).to_string(),
                        field: self.interner.resolve(field.name).to_string(),
                        span: SpanBounds::from_span(span),
                    });
                }
            }

            if si.type_params.is_empty() {
                return HirType::Named(struct_name);
            }

            // Solve type params from field values
            let field_value_types: Vec<(Symbol, HirType)> = fields.iter()
                .map(|(sym, _)| (*sym, self.expr_type(*sym).unwrap_or(HirType::Error)))
                .collect();

            let result = self.solve_struct_type_args(&si, &field_value_types, expected, span);
            if !result.concrete_args.is_empty() {
                self.record_call_type_args(_expr_id, result.concrete_args);
                return HirType::Generic(struct_name, result.concrete_args);
            }
        } else {
            for (_, fexpr) in fields { self.infer_expr(fexpr, None); }
        }

        HirType::Named(struct_name)
    }

    fn solve_struct_type_args(
        &mut self,
        struct_info: &registry::StructInfo,
        field_value_types: &[(Symbol, HirType)],
        expected: Option<&HirType>,
        span: glyim_diag::Span,
    ) -> SolveResult {
        // Build a synthetic FnInfo for the struct constructor
        let dummy_sym = struct_info.name; // Use struct name as base
        let constructor_ret = if struct_info.type_params.is_empty() {
            HirType::Named(struct_info.name)
        } else {
            HirType::Generic(
                struct_info.name,
                struct_info.type_params.iter().map(|tp| HirType::Named(*tp)).collect(),
            )
        };

        let fn_info = registry::FnInfo {
            name: dummy_sym,
            type_params: struct_info.type_params.clone(),
            params: struct_info.fields.iter().map(|f| (f.name, f.ty.clone())).collect(),
            ret: Some(constructor_ret),
            is_generic: !struct_info.type_params.is_empty(),
        };

        let arg_types: Vec<HirType> = field_value_types.iter().map(|(_, t)| t.clone()).collect();

        solve::solve_generic_params(
            &mut self.table, &fn_info, &arg_types, expected,
            &self.interner, SpanBounds::from_span(span),
        )
    }

    // ── Enum variant ────────────────────────────────────────────

    fn infer_enum_variant(
        &mut self,
        enum_name: Symbol,
        variant_name: Symbol,
        args: &[HirExpr],
        span: glyim_diag::Span,
    ) -> HirType {
        for a in args { self.infer_expr(a, None); }

        let enum_info = match self.registry.enums.get(&enum_name) {
            Some(info) => info.clone(),
            None => return HirType::Named(enum_name),
        };

        let variant = enum_info.variants.iter().find(|v| v.name == variant_name);
        if variant.is_none() {
            self.errors.push(TypeError::UnknownVariant {
                enum_name: self.interner.resolve(enum_name).to_string(),
                variant_name: self.interner.resolve(variant_name).to_string(),
                span: SpanBounds::from_span(span),
            });
            return HirType::Named(enum_name);
        }

        if enum_info.type_params.is_empty() {
            return HirType::Named(enum_name);
        }

        // Infer type params from variant field types
        let mut sub = HashMap::new();
        if let Some(variant) = variant {
            for (i, field) in variant.fields.iter().enumerate() {
                if let Some(arg) = args.get(i) {
                    let arg_ty = self.expr_type_from_id(arg.get_id());
                    // Unify field type (which may contain type params) with arg type
                    let field_span = SpanBounds::from_span(span);
                    self.unify_types(&field.ty, &arg_ty, field_span);
                }
            }
        }

        // Resolve type params from the unification table
        let concrete_args: Vec<HirType> = enum_info.type_params.iter()
            .map(|tp| {
                let named = HirType::Named(*tp);
                self.table.resolve(&named, &self.interner)
            })
            .collect();

        HirType::Generic(enum_name, concrete_args)
    }

    // ── Field access ────────────────────────────────────────────

    fn infer_field_access(&mut self, object: &HirExpr, field: Symbol, span: glyim_diag::Span) -> HirType {
        let obj_ty = self.infer_expr(object, None);
        match &obj_ty {
            HirType::Named(s) | HirType::Generic(s, _) => {
                if let Some(info) = self.registry.structs.get(s) {
                    if let Some(f) = info.fields.iter().find(|f| f.name == field) {
                        // If the struct is generic, substitute type params from the object type
                        if let HirType::Generic(_, type_args) = &obj_ty {
                            let sub: HashMap<Symbol, HirType> = info.type_params.iter()
                                .zip(type_args.iter())
                                .map(|(tp, ta)| (*tp, ta.clone()))
                                .collect();
                            return glyim_hir::types::substitute_type(&f.ty, &sub);
                        }
                        return f.ty.clone();
                    }
                    self.errors.push(TypeError::UnknownField {
                        struct_name: self.interner.resolve(*s).to_string(),
                        field: self.interner.resolve(field).to_string(),
                        span: SpanBounds::from_span(span),
                    });
                }
            }
            HirType::Tuple(elems) => {
                let field_str = self.interner.resolve(field);
                if let Some(idx) = field_str.strip_prefix('_')
                    .and_then(|s| s.parse::<usize>().ok())
                {
                    if idx < elems.len() {
                        return elems[idx].clone();
                    }
                }
            }
            _ => {}
        }
        HirType::Error
    }

    // ── Match ───────────────────────────────────────────────────

    fn infer_match(
        &mut self,
        scrutinee: &HirExpr,
        arms: &[glyim_hir::MatchArm],
        expected: Option<&HirType>,
        span: glyim_diag::Span,
    ) -> HirType {
        let scrutinee_ty = self.infer_expr(scrutinee, None);
        self.check_match_exhaustiveness(&scrutinee_ty, arms, span);

        let mut arm_types = Vec::new();
        for arm in arms {
            self.env.push_scope(); // Each arm gets its own scope
            self.bind_pattern(&arm.pattern, &scrutinee_ty, false);
            if let Some(ref g) = arm.guard { self.infer_expr(g, None); }
            let arm_ty = self.infer_expr(&arm.body, expected);
            arm_types.push(arm_ty);
            self.env.pop_scope();
        }

        // Unify all arm types with each other
        if arm_types.len() > 1 {
            for i in 1..arm_types.len() {
                let span = SpanBounds::from_span(arms[i].body.get_span());
                self.unify_types(&arm_types[0], &arm_types[i], span);
            }
        }

        arm_types.first().cloned().unwrap_or(HirType::Unit)
    }

    // ── ForIn ───────────────────────────────────────────────────

    fn infer_forin(
        &mut self,
        pattern: &HirPattern,
        iter: &HirExpr,
        body: &HirExpr,
        span: glyim_diag::Span,
    ) -> HirType {
        let iter_ty = self.infer_expr(iter, None);
        let elem_ty = self.element_type_for_iter(&iter_ty);

        self.env.push_scope();
        self.bind_pattern(pattern, &elem_ty, false);
        self.infer_expr(body, None);
        self.env.pop_scope();

        // Record type args for the iterator so the monomorphizer
        // can discover the necessary specializations.
        // We record under the ForIn expr's ID (which the monomorphizer
        // can look up) AND under the iter expr's ID.
        self.record_iter_type_args(&iter_ty, iter.get_id());

        HirType::Unit
    }

    /// Record the type arguments needed for iterator specialization.
    fn record_iter_type_args(&mut self, iter_ty: &HirType, iter_expr_id: ExprId) {
        match iter_ty {
            HirType::Generic(name, args) => {
                // The iterator type args are the specialization keys
                // e.g., Vec<i64> → args = [i64]
                if !args.is_empty() && !args.iter().any(|a| self.has_current_type_param(a)) {
                    self.call_type_args.insert(iter_expr_id, args.clone());
                }
            }
            HirType::RawPtr(inner) => self.record_iter_type_args(inner, iter_expr_id),
            _ => {}
        }
    }

    // ── Statement inference ─────────────────────────────────────

    fn infer_stmt(&mut self, stmt: &HirStmt, expected: Option<&HirType>) -> Option<HirType> {
        match stmt {
            HirStmt::Let { name, mutable, value, span } => {
                let ty = self.infer_expr(value, None);
                self.env.insert(*name, ty.clone(), *mutable);
                None
            }
            HirStmt::LetPat { pattern, mutable, value, ty: annotation, span } => {
                let inferred = self.infer_expr(value, annotation.as_ref());
                let ty = if let Some(annotated) = annotation {
                    let span = SpanBounds::from_span(*span);
                    self.unify_types(annotated, &inferred, span);
                    // Record call_type_args from the annotation
                    if let HirExpr::Call { id, callee, .. } = value {
                        if let HirType::Generic(_, type_args) = annotated {
                            if let Some(fn_info) = self.registry.find_fn(*callee) {
                                if fn_info.is_generic && type_args.len() == fn_info.type_params.len() {
                                    self.record_call_type_args(*id, type_args.clone());
                                }
                            }
                        }
                    }
                    annotated.clone()
                } else {
                    inferred
                };
                self.bind_pattern(pattern, &ty, *mutable);
                None
            }
            HirStmt::Assign { target, value, span } => {
                if !self.env
