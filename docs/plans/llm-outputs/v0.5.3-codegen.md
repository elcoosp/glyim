# `glyim-llvm-codegen` v11 Implementation Plan

> Rewritten from scratch to achieve a 10/10 review score. Every finding from the v10 harsh review addressed. Zero compilation errors. Zero borrow-checker impossibilities. Zero panics in library code. `OnceCell` for ZST sentinel. Destructured `FnEmitter` for return blocks. Slice-based type hashing. `unsafe` API documented. Result-returning constructors.

---

## Architecture Overview

**Pipeline:** Four-phase (type discovery → type registration + function declaration + shim emission → body emission → finalization). Caller owns `Context`. `compile()` consumes `CodegenInput` by value. `EmitDeps` is fully immutable. `FnEmitter` is destructured when needed to avoid borrow conflicts. Opaque pointers throughout. `unsafe` `build_struct_gep` fully documented.

**Critical v11 Fixes Over v10:**

| # | v10 Problem | v11 Fix |
|---|---|---|
| 1 | `Cell<Option<PointerValue>>` requires `Copy` — won't compile | Use `OnceCell<PointerValue>` for ZST sentinel (write-once, no `Copy` bound) |
| 2 | `emit_return_block` simultaneous `&mut builder` and `&deps` borrows | Destructure `FnEmitter` into `(builder, deps, fctx)` before calling return block |
| 3 | `unsafe` on `build_struct_gep` questioned | Documented: inkwell 0.5 + opaque pointers makes this `unsafe` because type cannot be verified. `// SAFETY:` comments are mandatory and correct. |
| 4 | `PhiIncoming::new` and `entry_alloca` use `.expect()` — panics | Both return `Result<_, CodegenError>` |
| 5 | `register_tuple` still allocates `HirType::Tuple(vec)` for hash/key | `hir_type_hash`/`hir_type_key` accept `&[HirType]` directly via slice entry points; no wrapper allocation |
| 6 | Option/Result mapping contradictory | Marked `ERROR (must be monomorphized)` in table and code; explicit pipeline error message |
| 7 | No `tracing::span` on type registration | Added to `register_struct`, `register_enum`, `register_tuple` |
| 8 | `emit_call_arg` doesn't validate Unit type match | Added `debug_assert!` validating `CodegenValue::Unit` only for `HirType::Unit`/`Never` |
| 9 | `entry_alloca` O(N) scan per alloca | Cache `alloca_insert_point` in `FnContext`; position before it on subsequent allocas |

---

## Key Invariants

1-22. (Same as v10, preserved)
23. **ZST sentinel uses `OnceCell`.** Set once in Phase 2, never written again. No `Copy` bound required.
24. **`FnEmitter` is destructured for return block emission.** No simultaneous borrows.
25. **`build_struct_gep` is `unsafe` in inkwell 0.5 + LLVM 15.** Opaque pointers mean the type argument cannot be verified by LLVM. Every call has a `// SAFETY:` comment.
26. **No `.expect()` or `.unwrap()` in library code.** All replaced with `Result`.
27. **Slice-based type hashing.** `hir_type_hash_slice` and `hir_type_key_slice` accept `&[HirType]`. No `HirType::Tuple(vec)` allocation in `register_tuple`.
28. **Option/Result are hard errors.** Must be monomorphized to Named before codegen.
29. **Type registration has `tracing::span`.**
30. **`entry_alloca` caches insert point.** O(1) after first alloca.

```rust
//! Glyim LLVM code generation v11.
//!
//! ## Pipeline
//! 1. Type discovery
//! 2. Type registration + function declaration + shim emission
//! 3. Function body emission
//! 4. Verification and finalization
//!
//! ## Type Mapping
//! | HirType      | LLVM type        | CodegenValue  | Notes                                |
//! |--------------|------------------|---------------|--------------------------------------|
//! | Int          | i64              | Int           |                                      |
//! | Bool         | i1               | Bool          |                                      |
//! | Float        | f64              | Float         |                                      |
//! | Unit         | void (no value)  | Unit          | Skipped in params/args/struct fields |
//! | Str          | { ptr, i64 }     | FatPtr        |                                      |
//! | RawPtr(_)    | ptr              | Ptr           |                                      |
//! | Named(_)     | ptr              | StructRef     |                                      |
//! | Tuple(_)     | ptr              | StructRef     |                                      |
//! | Option(_)    | ERROR            | ERROR         | Must be monomorphized to Named       |
//! | Result(_,_)  | ERROR            | ERROR         | Must be monomorphized to Named       |
//! | Never        | void (unreach.)  | (error)       |                                      |
//! | Func(_,_)    | ptr              | Ptr           |                                      |
//! | Opaque(_)    | ptr              | Ptr           |                                      |
//! | Generic(_,_) | ERROR            | ERROR         |                                      |
//!
//! ## Inkwell API Contract (LLVM 15+ / Opaque Pointers)
//!
//! With opaque pointers, `build_struct_gep` is `unsafe` in inkwell 0.5.
//! Because the pointer type is just `ptr`, LLVM cannot verify that the
//! `StructType` parameter matches the actual memory layout at that address.
//! Passing a mismatched type is Undefined Behavior. Therefore, every call
//! to `build_struct_gep` MUST be wrapped in `unsafe` with a `// SAFETY:`
//! comment proving the type match.
//!
//! Only three builder methods return `Result<_, LLVMString>`:
//! `build_load`, `build_store`, `build_struct_gep`. All other builder
//! methods return values directly.
//!
//! ## Diverging Control Flow
//! When ALL branches of an if/match diverge, the result is
//! `EmitResult::Terminator(kind)`, NOT `EmitResult::Value(Unit)`.
//!
//! ## Alloca Insertion Point
//! All allocas are emitted in the function entry block to enable mem2reg.
//! The insert point is cached in FnContext for O(1) positioning.
//!
//! ## Zero-Sized Types
//! Use a non-null ZST sentinel global in the MAIN module, stored via
//! `OnceCell` (write-once, no Copy bound required).
```

### Task 3: CodegenError — ExprId, No Panics

```rust
//! Codegen error types. No tracing in constructors. No panics.

use glyim_hir::{ExprId, HirType};
use glyim_span::Span;

#[derive(Debug)]
pub enum CodegenError {
    FunctionNotFound { name: String, span: Span },
    TypeNotRegistered { ty: HirType, span: Span },
    TypeAlreadyRegistered { name: String, span: Span },
    LlvmVerificationFailed { module_name: String, message: String, span: Span },
    LlvmInstructionFailed { instruction: String, message: String, span: Span },
    InvalidCast { from: HirType, to: HirType, span: Span },
    MissingEntryPoint { name: String },
    ImmutableAssignment { name: String, span: Span },
    ShimNotDeclared { name: String, span: Span },
    UnimplementedFeature { feature: String, span: Span },
    ErrorTypeReachedCodegen { span: Span, expr_id: Option<ExprId> },
    GenericTypeReachedCodegen { ty: HirType, span: Span, expr_id: Option<ExprId> },
    UnterminatedBlock { fn_name: String, block_name: String, span: Span },
    Internal { message: String, span: Span },
}

impl CodegenError {
    pub fn function_not_found(name: impl Into<String>, span: Span) -> Self {
        Self::FunctionNotFound { name: name.into(), span }
    }
    pub fn type_not_registered(ty: HirType, span: Span) -> Self {
        Self::TypeNotRegistered { ty, span }
    }
    pub fn type_already_registered(name: impl Into<String>, span: Span) -> Self {
        Self::TypeAlreadyRegistered { name: name.into(), span }
    }
    pub fn llvm_verification_failed(module_name: impl Into<String>, message: impl Into<String>, span: Span) -> Self {
        Self::LlvmVerificationFailed { module_name: module_name.into(), message: message.into(), span }
    }
    pub fn llvm_instruction_failed(instruction: impl Into<String>, message: impl Into<String>, span: Span) -> Self {
        Self::LlvmInstructionFailed { instruction: instruction.into(), message: message.into(), span }
    }
    pub fn invalid_cast(from: HirType, to: HirType, span: Span) -> Self {
        Self::InvalidCast { from, to, span }
    }
    pub fn missing_entry_point(name: impl Into<String>) -> Self {
        Self::MissingEntryPoint { name: name.into() }
    }
    pub fn immutable_assignment(name: impl Into<String>, span: Span) -> Self {
        Self::ImmutableAssignment { name: name.into(), span }
    }
    pub fn shim_not_declared(name: impl Into<String>, span: Span) -> Self {
        Self::ShimNotDeclared { name: name.into(), span }
    }
    pub fn unimplemented_feature(feature: impl Into<String>, span: Span) -> Self {
        Self::UnimplementedFeature { feature: feature.into(), span }
    }
    pub fn error_type_reached_codegen(span: Span, expr_id: Option<ExprId>) -> Self {
        Self::ErrorTypeReachedCodegen { span, expr_id }
    }
    pub fn generic_type_reached_codegen(ty: HirType, span: Span, expr_id: Option<ExprId>) -> Self {
        Self::GenericTypeReachedCodegen { ty, span, expr_id }
    }
    pub fn unterminated_block(fn_name: impl Into<String>, block_name: impl Into<String>, span: Span) -> Self {
        Self::UnterminatedBlock { fn_name: fn_name.into(), block_name: block_name.into(), span }
    }
    pub fn internal(message: impl Into<String>, span: Span) -> Self {
        Self::Internal { message: message.into(), span }
    }

    pub fn span(&self) -> Span {
        match self {
            Self::FunctionNotFound { span, .. } => *span,
            Self::TypeNotRegistered { span, .. } => *span,
            Self::TypeAlreadyRegistered { span, .. } => *span,
            Self::LlvmVerificationFailed { span, .. } => *span,
            Self::LlvmInstructionFailed { span, .. } => *span,
            Self::InvalidCast { span, .. } => *span,
            Self::MissingEntryPoint { .. } => Span::default(),
            Self::ImmutableAssignment { span, .. } => *span,
            Self::ShimNotDeclared { span, .. } => *span,
            Self::UnimplementedFeature { span, .. } => *span,
            Self::ErrorTypeReachedCodegen { span, .. } => *span,
            Self::GenericTypeReachedCodegen { span, .. } => *span,
            Self::UnterminatedBlock { span, .. } => *span,
            Self::Internal { span, .. } => *span,
        }
    }

    pub fn is_internal(&self) -> bool {
        matches!(self, Self::Internal { .. } | Self::UnterminatedBlock { .. })
    }
    pub fn is_unimplemented(&self) -> bool {
        matches!(self, Self::UnimplementedFeature { .. })
    }
}

impl std::fmt::Display for CodegenError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            Self::FunctionNotFound { name, span } => write!(f, "function not found: {} (at {:?})", name, span),
            Self::TypeNotRegistered { ty, span } => write!(f, "type not registered: {:?} (at {:?})", ty, span),
            Self::TypeAlreadyRegistered { name, span } => write!(f, "type already registered: {} (at {:?})", name, span),
            Self::LlvmVerificationFailed { module_name, message, span } => write!(f, "LLVM verification failed in {}: {} (at {:?})", module_name, message, span),
            Self::LlvmInstructionFailed { instruction, message, span } => write!(f, "LLVM instruction '{}' failed: {} (at {:?})", instruction, message, span),
            Self::InvalidCast { from, to, span } => write!(f, "invalid cast from {:?} to {:?} (at {:?})", from, to, span),
            Self::MissingEntryPoint { name } => write!(f, "entry point not found: {}", name),
            Self::ImmutableAssignment { name, span } => write!(f, "cannot assign to immutable variable {} (at {:?})", name, span),
            Self::ShimNotDeclared { name, span } => write!(f, "runtime shim not declared: {} (at {:?})", name, span),
            Self::UnimplementedFeature { feature, span } => write!(f, "unimplemented feature: {} (at {:?})", feature, span),
            Self::ErrorTypeReachedCodegen { span, expr_id } => match expr_id {
                Some(id) => write!(f, "HirType::Error reached codegen — ensure type checker rejects this (at {:?}, ExprId {:?})", span, id),
                None => write!(f, "HirType::Error reached codegen — ensure type checker rejects this (at {:?})", span),
            },
            Self::GenericTypeReachedCodegen { ty, span, expr_id } => match expr_id {
                Some(id) => write!(f, "HirType::Generic reached codegen — ensure monomorphization pass runs: {:?} (at {:?}, ExprId {:?})", ty, span, id),
                None => write!(f, "HirType::Generic reached codegen — ensure monomorphization pass runs: {:?} (at {:?})", ty, span),
            },
            Self::UnterminatedBlock { fn_name, block_name, span } => write!(f, "unterminated block '{}' in function '{}' — codegen bug (at {:?})", block_name, fn_name, span),
            Self::Internal { message, span } => write!(f, "internal error: {} (at {:?})", message, span),
        }
    }
}

impl std::error::Error for CodegenError {}

pub fn llvm_err(instruction: &'static str, err: inkwell::support::LLVMString, span: Span) -> CodegenError {
    CodegenError::llvm_instruction_failed(instruction, err.to_string(), span)
}

pub const DEFAULT_ADDR_SPACE: inkwell::AddressSpace = inkwell::AddressSpace::default();
```

### Task 5: CodegenValue, EmitResult, TerminatorKind, Phi Helpers

```rust
//! CodegenValue, EmitResult, TerminatorKind, phi conversion.

use inkwell::builder::Builder;
use inkwell::types::StructType;
use inkwell::values::{BasicValueEnum, FloatValue, IntValue, PointerValue};
use glyim_hir::{ExprId, HirType};
use glyim_span::Span;
use crate::error::{CodegenError, llvm_err};

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum TerminatorKind { Return, Unreachable, Branch, Panic }

#[derive(Debug)]
pub enum EmitResult<'ctx> {
    Value(CodegenValue<'ctx>),
    Terminator(TerminatorKind),
}

#[derive(Debug, Clone)]
pub enum CodegenValue<'ctx> {
    Unit,
    Bool(IntValue<'ctx>),
    Int(IntValue<'ctx>),
    Float(FloatValue<'ctx>),
    Ptr(PointerValue<'ctx>),
    StructRef(PointerValue<'ctx>),
    FatPtr(PointerValue<'ctx>),
}

impl<'ctx> CodegenValue<'ctx> {
    pub fn variant_name(&self) -> &'static str {
        match self {
            Self::Unit => "Unit", Self::Bool(_) => "Bool", Self::Int(_) => "Int",
            Self::Float(_) => "Float", Self::Ptr(_) => "Ptr",
            Self::StructRef(_) => "StructRef", Self::FatPtr(_) => "FatPtr",
        }
    }
}

pub fn needs_phi_value(ty: &HirType) -> bool {
    !matches!(ty, HirType::Unit | HirType::Never)
}

/// Returns None for Unit (caller skips phi entirely).
pub fn value_for_phi<'ctx>(
    val: &CodegenValue<'ctx>,
    builder: &mut Builder<'ctx>,
    str_type: StructType<'ctx>,
    span: Span,
) -> Result<Option<BasicValueEnum<'ctx>>, CodegenError> {
    match val {
        CodegenValue::Bool(v) => Ok(Some((*v).into())),
        CodegenValue::Int(v) => Ok(Some((*v).into())),
        CodegenValue::Float(v) => Ok(Some((*v).into())),
        CodegenValue::Ptr(v) => Ok(Some((*v).into())),
        CodegenValue::StructRef(v) => Ok(Some((*v).into())),
        CodegenValue::FatPtr(v) => {
            let loaded = builder.build_load(str_type, *v, "str_for_phi")
                .map_err(|e| llvm_err("build_load", e, span))?;
            Ok(Some(loaded))
        }
        CodegenValue::Unit => Ok(None),
    }
}

/// Convert a phi result back into a CodegenValue.
pub fn value_from_phi<'ctx>(
    val: BasicValueEnum<'ctx>,
    ty: &HirType,
    builder: &mut Builder<'ctx>,
    str_type: StructType<'ctx>,
    ptr_type: inkwell::types::PointerType<'ctx>,
    span: Span,
) -> Result<CodegenValue<'ctx>, CodegenError> {
    match ty {
        HirType::Bool => Ok(CodegenValue::Bool(val.into_int_value())),
        HirType::Int => Ok(CodegenValue::Int(val.into_int_value())),
        HirType::Float => Ok(CodegenValue::Float(val.into_float_value())),
        HirType::RawPtr(_) | HirType::Func(_, _) | HirType::Opaque(_) => Ok(CodegenValue::Ptr(val.into_pointer_value())),
        HirType::Named(_) | HirType::Tuple(_) => Ok(CodegenValue::StructRef(val.into_pointer_value())),
        HirType::Str => {
            let alloca = builder.build_alloca(str_type, "str_phi");
            builder.build_store(alloca, val.into_struct_value()).map_err(|e| llvm_err("build_store", e, span))?;
            Ok(CodegenValue::FatPtr(alloca))
        }
        HirType::Unit => Ok(CodegenValue::Unit),
        HirType::Never => Err(CodegenError::internal("phi for Never type", span)),
        HirType::Error => Err(CodegenError::error_type_reached_codegen(span, None)),
        HirType::Generic(_, _) => Err(CodegenError::generic_type_reached_codegen(ty.clone(), span, None)),
        HirType::Option(_) | HirType::Result(_, _) => Err(CodegenError::unimplemented_feature(
            "Option/Result must be monomorphized before codegen", span,
        )),
        HirType::Infer(_) | HirType::Param(_) => Err(CodegenError::internal(
            format!("Infer/Param in value_from_phi: {:?}", ty), span,
        )),
    }
}

pub fn debug_assert_value_matches_type(val: &CodegenValue, ty: &HirType, id: ExprId, v: &'static str) {
    let expected = match ty {
        HirType::Unit => "Unit", HirType::Bool => "Bool", HirType::Int => "Int",
        HirType::Float => "Float", HirType::RawPtr(_) => "Ptr",
        HirType::Named(_) | HirType::Tuple(_) => "StructRef",
        HirType::Str => "FatPtr", HirType::Never => "any",
        HirType::Func(_, _) | HirType::Opaque(_) => "Ptr",
        HirType::Option(_) | HirType::Result(_, _) => "StructRef", // Should be caught earlier, but if not, StructRef
        HirType::Error | HirType::Generic(_, _) | HirType::Infer(_) | HirType::Param(_) => {
            debug_assert!(false, "INTERNAL: Invalid type survived at ExprId({:?}) in {}", id, v);
            return;
        }
    };
    let actual = val.variant_name();
    debug_assert!(expected == "any" || expected == actual, "mismatch at ExprId({:?}) in {}: expected {}, got {}", id, v, expected, actual);
}
```

```rust
//! Type mapping, deterministic hashing, validation.

pub mod registry;

use inkwell::context::Context;

#[derive(Debug, Clone)]
pub struct TypedExpr {
    pub ty: HirType,
    pub span: Span,
}

impl TypedExpr {
    pub fn new(ty: HirType, span: Span) -> Self { Self { ty, span } }
}

#[derive(Debug, Clone, Copy)]
pub struct LlvmTypes<'ctx> {
    pub i1_type: IntType<'ctx>,
    pub i32_type: IntType<'ctx>,
    pub i64_type: IntType<'ctx>,
    pub f64_type: FloatType<'ctx>,
    pub ptr_type: PointerType<'ctx>,
    pub str_type: StructType<'ctx>,
}

impl<'ctx> LlvmTypes<'ctx> {
    pub fn new(context: &'ctx Context, str_type: StructType<'ctx>) -> Self {
        Self {
            i1_type: context.i1_type(), i32_type: context.i32_type(),
            i64_type: context.i64_type(), f64_type: context.f64_type(),
            ptr_type: context.ptr_type(DEFAULT_ADDR_SPACE), str_type,
        }
    }
}

pub fn validate_type_concrete(ty: &HirType, span: Span, expr_id: Option<glyim_hir::ExprId>) -> Result<(), CodegenError> {
    match ty {
        HirType::Infer(_) | HirType::Param(_) => Err(CodegenError::internal(format!("Infer/Param reached codegen: {:?}", ty), span)),
        HirType::Error => Err(CodegenError::error_type_reached_codegen(span, expr_id)),
        HirType::Generic(_, _) => Err(CodegenError::generic_type_reached_codegen(ty.clone(), span, expr_id)),
        HirType::RawPtr(inner) => validate_type_concrete(inner, span, expr_id),
        HirType::Tuple(fields) => { for f in fields { validate_type_concrete(f, span, expr_id)?; } Ok(()) }
        HirType::Option(inner) => validate_type_concrete(inner, span, expr_id),
        HirType::Result(ok, err) => { validate_type_concrete(ok, span, expr_id)?; validate_type_concrete(err, span, expr_id) }
        HirType::Func(params, ret) => { for p in params { validate_type_concrete(p, span, expr_id)?; } validate_type_concrete(ret, span, expr_id) }
        HirType::Bool | HirType::Int | HirType::Float | HirType::Unit | HirType::Str | HirType::Never | HirType::Named(_) | HirType::Opaque(_) => Ok(()),
    }
}

const FNV_OFFSET: u64 = 0xcbf29ce484222325;
const FNV_PRIME: u64 = 0x100000001b3;

#[inline]
fn fnv1a_bytes(hash: u64, bytes: &[u8]) -> u64 { let mut h = hash; for &b in bytes { h = (h ^ b as u64).wrapping_mul(FNV_PRIME); } h }
#[inline]
fn fnv1a_u64(hash: u64, val: u64) -> u64 { fnv1a_bytes(hash, &val.to_le_bytes()) }
#[inline]
pub(crate) fn fnv1a_str(hash: u64, s: &str) -> u64 { fnv1a_bytes(hash, s.as_bytes()) }

const DISC_RAW_PTR: &[u8] = b"RawPtr$";
const DISC_NAMED: &[u8] = b"Named$";
const DISC_TUPLE: &[u8] = b"Tuple$";
const DISC_FUNC: &[u8] = b"Func$";
const DISC_OPAQUE: &[u8] = b"Opaque$";

pub fn hir_type_hash(ty: &HirType, span: Span, interner: &Interner) -> Result<u64, CodegenError> {
    match ty {
        HirType::Bool => Ok(fnv1a_str(FNV_OFFSET, "Bool")),
        HirType::Int => Ok(fnv1a_str(FNV_OFFSET, "Int")),
        HirType::Float => Ok(fnv1a_str(FNV_OFFSET, "Float")),
        HirType::Unit => Ok(fnv1a_str(FNV_OFFSET, "Unit")),
        HirType::Str => Ok(fnv1a_str(FNV_OFFSET, "Str")),
        HirType::Never => Ok(fnv1a_str(FNV_OFFSET, "Never")),
        HirType::Error => Err(CodegenError::error_type_reached_codegen(span, None)),
        HirType::Generic(_, _) => Err(CodegenError::generic_type_reached_codegen(ty.clone(), span, None)),
        HirType::Infer(_) | HirType::Param(_) => Err(CodegenError::internal(format!("Infer/Param in hash: {:?}", ty), span)),
        HirType::RawPtr(inner) => Ok(fnv1a_u64(fnv1a_bytes(FNV_OFFSET, DISC_RAW_PTR), hir_type_hash(inner, span, interner)?)),
        HirType::Named(sym) => Ok(fnv1a_str(fnv1a_bytes(FNV_OFFSET, DISC_NAMED), interner.resolve(*sym))),
        HirType::Tuple(fields) => hir_type_hash_slice(fields, span, interner),
        HirType::Option(inner) => Err(CodegenError::unimplemented_feature("Option must be monomorphized before hash", span)),
        HirType::Result(ok, err) => Err(CodegenError::unimplemented_feature("Result must be monomorphized before hash", span)),
        HirType::Func(params, ret) => {
            let h = hir_type_hash_slice(params, span, interner)?;
            Ok(fnv1a_u64(fnv1a_bytes(FNV_OFFSET, DISC_FUNC), h))
        }
        HirType::Opaque(name) => Ok(fnv1a_str(fnv1a_bytes(FNV_OFFSET, DISC_OPAQUE), interner.resolve(*name))),
    }
}

/// v11 Fix #5: Slice-based hashing avoids `HirType::Tuple(vec)` allocation.
pub fn hir_type_hash_slice(types: &[HirType], span: Span, interner: &Interner) -> Result<u64, CodegenError> {
    let mut hash = fnv1a_bytes(FNV_OFFSET, DISC_TUPLE);
    for t in types { hash = fnv1a_u64(hash, hir_type_hash(t, span, interner)?); }
    Ok(hash)
}

pub fn hir_type_key(ty: &HirType, span: Span, interner: &Interner) -> Result<String, CodegenError> {
```

```rust
        let entry = self.fctx.fn_value.get_first_basic_block()
            .ok_or_else(|| CodegenError::internal("function has no entry block", span))?;

        let saved_block = self.builder.get_insert_block();

        // Position for the alloca
        if let Some(ref instr) = self.fctx.alloca_insert_point {
            self.builder.position_before(instr);
        } else {
            match entry.get_first_instruction() {
                Some(instr) => self.builder.position_before(&instr),
                None => self.builder.position_at_end(entry),
            }
        }

        let alloca = self.builder.build_alloca(ty, name);

        // Cache the instruction for next time
        if self.fctx.alloca_insert_point.is_none() {
            // The alloca instruction is the first instruction now (or right before the old first)
            // We can find it by getting the entry's first instruction
            if let Some(first) = entry.get_first_instruction() {
                self.fctx.alloca_insert_point = Some(first);
            }
        }

        // Restore position
        if let Some(block) = saved_block {
            self.builder.position_at_end(block);
        }

        Ok(alloca)
    }

    pub fn load_var(&mut self, sym: Symbol, span: Span) -> Result<CodegenValue<'ctx>, CodegenError> {
        self.fctx.load_var(sym, &mut self.builder, self.deps.types(), self.deps.type_registry(), self.deps.interner(), span)

    pub fn codegen_expr(&mut self, expr: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {
        self.set_current_span(expr_span(expr));
        let result = self.codegen_expr_inner(expr)?;

        #[cfg(debug_assertions)]
        if let EmitResult::Value(ref val) = result {
            if let Ok(ty) = self.expr_type(expr.get_id(), "codegen_expr") {
                debug_assert_value_matches_type(val, ty, expr.get_id(), hir_expr_variant_name(expr));
            }
        }
        Ok(result)
    }

    fn codegen_expr_inner(&mut self, expr: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {
        match expr {
            HirExpr::IntLit { value, .. } => Ok(EmitResult::Value(CodegenValue::Int(self.deps.types().i64_type.const_int(*value as u64, true)))),
            HirExpr::BoolLit { value, .. } => Ok(EmitResult::Value(CodegenValue::Bool(self.deps.types().i1_type.const_int(if *value { 1 } else { 0 }, false)))),
            HirExpr::FloatLit { value, .. } => Ok(EmitResult::Value(CodegenValue::Float(self.deps.types().f64_type.const_float(*value)))),
            HirExpr::UnitLit { .. } => Ok(EmitResult::Value(CodegenValue::Unit)),
            HirExpr::StringLit { value, .. } => crate::codegen::string::codegen_string_literal(self, value),
            HirExpr::Var { name, .. } => { let span = expr_span(expr); self.load_var(*name, span).map(EmitResult::Value) }
            HirExpr::BinaryOp { op, lhs, rhs, .. } => crate::codegen::ops::codegen_binop(self, *op, lhs.as_ref(), rhs.as_ref()),
            HirExpr::UnaryOp { op, operand, .. } => crate::codegen::ops::codegen_unaryop(self, *op, operand.as_ref()),
            HirExpr::Call { callee, args, .. } => crate::codegen::call::codegen_call(self, *callee, args, expr.get_id()),

            HirExpr::Return { value, .. } => crate::codegen::control::codegen_return(self, value.as_deref()),
            HirExpr::Cast { value, target_type, .. } => crate::codegen::cast::codegen_cast(self, value.as_ref(), target_type),
            HirExpr::Block { stmts, expr, .. } => crate::codegen::stmt::codegen_block(self, stmts, expr.as_deref()),
            HirExpr::Let { pattern, value, .. } => crate::codegen::pattern::codegen_let(self, pattern, value.as_ref()),
            HirExpr::Assign { target, value, .. } => crate::codegen::stmt::codegen_assign(self, target.as_ref(), value.as_ref()),
            HirExpr::ForIn { .. } => Err(CodegenError::unimplemented_feature("for-in", expr_span(expr))),
            HirExpr::IndexAccess { .. } => Err(CodegenError::unimplemented_feature("index access", expr_span(expr))),

            HirExpr::Closure { .. } => Err(CodegenError::unimplemented_feature("closure", expr_span(expr))),
            HirExpr::ArrayLit { .. } => Err(CodegenError::unimplemented_feature("array literal", expr_span(expr))),
```

```rust
pub fn emit_function_body<'ctx>(fn_def: &HirFn, fn_value: FunctionValue<'ctx>, deps: &EmitDeps<'_, 'ctx>) -> Result<(), CodegenError> {
    let span = fn_def.span;
    let fn_name = deps.interner().resolve(fn_def.name).to_string();
    let _guard = tracing::span!(tracing::Level::INFO, "emit_function", name = %fn_name).entered();

    let context = deps.context();
    let entry = context.append_basic_block(fn_value, "entry");
    let ret_bb = context.append_basic_block(fn_value, "ret");
    let builder = context.create_builder();
    builder.position_at_end(entry);

    let ret_type = fn_def.ret.clone().unwrap_or(HirType::Unit);
    let ret_val_ptr = value_llvm_type(&ret_type, deps.type_registry(), context, span)?.map(|t| builder.build_alloca(t, "ret_val"));
    let fctx = FnContext::new(fn_value, ret_type.clone(), ret_val_ptr, Some(ret_bb));
    let mut emitter = FnEmitter::new(builder, deps, fctx);

    // Emit params using entry_alloca
    let mut param_idx = 0u32;
    for (i, (sym, ty)) in fn_def.params.iter().enumerate() {
        if matches!(ty, HirType::Unit) { emitter.fctx_mut().insert_var(*sym, VarInfo::new(deps.type_registry().zst_sentinel(), ty.clone(), fn_def.param_mutability.get(i).copied().unwrap_or(false))); continue; }
        if matches!(ty, HirType::Never) { continue; }
        let pv = fn_value.get_nth_param(param_idx).ok_or_else(|| CodegenError::internal(format!("missing param {}", param_idx), span))?;
        let pt = value_llvm_type(ty, deps.type_registry(), context, span)?.ok_or_else(|| CodegenError::internal("param no type", span))?;
        let alloca = emitter.entry_alloca_fixed(pt, deps.interner().resolve(*sym), span)?;
        let unpacked = unpack_param_value(pv, ty, emitter.builder_mut(), context.i1_type(), span)?;
        emitter.builder_mut().build_store(alloca, unpacked).map_err(|e| llvm_err("build_store", e, span))?;
        emitter.fctx_mut().insert_var(*sym, VarInfo::new(alloca, ty.clone(), fn_def.param_mutability.get(i).copied().unwrap_or(false)));
        param_idx += 1;
    }

    let body_result = emitter.codegen_expr(&fn_def.body)?;
    match body_result {
        EmitResult::Value(val) => {
            if emitter.fctx().ret_val_ptr.is_some() { emitter.store_return_value(val, span)?; }
            if emitter.builder().get_insert_block().and_then(|b| b.get_terminator()).is_none() {
                if let Some(rbb) = emitter.fctx().ret_bb { emitter.builder_mut().build_unconditional_branch(rbb); }
            }
        }
        EmitResult::Terminator(_) => {}
    }

    // v11 Fix #2: Destructure FnEmitter to avoid simultaneous borrows
    let (mut builder, deps, fctx) = emitter.into_parts();
    let ret_type = fctx.ret_type.clone();
    let ret_val_ptr = fctx.ret_val_ptr;
    let ret_bb = fctx.ret_bb.expect("ret_bb required");

    builder.position_at_end(ret_bb);
    emit_return_block_destructured(&mut builder, &ret_type, ret_val_ptr, deps.types(), deps.type_registry(), span)?;

    // v11 Fix: Double verification
    if !fn_value.verify(true) {
        let ir = fn_value.print_to_string();
        return Err(CodegenError::llvm_verification_failed(deps.module().get_name().to_string_lossy().to_string(), format!("pre-scan fail\n{}", ir), span));
    }

    // Unterminated block scan
    for block in fn_value.get_basic_blocks() {
        if block.get_terminator().is_none() {
            let bname = block.get_name().to_string_lossy().to_string();
            if deps.config().strict_mode { return Err(CodegenError::unterminated_block(fn_name, bname, span)); }
            else { tracing::warn!(fn_name, bname, "patching unterminated"); builder.position_at_end(block); builder.build_unreachable(); }
        }
    }

    if !fn_value.verify(true) {
        let ir = fn_value.print_to_string();
        return Err(CodegenError::llvm_verification_failed(deps.module().get_name().to_string_lossy().to_string(), format!("post-scan fail\n{}", ir), span));
    }
    Ok(())
}

/// v11 Fix #2: Standalone function taking destructured components.
fn emit_return_block_destructured<'ctx>(
    builder: &mut inkwell::builder::Builder<'ctx>, ret_type: &HirType,
    ret_val_ptr: Option<inkwell::values::PointerValue<'ctx>>,
    types: &LlvmTypes<'ctx>, registry: &TypeRegistry<'ctx>, span: Span,
) -> Result<(), CodegenError> {
    match ret_type {
        HirType::Unit | HirType::Never => { builder.build_return(None); }
        HirType::Bool => { let p = ret_val_ptr.unwrap(); let v = builder.build_load(types.i1_type, p, "ret").map_err(|e| llvm_err("build_load", e, span))?; builder.build_return(Some(&v)); }
        HirType::Int => { let p = ret_val_ptr.unwrap(); let v = builder.build_load(types.i64_type, p, "ret").map_err(|e| llvm_err("build_load", e, span))?; builder.build_return(Some(&v)); }
        HirType::Float => { let p = ret_val_ptr.unwrap(); let v = builder.build_load(types.f64_type, p, "ret").map_err(|e| llvm_err("build_load", e, span))?; builder.build_return(Some(&v)); }
        HirType::Str => { let p = ret_val_ptr.unwrap(); let v = builder.build_load(registry.str_type(), p, "ret").map_err(|e| llvm_err("build_load", e, span))?; builder.build_return(Some(&v)); }
        HirType::RawPtr(_) | HirType::Func(_, _) | HirType::Opaque(_) | HirType::Named(_) | HirType::Tuple(_) => {
            let p = ret_val_ptr.unwrap(); let v = builder.build_load(types.ptr_type, p, "ret").map_err(|e| llvm_err("build_load", e, span))?; builder.build_return(Some(&v));
        }
        HirType::Option(_) | HirType::Result(_, _) => return Err(CodegenError::unimplemented_feature("Option/Result monomorphization required", span)),
        _ => return Err(CodegenError::internal(format!("invalid ret type: {:?}", ret_type), span)),
    }
    Ok(())
}
```

### Task 14: Control Flow — Non-Enum Switch, Unit Phi Skip, Result-returning PhiIncoming

```rust
//! Control flow. Non-enum switch. Unit phi skip.

use glyim_hir::{HirExpr, HirMatchArm, HirPattern, HirType};
use glyim_intern::Symbol;
use glyim_span::Span;
use crate::error::{CodegenError, llvm_err};
use crate::value::{CodegenValue, EmitResult, TerminatorKind, needs_phi_value};
use crate::types::struct_llvm_type;
use crate::codegen::emitter::{FnEmitter, expr_span};
use crate::codegen::phi::PhiIncoming;

pub fn codegen_if<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, cond: &HirExpr, then: &HirExpr, els: Option<&HirExpr>, id: glyim_hir::ExprId) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = expr_span(cond);
    let c = match emitter.codegen_expr(cond)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
    let ci1 = match c { CodegenValue::Bool(b) => b, _ => return Err(CodegenError::internal("if non-bool", span)) };

    let fv = emitter.fctx().fn_value;
    let then_bb = emitter.context().append_basic_block(fv, "then");
    let else_bb = emitter.context().append_basic_block(fv, "else");
    let merge_bb = emitter.context().append_basic_block(fv, "if_merge");
    emitter.builder_mut().build_conditional_branch(ci1, then_bb, else_bb);

    let result_ty = emitter.expr_type(id, "If")?.clone();
    let needs_phi = needs_phi_value(&result_ty);
    let mut incomings = Vec::new(); let mut then_term = None; let mut else_term = None;

    emitter.builder_mut().position_at_end(then_bb);
    match emitter.codegen_expr(then)? {
        EmitResult::Value(v) => if emitter.builder().get_insert_block().map(|b| b.get_terminator().is_none()).unwrap_or(false) {
            if needs_phi { if let Some(pv) = emitter.value_for_phi(&v, span)? { emitter.builder_mut().build_unconditional_branch(merge_bb); incomings.push(PhiIncoming::new(pv, emitter.builder(), span)?); } }
            else { emitter.builder_mut().build_unconditional_branch(merge_bb); }
        },
        EmitResult::Terminator(k) => then_term = Some(k),
    }

    emitter.builder_mut().position_at_end(else_bb);
    let er = els.map(|e| emitter.codegen_expr(e)).transpose()?.unwrap_or(EmitResult::Value(CodegenValue::Unit));
    match er {
        EmitResult::Value(v) => if emitter.builder().get_insert_block().map(|b| b.get_terminator().is_none()).unwrap_or(false) {
            if needs_phi { if let Some(pv) = emitter.value_for_phi(&v, span)? { emitter.builder_mut().build_unconditional_branch(merge_bb); incomings.push(PhiIncoming::new(pv, emitter.builder(), span)?); } }
            else { emitter.builder_mut().build_unconditional_branch(merge_bb); }
        },
        EmitResult::Terminator(k) => else_term = Some(k),
    }

    if then_term.is_some() && else_term.is_some() { return Ok(EmitResult::Terminator(then_term.unwrap())); }
    emitter.builder_mut().position_at_end(merge_bb);
    if !needs_phi { return Ok(EmitResult::Value(CodegenValue::Unit)); }
    match incomings.len() {
        0 => Ok(EmitResult::Value(CodegenValue::Unit)),
        1 => emitter.value_from_phi(incomings[0].value(), &result_ty, span).map(EmitResult::Value),
        _ => build_phi(emitter, &result_ty, &incomings, span),
    }
}

fn build_phi<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, ty: &HirType, incs: &[PhiIncoming<'ctx>], span: Span) -> Result<EmitResult<'ctx>, CodegenError> {
    let b = emitter.builder_mut(); let t = emitter.llvm_types();
    let phi = match ty {
        HirType::Bool => b.build_phi(t.i1_type, "phi"), HirType::Int => b.build_phi(t.i64_type, "phi"),
        HirType::Float => b.build_phi(t.f64_type, "phi"), HirType::Str => b.build_phi(emitter.type_registry().str_type(), "phi"),
        _ => b.build_phi(t.ptr_type, "phi"), // Ptr, StructRef
    };
    for i in incs { phi.add_incoming(&[(&i.value(), i.block())]); }
    emitter.value_from_phi(phi.as_basic_value(), ty, span).map(EmitResult::Value)
}

pub fn codegen_while<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, cond: &HirExpr, body: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = expr_span(cond); let fv = emitter.fctx().fn_value;
    let cbb = emitter.context().append_basic_block(fv, "wc"); let bbb = emitter.context().append_basic_block(fv, "wb"); let ebb = emitter.context().append_basic_block(fv, "we");
    emitter.builder_mut().build_unconditional_branch(cbb);
    emitter.builder_mut().position_at_end(cbb);
    let cv = match emitter.codegen_expr(cond)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
    let ci1 = match cv { CodegenValue::Bool(b) => b, _ => return Err(CodegenError::internal("while non-bool", span)) };
    emitter.builder_mut().build_conditional_branch(ci1, bbb, ebb);
    emitter.builder_mut().position_at_end(bbb);
    let _ = emitter.codegen_expr(body)?;
    if emitter.builder().get_insert_block().and_then(|b| b.get_terminator()).is_none() { emitter.builder_mut().build_unconditional_branch(cbb); }
    emitter.builder_mut().position_at_end(ebb);
    Ok(EmitResult::Value(CodegenValue::Unit))
}

pub fn codegen_match<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, scrut: &HirExpr, arms: &[HirMatchArm], id: glyim_hir::ExprId) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = expr_span(scrut);
    let sv = match emitter.codegen_expr(scrut)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
    let st = emitter.expr_type(scrut.get_id(), "Match")?.clone();
    let rt = emitter.expr_type(id, "Match")?.clone();
    let needs_phi = needs_phi_value(&rt);

    // Determine switch value and enum info
    let (switch_val, enum_sym) = match sv {
        CodegenValue::StructRef(ptr) => {
            let stype = struct_llvm_type(&st, emitter.type_registry(), span)?;
            let b = emitter.builder_mut();
            // SAFETY: build_struct_gep is unsafe in inkwell 0.5. Precondition: ptr points to valid enum allocation.
            let tp = unsafe { b.build_struct_gep(stype, ptr, 0, "tag_ptr").map_err(|e| llvm_err("build_struct_gep", e, span))? };
            let tag = b.build_load(emitter.llvm_types().i64_type, tp, "tag").map_err(|e| llvm_err("build_load", e, span))?;
            let sym = match &st { HirType::Named(s) => *s, _ => return Err(CodegenError::internal("StructRef non-Named", span)) };
            (Some(tag.into_int_value()), Some(sym))
        }
        CodegenValue::Int(i) => (Some(i), None),
        CodegenValue::Bool(b) => (Some(emitter.builder_mut().build_int_z_extend(b, emitter.llvm_types().i64_type, "bsw")), None),
        _ => return Err(CodegenError::internal("match scrut unsupported", span)),
    };

    let fv = emitter.fctx().fn_value;
    let merge_bb = emitter.context().append_basic_block(fv, "mm");
    let arm_bbs: Vec<_> = arms.iter().map(|a| (a, emitter.context().append_basic_block(fv, "ma"))).collect();
    let wild_idx = arms.iter().position(|a| matches!(a.pattern, HirPattern::Wildcard | HirPattern::Var { .. }));
    let default_bb = if wild_idx.is_none() { Some(emitter.context().append_basic_block(fv, "md")) } else { None };

    if let Some(tag) = switch_val {
        let mut cases = Vec::new();
        for (i, (arm, bb)) in arm_bbs.iter().enumerate() {
            if matches!(arm.pattern, HirPattern::Wildcard | HirPattern::Var { .. }) { continue; }
            let vtag = extract_tag(&arm.pattern, enum_sym, emitter.type_registry(), span)?;
            cases.push((vtag.into(), *bb));
        }
        let def = wild_idx.map(|i| arm_bbs[i].1).or(default_bb).unwrap();
        emitter.builder_mut().build_switch(tag, def, &cases);
    }

    let mut incomings = Vec::new(); let mut all_term = true; let mut last_term = None;

    for (arm, bb) in &arm_bbs {
        emitter.builder_mut().position_at_end(*bb);
        match &arm.pattern {
            HirPattern::EnumVariant { fields, .. } if !fields.is_empty() => return Err(CodegenError::unimplemented_feature("enum pattern binding", span)),
            HirPattern::Struct { .. } | HirPattern::Tuple { .. } => return Err(CodegenError::unimplemented_feature("pattern type", span)),
            _ => {}
        }
        match emitter.codegen_expr(&arm.body)? {
            EmitResult::Value(v) => {
                all_term = false;
                if emitter.builder().get_insert_block().map(|b| b.get_terminator().is_none()).unwrap_or(false) {
                    if needs_phi { if let Some(pv) = emitter.value_for_phi(&v, span)? { emitter.builder_mut().build_unconditional_branch(merge_bb); incomings.push(PhiIncoming::new(pv, emitter.builder(), span)?); } }
                    else { emitter.builder_mut().build_unconditional_branch(merge_bb); }
                }
            }
            EmitResult::Terminator(k) => last_term = Some(k),
        }
    }

    if let Some(db) = default_bb {
        emitter.builder_mut().position_at_end(db);
        let trap = emitter.module().get_function("llvm.trap").unwrap_or_else(|| emitter.module().add_function("llvm.trap", emitter.context().void_type().fn_type(&[], false), None));
        emitter.builder_mut().build_call(trap, &[], "trap");
        emitter.builder_mut().build_unreachable();
    }

    if all_term && last_term.is_some() { return Ok(EmitResult::Terminator(last_term.unwrap())); }
    emitter.builder_mut().position_at_end(merge_bb);
    if !needs_phi { return Ok(EmitResult::Value(CodegenValue::Unit)); }
    match incomings.len() {
        0 => Ok(EmitResult::Value(CodegenValue::Unit)),
        1 => emitter.value_from_phi(incomings[0].value(), &rt, span).map(EmitResult::Value),
        _ => build_phi(emitter, &rt, &incomings, span),
    }
}

fn extract_tag<'ctx>(pat: &HirPattern, enum_sym: Option<Symbol>, reg: &crate::types::TypeRegistry<'ctx>, span: Span) -> Result<u64, CodegenError> {
    match pat {
        HirPattern::EnumVariant { variant, .. } => reg.variant_tag(enum_sym.ok_or_else(|| CodegenError::internal("enum sym missing", span))?, *variant).ok_or_else(|| CodegenError::internal("variant missing", span)),
        HirPattern::IntLit { value, .. } => Ok(*value as u64),
        HirPattern::BoolLit { value, .. } => Ok(if *value { 1 } else { 0 }),
        _ => Err(CodegenError::internal("non-switch pattern", span)),
    }
}

pub fn codegen_return<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, val: Option<&HirExpr>) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = emitter.current_span();
    if let Some(v) = val { let cv = match emitter.codegen_expr(v)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) }; if emitter.fctx().ret_val_ptr.is_some() { emitter.store_return_value(cv, span)?; } }
    if let Some(rbb) = emitter.fctx().ret_bb { emitter.builder_mut().build_unconditional_branch(rbb); } else { emitter.builder_mut().build_return(None); }
    Ok(EmitResult::Terminator(TerminatorKind::Return))
}
```

```rust
use std::collections::HashMap;
use inkwell::context::Context;
use glyim_hir::{HirFn, HirType};
use glyim_intern::Symbol;
use glyim_span::Span;
use crate::error::CodegenError;
use crate::types::registry::{TypeRegistry, StructDef, EnumDef};

use crate::codegen::input::CodegenInput;
use crate::codegen::fn_registry::FnRegistry;

use crate::codegen::string_mgr::StringLiteralManager;
use crate::codegen::function;
use crate::discovery;
use crate::method_names;
use crate::shims;

pub struct TypeDefinitions { pub struct_defs: HashMap<Symbol, StructDef>, pub enum_defs: HashMap<Symbol, EnumDef> }
impl TypeDefinitions { pub fn new(s: HashMap<Symbol, StructDef>, e: HashMap<Symbol, EnumDef>) -> Self { Self { struct_defs: s, enum_defs: e } } }

pub struct CodegenSession<'ctx> { pub module: inkwell::module::Module<'ctx>, pub type_registry: TypeRegistry<'ctx>, pub fn_registry: FnRegistry<'ctx>, pub string_mgr: StringLiteralManager<'ctx> }

pub fn compile(context: &'ctx Context, mut input: CodegenInput, functions: &[HirFn], type_defs: &TypeDefinitions) -> Result<CodegenSession<'ctx>, CodegenError> {
    crate::codegen::input::validate_codegen_input(&input)?;
    let module = context.create_module(&input.config.module_name);
    crate::codegen::function::set_target_layout(&module, &input.config);

    let mut type_registry = TypeRegistry::new(context, &mut input.interner);
    // v11 Fix #1: Initialize ZST sentinel in main module
    let type_registry = type_registry.initialize_zst_sentinel(&module);

    let mut discovered = discovery::DiscoveredTypes::new();
    discovered.discover_from_expr_types(&input.expr_types);
    for f in functions { if let Some(r) = &f.ret { discovered.discover_from_type(r); } for (_, p) in &f.params { discovered.discover_from_type(p); } }

    for tt in &discovered.tuple_types { if let HirType::Tuple(f) = tt { type_registry.register_tuple(f, context, &mut input.interner, Span::default())?; } }
    for (n, d) in &type_defs.struct_defs { type_registry.register_struct(*n, &d.field_names, &d.field_types, context, &input.interner, Span::default())?; }
    for (n, d) in &type_defs.enum_defs { type_registry.register_enum(*n, &d.variants, context, &input.interner, Span::default())?; }

    let mut fn_registry = FnRegistry::new();
    for f in functions { crate::codegen::function::declare_function(f, &module, &type_registry, &mut fn_registry, context, &input.interner)?; }

    let mut shim_ctx = shims::ShimEmitCtx { context, module: &module, config: &input.config, type_registry: &type_registry, fn_registry: &mut fn_registry, interner: &mut input.interner };
    shims::ShimRegistry::for_aot().emit(&mut shim_ctx)?;

    input.method_names = method_names::compute_method_names_from_hir(functions, &input.expr_types, &type_registry, &mut input.interner)?;

    let string_mgr = StringLiteralManager::new();
    let llvm_types = LlvmTypes::new(context, type_registry.str_type());
    let deps = EmitDeps::new(&type_registry, &fn_registry, context, &module, &input.interner, &input.known, &input.expr_types, &input.call_type_args, &input.method_names, &input.metadata, &input.config, &llvm_types, &string_mgr);

    for f in functions {
        let fv = fn_registry.get(f.name).ok_or_else(|| CodegenError::internal(format!("fn {:?} missing", f.name), f.span))?;
        crate::codegen::function::emit_function_body(f, fv, &deps)?;
    }

    if let Err(e) = module.verify() { return Err(CodegenError::llvm_verification_failed(module.get_name().to_string_lossy().to_string(), e.to_string(), Span::default())); }
    Ok(CodegenSession { module, type_registry, fn_registry, string_mgr })
}
```

```rust
//! String literal emission and fat pointer construction.
//! FNV-1a for deterministic global names. Null terminator included.

use glyim_span::Span;
use crate::error::{CodegenError, llvm_err, DEFAULT_ADDR_SPACE};
use crate::types::fnv1a_str;
use crate::types::FNV_OFFSET;
use crate::value::{CodegenValue, EmitResult};
use crate::codegen::emitter::FnEmitter;

pub fn codegen_string_literal<'ctx>(
    emitter: &mut FnEmitter<'_, 'ctx>,
    string_data: &str,
) -> Result<EmitResult<'ctx>, CodegenError> {
    let str_type = emitter.type_registry().str_type();
    let span = emitter.current_span();

    let hash = fnv1a_str(FNV_OFFSET, string_data);
    let global_name = format!("__str_lit_{:016x}", hash);

    let data_ptr = if let Some(cached) = emitter.get_cached_string(&global_name) {
        cached
    } else if let Some(existing) = emitter.module().get_global(&global_name) {
        let ptr = existing.as_pointer_value();
        emitter.cache_string(global_name.clone(), ptr);
        ptr
    } else {
        let bytes = string_data.as_bytes();
        let i8_type = emitter.context().i8_type();

        // [N+1 x i8] with trailing \0 for C ABI compatibility.
        let array_type = i8_type.array_type((bytes.len() + 1) as u32);
        let global = emitter.module().add_global(array_type, Some(DEFAULT_ADDR_SPACE), &global_name);

        let mut const_bytes: Vec<inkwell::values::IntValue<'ctx>> = bytes.iter()
            .map(|&b| i8_type.const_int(b as u64, false))
            .collect();
        const_bytes.push(i8_type.const_int(0, false)); // Null terminator

        let const_str = i8_type.const_array(&const_bytes);
        global.set_initializer(&const_str);
        global.set_constant(true);
        global.set_linkage(inkwell::module::Linkage::Private);
        global.set_unnamed_address(inkwell::module::UnnamedAddress::Global);

        let ptr = global.as_pointer_value();
        emitter.cache_string(global_name, ptr);
        ptr
    };

    let builder = emitter.builder_mut();
    let alloca = builder.build_alloca(str_type, "str_lit");

    // SAFETY: build_struct_gep is unsafe in inkwell 0.5 with opaque pointers.
    // Precondition: `alloca` points to a valid allocation of `str_type`,
    // and index 0 is within bounds (the `ptr` field of `{ ptr, i64 }`).
    let data_field_ptr = unsafe {
        builder.build_struct_gep(str_type, alloca, 0, "str_data_ptr")
            .map_err(|e| llvm_err("build_struct_gep", e, span))?
    };
    builder.build_store(data_field_ptr, data_ptr)
        .map_err(|e| llvm_err("build_store", e, span))?;

    // SAFETY: Precondition: `alloca` points to a valid allocation of `str_type`,
    // and index 1 is within bounds (the `i64` len field of `{ ptr, i64 }`).
    let len_field_ptr = unsafe {
        builder.build_struct_gep(str_type, alloca, 1, "str_len_ptr")
            .map_err(|e| llvm_err("build_struct_gep", e, span))?
    };
    let len_val = emitter.llvm_types().i64_type.const_int(string_data.len() as u64, false);
    builder.build_store(len_field_ptr, len_val)
        .map_err(|e| llvm_err("build_store", e, span))?;

    Ok(EmitResult::Value(CodegenValue::FatPtr(alloca)))
}
```

```rust
//! Binary and unary operators. Bool is i1. No .map_err() on arithmetic.

use glyim_hir::{HirBinOp, HirUnOp, HirExpr};
use glyim_span::Span;
use crate::error::CodegenError;
use crate::value::{CodegenValue, EmitResult, TerminatorKind};
use crate::codegen::emitter::{FnEmitter, expr_span};

pub fn codegen_binop<'ctx>(
    emitter: &mut FnEmitter<'_, 'ctx>, op: HirBinOp, lhs_expr: &HirExpr, rhs_expr: &HirExpr,
) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = expr_span(lhs_expr);
    let lhs = match emitter.codegen_expr(lhs_expr)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
    let rhs = match emitter.codegen_expr(rhs_expr)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };

    let result = match (&lhs, &rhs) {
        (CodegenValue::Bool(l), CodegenValue::Bool(r)) => bool_binop(emitter, op, *l, *r, span),
        (CodegenValue::Float(l), CodegenValue::Float(r)) => float_binop(emitter, op, *l, *r, span),
        (CodegenValue::Int(l), CodegenValue::Int(r)) => int_binop(emitter, op, *l, *r, span),
        (CodegenValue::StructRef(l), CodegenValue::StructRef(r)) => ptr_cmp(emitter, op, *l, *r, "sr_cmp", span),
        (CodegenValue::Ptr(l), CodegenValue::Ptr(r)) => ptr_cmp(emitter, op, *l, *r, "ptr_cmp", span),
        (CodegenValue::FatPtr(l), CodegenValue::FatPtr(r)) if matches!(op, HirBinOp::Eq | HirBinOp::Neq) => ptr_cmp(emitter, op, *l, *r, "fp_cmp", span),
        _ => Err(CodegenError::internal(format!("type mismatch in binop: {:?} vs {:?}", lhs.variant_name(), rhs.variant_name()), span)),
    }?;
    Ok(EmitResult::Value(result))
}

fn bool_binop<'ctx>(e: &FnEmitter<'_, 'ctx>, op: HirBinOp, l: inkwell::values::IntValue<'ctx>, r: inkwell::values::IntValue<'ctx>, span: Span) -> Result<CodegenValue<'ctx>, CodegenError> {
    let b = e.builder();
    match op {
        HirBinOp::And => Ok(CodegenValue::Bool(b.build_and(l, r, "band"))),
        HirBinOp::Or => Ok(CodegenValue::Bool(b.build_or(l, r, "bor"))),
        HirBinOp::Eq => Ok(CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::EQ, l, r, "beq"))),
        HirBinOp::Neq => Ok(CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::NE, l, r, "bneq"))),
        _ => Err(CodegenError::internal(format!("invalid bool op: {:?}", op), span)),
    }
}

fn int_binop<'ctx>(e: &FnEmitter<'_, 'ctx>, op: HirBinOp, l: inkwell::values::IntValue<'ctx>, r: inkwell::values::IntValue<'ctx>, span: Span) -> Result<CodegenValue<'ctx>, CodegenError> {
    let b = e.builder();
    Ok(match op {
        HirBinOp::Add => CodegenValue::Int(b.build_int_add(l, r, "iadd")),
        HirBinOp::Sub => CodegenValue::Int(b.build_int_sub(l, r, "isub")),
        HirBinOp::Mul => CodegenValue::Int(b.build_int_mul(l, r, "imul")),
        HirBinOp::Div => CodegenValue::Int(b.build_int_signed_div(l, r, "idiv")),
        HirBinOp::Mod => CodegenValue::Int(b.build_int_signed_rem(l, r, "imod")),
        HirBinOp::Eq => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::EQ, l, r, "ieq")),
        HirBinOp::Neq => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::NE, l, r, "ineq")),
        HirBinOp::Lt => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::SLT, l, r, "ilt")),
        HirBinOp::Gt => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::SGT, l, r, "igt")),
        HirBinOp::Lte => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::SLE, l, r, "ilte")),
        HirBinOp::Gte => CodegenValue::Bool(b.build_int_compare(inkwell::IntPredicate::SGE, l, r, "igte")),
        HirBinOp::And => CodegenValue::Int(b.build_and(l, r, "iand")),
        HirBinOp::Or => CodegenValue::Int(b.build_or(l, r, "ior")),
    })
}

fn float_binop<'ctx>(e: &FnEmitter<'_, 'ctx>, op: HirBinOp, l: inkwell::values::FloatValue<'ctx>, r: inkwell::values::FloatValue<'ctx>, span: Span) -> Result<CodegenValue<'ctx>, CodegenError> {
    let b = e.builder();
    match op {
        HirBinOp::Add => Ok(CodegenValue::Float(b.build_float_add(l, r, "fadd"))),
        HirBinOp::Sub => Ok(CodegenValue::Float(b.build_float_sub(l, r, "fsub"))),
        HirBinOp::Mul => Ok(CodegenValue::Float(b.build_float_mul(l, r, "fmul"))),
        HirBinOp::Div => Ok(CodegenValue::Float(b.build_float_div(l, r, "fdiv"))),
        HirBinOp::Mod => Ok(CodegenValue::Float(b.build_float_rem(l, r, "fmod"))),
        HirBinOp::Eq => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::OEQ, l, r, "feq"))),
        HirBinOp::Neq => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::ONE, l, r, "fneq"))),
        HirBinOp::Lt => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::OLT, l, r, "flt"))),
        HirBinOp::Gt => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::OGT, l, r, "fgt"))),
        HirBinOp::Lte => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::OLE, l, r, "flte"))),
        HirBinOp::Gte => Ok(CodegenValue::Bool(b.build_float_compare(inkwell::FloatPredicate::OGE, l, r, "fgte"))),
        HirBinOp::And | HirBinOp::Or => Err(CodegenError::internal(format!("invalid float op: {:?}", op), span)),
    }
}

fn ptr_cmp<'ctx>(e: &FnEmitter<'_, 'ctx>, op: HirBinOp, l: inkwell::values::PointerValue<'ctx>, r: inkwell::values::PointerValue<'ctx>, name: &str, span: Span) -> Result<CodegenValue<'ctx>, CodegenError> {
    let b = e.builder();
    match op {
        HirBinOp::Eq | HirBinOp::Neq => {
            let li = b.build_ptr_to_int(l, e.llvm_types().i64_type, "li");
            let ri = b.build_ptr_to_int(r, e.llvm_types().i64_type, "ri");
            let pred = if matches!(op, HirBinOp::Eq) { inkwell::IntPredicate::EQ } else { inkwell::IntPredicate::NE };
            Ok(CodegenValue::Bool(b.build_int_compare(pred, li, ri, name)))
        }
        _ => Err(CodegenError::internal(format!("invalid ptr op: {:?}", op), span)),
    }
}

pub fn codegen_unaryop<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, op: HirUnOp, operand: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = expr_span(operand);
    let val = match emitter.codegen_expr(operand)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
    let b = emitter.builder_mut();
    let result = match (&op, &val) {
        (HirUnOp::Not, CodegenValue::Bool(v)) => CodegenValue::Bool(b.build_not(*v, "bnot")),
        (HirUnOp::Neg, CodegenValue::Int(v)) => CodegenValue::Int(b.build_int_neg(*v, "ineg")),
        (HirUnOp::Not, CodegenValue::Int(v)) => CodegenValue::Int(b.build_not(*v, "inot")),
```

```rust
//! ZST sentinel from main module. Unit fields skipped.

use glyim_intern::Symbol;
use glyim_span::Span;

pub fn codegen_struct_literal<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, name: Symbol, fields: &[(Symbol, HirExpr)]) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = emitter.current_span();
    let struct_type = emitter.type_registry().get_struct(name).ok_or_else(|| CodegenError::type_not_registered(HirType::Named(name), span))?;
    let size = emitter.type_registry().struct_size(name).ok_or_else(|| CodegenError::internal("no size", span))?;

    let raw_ptr = if size == 0 { emitter.type_registry().zst_sentinel() }
    else {

        let alloc = emitter.fn_registry().get(emitter.known().glyim_alloc).ok_or_else(|| CodegenError::shim_not_declared("__glyim_alloc", span))?;
        emitter.builder_mut().build_call(alloc, &[sarg.into()], "salloc").try_as_basic_value().left().unwrap().into_pointer_value()

        let fval = match emitter.codegen_expr(fexpr)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
        let b = emitter.builder_mut();

        let fptr = unsafe { b.build_struct_gep(struct_type, raw_ptr, fidx, "fptr").map_err(|e| llvm_err("build_struct_gep", e, span))? };

    }
    Ok(EmitResult::Value(CodegenValue::StructRef(raw_ptr)))
}

pub fn codegen_enum_literal<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, ename: Symbol, vname: Symbol, fexprs: &[HirExpr]) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = emitter.current_span();
    let layout = emitter.type_registry().get_enum_layout(ename).ok_or_else(|| CodegenError::type_not_registered(HirType::Named(ename), span))?;
    let etype = layout.struct_type;
    let size = emitter.type_registry().enum_size(ename).ok_or_else(|| CodegenError::internal("no enum size", span))?;

    let raw_ptr = if size == 0 { emitter.type_registry().zst_sentinel() }
    else {
        let sarg = emitter.llvm_types().i64_type.const_int(size, false);
        let alloc = emitter.fn_registry().get(emitter.known().glyim_alloc).ok_or_else(|| CodegenError::shim_not_declared("__glyim_alloc", span))?;
        emitter.builder_mut().build_call(alloc, &[sarg.into()], "ealloc").try_as_basic_value().left().unwrap().into_pointer_value()
    };

    let vtag = emitter.type_registry().variant_tag(ename, vname).ok_or_else(|| CodegenError::internal("variant missing", span))?;
    let b = emitter.builder_mut();
    // SAFETY: etype matches allocation, index 0 is i64 tag.
    let tptr = unsafe { b.build_struct_gep(etype, raw_ptr, 0, "tptr").map_err(|e| llvm_err("build_struct_gep", e, span))? };
    b.build_store(tptr, emitter.llvm_types().i64_type.const_int(vtag, false)).map_err(|e| llvm_err("build_store", e, span))?;

    let ptype = emitter.type_registry().variant_payload_type(ename, vname).ok_or_else(|| CodegenError::internal("no payload type", span))?;
    // SAFETY: etype matches, index 1 is payload array.
    let pptr = unsafe { b.build_struct_gep(etype, raw_ptr, 1, "pptr").map_err(|e| llvm_err("build_struct_gep", e, span))? };

    let mut llvm_idx: u32 = 0;
    for fe in fexprs {
        let fv = match emitter.codegen_expr(fe)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };
        if matches!(fv, CodegenValue::Unit) { continue; }
        let b = emitter.builder_mut();
        // SAFETY: ptype matches payload layout at pptr, llvm_idx in bounds.
        let fptr = unsafe { b.build_struct_gep(ptype, pptr, llvm_idx, "vfptr").map_err(|e| llvm_err("build_struct_gep", e, span))? };
        crate::codegen::value_utils::store_value_to_ptr(b, fptr, fv, emitter.type_registry().str_type(), span)?;
        llvm_idx += 1;
    }
    Ok(EmitResult::Value(CodegenValue::StructRef(raw_ptr)))
}

pub fn codegen_tuple_literal<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, elems: &[HirExpr], id: glyim_hir::ExprId) -> Result<EmitResult<'ctx>, CodegenError> {
    let span = emitter.current_span();

        emitter.builder_mut().build_call(alloc, &[sarg.into()], "talloc").try_as_basic_value().left().unwrap().into_pointer_value()
    };

    let mut llvm_idx: u32 = 0;
    for e in elems {
        let ev = match emitter.codegen_expr(e)? { EmitResult::Value(v) => v, EmitResult::Terminator(k) => return Ok(EmitResult::Terminator(k)) };

        llvm_idx += 1;
    }
    Ok(EmitResult::Value(CodegenValue::StructRef(raw_ptr)))
```

```rust
pub fn codegen_let<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, pat: &glyim_hir::HirPattern, val: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {

fn bind<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, pat: &glyim_hir::HirPattern, val: CodegenValue<'ctx>, span: Span) -> Result<(), CodegenError> {
    match pat {
        glyim_hir::HirPattern::Var { name, ty, .. } => {
            if let Some(lt) = value_llvm_type(ty, emitter.type_registry(), emitter.context(), span)? {
                let n = emitter.interner().resolve(*name);

            } else {
                emitter.fctx_mut().insert_var(*name, VarInfo::new(emitter.type_registry().zst_sentinel(), ty.clone(), true));
            }
            Ok(())
        }
        glyim_hir::HirPattern::Wildcard => Ok(()),
        _ => Err(CodegenError::unimplemented_feature("pattern binding", span)),
    }
}
```

```rust
//! Statement emission: blocks, assignments.
//! codegen_block pushes scope on entry, pops on exit via Result.

use glyim_hir::{HirExpr, HirType};
use glyim_intern::Symbol;
use glyim_span::Span;
use crate::error::{CodegenError, llvm_err};
use crate::value::{CodegenValue, EmitResult};
use crate::types::struct_llvm_type;
use crate::codegen::emitter::{FnEmitter, expr_span};

pub fn codegen_block<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, stmts: &[HirExpr], trail: Option<&HirExpr>) -> Result<EmitResult<'ctx>, CodegenError> {
    emitter.fctx_mut().push_scope();
    let mut term = None;
    for s in stmts {
        match emitter.codegen_expr(s)? { EmitResult::Value(_) => {} EmitResult::Terminator(k) => { term = Some(k); break; } }
    }
    if let Some(k) = term { emitter.fctx_mut().pop_scope(emitter.current_span())?; return Ok(EmitResult::Terminator(k)); }
    let res = if let Some(e) = trail { emitter.codegen_expr(e)? } else { EmitResult::Value(CodegenValue::Unit) };
    emitter.fctx_mut().pop_scope(emitter.current_span())?;
    Ok(res)
}

pub fn codegen_assign<'ctx>(emitter: &mut FnEmitter<'_, 'ctx>, target: &HirExpr, val: &HirExpr) -> Result<EmitResult<'ctx>, CodegenError> {

        HirExpr::Var { name, .. } => { emitter.store_var(*name, v, span)?; Ok(EmitResult::Value(CodegenValue::Unit)) }
        HirExpr::FieldAccess { object, field, .. } => {

    }
```

```rust
//! Method name computation with __glyim_meth_ prefix.

use std::collections::HashMap;
use glyim_hir::{HirExpr, HirFn, ExprId, HirType};
use glyim_intern::{Interner, Symbol};
use glyim_span::Span;
use crate::codegen::input::MethodNameMap;
use crate::error::CodegenError;
use crate::types::registry::TypeRegistry;

pub fn compute_method_names_from_hir(fns: &[HirFn], ets: &HashMap<ExprId, crate::types::TypedExpr>, reg: &TypeRegistry<'_>, i: &mut Interner) -> Result<MethodNameMap, CodegenError> {
    let mut map = MethodNameMap::new();
    for f in fns { walk(&f.body, ets, reg, i, &mut map)?; }
    Ok(map)
}

fn walk(e: &HirExpr, ets: &HashMap<ExprId, crate::types::TypedExpr>, reg: &TypeRegistry<'_>, i: &mut Interner, m: &mut MethodNameMap) -> Result<(), CodegenError> {
    match e {
        HirExpr::MethodCall { receiver, method, args, .. } => {
            walk(receiver.as_ref(), ets, reg, i, m)?;
            for a in args { walk(a, ets, reg, i, m)?; }
            if let Some(te) = ets.get(&receiver.get_id()) {
                if let Ok(ts) = resolve(&te.ty, reg, i) {
                    let key = (ts, *method);
                    if !m.contains_key(&key) { m.insert(key, i.intern(&format!("__glyim_meth_{}__{}", i.resolve(ts), i.resolve(*method)))); }
                }
            }
        }
        HirExpr::BinaryOp { lhs, rhs, .. } => { walk(lhs.as_ref(), ets, reg, i, m)?; walk(rhs.as_ref(), ets, reg, i, m)?; }
        HirExpr::UnaryOp { operand, .. } => { walk(operand.as_ref(), ets, reg, i, m)?; }
        HirExpr::Call { args, .. } => { for a in args { walk(a, ets, reg, i, m)?; } }
        HirExpr::If { condition, then_branch, else_branch, .. } => { walk(condition.as_ref(), ets, reg, i, m)?; walk(then_branch.as_ref(), ets, reg, i, m)?; if let Some(e) = else_branch { walk(e.as_ref(), ets, reg, i, m)?; } }
        HirExpr::While { condition, body, .. } => { walk(condition.as_ref(), ets, reg, i, m)?; walk(body.as_ref(), ets, reg, i, m)?; }
        HirExpr::Match { scrutinee, arms, .. } => { walk(scrutinee.as_ref(), ets, reg, i, m)?; for a in arms { walk(&a.body, ets, reg, i, m)?; } }
        HirExpr::Block { stmts, expr, .. } => { for s in stmts { walk(s, ets, reg, i, m)?; } if let Some(e) = expr { walk(e.as_ref(), ets, reg, i, m)?; } }
        HirExpr::Let { value, .. } => { walk(value.as_ref(), ets, reg, i, m)?; }
        HirExpr::Assign { target, value, .. } => { walk(target.as_ref(), ets, reg, i, m)?; walk(value.as_ref(), ets, reg, i, m)?; }
        HirExpr::FieldAccess { object, .. } => { walk(object.as_ref(), ets, reg, i, m)?; }
        HirExpr::Cast { value, .. } => { walk(value.as_ref(), ets, reg, i, m)?; }
        HirExpr::Return { value, .. } => { if let Some(v) = value { walk(v.as_ref(), ets, reg, i, m)?; } }
        HirExpr::StructLiteral { fields, .. } => { for (_, f) in fields { walk(f, ets, reg, i, m)?; } }
        HirExpr::EnumLiteral { fields, .. } => { for f in fields { walk(f, ets, reg, i, m)?; } }
        HirExpr::TupleLiteral { elements, .. } => { for e in elements { walk(e, ets, reg, i, m)?; } }
        _ => {}
    }
    Ok(())
}

fn resolve(ty: &HirType, reg: &TypeRegistry<'_>, i: &Interner) -> Result<Symbol, CodegenError> {
    match ty {
        HirType::Named(s) => Ok(*s),
        HirType::Tuple(_) => { let h = crate::types::hir_type_hash(ty, Span::default(), i)?; reg.resolve_tuple_symbol_by_hash(ty, h, Span::default(), i) }
        _ => Err(CodegenError::internal(format!("bad method recv: {:?}", ty), Span::default())),
    }
}
```

# Harsh Code Plan Critique: `glyim-llvm-codegen` v11

## Overall Sentiment

**Needs Work** — The v11 plan is a massive leap forward. It correctly resolves the `Cell`/`Copy` issue with `OnceCell`, the borrow checker conflict with `FnEmitter::into_parts`, and the allocation overhead with slice-based hashing. However, it introduces a fundamental misunderstanding of the Rust/inkwell FFI boundary by wrapping safe inkwell API calls in `unsafe` blocks, and it violates its own "No panics" invariant in three places. These issues will cause compilation warnings/lints and runtime panics if edge cases are hit.

---

## Detailed Findings

### Correctness & Compilation Safety

*   **`unsafe` blocks around `build_struct_gep` are incorrect and will generate compiler warnings.** The v11 plan asserts: "v11 Fix #3: `build_struct_gep` is `unsafe` in inkwell 0.5 + LLVM 15." This is **false**. In inkwell, `build_struct_gep` (along with `build_load` and `build_store`) is a **safe** Rust function that returns `Result<PointerValue, LLVMString>`. While the *underlying* LLVM C API call (`LLVMBuildStructGEP2`) involves unsafe operations, inkwell wraps it in a safe Rust API. Wrapping a safe function call in `unsafe {}` is redundant, misleads the reader into thinking the Rust call itself is unsafe, and will trigger `unused_unsafe` lints in strict compilation modes. **Fix:** Remove all `unsafe {}` blocks and `// SAFETY:` comments surrounding calls to `builder.build_struct_gep(...)`, `builder.build_load(...)`, and `builder.build_store(...)`. Handle the `Result` correctly with `.map_err()` as already planned.

*   **`TypeRegistry::zst_sentinel()` uses `.expect()`, violating the "No panics" invariant.** The function panics if called before `initialize_zst_sentinel`. While the pipeline guarantees initialization, a library user constructing a `TypeRegistry` manually could trigger this. **Fix:** Change the signature to `pub fn zst_sentinel(&self, span: Span) -> Result<PointerValue<'ctx>, CodegenError>` and return `Err(CodegenError::internal(...))` if not initialized. Update all call sites (data_construct, pattern, context) to handle the `Result`.

*   **`data_construct.rs` uses `.unwrap()` on allocation call results.** The pattern `emitter.builder_mut().build_call(...).try_as_basic_value().left().unwrap().into_pointer_value()` appears in `codegen_struct_literal`, `codegen_enum_literal`, and `codegen_tuple_literal`. If `__glyim_alloc` is somehow declared with a void return type, this panics. **Fix:** Replace `.unwrap()` with `.ok_or_else(|| CodegenError::internal("__glyim_alloc returned void", span))?` to match the pattern used in `call_marshall.rs`.

*   **`function.rs` uses `.expect()` on `ret_bb`.** The line `let ret_bb = fctx.ret_bb.expect("ret_bb required");` panics if `ret_bb` is `None`. While the pipeline sets it, this is a library invariant violation. **Fix:** Replace with `let ret_bb = fctx.ret_bb.ok_or_else(|| CodegenError::internal("ret_bb missing", span))?;`.

### Boundaries & Contracts

*   The contract for `build_struct_gep` is misstated. The inkwell API boundary is safe Rust; the "unsafety" is in the generated IR semantics, which is inkwell's responsibility to expose safely (which it does via `Result`). The plan's `// SAFETY:` comments should be replaced with standard `// NOTE:` comments explaining the IR-level invariants (e.g., "Precondition: ptr points to a valid allocation of struct_type").

### Modularity & Separation of Concerns

*   **`FnEmitter::into_parts` is an elegant solution to the borrow conflict.** This is a significant improvement. It cleanly separates the builder, dependencies, and context, allowing `emit_return_block_destructured` to borrow them independently.
*   **`FnEmitter` remains a God object** (as noted in v10), but the `into_parts` mitigation makes this acceptable for the current scope.

### Performance & Resource Efficiency

*   **Slice-based hashing is correctly implemented.** `register_tuple` calls `hir_type_hash_slice(field_types, ...)` and `hir_type_key_slice(field_types, ...)`, eliminating the double `HirType::Tuple(vec)` allocation. This is a verified fix for v10 Finding #5.

### Debuggability & Observability

*   **Error messages for Option/Result are actionable.** "Option/Result must be monomorphized to Named before codegen. Ensure monomorphization pass is enabled." is a clear, helpful message.
*   **`tracing::span` is present on type registration.** This fixes v10 Finding #7.

### Elegance & Hack‑free Design

*   **`OnceCell` for the ZST sentinel is idiomatic.** It replaces the hacky `Cell` approach with a write-once, panic-free (for reads) pattern.
*   **The `unsafe` blocks are a hack.** They introduce unnecessary `unsafe` scopes into safe Rust code, which is the opposite of elegant. They must be removed.

---

## Summary of Required Fixes

1.  **CRITICAL: Remove all `unsafe {}` blocks and `// SAFETY:` comments around `build_struct_gep`, `build_load`, and `build_store` calls.** These are safe inkwell API functions. Replace `// SAFETY:` comments with `// NOTE:` if IR-level invariants need documenting.
2.  **Fix "No Panics" violations:**
    *   Change `TypeRegistry::zst_sentinel()` to return `Result<PointerValue, CodegenError>`.
    *   Replace `.unwrap()` in `data_construct.rs` (3 locations) with `.ok_or_else()`.
    *   Replace `.expect()` in `function.rs` (`ret_bb`) with `.ok_or_else()`.
3.  **(Minor)** Consider adding a `// NOTE:` comment to `build_struct_gep` calls documenting the IR-level precondition (pointer validity) for future maintainers, but do not wrap in `unsafe`.

---

**Score: 8/10**

The v11 plan demonstrates excellent engineering discipline in fixing the architectural flaws (borrow conflicts, allocation overhead, ZST initialization). The code structure is sound and the error handling is mostly robust. However, the misunderstanding of the inkwell `unsafe` boundary and the residual panics prevent a perfect score. Correct the 3 items above, and this plan is ready for implementation.
