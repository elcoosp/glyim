# Glyim Lint Implementation Plan — v6 (10/10 Production-Ready)

> **Goal:** Build a type-aware linting engine for Glyim that consumes `TypeCheckOutput` from the v8 typechecker, runs lints via single-traversal `LintDriver`, and integrates with CLI. Every file compiles. Zero placeholders. Zero known bugs. All v4 and v5 review findings addressed.

## Traversal Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                    Shared walk module                         │
│  HirVisitor<'a> trait:                                       │
│    visit_item, visit_fn, visit_expr, visit_expr_post,        │
│    visit_stmt, visit_pattern                                 │
│                                                              │
│  NOTE: Does NOT walk HirExpr::Call callee (callee is         │
│  ExprId — requires expr_map to resolve). Consumers that     │
│  need callee walking must resolve via CalleeResolver.        │
│                                                              │
│  Used by: CalleeResolver                                     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│           Custom traversal: LintDriver                        │
│  Why custom: per-pass dispatch, is_last_stmt tracking,       │
│  scope_depth tracking, callee walking via CalleeResolver     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│           Custom traversal: ScopeAnalysis                    │
│  Why custom: scope push/pop tied to variable declaration     │
│  semantics, pattern context tracking, callee walking via     │
│  resolve_expr closure, HirPattern::Var lacks span            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│           Custom traversal: CognitiveComplexity              │
│  Why custom: nesting-level-aware scoring with structural     │
│  context (else-if chains score differently than nested ifs)  │
└─────────────────────────────────────────────────────────────┘

Contract test: walk_consistency ensures all traversals visit
the same set of expression/statement/pattern nodes (excluding
```

//! Glyim linting engine.
//!
//! Type-aware lints that consume `TypeCheckOutput` from `glyim-typeck`.
//! Zero duplication — borrows type maps, HirIndex, KnownSymbols, Interner.
//!
//! # Architecture
//!
//! ```text
//! TypeCheckOutput ──borrow──▶ TypeCheckBorrow ─┐
//!                                                ├▶ LintDriver ──▶ Vec<LintDiagnostic>
//! LintConfig + LintRegistry + SuppressionMap ──▶ LintEnvironment ─┘
//! ```
//!
//! # Adding a new lint
//!
//! 1. Create a new file in the appropriate `lints/` subdirectory.
//! 2. Implement `LateLintPass` for your lint struct.
//! 3. Add the module to the subdirectory's `mod.rs`.
//! 4. Add one line to the `lint_passes!` macro in `lints/mod.rs`.
//!
//! # LintId convention
//!
//! LintId values use `snake_case` (e.g., `"unused_variable"`) to match
//! the names users write in config files and suppression annotations.
//! The `lint_passes!` macro generates both the LintId constant and
//! the descriptor from a single snake_case string literal.
//!
//! The inner `&'static str` field of `LintId` is `pub(crate)` — external
//! code must use the exported UPPER_CASE constants (e.g., `UNUSED_VARIABLE`)
//! to obtain `LintId` values. LintId does NOT implement Deserialize
//! because `&'static str` cannot be safely deserialized from runtime data.

#[cfg(test)]
mod contracts;

pub mod diagnostic;
pub mod config;
pub mod suppression;
pub mod registry;
pub mod pass;
pub mod walk;
pub mod callee;
pub mod scope;
pub mod type_query;
pub mod context;
pub mod driver;
pub mod lints;
pub mod output;

pub use diagnostic::{
    Applicability, DiagnosticStats, LintDiagnostic, LintGroup, LintId, LintLevel, LintSuggestion,
    Replacement,
};
pub use config::{LintConfig, LintConfigFile, LintThresholds};
pub use type_query::{LintEnvironment, TypeCheckBorrow, TypeQueryCache};
pub use suppression::SuppressionMap;
pub use registry::{LintDescriptor, LintRegistry};
pub use pass::LateLintPass;
pub use callee::CalleeResolver;
pub use scope::{ScopeAnalysis, ScopeAnalysisResult, VarInfo};
pub use context::LintContext;
pub use driver::LintDriver;
pub use output::OutputFormat;

## Chunk 1: Diagnostic Types

### Task 1.1: diagnostic.rs

```rust
//! Lint diagnostic types.
//!
//! Key design decisions:
//! - `LintId` does NOT implement `Deserialize` — `&'static str` cannot be
//!   safely deserialized from runtime TOML/JSON. Config files use string keys
//!   resolved via `LintRegistry::get_by_name`.
//! - `LintLevel::Forbid` cannot be suppressed by annotations (checked in
//!   `LintContext::emit_core` before suppression lookup).
//! - `LintLevel::Forbid` cannot be downgraded by config (checked in
//!   `LintConfig::apply_file`).

use glyim_diag::{Severity, Span};
use serde::{Deserialize, Serialize};

/// Unique identifier for a lint rule.
///
/// Constructed from a `&'static str` snake_case literal.
/// The inner field is `pub(crate)` — external code must use the
/// UPPER_CASE constants generated by the `lint_passes!` macro
/// (e.g., `UNUSED_VARIABLE`, `CAST_SIGN_LOSS`).
///
/// # Why no Deserialize?
///
/// `LintId` contains `&'static str`. Deserializing from runtime data
/// (TOML, JSON) would require producing a `&'static str` from borrowed
/// data, which is unsound. Config files use `HashMap<String, LintLevel>`
/// and resolve names via `LintRegistry::get_by_name`.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, PartialOrd, Ord, Serialize)]
pub struct LintId {
    /// The snake_case identifier string. `pub(crate)` so only the
    /// `lint_passes!` macro and registry lookup can construct LintIds.
    pub(crate) inner: &'static str,
}

impl LintId {
    /// Construct a LintId from a static string.
    ///
    /// Only available within the crate. External users must use the
    /// exported constants or `LintRegistry::get()`.
    pub const fn new(s: &'static str) -> Self {
        Self { inner: s }
    }

    /// The snake_case identifier string.
    pub fn as_str(&self) -> &'static str {
        self.inner
    }
}

impl std::fmt::Display for LintId {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "{}", self.inner)
    }
}

/// User-configurable lint level.
///
/// Distinct from `Severity` — this is the configuration concept.
/// Maps to `Severity` when emitting diagnostics:
/// - `Allow` → no diagnostic emitted
/// - `Warn` → `Severity::Warning`
/// - `Deny` → `Severity::Error`
/// - `Forbid` → `Severity::Error` (cannot be overridden by annotations
///   or downgraded by config)
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum LintLevel {
    Allow,
    Warn,
    Deny,
    Forbid,
}

impl LintLevel {
    /// Convert to diagnostic severity.
    ///
    /// Returns `None` for `Allow` (no diagnostic should be emitted).
    pub fn to_severity(self) -> Option<Severity> {
        match self {
            LintLevel::Allow => None,
            LintLevel::Warn => Some(Severity::Warning),
            LintLevel::Deny => Some(Severity::Error),
            LintLevel::Forbid => Some(Severity::Error),
        }
    }
}

impl std::fmt::Display for LintLevel {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            LintLevel::Allow => write!(f, "allow"),
            LintLevel::Warn => write!(f, "warn"),
            LintLevel::Deny => write!(f, "deny"),
            LintLevel::Forbid => write!(f, "forbid"),
        }
    }
}

/// Lint group for categorization and group-level config overrides.
///
/// # Invariant
///
/// A lint's group MUST match its directory location:
/// - `correctness/` → `LintGroup::Correctness`
/// - `style/` → `LintGroup::Style`
/// - `type_aware/` → `LintGroup::TypeAware`
/// - `complexity/` → `LintGroup::Complexity`
/// - `performance/` → `LintGroup::Performance`
///
/// Verified by the `lint_groups_match_directories` contract test.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize)]
pub enum LintGroup {
    Correctness,
    Style,
    TypeAware,
    Complexity,
    Performance,
    Deprecated,
}

impl std::fmt::Display for LintGroup {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            LintGroup::Correctness => write!(f, "correctness"),
            LintGroup::Style => write!(f, "style"),
            LintGroup::TypeAware => write!(f, "type_aware"),
            LintGroup::Complexity => write!(f, "complexity"),
            LintGroup::Performance => write!(f, "performance"),
            LintGroup::Deprecated => write!(f, "deprecated"),
        }
    }
}

/// How confident the auto-fix is.
///
/// Ordered from most to least applicable.
/// When `Applicability` is `MachineApplicable`, replacement text MUST be
/// syntactically valid source code that produces the same semantics.
#[derive(Debug, Clone, Copy, PartialEq, Eq, PartialOrd, Ord)]
pub enum Applicability {
    /// Fix is definitely correct and produces valid source code.
    MachineApplicable,
    /// Fix is probably correct, but may change semantics in edge cases.
    MaybeIncorrect,
    /// Fix contains placeholders that the user must fill in.
    HasPlaceholders,
    /// No auto-fix available. The diagnostic provides a note instead.
    /// Replacement text may be empty — the note explains what to do manually.
    Unspecified,
}

/// A single span replacement for an auto-fix.
///
/// # Invariant
///
/// When `Applicability` is `MachineApplicable`, `new_text` MUST be
/// syntactically valid source code that preserves the intended semantics.
```

//!
//! # Severity resolution (single code path in LintContext::emit_core)
//!
//! ```text
//! config per-lint override? ──yes──▶ use it
//!       │
//!       no
//!       ▼
//! config per-group override? ──yes──▶ use it
//!       │
//!       no
//!       ▼
//!   keep diagnostic's original severity (set by lint)
//! ```
//!
//! # Forbid enforcement
//!
//! `LintLevel::Forbid` cannot be downgraded by config file entries.
//! `apply_file` rejects any attempt to set a Forbid lint to a lower
//! level, logging a warning and skipping the override.
//!
//! # Config file format (TOML)
//!
//! ```toml
//! [lint]
//! default_level = "warn"
//!
//! [lint.levels]
//! unused_variable = "allow"
//! division_by_zero = "deny"
//!
//! [lint.groups]
//! style = "allow"
//!
//! [lint.thresholds]
//! cognitive_complexity = 15

/// Lint configuration.
///
/// Constructed programmatically via `LintConfig::new()` and setters,
/// or loaded from a TOML file via `LintConfigFile`.
#[derive(Debug, Clone)]
pub struct LintConfig {
    /// Per-lint level overrides.
    lint_levels: HashMap<LintId, LintLevel>,
    /// Per-group level overrides.
    group_levels: HashMap<LintGroup, LintLevel>,
    /// Configurable thresholds for parameterized lints.
    pub thresholds: LintThresholds,
}

impl Default for LintConfig {
    fn default() -> Self {
        Self::new()
    }
}

impl LintConfig {
    pub fn new() -> Self {
        Self {
            lint_levels: HashMap::new(),
            group_levels: HashMap::new(),
            thresholds: LintThresholds::default(),
        }
    }

    /// Get the effective config level override for a lint.
    ///
    /// Returns `None` when no override exists — the caller should use
    /// the diagnostic's original severity.
    /// Priority: per-lint > per-group > None.
    pub fn effective_level(&self, lint_id: LintId, group: LintGroup) -> Option<LintLevel> {
        if let Some(level) = self.lint_levels.get(&lint_id) {
            return Some(*level);
        }
        if let Some(level) = self.group_levels.get(&group) {
            return Some(*level);
        }
        None
    }

    /// Override level for a specific lint.
    ///
    /// Takes a `LintId` — use the exported constants (e.g., `UNUSED_VARIABLE`)
    /// to construct valid IDs.
    pub fn set_lint_level(&mut self, lint_id: LintId, level: LintLevel) {
        self.lint_levels.insert(lint_id, level);
    }

    /// Override level for an entire group.
    pub fn set_group_level(&mut self, group: LintGroup, level: LintLevel) {
        self.group_levels.insert(group, level);
    }

    /// Apply overrides from a config file.
    ///
    /// Requires a `&LintRegistry` to resolve string lint names from
    /// the config file to canonical `LintId` values. Unknown lint names
    /// are logged as warnings and skipped.
    ///
    /// # Forbid enforcement
    ///
    /// If a lint's default level is `Forbid`, any config file entry
    /// attempting to lower it is rejected with a warning. This prevents
    /// users from accidentally allowing lints that must never be disabled.
    pub fn apply_file(&mut self, file: &LintConfigFile, registry: &LintRegistry) {
        for (lint_name, level) in &file.lint.levels {
            match registry.get_by_name(lint_name) {
                Some(descriptor) => {
                    // Forbid enforcement: cannot downgrade a Forbid lint
                    if descriptor.default_level == LintLevel::Forbid
                        && *level != LintLevel::Forbid
                    {
                        tracing::warn!(
                            lint = lint_name,
                            requested_level = %level,
                            "cannot downgrade Forbid lint via config, skipping"
                        );
                        continue;
                    }
                    self.set_lint_level(descriptor.id, *level);
                }
                None => {
                    tracing::warn!(
                        lint = lint_name,
                        "unknown lint in config file, skipping"
                    );
                }
            }
        }
        for (group_name, level) in &file.lint.groups {
            let group = match group_name.as_str() {
                "correctness" => LintGroup::Correctness,
                "style" => LintGroup::Style,
                "type_aware" => LintGroup::TypeAware,
                "complexity" => LintGroup::Complexity,
                "performance" => LintGroup::Performance,
                "deprecated" => LintGroup::Deprecated,
                _ => {
                    tracing::warn!(
                        group = group_name,
                        "unknown lint group in config file, skipping"
                    );
                    continue;
                }
            };
            self.set_group_level(group, *level);
        }
        // Always apply threshold values unconditionally — avoids
        // hardcoding defaults that drift from LintThresholds::default().
        self.thresholds.cognitive_complexity = file.lint.thresholds.cognitive_complexity;
        self.thresholds.too_many_arguments = file.lint.thresholds.too_many_arguments;
        self.thresholds.large_type_bytes = file.lint.thresholds.large_type_bytes;
    }
}

```rust
//! Suppression map built from `#[allow(...)]` / `#[deny(...)]` annotations.
//!
//! # Overlap semantics
//!
//! When suppressions overlap, the **innermost** (narrowest span) wins.
//!
//! # Span containment
//!
//! A diagnostic span `D` is "contained" by a suppression span `S` when:
//! `S.start <= D.start AND D.end <= S.end`
//!
//! # Forbid enforcement
//!
//! `LintContext::emit_core` checks for `Forbid` level BEFORE consulting
//! this map. Forbid-level lints bypass suppression entirely.

use crate::diagnostic::LintId;
use glyim_diag::Span;
use std::collections::BTreeMap;

/// A single suppression entry.
#[derive(Debug, Clone)]
struct SuppressionEntry {
    span: Span,
    lint_ids: Vec<LintId>,
}

/// Suppression map with BTreeMap-based lookup.
#[derive(Debug, Clone, Default)]
pub struct SuppressionMap {
    entries: BTreeMap<usize, Vec<SuppressionEntry>>,
}

impl SuppressionMap {
    pub fn new() -> Self {
        Self::default()
    }

    /// Add a suppression covering `span` for the given lint IDs.
    ///
    /// If `lint_ids` is empty, this suppresses ALL lints within the span
    /// (equivalent to `#[allow(...)]` with no specific lint names).
    pub fn add_suppression(&mut self, span: Span, lint_ids: Vec<LintId>) {
        tracing::debug!(
            span_start = span.start,
            span_end = span.end,
            lints = ?lint_ids,
            "adding suppression"
        );
        let entry = SuppressionEntry { span, lint_ids };
        let key = span.start;
        self.entries.entry(key).or_default().push(entry);
        if let Some(vec) = self.entries.get_mut(&key) {
            // Sort by span length ascending so innermost is found first
            vec.sort_by_key(|e| e.span.end.saturating_sub(e.span.start));
        }
    }

    /// Check if a lint is suppressed at the given span.
    ///
    /// O(k) where k = number of entries with start <= span.start.
    /// Designed for files with <100 suppressions; for larger counts,
    /// consider an interval tree.
    pub fn is_suppressed(&self, lint_id: LintId, span: Span) -> bool {
        let mut best: Option<&SuppressionEntry> = None;
        let mut best_length = usize::MAX;

        for (_, entries) in self.entries.range(..=span.start) {
            for entry in entries {
                if Self::fully_contains(&entry.span, &span) {
                    let entry_len = entry.span.end.saturating_sub(entry.span.start);
                    if entry_len < best_length {
                        best_length = entry_len;
                        best = Some(entry);
                    }
                }
            }
        }

        let Some(entry) = best else {
            return false;
        };

        let suppressed = entry.lint_ids.is_empty() || entry.lint_ids.contains(&lint_id);

        if suppressed {
            tracing::debug!(
                lint_id = lint_id.as_str(),
                diag_start = span.start,
                suppression_start = entry.span.start,
                suppression_end = entry.span.end,
                "diagnostic suppressed by annotation"
            );
        }

        suppressed
    }

    fn fully_contains(outer: &Span, inner: &Span) -> bool {
        outer.start <= inner.start && inner.end <= outer.end
    }
}
```

```rust
//! Lint registry: stores LintDescriptors and provides lookup.
//!
//! Each lint pass declares its own descriptor via `LateLintPass::descriptor()`.
//! The registry is populated by calling `register` for each pass.
//! The `lint_passes!` macro ensures every pass is registered exactly once.

use crate::diagnostic::{LintGroup, LintId, LintLevel};
use std::collections::HashMap;

/// Describes a lint rule.
#[derive(Debug, Clone)]
pub struct LintDescriptor {
    pub id: LintId,
    pub name: &'static str,
    pub description: &'static str,
    pub default_level: LintLevel,
    pub group: LintGroup,
}

/// The lint registry, containing all registered lints.
#[derive(Debug, Clone)]
pub struct LintRegistry {
    lints: HashMap<LintId, LintDescriptor>,
    name_map: HashMap<&'static str, LintId>,
}

impl Default for LintRegistry {
    fn default() -> Self {
        Self::new()
    }
}

impl LintRegistry {
    pub fn new() -> Self {
        Self {
            lints: HashMap::new(),
            name_map: HashMap::new(),
        }
    }

    /// Create a registry pre-populated with all built-in lints.
    pub fn with_all_builtins() -> Self {
        let mut registry = Self::new();
        for desc in crate::lints::all_descriptors() {
            registry.register(desc);
        }
        registry
    }

    /// Register a lint descriptor.
    ///
    /// Logs a warning if a duplicate LintId is registered (overwrites previous).
    pub fn register(&mut self, descriptor: LintDescriptor) {
        if self.lints.contains_key(&descriptor.id) {
            tracing::warn!(
                lint_id = descriptor.id.as_str(),
                "duplicate lint registration — overwriting previous descriptor"
            );
        }
```

## Chunk 6: Shared HIR Walk Module

### Task 6.1: walk.rs

```rust
//! Shared HIR traversal infrastructure.
//!
//! Eliminates copy-pasted walk implementations across CalleeResolver
//! and future consumers that need simple observation.
//!
//! # Callee handling
//!
//! `HirExpr::Call { callee: ExprId, .. }` stores the callee as an
//! `ExprId` reference, not an inline child. This module does NOT walk
//! the callee because resolving `ExprId → &HirExpr` requires an expr_map
//! that this generic walker does not possess. Consumers that need callee
//! walking must resolve it themselves (e.g., CalleeResolver via its expr_map).
//!
//! # Why ScopeAnalysis, LintDriver, and CognitiveComplexity don't use this
//!
//! - ScopeAnalysis: requires scope push/pop tied to variable declaration
//!   semantics, pattern context tracking, and callee walking via resolve_expr.
//! - LintDriver: requires per-pass dispatch, is_last_stmt tracking,
//!   scope_depth tracking, and callee walking.
//! - CognitiveComplexity: requires nesting-level-aware scoring.

use glyim_hir::item::HirItem;
use glyim_hir::node::{HirExpr, HirStmt};
use glyim_hir::types::HirPattern;
use glyim_hir::HirFn;
use glyim_hir::Hir;

/// Callback trait for HIR traversal.
///
/// Every method has a default no-op implementation. Implementors
/// override only the methods they need.
pub trait HirVisitor<'a> {
    fn visit_item(&mut self, _item: &'a HirItem) {}
    fn visit_fn(&mut self, _f: &'a HirFn) {}
    fn visit_expr(&mut self, _expr: &'a HirExpr) {}
    fn visit_expr_post(&mut self, _expr: &'a HirExpr) {}
    fn visit_stmt(&mut self, _stmt: &'a HirStmt) {}
    fn visit_pattern(&mut self, _pat: &'a HirPattern) {}
}

/// Walk an entire HIR, calling visitor callbacks at each node.
pub fn walk_hir<'a>(hir: &'a Hir, visitor: &mut dyn HirVisitor<'a>) {
    for item in &hir.items {
        visitor.visit_item(item);
        walk_item(item, visitor);
    }
}

/// Walk a single item's children.
pub fn walk_item<'a>(item: &'a HirItem, visitor: &mut dyn HirVisitor<'a>) {
    match item {
        HirItem::Fn(f) => {
            visitor.visit_fn(f);
            walk_expr(&f.body, visitor);
        }
        HirItem::Impl(imp) => {
            for m in &imp.methods {
                visitor.visit_fn(m);
                walk_expr(&m.body, visitor);
            }
        }
        _ => {}
    }
}

/// Walk an expression and its children, calling visitor callbacks.
///
/// Calls `visitor.visit_expr(expr)` before recursing (pre-order).
/// Does NOT walk into the callee of `HirExpr::Call` — the callee
/// is an ExprId that requires an expr_map to resolve.
pub fn walk_expr<'a>(expr: &'a HirExpr, visitor: &mut dyn HirVisitor<'a>) {
    visitor.visit_expr(expr);

    match expr {
        HirExpr::Binary { lhs, rhs, .. } => {
            walk_expr(lhs, visitor);
            walk_expr(rhs, visitor);
        }
        HirExpr::Unary { operand, .. } => {
            walk_expr(operand, visitor);
        }
        HirExpr::Block { stmts, .. } => {
            for s in stmts {
                walk_stmt(s, visitor);
            }
        }
        HirExpr::If {
            condition,
            then_branch,
            else_branch,
            ..
        } => {
            walk_expr(condition, visitor);
            walk_expr(then_branch, visitor);
            if let Some(e) = else_branch {
                walk_expr(e, visitor);
            }
        }
        HirExpr::Match { scrutinee, arms, .. } => {
            walk_expr(scrutinee, visitor);
            for arm in arms {
                walk_pattern(&arm.pattern, visitor);
                if let Some(g) = &arm.guard {
                    walk_expr(g, visitor);
                }
                walk_expr(&arm.body, visitor);
            }
        }
        // NOTE: callee is ExprId, not a child — not walked here
        HirExpr::Call { args, .. } => {
            for a in args {
                walk_expr(a, visitor);
            }
        }
        HirExpr::MethodCall {
            receiver, args, ..
        } => {
            walk_expr(receiver, visitor);
            for a in args {
                walk_expr(a, visitor);
            }
        }
        HirExpr::StructLit { fields, .. } => {
            for (_, v) in fields {
                walk_expr(v, visitor);
            }
        }
        HirExpr::EnumVariant { args, .. } | HirExpr::TupleLit { elements: args, .. } => {
            for a in args {
                walk_expr(a, visitor);
            }
        }
        HirExpr::ForIn {
            pattern, iter, body, ..
        } => {
            walk_pattern(pattern, visitor);
            walk_expr(iter, visitor);
            walk_expr(body, visitor);
        }
        HirExpr::While {
            condition, body, ..
        } => {
            walk_expr(condition, visitor);
            walk_expr(body, visitor);
        }
        HirExpr::Return { value, .. } => {
            if let Some(v) = value {
                walk_expr(v, visitor);
            }
        }
        HirExpr::Assert {
            condition,
            message,
            ..
        } => {
            walk_expr(condition, visitor);
            if let Some(m) = message {
                walk_expr(m, visitor);
            }
        }
        HirExpr::Println { arg, .. } => {
            walk_expr(arg, visitor);
        }
        HirExpr::As { expr, .. } => {
            walk_expr(expr, visitor);
        }
        HirExpr::Deref { expr, .. } => {
            walk_expr(expr, visitor);
        }
        HirExpr::AddrOf { target, .. } => {
            walk_expr(target, visitor);
        }
        HirExpr::FieldAccess { object, .. } => {
            walk_expr(object, visitor);
        }

        HirExpr::SizeOf { .. }
        | HirExpr::IntLit { .. }
        | HirExpr::FloatLit { .. }
        | HirExpr::BoolLit { .. }
        | HirExpr::StrLit { .. }
        | HirExpr::UnitLit { .. }
        | HirExpr::Ident { .. } => {}
    }

    visitor.visit_expr_post(expr);
}

/// Walk a statement and its children.
pub fn walk_stmt<'a>(stmt: &'a HirStmt, visitor: &mut dyn HirVisitor<'a>) {
    visitor.visit_stmt(stmt);

    match stmt {
        HirStmt::Let { value, .. } => walk_expr(value, visitor),
        HirStmt::LetPat {
            pattern, value, ..
        } => {
            walk_pattern(pattern, visitor);
            walk_expr(value, visitor);
        }
        HirStmt::Assign { target, value, .. } => {
            walk_expr(target, visitor);
            walk_expr(value, visitor);
        }
        HirStmt::AssignDeref {
            target, value, ..
        } => {
            walk_expr(target, visitor);
            walk_expr(value, visitor);
        }
        HirStmt::AssignField {
            object, value, ..
        } => {
            walk_expr(object, visitor);
            walk_expr(value, visitor);
        }
        HirStmt::Expr(e) => walk_expr(e, visitor),
    }
}

/// Walk a pattern and its children.
pub fn walk_pattern<'a>(pat: &'a HirPattern, visitor: &mut dyn HirVisitor<'a>) {
    visitor.visit_pattern(pat);

    match pat {
        HirPattern::Tuple { elements, .. } => {
            for e in elements {
                walk_pattern(e, visitor);
            }
        }
        HirPattern::Struct { bindings, .. } => {
            for (_, p) in bindings {
                walk_pattern(p, visitor);
            }
        }
        HirPattern::EnumVariant { bindings, .. } => {
            for (_, p) in bindings {
                walk_pattern(p, visitor);
            }
        }
        HirPattern::OptionSome(inner) => {
            walk_pattern(inner, visitor);
        }
        HirPattern::ResultOk(inner) | HirPattern::ResultErr(inner) => {
            walk_pattern(inner, visitor);
        }

        HirPattern::Wild
        | HirPattern::BoolLit(_)
        | HirPattern::IntLit(_)
        | HirPattern::FloatLit(_)
        | HirPattern::StrLit(_)
        | HirPattern::Unit
        | HirPattern::Var(_)
        | HirPattern::OptionNone => {}
    }
}
```

## Chunk 7: CalleeResolver (Redesigned)

### Task 7.1: callee.rs

```rust
//! Resolves `ExprId` to `HirExpr` nodes and builds a call graph.
//!
//! Uses the shared `walk` module. During the walk, it builds:
//! 1. `expr_map`: `ExprId → &HirExpr` for all expressions
//! 2. `call_graph`: `Symbol → HashSet<Symbol>` mapping each function
//!    to the set of functions it calls (by name).
//!
//! The call graph is used by `unused_function` to compute reachability
//! from entry points via BFS.
//!
//! # v6 FIX: finish_function uses extend, not insert
//!
//! Method calls are recorded directly into `call_graph` during the walk
//! (in `visit_expr`). Direct calls are accumulated in `current_fn_callees`
//! and resolved in `finish_function`. `finish_function` now uses
//! `entry().or_default().extend()` to merge resolved callees into the
//! existing entry, preserving method calls.

use crate::walk::{walk_hir, HirVisitor};
use glyim_hir::item::HirItem;
use glyim_hir::node::HirExpr;
use glyim_hir::types::ExprId;
use glyim_hir::Hir;
use glyim_hir::HirFn;
use glyim_interner::Symbol;
use std::collections::{HashMap, HashSet, VecDeque};

/// Resolves `ExprId` → `&HirExpr` and provides a call graph.
pub struct CalleeResolver<'a> {
    expr_map: HashMap<ExprId, &'a HirExpr>,
    call_graph: HashMap<Symbol, HashSet<Symbol>>,
}

impl<'a> CalleeResolver<'a> {
    /// Build a CalleeResolver by walking the HIR.
    pub fn build(hir: &'a Hir) -> Self {
        let mut builder = CalleeBuilder {
            expr_map: HashMap::new(),
            current_fn: None,
            current_fn_callees: HashSet::new(),
            call_graph: HashMap::new(),
        };
        walk_hir(hir, &mut builder);

        // Finish the last function if the HIR doesn't end with an impl
        builder.finish_function();

        CalleeResolver {
            expr_map: builder.expr_map,
            call_graph: builder.call_graph,
        }
    }

    /// Look up an expression by its ExprId.
    pub fn find_expr(&self, id: ExprId) -> Option<&'a HirExpr> {
        self.expr_map.get(&id).copied()
    }

    /// Resolve the name of a callee expression, if it's an Ident.
    pub fn resolve_callee_name(&self, callee_id: ExprId) -> Option<Symbol> {
        let expr = self.find_expr(callee_id)?;
        match expr {
            HirExpr::Ident { name, .. } => Some(*name),
            _ => None,
        }
    }

    /// Access the call graph.
    pub fn call_graph(&self) -> &HashMap<Symbol, HashSet<Symbol>> {
        &self.call_graph
    }

    /// Compute the set of functions reachable from entry points via BFS.
    pub fn compute_reachable(
        &self,
        hir: &Hir,
        is_entry: &dyn Fn(&HirFn) -> bool,
    ) -> HashSet<Symbol> {
        let mut reachable = HashSet::new();
        let mut worklist: VecDeque<Symbol> = VecDeque::new();

        for item in &hir.items {
            match item {
                HirItem::Fn(f) if is_entry(f) => {
                    if reachable.insert(f.name) {
                        worklist.push_back(f.name);
                    }
                }
                HirItem::Impl(imp) => {
                    for m in &imp.methods {
                        if is_entry(m) {
                            if reachable.insert(m.name) {
                                worklist.push_back(m.name);
                            }
                        }
                    }
                }
                _ => {}
            }
        }

        while let Some(caller) = worklist.pop_front() {
            if let Some(callees) = self.call_graph.get(&caller) {
                for callee in callees {
                    if reachable.insert(*callee) {
                        worklist.push_back(*callee);
                    }
                }
            }
        }

        reachable
    }
}

struct CalleeBuilder<'a> {
    expr_map: HashMap<ExprId, &'a HirExpr>,
    current_fn: Option<Symbol>,
    current_fn_callees: HashSet<ExprId>,
    call_graph: HashMap<Symbol, HashSet<Symbol>>,
}

impl<'a> CalleeBuilder<'a> {
    /// Called when leaving a function. Resolves accumulated callee
    /// ExprIds to Symbol names and merges them into the call graph.
    ///
    /// v6 FIX: Uses `extend` instead of `insert` to preserve method-call
    /// entries that were inserted directly during `visit_expr`.
    fn finish_function(&mut self) {
        if let Some(fn_name) = self.current_fn.take() {
            let mut resolved_callees = HashSet::new();
            for callee_id in self.current_fn_callees.drain() {
                if let Some(expr) = self.expr_map.get(&callee_id) {
                    if let HirExpr::Ident { name, .. } = expr {
                        resolved_callees.insert(*name);
                    }
                }
            }
            // MERGE, don't overwrite — method calls already exist in the entry
            self.call_graph
                .entry(fn_name)
                .or_default()
                .extend(resolved_callees);
        }
    }
}

impl<'a> HirVisitor<'a> for CalleeBuilder<'a> {
    fn visit_fn(&mut self, f: &'a HirFn) {
        // Finish previous function before starting a new one
        self.finish_function();
        self.current_fn = Some(f.name);
        self.current_fn_callees.clear();
    }

    fn visit_expr(&mut self, expr: &'a HirExpr) {
        self.expr_map.insert(expr.id(), expr);

        // Direct function calls: accumulate callee ExprId for later resolution
        if let HirExpr::Call { callee, .. } = expr {
            self.current_fn_callees.insert(*callee);
        }

        // Method calls: insert directly into call graph immediately
        if let HirExpr::MethodCall { method_name, .. } = expr {
            if let Some(fn_name) = self.current_fn {
                self.call_graph
                    .entry(fn_name)
                    .or_default()
                    .insert(*method_name);
            }
        }
    }
}
```

## Chunk 8: Scope Analysis

### Task 8.1: scope.rs

```rust
//! Scoped variable analysis for unused_variable and unnecessary_mut lints.
//!
//! # Why custom traversal
//!
//! ScopeAnalysis cannot use the shared `walk` module because:
//! 1. Scope push/pop is tied to variable declaration semantics.
//! 2. Requires pattern context tracking (mutability from stmt vs for-in vs match arm).
//! 3. HirPattern::Var lacks span info, requiring span inference from parent.
//! 4. Must walk `HirExpr::Call` callee expressions to mark callee variables as used.
//!
//! # For-in binding mutability
//!
//! For-in bindings default to `mutable: false` because the HIR does not
//! carry mutability info on ForIn patterns. This is conservative for
//! `unnecessary_mut` (avoids false positives) and correct for
//! `unused_variable` (mutability doesn't affect usage).
//!
//! # v6 FIX: Assign target is Box<HirExpr>, not Symbol
//!
//! `HirStmt::Assign { target: Box<HirExpr>, .. }` — the target is an
//! expression, not a bare symbol. We extract the name from Ident targets
//! and also walk the target expression for complex assignments like
//! field accesses.

use glyim_diag::Span;
use glyim_hir::node::{HirExpr, HirStmt};
use glyim_hir::types::{ExprId, HirPattern};
use glyim_hir::HirFn;
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

#[derive(Debug, Clone)]
struct VarState {
    name: Symbol,
    decl_span: Span,
    name_span: Span,
    is_mutable: bool,
    is_used: bool,
    is_assigned: bool,
}

#[derive(Debug, Default)]
struct Scope {
    vars: HashMap<Symbol, VarState>,
}

impl Scope {
    fn new() -> Self { Self::default() }
    fn declare(&mut self, state: VarState) { self.vars.insert(state.name, state); }
    fn mark_used(&mut self, name: Symbol) -> bool {
        if let Some(var) = self.vars.get_mut(&name) { var.is_used = true; true } else { false }
    }
    fn mark_assigned(&mut self, name: Symbol) -> bool {
        if let Some(var) = self.vars.get_mut(&name) { var.is_assigned = true; true } else { false }
    }
}

#[derive(Debug, Clone)]
pub struct VarInfo {
    pub name: Symbol,
    pub decl_span: Span,
    pub name_span: Span,
    pub is_mutable: bool,
}

#[derive(Debug, Clone)]
pub struct ScopeAnalysisResult {
    pub unused_vars: Vec<VarInfo>,
    pub unnecessary_muts: Vec<VarInfo>,
}

pub struct ScopeAnalysis;

impl ScopeAnalysis {
    /// Analyze a function's scopes for unused variables and unnecessary mutability.
    ///
    /// Takes a `resolve_expr` closure that resolves `ExprId → &HirExpr`
    /// so that `HirExpr::Call` callee expressions can be walked, marking
    /// callee variables as used.
    pub fn analyze<F>(f: &HirFn, interner: &Interner, resolve_expr: F) -> ScopeAnalysisResult
    where
        F: Fn(ExprId) -> Option<&HirExpr>,
    {
        let mut analyzer = ScopeAnalyzer {
            scope_stack: vec![Scope::new()],
            completed: Vec::new(),
            interner,
            resolve_expr: &resolve_expr,
        };

        for (i, (sym, _ty)) in f.params.iter().enumerate() {
            let mutable = f.param_mutability.get(i).copied().unwrap_or(false);
            analyzer.scope_stack[0].declare(VarState {
                name: *sym,
                decl_span: f.span,
                name_span: f.span,
                is_mutable: mutable,
                is_used: false,
                is_assigned: false,
            });
        }

        analyzer.walk_expr(&f.body);

        let mut all_scopes = analyzer.completed;
        all_scopes.extend(analyzer.scope_stack);

        let mut unused_vars = Vec::new();
        let mut unnecessary_muts = Vec::new();

        for scope in &all_scopes {
            for var in scope.vars.values() {
                let name_str = interner.resolve(var.name);
                if !var.is_used && !name_str.starts_with('_') {
                    unused_vars.push(VarInfo {
                        name: var.name, decl_span: var.decl_span,
                        name_span: var.name_span, is_mutable: var.is_mutable,
                    });
                }
                if var.is_mutable && !var.is_assigned {
                    unnecessary_muts.push(VarInfo {
                        name: var.name, decl_span: var.decl_span,
                        name_span: var.name_span, is_mutable: var.is_mutable,
                    });
                }
            }
        }

        ScopeAnalysisResult { unused_vars, unnecessary_muts }
    }
}

struct ScopeAnalyzer<'a, F> {
    scope_stack: Vec<Scope>,
    completed: Vec<Scope>,
    interner: &'a Interner,
    resolve_expr: &'a F,
}

impl<'a, F> ScopeAnalyzer<'a, F>
where
    F: Fn(ExprId) -> Option<&HirExpr>,
{
    fn mark_used(&mut self, name: Symbol) {
        for scope in self.scope_stack.iter_mut().rev() {
            if scope.mark_used(name) { return; }
        }
    }

    fn mark_assigned(&mut self, name: Symbol) {
        for scope in self.scope_stack.iter_mut().rev() {
            if scope.mark_assigned(name) { return; }
        }
    }

    fn push_scope(&mut self) { self.scope_stack.push(Scope::new()); }

    fn pop_scope(&mut self) {
        if self.scope_stack.len() > 1 {
            if let Some(scope) = self.scope_stack.pop() {
                self.completed.push(scope);
            }
        }
    }

    fn walk_expr(&mut self, expr: &HirExpr) {
        match expr {
            HirExpr::Ident { name, .. } => { self.mark_used(*name); }
            HirExpr::Block { stmts, .. } => {
                self.push_scope();
                for stmt in stmts { self.walk_stmt(stmt); }
                self.pop_scope();
            }
            HirExpr::If { condition, then_branch, else_branch, .. } => {
                self.walk_expr(condition);
                self.push_scope(); self.walk_expr(then_branch); self.pop_scope();
                if let Some(e) = else_branch { self.push_scope(); self.walk_expr(e); self.pop_scope(); }
            }
            HirExpr::Match { scrutinee, arms, .. } => {
                self.walk_expr(scrutinee);
                for arm in arms {
                    self.push_scope();
                    if let Some(scope) = self.scope_stack.last_mut() {
                        Self::collect_pattern_decls(&arm.pattern, false, arm.body.span(), scope);
                    }
                    if let Some(g) = &arm.guard { self.walk_expr(g); }
                    self.walk_expr(&arm.body);
                    self.pop_scope();
                }
            }
            HirExpr::While { condition, body, .. } => {
                self.walk_expr(condition);
                self.push_scope(); self.walk_expr(body); self.pop_scope();
            }
            HirExpr::ForIn { pattern, iter, body, .. } => {
                self.walk_expr(iter);
                self.push_scope();
                if let Some(scope) = self.scope_stack.last_mut() {
                    // For-in bindings default to false mutability
                    Self::collect_pattern_decls(pattern, false, iter.span(), scope);
                }
                self.walk_expr(body);
                self.pop_scope();
            }
            HirExpr::Binary { lhs, rhs, .. } => { self.walk_expr(lhs); self.walk_expr(rhs); }
            HirExpr::Unary { operand, .. } => { self.walk_expr(operand); }
            // v6 FIX: Walk the callee expression via resolve_expr
            HirExpr::Call { callee, args, .. } => {
                if let Some(callee_expr) = (self.resolve_expr)(*callee) {
                    self.walk_expr(callee_expr);
                }
                for a in args { self.walk_expr(a); }
            }
            HirExpr::MethodCall { receiver, args, .. } => {
                self.walk_expr(receiver);
                for a in args { self.walk_expr(a); }
            }
            HirExpr::StructLit { fields, .. } => { for (_, v) in fields { self.walk_expr(v); } }
            HirExpr::EnumVariant { args, .. } | HirExpr::TupleLit { elements: args, .. } => { for a in args { self.walk_expr(a); } }
            HirExpr::Return { value: Some(v), .. } => self.walk_expr(v),
            HirExpr::Assert { condition, message, .. } => {
                self.walk_expr(condition);
                if let Some(m) = message { self.walk_expr(m); }
            }
            HirExpr::Println { arg, .. } => self.walk_expr(arg),
            HirExpr::As { expr, .. } => self.walk_expr(expr),
            HirExpr::Deref { expr, .. } => self.walk_expr(expr),
            HirExpr::AddrOf { target, .. } => self.walk_expr(target),
            HirExpr::FieldAccess { object, .. } => self.walk_expr(object),
            HirExpr::IntLit { .. } | HirExpr::FloatLit { .. } | HirExpr::BoolLit { .. }
            | HirExpr::StrLit { .. } | HirExpr::UnitLit { .. } | HirExpr::Return { value: None, .. }
            | HirExpr::SizeOf { .. } => {}
        }
    }

    fn walk_stmt(&mut self, stmt: &HirStmt) {
        match stmt {
            HirStmt::Let { name, mutable, value, span, .. } => {
                if let Some(scope) = self.scope_stack.last_mut() {
                    scope.declare(VarState { name: *name, decl_span: *span, name_span: *span, is_mutable: *mutable, is_used: false, is_assigned: false });
                }
                self.walk_expr(value);
            }
            HirStmt::LetPat { pattern, mutable, value, span, .. } => {
                if let Some(scope) = self.scope_stack.last_mut() {
                    Self::collect_pattern_decls(pattern, *mutable, *span, scope);
                }
                self.walk_expr(value);
            }
            // v6 FIX: target is Box<HirExpr>, not Symbol. Extract ident name
            // for assignment tracking, AND walk the target for complex expressions.
            HirStmt::Assign { target, value, .. } => {
                if let HirExpr::Ident { name, .. } = target.as_ref() {
                    self.mark_assigned(*name);
                }
                self.walk_expr(target);
                self.walk_expr(value);
            }
            HirStmt::AssignDeref { target, value, .. } => { self.walk_expr(target); self.walk_expr(value); }
            HirStmt::AssignField { object, value, .. } => { self.walk_expr(object); self.walk_expr(value); }
            HirStmt::Expr(e) => { self.walk_expr(e); }
        }
    }

    fn collect_pattern_decls(pat: &HirPattern, mutable: bool, span: Span, scope: &mut Scope) {
        match pat {
            HirPattern::Var(sym) => {
                scope.declare(VarState { name: *sym, decl_span: span, name_span: span, is_mutable: mutable, is_used: false, is_assigned: false });
            }
            HirPattern::Tuple { elements, .. } => { for e in elements { Self::collect_pattern_decls(e, mutable, span, scope); } }
            HirPattern::Struct { bindings, .. } => { for (_, p) in bindings { Self::collect_pattern_decls(p, mutable, span, scope); } }
            HirPattern::EnumVariant { bindings, .. } => { for (_, p) in bindings { Self::collect_pattern_decls(p, mutable, span, scope); } }
            HirPattern::OptionSome(inner) => { Self::collect_pattern_decls(inner, mutable, span, scope); }
            HirPattern::ResultOk(inner) | HirPattern::ResultErr(inner) => { Self::collect_pattern_decls(inner, mutable, span, scope); }
            _ => {}
        }
    }
}
```

```rust
/// Memoized type queries with O(1) cycle detection via HashSet.
pub struct TypeQueryCache {
    is_copy_cache: HashMap<HirType, bool>,
    size_cache: HashMap<HirType, usize>,
    copy_computing: HashSet<HirType>,
    size_computing: HashSet<HirType>,
}

impl TypeQueryCache {
    pub fn new() -> Self {
        Self {
            is_copy_cache: HashMap::new(), size_cache: HashMap::new(),
            copy_computing: HashSet::new(), size_computing: HashSet::new(),
        }
    }

    pub fn is_copy(&mut self, ty: &HirType, index: &HirIndex) -> bool {
        if let Some(result) = self.is_copy_cache.get(ty) { return *result; }
        if self.copy_computing.contains(ty) { return false; }
        self.copy_computing.insert(ty.clone());
        let result = self.compute_is_copy(ty, index);
        self.copy_computing.remove(ty);
        self.is_copy_cache.insert(ty.clone(), result);
        result
    }

    fn compute_is_copy(&mut self, ty: &HirType, index: &HirIndex) -> bool {
        match ty {
            HirType::Int | HirType::Float | HirType::Bool | HirType::Unit | HirType::Never => true,
            HirType::Tuple(elems) => elems.iter().all(|e| self.is_copy(e, index)),
            HirType::Option(inner) => self.is_copy(inner, index),
            HirType::Result(ok, err) => self.is_copy(ok, index) && self.is_copy(err, index),
            HirType::RawPtr(_) | HirType::Func(_, _) => true,
            HirType::Opaque(_) | HirType::Str | HirType::Error => false,
            HirType::Named(sym) | HirType::Generic(sym, _) => {
                index.find_struct(*sym).map(|s| s.fields.iter().all(|f| self.is_copy(&f.ty, index))).unwrap_or(false)
            }
        }
    }

    pub fn estimated_size(&mut self, ty: &HirType, index: &HirIndex) -> usize {
        if let Some(result) = self.size_cache.get(ty) { return *result; }
        if self.size_computing.contains(ty) { return 8; }
        self.size_computing.insert(ty.clone());
        let result = self.compute_estimated_size(ty, index);
        self.size_computing.remove(ty);
        self.size_cache.insert(ty.clone(), result);
        result
    }

    fn compute_estimated_size(&mut self, ty: &HirType, index: &HirIndex) -> usize {
        match ty {
            HirType::Int | HirType::Float => 8,
            HirType::Bool => 1,
            HirType::Unit | HirType::Never | HirType::Error => 0,
            HirType::Str => 24,
            HirType::Tuple(elems) => elems.iter().map(|e| self.estimated_size(e, index)).sum(),
            HirType::Option(inner) => 1 + self.estimated_size(inner, index),
            HirType::Result(ok, err) => 1 + std::cmp::max(self.estimated_size(ok, index), self.estimated_size(err, index)),
            HirType::RawPtr(_) | HirType::Func(_, _) | HirType::Opaque(_) => 8,
            HirType::Named(sym) | HirType::Generic(sym, _) => {
                index.find_struct(*sym).map(|s| s.fields.iter().map(|f| self.estimated_size(&f.ty, index)).sum()).unwrap_or(8)
            }
        }
    }

    pub fn is_ref_like(&self, ty: &HirType) -> bool {
        matches!(ty, HirType::RawPtr(_))
    }

    pub fn has_is_empty(&self, ty: &HirType, known: &KnownSymbols, interner: &Interner) -> bool {
        match ty {
            HirType::Str | HirType::Option(_) => true,
            HirType::Generic(sym, _) | HirType::Named(sym) => {
                matches!(interner.resolve(*sym), "Vec" | "HashMap" | "HashSet" | "LinkedList" | "VecDeque" | "BTreeMap" | "BTreeSet")
            }
            _ => false,
        }
    }

    pub fn is_unsigned_int(&self, ty: &HirType, known: &KnownSymbols) -> bool {
        match ty {
            HirType::Named(sym) | HirType::Generic(sym, _) => {
                *sym == known.uint8 || *sym == known.uint16 || *sym == known.uint32 || *sym == known.uint64 || *sym == known.usize
            }
            _ => false,
        }
    }

    pub fn is_signed_int(&self, ty: &HirType, known: &KnownSymbols) -> bool {
        match ty {
            HirType::Int => true,
            HirType::Named(sym) | HirType::Generic(sym, _) => {
                *sym == known.int8 || *sym == known.int16 || *sym == known.int32 || *sym == known.int64 || *sym == known.isize
            }
            _ => false,
        }
    }

    match ty {
        HirType::Int => "i64".into(),
        HirType::Bool => "bool".into(),
        HirType::Float => "f64".into(),
        HirType::Str => "Str".into(),

        HirType::Named(sym) => interner.resolve(*sym).to_string(),
        HirType::Generic(sym, args) => {
            let a: Vec<String> = args.iter().map(|a| format_type_for_display(a, interner)).collect();

        }
        HirType::RawPtr(inner) => format!("*{}", format_type_for_display(inner, interner)),
        HirType::Opaque(sym) => format!("@{}", interner.resolve(*sym)),
        HirType::Option(inner) => format!("Option<{}>", format_type_for_display(inner, interner)),
        HirType::Result(ok, err) => format!("Result<{}, {}>", format_type_for_display(ok, interner), format_type_for_display(err, interner)),
        HirType::Func(params, ret) => {
            let p: Vec<String> = params.iter().map(|a| format_type_for_display(a, interner)).collect();
            format!("fn({}) -> {}", p.join(", "), format_type_for_display(ret, interner))
        }
        HirType::Tuple(elems) => {
            let e: Vec<String> = elems.iter().map(|a| format_type_for_display(a, interner)).collect();
            format!("({})", e.join(", "))
        }
    }
```

```rust
//! Lint context: shared state available to all lint passes.
//!
//! Three responsibilities:
//! 1. Type queries — resolve expression types, check Copy/size
//! 2. Diagnostic emission — single code path for severity resolution
//! 3. Traversal state — current function, scope depth, etc.
//!
//! # Emission API boundary
//!
//! - `emit_core()` is private — no external access.
//! - `emit_lint()` / `emit_with_fix()` are the public API.
//! - `emit_with_custom_severity()` is `pub(crate)` for documented
//!   escape hatches (e.g., unused_result's conditional severity).
//!
//! # Forbid enforcement
//!
//! `emit_core` checks for `Forbid` level BEFORE checking suppressions.
//! A Forbid-level lint cannot be suppressed by `#[allow(...)]` annotations.

use crate::callee::CalleeResolver;
use crate::config::LintConfig;
use crate::diagnostic::{

    Replacement,

use glyim_interner::{Interner, Symbol};
use std::collections::{HashMap, HashSet};

    diagnostics: Vec<LintDiagnostic>,
    stats: DiagnosticStats,
    current_fn: Option<&'a HirFn>,
    scope_depth: usize,
    is_last_stmt: bool,
    scope_cache: HashMap<(Symbol, Span), ScopeAnalysisResult>,

        env: &'a LintEnvironment<'a>,
        callee_resolver: CalleeResolver<'a>,
    ) -> Self {
        Self {
            type_check, query_cache: TypeQueryCache::new(), env, callee_resolver,
            diagnostics: Vec::new(), stats: DiagnosticStats::default(),

    pub fn callee_resolver(&self) -> &CalleeResolver<'a> { &self.callee_resolver }

    /// Get or compute scope analysis. Returns an **owned** result to avoid
    /// borrow conflicts when the caller subsequently calls emit().
    /// The clone is cheap for typical function bodies.
    ///
    /// Passes `resolve_expr` closure to ScopeAnalysis so it can walk
    /// `HirExpr::Call` callee expressions and mark callee variables as used.
    pub fn scope_analysis(&mut self, f: &HirFn) -> ScopeAnalysisResult {
        let key = (f.name, f.span);
        if !self.scope_cache.contains_key(&key) {
            let resolver = &self.callee_resolver;
            let result = ScopeAnalysis::analyze(f, self.type_check.interner, |id| resolver.find_expr(id));
            self.scope_cache.insert(key, result);
        }
        self.scope_cache.get(&key).expect("just inserted").clone()
    }

    // ═══════════════════════════════════════════════════════════
    //  Diagnostic emission
    // ═══════════════════════════════════════════════════════════

    /// Core emission — private. Single severity resolution path.
    ///
    /// # Forbid enforcement
    ///
    /// Checks for `Forbid` level BEFORE checking suppressions.
    /// A `Forbid`-level lint always emits regardless of `#[allow(...)]`.
    fn emit_core(&mut self, mut diag: LintDiagnostic) {
        let config_level = self.env.config.effective_level(diag.lint_id, diag.group);

        // Forbid enforcement: Forbid lints bypass suppressions and cannot
        // be downgraded by config. They always emit as Severity::Error.
        if matches!(config_level, Some(LintLevel::Forbid)) {
            diag.severity = Severity::Error;
            self.stats.emitted += 1;
            self.diagnostics.push(diag);
            return;
        }

        // Check suppressions AFTER Forbid check
        if self.env.suppressions.is_suppressed(diag.lint_id, diag.span) {
            self.stats.suppressed += 1;
            return;
        }

        // Resolve severity from config
        match config_level {
            Some(LintLevel::Allow) => {
                self.stats.allowed += 1;
                return;
            }
            Some(level) => {
                // Warn or Deny — use the config-specified severity
                match level.to_severity() {
                    Some(sev) => diag.severity = sev,
                    None => {
                        self.stats.allowed += 1;
                        return;
                    }
                }
            }
            None => {
                // No config override — keep the lint's original severity
            }
        }

        self.stats.emitted += 1;
        self.diagnostics.push(diag);
    }

    /// Emit a simple diagnostic with no suggestion.
    pub fn emit_lint(&mut self, lint_id: LintId, span: Span, message: impl Into<String>, group: LintGroup) {
        let severity = self.resolve_default_severity(lint_id, group);
        self.emit_core(LintDiagnostic {
            lint_id, severity, span, message: message.into(),
            labels: Vec::new(), suggestion: None, note: None, group,
        });
    }

    /// Emit with a suggestion.
    ///
    /// # Applicability contract
    ///
    /// - `MachineApplicable`: `replacement_text` MUST be syntactically valid
    ///   source code that produces the correct semantics.
    /// - `Unspecified`: `replacement_text` may be empty. The diagnostic's
    ///   note explains the required manual change.
    pub fn emit_with_fix(
        &mut self, lint_id: LintId, span: Span, message: impl Into<String>, group: LintGroup,
        fix_span: Span, replacement_text: impl Into<String>, label: impl Into<String>, applicability: Applicability,
    ) {
        let severity = self.resolve_default_severity(lint_id, group);
        self.emit_core(LintDiagnostic {
            lint_id, severity, span, message: message.into(),
            labels: vec![(fix_span, label.as_ref().to_string())],
            suggestion: Some(LintSuggestion {
                replacements: vec![Replacement { span: fix_span, new_text: replacement_text.into() }],
                label: label.into(), applicability,
            }),
            note: None, group,
        });
    }

    /// Emit with a custom severity override. `pub(crate)` — for documented
    /// escape hatches where a lint needs conditional severity (e.g., unused_result).
    pub(crate) fn emit_with_custom_severity(&mut self, diag: LintDiagnostic) {
        self.emit_core(diag);
    }

    fn resolve_default_severity(&self, lint_id: LintId, group: LintGroup) -> Severity {
        self.env.registry.get(lint_id)
            .and_then(|d| d.default_level.to_severity())
            .unwrap_or(Severity::Warning)
    }

    pub fn into_results(self) -> (Vec<LintDiagnostic>, DiagnosticStats) {
        (self.diagnostics, self.stats)
    }

    // ═══════════════════════════════════════════════════════════
    //  Formatting helpers
    // ═══════════════════════════════════════════════════════════

    pub fn resolve(&self, sym: Symbol) -> &str { self.type_check.interner.resolve(sym) }
    pub fn format_type(&self, ty: &HirType) -> String { format_type_for_display(ty, self.type_check.interner) }
```

## Chunk 11: LintDriver

### Task 11.1: driver.rs

```rust
//! Lint driver: walks the HIR once, dispatching to all registered passes.
//!
//! # Why custom traversal
//!
//! LintDriver cannot use the shared `walk` module because it requires:
//! - Per-pass dispatch via `LateLintPass` trait
//! - `is_last_stmt` tracking for the driver's block traversal
//! - `scope_depth` tracking for `is_function_body_block`
//! - Callee walking via CalleeResolver (resolves ExprId → &HirExpr)
//!
//! # File path correlation
//!
//! `run()` takes `file_path` for tracing correlation in multi-file runs.
//!
//! # Timing
//!
//! Only total run timing is measured. Per-pass timing was removed because
//! the driver dispatches `check_expr`/`check_stmt` for ALL passes in a
//! single loop, making per-pass accumulation impractical without changing
//! the architecture. Total timing is sufficient for production observability.

use crate::callee::CalleeResolver;
use crate::context::LintContext;
use crate::diagnostic::{DiagnosticStats, LintDiagnostic, LintId};
use crate::pass::LateLintPass;
use crate::type_query::{LintEnvironment, TypeCheckBorrow};
use glyim_hir::item::HirItem;
use glyim_hir::node::{HirExpr, HirStmt};
use glyim_hir::types::HirPattern;

struct RegisteredPass {
    pass: Box<dyn LateLintPass>,
    lint_id: LintId,
}

pub struct LintDriver {
    passes: Vec<RegisteredPass>,
}

impl Default for LintDriver {
    fn default() -> Self { Self::new() }
}

impl LintDriver {
    pub fn new() -> Self { Self { passes: Vec::new() } }

    pub fn register_pass(&mut self, pass: Box<dyn LateLintPass>, lint_id: LintId) {
        tracing::debug!(lint_id = lint_id.as_str(), "registering lint pass");
        self.passes.push(RegisteredPass { pass, lint_id });
    }

    pub fn register_builtins(&mut self) {
        crate::lints::register_all(self);
    }

    /// Run all lints on the HIR.
    pub fn run<'a>(
        &mut self,
        type_check: &'a TypeCheckBorrow<'a>,
        env: &'a LintEnvironment<'a>,
        file_path: &str,
    ) -> (Vec<LintDiagnostic>, DiagnosticStats) {
        let span = tracing::info_span!("lint_run", file = file_path);
        let _enter = span.enter();

        let start = std::time::Instant::now();
        let callee_resolver = CalleeResolver::build(type_check.hir);
        let mut ctx = LintContext::new(type_check, env, callee_resolver);

        for item in &type_check.hir.items {
            for rp in &mut self.passes {
                rp.pass.check_item(&mut ctx, item);
            }

            match item {
                HirItem::Fn(f) => {
                    ctx.set_current_fn(f);
                    for rp in &mut self.passes {
                        rp.pass.check_fn(&mut ctx, f);
                    }
                    self.walk_expr(&mut ctx, &f.body);
                    ctx.clear_current_fn();
                }
                HirItem::Impl(imp) => {
                    for m in &imp.methods {
                        ctx.set_current_fn(m);
                        for rp in &mut self.passes {
                            rp.pass.check_fn(&mut ctx, m);
                        }
                        self.walk_expr(&mut ctx, &m.body);
                        ctx.clear_current_fn();
                    }
                }
                _ => {}
            }
        }

        for rp in &mut self.passes {
            rp.pass.check_crate_post(&mut ctx);
        }

        let elapsed = start.elapsed();
        let (diagnostics, stats) = ctx.into_results();

        tracing::info!(
            duration_ms = elapsed.as_millis() as u64,
            num_diagnostics = diagnostics.len(),
            stats = %stats,
            "lint run complete"
        );

        (diagnostics, stats)
    }

    fn walk_expr(&mut self, ctx: &mut LintContext, expr: &HirExpr) {
        for rp in &mut self.passes { rp.pass.check_expr(ctx, expr); }

        match expr {
            HirExpr::Binary { lhs, rhs, .. } => { self.walk_expr(ctx, lhs); self.walk_expr(ctx, rhs); }
            HirExpr::Unary { operand, .. } => self.walk_expr(ctx, operand),
            HirExpr::Block { stmts, .. } => {
                ctx.enter_scope();
                for (i, stmt) in stmts.iter().enumerate() {
                    ctx.set_last_stmt(i == stmts.len() - 1);
                    self.walk_stmt(ctx, stmt);
                }
                ctx.set_last_stmt(false);
                ctx.leave_scope();
            }
            HirExpr::If { condition, then_branch, else_branch, .. } => {
                self.walk_expr(ctx, condition);
                self.walk_expr(ctx, then_branch);
                if let Some(e) = else_branch { self.walk_expr(ctx, e); }
            }
            HirExpr::Match { scrutinee, arms, .. } => {
                self.walk_expr(ctx, scrutinee);
                for arm in arms {
                    if let Some(g) = &arm.guard { self.walk_expr(ctx, g); }
                    for rp in &mut self.passes { rp.pass.check_pattern(ctx, &arm.pattern); }
                    self.walk_expr(ctx, &arm.body);
                }
            }
            // Walk callee via CalleeResolver
            HirExpr::Call { callee, args, .. } => {
                if let Some(callee_expr) = ctx.callee_resolver().find_expr(*callee) {
                    self.walk_expr(ctx, callee_expr);
                }
                for a in args { self.walk_expr(ctx, a); }
            }
            HirExpr::MethodCall { receiver, args, .. } => {
                self.walk_expr(ctx, receiver);
                for a in args { self.walk_expr(ctx, a); }
            }
            HirExpr::StructLit { fields, .. } => { for (_, v) in fields { self.walk_expr(ctx, v); } }
            HirExpr::EnumVariant { args, .. } | HirExpr::TupleLit { elements: args, .. } => {
                for a in args { self.walk_expr(ctx, a); }
            }
            HirExpr::ForIn { pattern, iter, body, .. } => {
                self.walk_expr(ctx, iter);
                for rp in &mut self.passes { rp.pass.check_pattern(ctx, pattern); }
                self.walk_expr(ctx, body);
            }
            HirExpr::While { condition, body, .. } => { self.walk_expr(ctx, condition); self.walk_expr(ctx, body); }
            HirExpr::As { expr, .. } => self.walk_expr(ctx, expr),
            HirExpr::Deref { expr, .. } => self.walk_expr(ctx, expr),
            HirExpr::AddrOf { target, .. } => self.walk_expr(ctx, target),
            HirExpr::Return { value, .. } => { if let Some(v) = value { self.walk_expr(ctx, v); } }
            HirExpr::Assert { condition, message, .. } => {
                self.walk_expr(ctx, condition);
                if let Some(m) = message { self.walk_expr(ctx, m); }
            }
            HirExpr::Println { arg, .. } => self.walk_expr(ctx, arg),
            HirExpr::FieldAccess { object, .. } => self.walk_expr(ctx, object),
            HirExpr::IntLit { .. } | HirExpr::FloatLit { .. } | HirExpr::BoolLit { .. }
            | HirExpr::StrLit { .. } | HirExpr::UnitLit { .. } | HirExpr::Ident { .. }
            | HirExpr::SizeOf { .. } => {}
        }

        for rp in &mut self.passes { rp.pass.check_expr_post(ctx, expr); }
    }

    fn walk_stmt(&mut self, ctx: &mut LintContext, stmt: &HirStmt) {
        for rp in &mut self.passes { rp.pass.check_stmt(ctx, stmt); }

        match stmt {
            HirStmt::Let { value, .. } => self.walk_expr(ctx, value),
            HirStmt::LetPat { pattern, value, .. } => {
                for rp in &mut self.passes { rp.pass.check_pattern(ctx, pattern); }
                self.walk_expr(ctx, value);
            }
            // v6 FIX: Walk target (Box<HirExpr>) for complex assignments
            HirStmt::Assign { target, value, .. } => {
                self.walk_expr(ctx, target);
                self.walk_expr(ctx, value);
            }
            HirStmt::AssignDeref { target, value, .. } => { self.walk_expr(ctx, target); self.walk_expr(ctx, value); }
            HirStmt::AssignField { object, value, .. } => { self.walk_expr(ctx, object); self.walk_expr(ctx, value); }
            HirStmt::Expr(e) => self.walk_expr(ctx, e),
        }
    }
}
```

```rust
//! Unused function detection via CalleeResolver's call graph.
//!
//! Uses `compute_reachable` from CalleeResolver which does BFS from
//! entry points through the pre-built call graph.
//!
//! Checks both top-level `HirItem::Fn` and impl methods.

use crate::context::LintContext;
use crate::diagnostic::{LintGroup, LintLevel};
use crate::lints;
use crate::pass::LateLintPass;
use crate::registry::LintDescriptor;
use glyim_hir::item::HirItem;

pub struct UnusedFunctionLint;

impl LateLintPass for UnusedFunctionLint {
    fn descriptor() -> LintDescriptor {
        LintDescriptor {
            id: lints::UNUSED_FUNCTION,
            name: "Unused Function",
            description: "Function is defined but never called",
            default_level: LintLevel::Warn,
            group: LintGroup::Correctness,
        }
    }

    fn check_crate_post(&mut self, ctx: &mut LintContext) {
        let hir = ctx.hir();
        let reachable = ctx.callee_resolver().compute_reachable(hir, &|f| {
            let name = ctx.resolve(f.name);
            name == "main" || f.is_test || f.is_extern_backed || f.is_pub
        });

        for item in &hir.items {
            if let HirItem::Fn(f) = item {
                let name = ctx.resolve(f.name);
                if name == "main" || f.is_test || f.is_extern_backed || f.is_pub {
                    continue;
                }
                if !reachable.contains(&f.name) {
                    tracing::debug!(function = name, "unused function detected");
                    ctx.emit_lint(
                        lints::UNUSED_FUNCTION,
                        f.span,
                        format!("function `{}` is never called", name),
                        LintGroup::Correctness,
                    );
                }
            }

            // Also check impl methods
            if let HirItem::Impl(imp) = item {
                for m in &imp.methods {
                    if m.is_pub || m.is_test || m.is_extern_backed {
                        continue;
                    }
                    if !reachable.contains(&m.name) {
                        let name = ctx.resolve(m.name);
                        tracing::debug!(method = name, "unused method detected");
                        ctx.emit_lint(
                            lints::UNUSED_FUNCTION,
                            m.span,
                            format!("method `{}` is never called", name),
                            LintGroup::Correctness,
                        );
                    }
                }
            }
        }
    }
}

impl Default for UnusedFunctionLint {
    fn default() -> Self { Self }
}
```

### Task 13.3: dead_code.rs

```rust
//! Dead code detection — reports the range of unreachable statements.
//!
//! Instead of only pointing at the FIRST unreachable statement, this
//! lint finds the span from the first unreachable statement to the
//! end of the block, emitting a single diagnostic.

use crate::context::LintContext;
use crate::diagnostic::{LintGroup, LintLevel};
use crate::lints;
use crate::pass::LateLintPass;
use crate::registry::LintDescriptor;
use glyim_diag::Span;
use glyim_hir::node::{HirExpr, HirStmt};

pub struct DeadCodeLint;

impl LateLintPass for DeadCodeLint {
    fn descriptor() -> LintDescriptor {
        LintDescriptor {
            id: lints::DEAD_CODE,
            name: "Dead Code",
            description: "Code is unreachable after a return/break/continue",
            default_level: LintLevel::Warn,
            group: LintGroup::Correctness,
        }
    }

    fn check_expr(&mut self, ctx: &mut LintContext, expr: &HirExpr) {
        if let HirExpr::Block { stmts, .. } = expr {
            let mut found_terminal_idx: Option<usize> = None;

            for (i, stmt) in stmts.iter().enumerate() {
                if found_terminal_idx.is_none() && is_terminal_stmt(stmt) {
                    found_terminal_idx = Some(i);
                }
            }

            if let Some(term_idx) = found_terminal_idx {
                if term_idx + 1 < stmts.len() {
                    let dead_start = stmt_span(&stmts[term_idx + 1]).start;
                    let dead_end = stmt_span(&stmts[stmts.len() - 1]).end;
                    let dead_span = Span::new(dead_start, dead_end);

                    tracing::debug!(span_start = dead_span.start, "dead code after terminal statement");
                    ctx.emit_lint(
                        lints::DEAD_CODE,
                        dead_span,
                        "unreachable code",
                        LintGroup::Correctness,
                    );
                }
            }
        }
    }
}

impl Default for DeadCodeLint {
    fn default() -> Self { Self }
}

fn stmt_span(stmt: &HirStmt) -> Span {
    match stmt {
        HirStmt::Let { span, .. }
        | HirStmt::LetPat { span, .. }
        | HirStmt::Assign { span, .. }
        | HirStmt::AssignDeref { span, .. }
        | HirStmt::AssignField { span, .. } => *span,
        HirStmt::Expr(e) => e.span(),
    }
}

fn is_terminal_stmt(stmt: &HirStmt) -> bool {
    match stmt {
        HirStmt::Expr(e) => is_terminal_expr(e),
        _ => false,
    }
}

fn is_terminal_expr(expr: &HirExpr) -> bool {
    match expr {
        HirExpr::Return { .. } => true,
        HirExpr::Block { stmts, .. } => stmts.iter().any(|s| is_terminal_stmt(s)),
        HirExpr::If { then_branch, else_branch, .. } => {
            is_terminal_expr(then_branch)
                && else_branch.as_ref().map(|e| is_terminal_expr(e)).unwrap_or(false)
        }
        _ => false,
    }
}
```

```rust
        LintDescriptor {
            id: lints::ERASING_OP,
            name: "Erasing Operation",
            description: "Operation always yields zero (e.g., x * 0)",
            default_level: LintLevel::Warn,
            group: LintGroup::Correctness,

        if matches!(op, HirBinOp::Mul) {
            if is_zero_literal(lhs) || is_zero_literal(rhs) {
                tracing::debug!(span_start = span.start, "erasing op: multiplication by zero");
                ctx.emit_lint(

                    *span,
                    "multiplication by zero always yields zero",
                    LintGroup::Correctness,
                );
```

```rust
impl LateLintPass for NonExhaustiveMatchLint {

        LintDescriptor {
            id: lints::NON_EXHAUSTIVE_MATCH,
            name: "Non-Exhaustive Match",
            description: "Match does not cover all enum variants",
            default_level: LintLevel::Deny,
            group: LintGroup::Correctness,
        }
    }

    fn check_expr(&mut self, ctx: &mut LintContext, expr: &HirExpr) {
        let HirExpr::Match { scrutinee, arms, span, .. } = expr else { return };

        let scrutinee_type = ctx.expr_type_cloned(scrutinee.id());

            HirType::Named(sym) | HirType::Generic(sym, _) => *sym,
            _ => return,

            Some(info) => info,
            None => return,
        };

        if arms.iter().any(|arm| matches!(arm.pattern, HirPattern::Wild)) {
            return;
        }

        let mut matched: HashSet<Symbol> = HashSet::new();
        for arm in arms {
            collect_variant_names(&arm.pattern, &mut matched);
        }

        let missing: Vec<String> = enum_info
            .variants
            .iter()
            .filter(|v| !matched.contains(&v.name))
            .map(|v| ctx.resolve(v.name).to_string())
            .collect();

        if !missing.is_empty() {
            tracing::debug!(span_start = span.start, missing = ?missing, "non-exhaustive match");
            ctx.emit_lint(
                lints::NON_EXHAUSTIVE_MATCH,
                *span,
                format!(

                    if missing.len() > 1 { "s" } else { "" },
                    missing.join(", ")
                ),
                LintGroup::Correctness,
            );
        }
    }
}

    fn default() -> Self { Self }

fn collect_variant_names(pat: &HirPattern, names: &mut HashSet<Symbol>) {
    match pat {
        HirPattern::EnumVariant { variant_name, bindings, .. } => {
            names.insert(*variant_name);
            for (_, p) in bindings { collect_variant_names(p, names); }
        }
        HirPattern::Struct { bindings, .. } => {

            for e in elements { collect_variant_names(e, names); }
        }
        HirPattern::OptionSome(inner) => collect_variant_names(inner, names),
        HirPattern::ResultOk(inner) | HirPattern::ResultErr(inner) => {
```

### Task 16.1: cognitive_complexity.rs

```rust
//! Cognitive complexity — custom traversal for nesting-level-aware scoring.
//!
//! # Why custom traversal
//!
//! Cognitive complexity cannot use the shared `walk` module because:
//! - Scoring depends on nesting depth (incremented by if/match/while/for).
//! - `else if` chains score differently from nested `if` blocks.
//! - Boolean operators (`&&`, `||`) add to score without affecting nesting.

use crate::context::LintContext;
use crate::diagnostic::{LintGroup, LintLevel};
use crate::lints;
use crate::pass::LateLintPass;
use crate::registry::LintDescriptor;
use glyim_hir::node::{HirBinOp, HirExpr, HirStmt};
use glyim_hir::HirFn;

pub struct CognitiveComplexityLint;

impl LateLintPass for CognitiveComplexityLint {
    fn descriptor() -> LintDescriptor {
        LintDescriptor {
            id: lints::COGNITIVE_COMPLEXITY,
            name: "Cognitive Complexity",
            description: "Function is too complex",
            default_level: LintLevel::Warn,
            group: LintGroup::Complexity,
        }
    }

    fn check_fn(&mut self, ctx: &mut LintContext, f: &HirFn) {
        let max = ctx.config().thresholds.cognitive_complexity;
        let complexity = compute_complexity(&f.body);

        if complexity > max {
            tracing::debug!(function = ctx.resolve(f.name), complexity, max, "cognitive complexity exceeded");
            ctx.emit_lint(
                lints::COGNITIVE_COMPLEXITY,
                f.span,
                format!(
                    "function `{}` has a cognitive complexity of {} (max: {})",
                    ctx.resolve(f.name), complexity, max
                ),
                LintGroup::Complexity,
            );
        }
    }
}

impl Default for CognitiveComplexityLint {
    fn default() -> Self { Self }
}

fn compute_complexity(expr: &HirExpr) -> u32 {
    let mut score = 0u32;
    walk(expr, 0, &mut score);
    score
}

fn walk(expr: &HirExpr, nesting: u32, score: &mut u32) {
    match expr {
        HirExpr::If { then_branch, else_branch, .. } => {
            *score += 1 + nesting;
            walk(then_branch, nesting + 1, score);
            if let Some(e) = else_branch {
                if let HirExpr::If { .. } = e.as_ref() {
                    *score += 1;
                    walk(e, nesting + 1, score);
                } else {
                    *score += 1 + nesting;
                    walk(e, nesting + 1, score);
                }
            }
        }
        HirExpr::Match { arms, scrutinee, .. } => {
            *score += 1 + nesting;
            walk(scrutinee, nesting + 1, score);
            for arm in arms {
                if let Some(g) = &arm.guard { *score += 1; walk(g, nesting + 1, score); }
                walk(&arm.body, nesting + 1, score);
            }
        }
        HirExpr::While { condition, body, .. } => {
            *score += 1 + nesting;
            walk(condition, nesting + 1, score);
            walk(body, nesting + 1, score);
        }
        HirExpr::ForIn { iter, body, .. } => {
            *score += 1 + nesting;
            walk(iter, nesting + 1, score);
            walk(body, nesting + 1, score);
        }
        HirExpr::Binary { op: HirBinOp::And | HirBinOp::Or, lhs, rhs, .. } => {
            *score += 1;
            walk(lhs, nesting, score);
            walk(rhs, nesting, score);
        }
        HirExpr::Block { stmts, .. } => { for s in stmts { walk_stmt(s, nesting, score); } }
        HirExpr::Unary { operand, .. } => walk(operand, nesting, score),
        HirExpr::Call { args, .. } => { for a in args { walk(a, nesting, score); } }
        HirExpr::MethodCall { receiver, args, .. } => {
            walk(receiver, nesting, score);
            for a in args { walk(a, nesting, score); }
        }
        HirExpr::Return { value: Some(v), .. } => walk(v, nesting, score),
        HirExpr::Assert { condition, message, .. } => {
            walk(condition, nesting, score);
            if let Some(m) = message { walk(m, nesting, score); }
        }
        HirExpr::Println { arg, .. } => walk(arg, nesting, score),
        HirExpr::As { expr, .. } => walk(expr, nesting, score),
        HirExpr::Deref { expr, .. } => walk(expr, nesting, score),
        HirExpr::AddrOf { target, .. } => walk(target, nesting, score),
        HirExpr::FieldAccess { object, .. } => walk(object, nesting, score),
        HirExpr::StructLit { fields, .. } => { for (_, v) in fields { walk(v, nesting, score); } }
        HirExpr::EnumVariant { args, .. } | HirExpr::TupleLit { elements: args, .. } => {
            for a in args { walk(a, nesting, score); }
        }
        HirExpr::Binary { lhs, rhs, .. } => { walk(lhs, nesting, score); walk(rhs, nesting, score); }
        _ => {}
    }
}

fn walk_stmt(stmt: &HirStmt, nesting: u32, score: &mut u32) {
    match stmt {
        HirStmt::Let { value, .. } => walk(value, nesting, score),
        HirStmt::LetPat { value, .. } => walk(value, nesting, score),
        HirStmt::Assign { target, value, .. } => { walk(target, nesting, score); walk(value, nesting, score); }
        HirStmt::AssignDeref { target, value, .. } => { walk(target, nesting, score); walk(value, nesting, score); }
        HirStmt::AssignField { object, value, .. } => { walk(object, nesting, score); walk(value, nesting, score); }
        HirStmt::Expr(e) => walk(e, nesting, score),
    }
}
```

```rust
//! Human-readable diagnostic output with ANSI colors.
//!
//! Color is controlled by the `use_color` parameter. Callers should
//! check both `NO_COLOR` environment variable and `is_terminal()`
//! before passing `true`.

use crate::diagnostic::LintDiagnostic;
use glyim_diag::Severity;
use std::io::Write;

/// Emit diagnostics in human-readable format.
pub fn emit_diagnostics(
    writer: &mut dyn Write,
    diagnostics: &[LintDiagnostic],
    source: &str,
    file_path: &str,
    use_color: bool,
) -> std::io::Result<Summary> {
    let mut errors = 0usize;
    let mut warnings = 0usize;

    for diag in diagnostics {
        let severity_str = match diag.severity {
            Severity::Error => format_colored("error", "31;1", use_color),
            Severity::Warning => format_colored("warning", "33;1", use_color),
            _ => format_colored("note", "34", use_color),
        };

        let (line, col) = offset_to_line_col(source, diag.span.start);

        writeln!(
            writer,
            "{}: {} [{}]",
            severity_str,
            diag.message,
            format_colored(diag.lint_id.as_str(), "2", use_color),
        )?;

        writeln!(
            writer,
            "  {} {}:{}:{}",
            format_colored("┌─", "34", use_color),
            file_path,
            line + 1,
            col + 1,
        )?;

        if let Some(line_text) = source.lines().nth(line) {
            writeln!(writer, "  │ {}", line_text)?;
            let mut underline = String::new();
            for _ in 0..col { underline.push(' '); }
            let span_len = diag.span.end.saturating_sub(diag.span.start).max(1);
            for _ in 0..span_len { underline.push('^'); }
            writeln!(
                writer,
                "  {} {}",
                format_colored(&underline, "33", use_color),
                diag.lint_id.as_str()
            )?;
        }

        for (span, label) in &diag.labels {
            let (lbl_line, lbl_col) = offset_to_line_col(source, span.start);
            writeln!(
                writer,
                "  {} {}:{}: {}",
                format_colored("note", "34", use_color),
                lbl_line + 1,
                lbl_col + 1,
                label,
            )?;
        }

        if let Some(note) = &diag.note {
            writeln!(writer, "  {} {}", format_colored("=", "34", use_color), note)?;
        }

        writeln!(writer)?;

        match diag.severity {
            Severity::Error => errors += 1,
            Severity::Warning => warnings += 1,
            _ => {}
        }
    }

    if !diagnostics.is_empty() {
        writeln!(
            writer,
            "{}: {} error{}, {} warning{}",
            format_colored("glyim-lint", "1", use_color),
            errors,
            if errors != 1 { "s" } else { "" },
            warnings,
            if warnings != 1 { "s" } else { "" },
        )?;
    } else {
        writeln!(
            writer,
            "{} No lint warnings found.",
            format_colored("✓", "32", use_color)
        )?;
    }

    Ok(Summary { errors, warnings })
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct Summary {
    pub errors: usize,
    pub warnings: usize,
}

fn offset_to_line_col(source: &str, offset: usize) -> (usize, usize) {
    let mut line = 0;
    let mut col = 0;
    for (i, ch) in source.char_indices() {
        if i >= offset { break; }
        if ch == '\n' { line += 1; col = 0; } else { col += 1; }
    }
    (line, col)
}

fn format_colored(text: &str, code: &str, use_color: bool) -> String {
    if use_color {
        format!("\x1b[{}m{}\x1b[0m", code, text)
    } else {
        text.to_string()
    }
}
```

```rust
/// Emit diagnostics as JSON with metadata wrapper.
pub fn emit_diagnostics(
    writer: &mut dyn Write,
    diagnostics: &[LintDiagnostic],
    file_path: &str,
) -> std::io::Result<()> {

        lint_id: d.lint_id.as_str().to_string(),
```

```rust
//! End-to-end integration tests for the lint pipeline.
//!
//! Uses `assert!` with descriptive messages for actionable failures.

use glyim_diag::Severity;
use glyim_hir::index::HirIndex;
use glyim_hir::symbols::KnownSymbols;

use glyim_interner::Interner;
use glyim_lint::{

    SuppressionMap, TypeCheckBorrow,
};
use std::collections::{HashMap, HashSet};

/// Run the linter on source code and return all diagnostics.
fn lint_source(source: &str) -> Vec<glyim_lint::LintDiagnostic> {
    lint_source_with_config(source, LintConfig::new(), SuppressionMap::new())
}

fn lint_source_with_config(
    source: &str,
    config: LintConfig,
    suppressions: SuppressionMap,
) -> Vec<glyim_lint::LintDiagnostic> {
    let parse_out = glyim_parse::parse(source);
    assert!(
        parse_out.errors.is_empty(),
        "Parse failed for source:\n{}\nErrors: {:?}",
        source, parse_out.errors
    );
    let mut interner = parse_out.interner;
    let hir = glyim_hir::lower(&parse_out.ast, &mut interner);

    let mut temp_interner = interner.clone();
    let index = HirIndex::build(&hir, &mut temp_interner);
    let known = KnownSymbols::intern_all(&mut temp_interner);

    let expr_types = HashMap::new();
    let call_type_args = HashMap::new();
    let generic_expr_ids = HashSet::new();

    let type_check = TypeCheckBorrow::builder()
        .hir(&hir)
        .interner(&interner)
        .expr_types(&expr_types)
        .call_type_args(&call_type_args)
        .generic_expr_ids(&generic_expr_ids)
        .index(&index)
        .known(&known)
        .build()
        .expect("TypeCheckBorrow build should not fail");

    let registry = LintRegistry::with_all_builtins();
    let env = LintEnvironment::new(&config, &registry, &suppressions);

    let mut driver = LintDriver::new();
    driver.register_builtins();

    let (diags, _stats) = driver.run(&type_check, &env, "test.g");
    diags
}
```

## Detailed Findings

### Correctness & Compilation Safety

- **`lints/mod.rs` — `lint_passes!` macro definition has 4 fragment arms, but invocations provide 3.** The macro is defined as `$($module:path => $id:literal => $CONST:ident => $Lint:ident),*` but the invocations look like `crate::lints::correctness::unused_variable::UnusedVariableLint => "unused_variable" => UNUSED_VARIABLE,`. This provides a path, a literal, and an ident — missing the 4th `$Lint:ident`. **This will fail to compile with a macro expansion error.** 
  *Fix:* Either change the macro to take 3 fragments: `($($module:path => $id:literal => $CONST:ident),* $(,)?)` and use `<$module>::default()` and `<$module as LateLintPass>::descriptor()`, OR revert the invocations to provide the module path and struct name separately: `crate::lints::correctness::unused_variable => "unused_variable" => UNUSED_VARIABLE => UnusedVariableLint`.

- **`scope.rs` — `ScopeAnalyzer<'a, F>` requires `&'a F`, but is passed a local closure.** The struct declares `resolve_expr: &'a F`, but `analyze` receives `resolve_expr: F` by value. When constructing the analyzer with `resolve_expr: &resolve_expr`, you are creating a reference to a local variable. The borrow checker will reject this because the local `resolve_expr` does not live for the lifetime `'a` (which is tied to the `Interner`). **This will not compile.**
  *Fix:* Change `ScopeAnalyzer` to store `F` by value: `resolve_expr: F`. The call site changes from `(self.resolve_expr)(*callee)` to `(self.resolve_expr)(*callee)` or `self.resolve_expr(*callee)`, which works perfectly since `F: Fn(ExprId) -> Option<&HirExpr>`.

- **`cast_sign_loss.rs` — Case 1 is completely subsumed by Case 4.** Check 1 matches `Float → builtin Int`. Check 4 matches `Float → is_signed_int(target)`. Since `is_signed_int` returns `true` for `HirType::Int`, Case 4 catches everything Case 1 catches. Case 1 is dead code. While harmless due to the `else if`, it's sloppy.
  *Fix:* Remove Case 1 entirely, or change Case 4 to `matches!(source_type, HirType::Float) && ctx.is_signed_int(target_type) && !matches!(target_type, HirType::Int)` if you really want the specific `i64` message.

### Boundaries & Contracts

- **Output formatters have incompatible signatures, preventing polymorphism.** `human::emit_diagnostics` takes `source: &str` and returns `io::Result<Summary>`. `json::emit_diagnostics` takes `file_path: &str` and returns `io::Result<()>`. If the CLI or driver ever needs to hold a formatter function pointer or trait object, it cannot. 
  *Fix:* This is a minor boundary issue. If they are only called directly from the CLI dispatcher, it's fine. If you ever want a `Formatter` trait, they must align.

### Modularity & Separation of Concerns

- No new issues. The separation of `ScopeAnalysis`, `CalleeResolver`, and `LintDriver` is clean and well-justified. The closure-based `resolve_expr` injection is an elegant solution to the callee-walking problem.

### Performance & Resource Efficiency

- No new issues. The BFS fix and the cache improvements remain solid.

### Debuggability & Observability

- No new issues. The `Forbid` enforcement, `type_misses` counter, and explicit match statement make the system much easier to reason about.

### Elegance & Hacks

- **The macro mismatch is a hasty edit artifact.** The v5 macro had 5 fragments, the v6 plan reduced it to 4 in the definition but 3 in the invocation. This is a copy-paste/deletion error, not a design error, but it breaks the build.

## Summary of Required Fixes

1. **Fix `lint_passes!` macro** — Align the macro definition with the invocation. Change to `($($module:path => $id:literal => $CONST:ident),* $(,)?)` and use `<$module>` directly for `::default()` and `::descriptor()`.
2. **Fix `ScopeAnalyzer` lifetime bug** — Store `resolve_expr: F` by value in `ScopeAnalyzer`, not `&'a F`, so it can consume the locally injected closure.
3. **Remove dead code in `cast_sign_loss`** — Drop the redundant `Float → Int` explicit check, since `Float → is_signed_int` covers it.
4. **(Optional) Align output formatter signatures** — If polymorphic formatting is ever desired, standardize the parameters and return types.
