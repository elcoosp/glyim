# Monomorphizer v2 & Codegen Soundness — Implementation Plan

> **For agentic workers:** Use `superpowers:executing-plans` to implement. Steps use checkbox syntax for tracking.

**Goal:** Replace the inference-heavy monomorphizer with a demand-driven BFS substitution engine that produces a fully concrete HIR + `expr_types`, then simplify the codegen to consume that concrete output directly — eliminating `type_overrides`, `concretize_type`, string-based fallback lookups, and the `Generic → Named` resolution scattered across the codegen.

**Key Invariant:** The monomorphizer is the **only** component that converts `Generic("Vec", [Int])` → `Named("Vec__i64")`. The typechecker works with logical types. The codegen works with concrete names. No other component performs mangling or `Generic` resolution.

**Constraint:** Do not touch the typechecker.

---

## Architecture

```
                 ┌──────────────┐
                 │  Typechecker  │  (unchanged)
                 └──────┬───────┘
                        │ expr_types (may contain Generic)
                        │ call_type_args
                        ▼
            ┌───────────────────────────────┐
            │       Monomorphizer v2         │
            │                               │
            │  • MonoIndex (O(1) lookups)   │
            │  • WorkQueue (dedup BFS)      │
            │  • MangleTable (deterministic)│
            │  • substitute_expr + renumber │
            │  • concretize (ONLY HERE)     │
            │                               │
            └──────────────┬────────────────┘
                           │ MonoResult {
                           │   hir: Hir,           // no Generic in bodies
                           │   expr_types: Vec,    // all concrete, fresh ExprIds
                           │ }
                           ▼
                     ┌──────────┐
                     │  Codegen  │
                     │           │
                     │  • struct lookup by Named symbol only
                     │  • direct fn call by mangled name
                     │  • no Generic resolution
                     │  • no type_overrides
                     │  • no string-based fallbacks
                     └──────────┘
```

```rust
// crates/glyim-hir/src/monomorphize/index.rs
//! O(1) lookup tables for HIR items, eliminating linear scans.

use crate::item::{EnumDef, HirImplDef, StructDef};
use crate::node::HirFn;
use crate::Hir;
use crate::HirItem;
use glyim_interner::Symbol;
use std::collections::HashMap;

/// Pre-built index over a Hir for O(1) lookups during monomorphization.
pub struct MonoIndex {
    /// Struct definitions indexed by name symbol.
    structs: HashMap<Symbol, StructDef>,
    /// Enum definitions indexed by name symbol.
    enums: HashMap<Symbol, EnumDef>,
    /// Top-level function definitions indexed by name symbol.
    fns: HashMap<Symbol, HirFn>,
    /// Impl methods indexed by (target_type_name, mangled_method_name).
    /// The mangled method name is "TypeName_method".
    impl_methods: HashMap<Symbol, HirFn>,
    /// All impl blocks, indexed by target type name.
    impls: HashMap<Symbol, HirImplDef>,
}

impl MonoIndex {
    /// Build the index by scanning all items in the Hir.
    pub fn build(hir: &Hir) -> Self {
        let mut index = Self {
            structs: HashMap::new(),
            enums: HashMap::new(),
            fns: HashMap::new(),
            impl_methods: HashMap::new(),
            impls: HashMap::new(),
        };

        for item in &hir.items {
            match item {
                HirItem::Struct(s) => {
                    index.structs.insert(s.name, s.clone());
                }
                HirItem::Enum(e) => {
                    index.enums.insert(e.name, e.clone());
                }
                HirItem::Fn(f) => {
                    index.fns.insert(f.name, f.clone());
                }
                HirItem::Impl(imp) => {
                    for m in &imp.methods {
                        // Index by the mangled method name (e.g., "Vec_push")
                        index.impl_methods.insert(m.name, m.clone());
                    }
                    index.impls.insert(imp.target_name, imp.clone());
                }
                _ => {}
            }
        }

        index
    }

    pub fn find_struct(&self, name: Symbol) -> Option<&StructDef> {
        self.structs.get(&name)
    }

    pub fn find_enum(&self, name: Symbol) -> Option<&EnumDef> {
        self.enums.get(&name)
    }

    pub fn find_fn(&self, name: Symbol) -> Option<&HirFn> {
        // First try top-level functions, then impl methods
        self.fns.get(&name).or_else(|| self.impl_methods.get(&name))
    }

    pub fn find_impl(&self, target: Symbol) -> Option<&HirImplDef> {
        self.impls.get(&target)
    }

    /// Return true if the given name refers to a generic item (has type params).
    pub fn is_generic_fn(&self, name: Symbol) -> bool {
        self.find_fn(name).map(|f| !f.type_params.is_empty()).unwrap_or(false)
    }

    pub fn is_generic_struct(&self, name: Symbol) -> bool {
        self.find_struct(name).map(|s| !s.type_params.is_empty()).unwrap_or(false)
    }

    pub fn is_generic_enum(&self, name: Symbol) -> bool {
        self.find_enum(name).map(|e| !e.type_params.is_empty()).unwrap_or(false)
    }

    /// Iterate over all top-level function names.
    pub fn fn_names(&self) -> impl Iterator<Item = Symbol> + '_ {
        self.fns.keys().copied()
    }

    /// Iterate over all struct names.
    pub fn struct_names(&self) -> impl Iterator<Item = Symbol> + '_ {
        self.structs.keys().copied()
    }

    /// Iterate over all enum names.
    pub fn enum_names(&self) -> impl Iterator<Item = Symbol> + '_ {
        self.enums.keys().copied()
    }
}
```

```rust
// Add to crates/glyim-hir/src/monomorphize/mod.rs

use crate::types::HirType;
use glyim_interner::Symbol;
use std::collections::{HashSet, VecDeque};

/// A unit of work for the monomorphizer: specialize an item with concrete type args.
#[derive(Clone, Debug, Hash, Eq, PartialEq)]
pub struct WorkItem {
    pub def_id: Symbol,
    pub kind: ItemKind,
    pub type_args: Vec<HirType>,
}

/// Whether the item is a fresh specialization or a passthrough of a non-generic item.
#[derive(Clone, Copy, Debug, Hash, Eq, PartialEq)]
pub enum ItemKind {
    /// Specialize a generic function with the given type args.
    FnSpecialize,
    /// Walk a non-generic function body to rewrite call targets.
    FnPassthrough,
    /// Specialize a generic struct.
    StructSpecialize,
    /// Passthrough a non-generic struct.
    StructPassthrough,
    /// Specialize a generic enum.
    EnumSpecialize,
    /// Passthrough a non-generic enum.
    EnumPassthrough,
}

impl WorkItem {
    pub fn fn_specialize(id: Symbol, args: Vec<HirType>) -> Self {
        Self { def_id: id, kind: ItemKind::FnSpecialize, type_args: args }
    }
    pub fn fn_passthrough(id: Symbol) -> Self {
        Self { def_id: id, kind: ItemKind::FnPassthrough, type_args: vec![] }
    }
    pub fn struct_specialize(id: Symbol, args: Vec<HirType>) -> Self {
        Self { def_id: id, kind: ItemKind::StructSpecialize, type_args: args }
    }
    pub fn struct_passthrough(id: Symbol) -> Self {
        Self { def_id: id, kind: ItemKind::StructPassthrough, type_args: vec![] }
    }
    pub fn enum_specialize(id: Symbol, args: Vec<HirType>) -> Self {
        Self { def_id: id, kind: ItemKind::EnumSpecialize, type_args: args }
    }
    pub fn enum_passthrough(id: Symbol) -> Self {
        Self { def_id: id, kind: ItemKind::EnumPassthrough, type_args: vec![] }
    }
}

/// Deduplicating BFS work queue.
pub struct WorkQueue {
    queue: VecDeque<WorkItem>,
    seen: HashSet<WorkItem>,
}

impl WorkQueue {
    pub fn new() -> Self {
        Self {
            queue: VecDeque::new(),
            seen: HashSet::new(),
        }
    }

    /// Push an item. If an identical item was already pushed, this is a no-op.
    pub fn push(&mut self, item: WorkItem) {
        if self.seen.insert(item.clone()) {
            self.queue.push_back(item);
        }
    }

    pub fn pop(&mut self) -> Option<WorkItem> {
        self.queue.pop_front()
    }

    pub fn is_empty(&self) -> bool {
        self.queue.is_empty()
    }

    pub fn len(&self) -> usize {
        self.queue.len()
    }
}

impl Default for WorkQueue {
    fn default() -> Self {
        Self::new()
    }
}
```

```rust
// crates/glyim-hir/src/monomorphize/subst.rs
//! Single-pass type substitution and expression walker for monomorphization.
//!
//! This module is the core of the new monomorphizer. It walks HIR expressions,
//! applies type substitutions, rewrites names to mangled forms, discovers new
//! specializations, renumbers ExprIds, and builds a fully concrete expr_types vec.

use crate::node::{HirExpr, HirFn, HirStmt, MatchArm};
use crate::types::{ExprId, HirPattern, HirType};
use crate::HirItem;
use glyim_interner::{Interner, Symbol};
use std::collections::HashMap;

/// Context for a single substitution pass over a function body.
///
/// Holds references to the input data (expr_types, call_type_args, index)
/// and accumulates the output (output_expr_types, work queue discoveries).
pub struct SubstContext<'a> {
    pub interner: &'a mut Interner,
    pub input_expr_types: &'a [HirType],
    pub call_type_args: &'a HashMap<ExprId, Vec<HirType>>,
    pub index: &'a super::index::MonoIndex,
    pub mangle_table: &'a mut super::mangle_table::MangleTable,

    /// ExprId counter for renumbering output expressions.
    next_expr_id: u32,

    /// Concrete expr_types being built, indexed by output ExprId.
    pub output_expr_types: Vec<HirType>,

    /// Work items discovered during this walk (to be enqueued by the caller).
    pub discovered: Vec<super::WorkItem>,
}

impl<'a> SubstContext<'a> {
    pub fn new(
        interner: &'a mut Interner,
        input_expr_types: &'a [HirType],
        call_type_args: &'a HashMap<ExprId, Vec<HirType>>,
        index: &'a super::index::MonoIndex,
        mangle_table: &'a mut super::mangle_table::MangleTable,
    ) -> Self {
        Self {
            interner,
            input_expr_types,
            call_type_args,
            index,
            mangle_table,
            next_expr_id: 0,
            output_expr_types: Vec::new(),
            discovered: Vec::new(),
        }
    }

    /// Allocate a fresh ExprId for the output HIR.
    fn fresh_id(&mut self) -> ExprId {
        let id = ExprId::new(self.next_expr_id);
        self.next_expr_id += 1;
        id
    }

    /// Store a concrete type for the given output ExprId.
    fn store_type(&mut self, id: ExprId, ty: HirType) {
        let idx = id.as_usize();
        if idx >= self.output_expr_types.len() {
            self.output_expr_types.resize(idx + 1, HirType::Error);
        }
        self.output_expr_types[idx] = ty;
    }

    /// Compute the concrete type for an expression, given the original ExprId
    /// and a type substitution map.
    ///
    /// This applies substitution then concretization, which is the ONLY place
    /// where Generic → Named(mangled) conversion happens.
    fn concrete_type_for(
        &mut self,
        original_id: ExprId,
        sub: &HashMap<Symbol, HirType>,
    ) -> HirType {
        let orig_ty = self.input_expr_types.get(original_id.as_usize())
            .cloned()
            .unwrap_or(HirType::Error);
        let substituted = crate::types::substitute_type(&orig_ty, sub);
        self.concretize(substituted)
    }

    /// Convert a type to its fully concrete form.
    ///
    /// - `Generic("Vec", [Int])` → `Named("Vec__i64")`
    /// - `Generic("Option", [Generic("Vec", [Int])])` → `Named("Option_Vec__i64")`
    /// - `Named("Int")` → `Named("Int")` (unchanged)
    /// - Primitives → themselves
    ///
    /// This is the ONLY place where Generic → Named conversion happens.
    /// The codegen never sees Generic types.
    pub fn concretize(&mut self, ty: HirType) -> HirType {
        match ty {
            HirType::Generic(sym, args) => {
                // First, concretize the arguments recursively
                let concrete_args: Vec<HirType> = args.into_iter()
                    .map(|a| self.concretize(a))
                    .collect();

                // Check if this is a known struct or enum
                if self.index.find_struct(sym).is_some() || self.index.find_enum(sym).is_some() {
                    // Mangle the name
                    let mangled = self.mangle_table.mangle(sym, &concrete_args, self.interner);
                    HirType::Named(mangled)
                } else {
                    // Unknown generic — leave as Generic (shouldn't happen in sound output)
                    HirType::Generic(sym, concrete_args)
                }
            }
            HirType::RawPtr(inner) => HirType::RawPtr(Box::new(self.concretize(*inner))),
            HirType::Option(inner) => {
                let concrete_inner = self.concretize(*inner);
                // Check if Option is a known enum
                let opt_sym = self.interner.intern("Option");
                if self.index.find_enum(opt_sym).is_some() {
                    let mangled = self.mangle_table.mangle(opt_sym, &[concrete_inner], self.interner);
                    HirType::Named(mangled)
                } else {
                    HirType::Option(Box::new(concrete_inner))
                }
            }
            HirType::Result(ok, err) => {
                let concrete_ok = self.concretize(*ok);
                let concrete_err = self.concretize(*err);
                let res_sym = self.interner.intern("Result");
                if self.index.find_enum(res_sym).is_some() {
                    let mangled = self.mangle_table.mangle(
                        res_sym,
                        &[concrete_ok, concrete_err],
                        self.interner,
                    );
                    HirType::Named(mangled)
                } else {
                    HirType::Result(Box::new(concrete_ok), Box::new(concrete_err))
                }
            }
            HirType::Tuple(elems) => {
                HirType::Tuple(elems.into_iter().map(|e| self.concretize(e)).collect())
            }
            HirType::Func(params, ret) => HirType::Func(
                params.into_iter().map(|p| self.concretize(p)).collect(),
                Box::new(self.concretize(*ret)),
            )
            // Primitives, Named, Opaque, Never, Error, Bool, Str, Unit, Float, Int
            // are already concrete
            other => other,
        }
    }

    /// Check if a type contains unresolved type parameters (single uppercase letters).
    pub fn has_unresolved_type_param(&self, ty: &HirType) -> bool {
        match ty {
            HirType::Named(sym) => {
                let s = self.interner.resolve(*sym);
                s.len() == 1 && s.chars().next().is_some_and(|c| c.is_uppercase())
            }
            HirType::Generic(_, args) => args.iter().any(|a| self.has_unresolved_type_param(a)),
            HirType::RawPtr(inner) => self.has_unresolved_type_param(inner),
            HirType::Option(inner) => self.has_unresolved_type_param(inner),
            HirType::Result(ok, err) => {
                self.has_unresolved_type_param(ok) || self.has_unresolved_type_param(err)
            }
            HirType::Tuple(elems) => elems.iter().any(|e| self.has_unresolved_type_param(e)),
            _ => false,
        }
    }

    /// Discover and enqueue a type for specialization if it's generic and concrete.
    pub fn discover_type(&mut self, ty: &HirType) {
        match ty {
            HirType::Generic(sym, args) => {
                if !args.iter().any(|a| self.has_unresolved_type_param(a)) {
                    if self.index.find_struct(*sym).is_some() {
                        self.discovered.push(
                            super::WorkItem::struct_specialize(*sym, args.clone())
                        );
                    } else if self.index.find_enum(*sym).is_some() {
                        self.discovered.push(
                            super::WorkItem::enum_specialize(*sym, args.clone())
                        );
                    }
                }
            }
            HirType::Option(inner) => {
                if !self.has_unresolved_type_param(inner) {
                    let opt_sym = self.interner.intern("Option");
                    if self.index.find_enum(opt_sym).is_some() {
                        self.discovered.push(
                            super::WorkItem::enum_specialize(opt_sym, vec![inner.as_ref().clone()])
                        );
                    }
                }
            }

                }
            }
            _ => {}

        args: &[HirType],
    ) -> HashMap<Symbol, HirType> {
```

```rust
    pub fn substitute_expr(
        &mut self,

        let original_id = expr.get_id();
        let new_id = self.fresh_id();

        self.store_type(new_id, concrete_ty.clone());

        &mut self,
```

```rust
        let new_args: Vec<HirExpr> = args.iter()
            .map(|a| self.substitute_expr(a, sub))
            .collect();

        let type_args = self.call_type_args.get(&original_id).cloned().unwrap_or_default();
        let concrete_type_args: Vec<HirType> = type_args.iter()
            .map(|t| self.concretize(crate::types::substitute_type(t, sub)))
            .collect();

        if !concrete_type_args.is_empty()
            && !concrete_type_args.iter().any(|t| self.has_unresolved_type_param(t))
        {
            // We can resolve this method call. Determine the mangled callee.
            // The method name in the index is "Type_method"
            let receiver_ty = self.input_expr_types.get(receiver.get_id().as_usize());
            let type_name = match receiver_ty {
                Some(HirType::Generic(name, _)) => *name,
                Some(HirType::Named(name)) => *name,
                Some(HirType::RawPtr(inner)) => match inner.as_ref() {
                    HirType::Generic(name, _) => *name,
                    HirType::Named(name) => *name,

                        return HirExpr::MethodCall {
                            id: new_id,
                            receiver: new_receiver,
                            method_name,
                            resolved_callee: None,
                            args: new_args,

                        id: new_id,
                        receiver: new_receiver,
                        method_name,
                        resolved_callee: None,
                        args: new_args,
                        span,
                    };

            let method_str = self.interner.resolve(method_name);
            let type_str = self.interner.resolve(type_name);
```

```rust
use crate::node::HirFn;
use crate::types::HirType;
use crate::HirItem;
use glyim_interner::Interner;
use glyim_interner::Symbol;
use std::collections::HashMap;

/// The new monomorphizer context. Drives BFS specialization and produces
/// a self-contained output.
pub struct MonoContext<'a> {
    pub interner: &'a mut Interner,
    pub input_expr_types: &'a [HirType],
    pub call_type_args: &'a HashMap<crate::types::ExprId, Vec<HirType>>,
    pub index: index::MonoIndex,
    pub mangle_table: mangle_table::MangleTable,

    work_queue: WorkQueue,
    output_items: Vec<HirItem>,
    output_expr_types: Vec<HirType>,
}

impl<'a> MonoContext<'a> {
    pub fn new(
        interner: &'a mut Interner,
        input_expr_types: &'a [HirType],
        call_type_args: &'a HashMap<crate::types::ExprId, Vec<HirType>>,
        hir: &crate::Hir,
    ) -> Self {
        let index = index::MonoIndex::build(hir);
        Self {
            interner,
            input_expr_types,
            call_type_args,
            index,
            mangle_table: mangle_table::MangleTable::new(),
            work_queue: WorkQueue::new(),
            output_items: Vec::new(),
            output_expr_types: Vec::new(),
        }
    }

    /// Process a function specialization.
    fn process_fn_specialize(&mut self, name: Symbol, type_args: Vec<HirType>) {
        let generic_fn = match self.index.find_fn(name) {
            Some(f) => f.clone(),
            None => {
                // This can happen if the function was already processed or doesn't exist
                return;
            }
        };

        let sub = if type_args.is_empty() {
            HashMap::new()
        } else {
            subst::SubstContext::build_subst_static(&generic_fn.type_params, &type_args)
        };

        let mut subst_ctx = subst::SubstContext::new(
            self.interner,
            self.input_expr_types,
            self.call_type_args,
            &self.index,
            &mut self.mangle_table,
        );

        let mut concrete_fn = generic_fn.clone();
        concrete_fn.type_params.clear();
        concrete_fn.name = self.mangle_table.mangle_fn(name, &type_args, self.interner);

        // Substitute signature types
        for (_, param_ty) in &mut concrete_fn.params {
            *param_ty = subst_ctx.concretize(
                crate::types::substitute_type(param_ty, &sub)
            );
        }
        if let Some(ret) = &mut concrete_fn.ret {
            *ret = subst_ctx.concretize(
                crate::types::substitute_type(ret, &sub)
            );
        }

        // Substitute body (with ExprId renumbering and type computation)
        concrete_fn.body = subst_ctx.substitute_expr(&concrete_fn.body, &sub);

        // Merge output expr_types
        self.merge_expr_types(subst_ctx.output_expr_types);

        // Enqueue discovered specializations
        for item in subst_ctx.discovered {
            self.work_queue.push(item);
        }

        self.output_items.push(HirItem::Fn(concrete_fn));
    }

    /// Process a non-generic function passthrough.
    /// Walks the body to rewrite call targets and build concrete expr_types.
    fn process_fn_passthrough(&mut self, name: Symbol) {
        let fn_def = match self.index.find_fn(name) {
            Some(f) => f.clone(),
            None => return,
        };

        let mut subst_ctx = subst::SubstContext::new(
            self.interner,
            self.input_expr_types,
            self.call_type_args,
            &self.index,
            &mut self.mangle_table,
        );

        let mut rewritten_fn = fn_def.clone();
        rewritten_fn.body = subst_ctx.substitute_expr(&rewritten_fn.body, &HashMap::new());

        self.merge_expr_types(subst_ctx.output_expr_types);

        for item in subst_ctx.discovered {
            self.work_queue.push(item);
        }

        self.output_items.push(HirItem::Fn(rewritten_fn));
    }

    /// Process a struct specialization.
    fn process_struct_specialize(&mut self, name: Symbol, type_args: Vec<HirType>) {
        let generic_struct = match self.index.find_struct(name) {
            Some(s) => s.clone(),
            None => return,
        };

        let sub = subst::SubstContext::build_subst_static(&generic_struct.type_params, &type_args);

        let mut subst_ctx = subst::SubstContext::new(
            self.interner,
            self.input_expr_types,
            self.call_type_args,
            &self.index,
            &mut self.mangle_table,
        );

        let mut concrete_struct = generic_struct.clone();
        concrete_struct.type_params.clear();
        concrete_struct.name = self.mangle_table.mangle(name, &type_args, self.interner);

        for field in &mut concrete_struct.fields {
            field.ty = subst_ctx.concretize(
                crate::types::substitute_type(&field.ty, &sub)
            );
            subst_ctx.discover_type(&field.ty);
        }

        for item in subst_ctx.discovered {
            self.work_queue.push(item);
        }

        self.output_items.push(HirItem::Struct(concrete_struct));
    }

    /// Process a non-generic struct passthrough.
    fn process_struct_passthrough(&mut self, name: Symbol) {
        let struct_def = match self.index.find_struct(name) {
            Some(s) => s.clone(),
            None => return,
        };
        self.output_items.push(HirItem::Struct(struct_def));
    }

    /// Process an enum specialization.
    fn process_enum_specialize(&mut self, name: Symbol, type_args: Vec<HirType>) {
        let generic_enum = match self.index.find_enum(name) {
            Some(e) => e.clone(),
            None => return,
        };

        let sub = subst::SubstContext::build_subst_static(&generic_enum.type_params, &type_args);

        let mut subst_ctx = subst::SubstContext::new(
            self.interner,
            self.input_expr_types,
            self.call_type_args,
            &self.index,
            &mut self.mangle_table,
        );

        let mut concrete_enum = generic_enum.clone();
        concrete_enum.type_params.clear();
        concrete_enum.name = self.mangle_table.mangle(name, &type_args, self.interner);

        for variant in &mut concrete_enum.variants {
            for field in &mut variant.fields {
                field.ty = subst_ctx.concretize(
                    crate::types::substitute_type(&field.ty, &sub)
                );
                subst_ctx.discover_type(&field.ty);
            }
        }

        for item in subst_ctx.discovered {
            self.work_queue.push(item);
        }

        self.output_items.push(HirItem::Enum(concrete_enum));
    }

    /// Process a non-generic enum passthrough.
    fn process_enum_passthrough(&mut self, name: Symbol) {
        let enum_def = match self.index.find_enum(name) {
            Some(e) => e.clone(),
            None => return,
        };
        self.output_items.push(HirItem::Enum(enum_def));
    }

    /// Merge expr_types from a subst_ctx into the global output.
    fn merge_expr_types(&mut self, local_types: Vec<HirType>) {
        // Since we renumber ExprIds globally, the local types are already
        // using IDs from the global counter. We just need to extend the
        // output vec to fit them.
        if local_types.len() > self.output_expr_types.len() {
            self.output_expr_types.resize(local_types.len(), HirType::Error);
        }
        for (i, ty) in local_types.into_iter().enumerate() {
            if i < self.output_expr_types.len() && self.output_expr_types[i] == HirType::Error {
                self.output_expr_types[i] = ty;
            } else if i >= self.output_expr_types.len() {
                self.output_expr_types.push(ty);
            }
            // If there's already a non-Error value at this index, it means
            // two different functions produced the same ExprId, which shouldn't
            // happen with proper renumbering.
        }
    }

    /// Run the BFS monomorphization loop.
    pub fn run(&mut self, hir: &crate::Hir, entry_points: &[Symbol]) {
        // Seed the queue with entry points
        for &ep in entry_points {
            self.work_queue.push(WorkItem::fn_passthrough(ep));
        }

        // Also enqueue all non-generic items for passthrough processing
        for name in self.index.fn_names() {
            if !self.index.is_generic_fn(name) {
                self.work_queue.push(WorkItem::fn_passthrough(name));
            }
        }
        for name in self.index.struct_names() {
            if !self.index.is_generic_struct(name) {
                self.work_queue.push(WorkItem::struct_passthrough(name));
            }
        }
        for name in self.index.enum_names() {
            if !self.index.is_generic_enum(name) {
                self.work_queue.push(WorkItem::enum_passthrough(name));
            }
        }

        // Pass through extern blocks
        for item in &hir.items {
            if let HirItem::Extern(ext) = item {
                self.output_items.push(HirItem::Extern(ext.clone()));
            }
        }

        // BFS loop
        while let Some(item) = self.work_queue.pop() {
            match item.kind {
                ItemKind::FnSpecialize => self.process_fn_specialize(item.def_id, item.type_args),
                ItemKind::FnPassthrough => self.process_fn_passthrough(item.def_id),
                ItemKind::StructSpecialize => self.process_struct_specialize(item.def_id, item.type_args),
                ItemKind::StructPassthrough => self.process_struct_passthrough(item.def_id),
                ItemKind::EnumSpecialize => self.process_enum_specialize(item.def_id, item.type_args),
                ItemKind::EnumPassthrough => self.process_enum_passthrough(item.def_id),
            }
        }
    }

    /// Consume the context and return the result.
    pub fn into_result(self) -> MonoResult {
        MonoResult {
            hir: crate::Hir { items: self.output_items },
            expr_types: self.output_expr_types,
        }
    }
}
```

### Task 10: Simplify `MethodCall` handling in codegen

**Files:** Modify `crates/glyim-codegen-llvm/src/codegen/expr/mod.rs`

- [ ] **Step 1: Simplify `MethodCall` arm**

The monomorphizer now converts most `MethodCall` expressions to `Call` expressions with mangled callee names. The codegen's `MethodCall` handling can be simplified to a fallback for cases where mono hasn't run (simple compilation path).

```rust
// In expr/mod.rs, the MethodCall arm:
HirExpr::MethodCall { receiver, method_name, args, .. } => {
    // The monomorphizer should have converted most MethodCalls to Calls.
    // This handler is a fallback for when mono hasn't run.

    // Check if this method is backed by an extern function
    if let Some(extern_name) = cg.extern_methods.get(method_name).copied() {
        let mut all_args = vec![receiver.as_ref().clone()];
        all_args.extend(args.clone());
        return super::string::codegen_call(cg, &extern_name, &all_args, fctx);
    }

    let receiver_val = codegen_expr(cg, receiver, fctx)?;

    // Try mangled name lookup
    let receiver_ty = cg.expr_types.get(receiver.get_id().as_usize()).cloned();
    let inner_ty = match &receiver_ty {
        Some(HirType::RawPtr(inner)) => inner.as_ref().clone(),
        other => other.clone().unwrap_or(HirType::Int),
    };

    // Build mangled name from type
    let mangled_name = match &inner_ty {
        HirType::Named(type_name) | HirType::Generic(type_name, _) => {
            format!("{}_{}", cg.interner.resolve(*type_name), cg.interner.resolve(*method_name))
        }
        _ => {
            cg.report_error(format!(
                "cannot resolve method '{}' on type {:?}",
                cg.interner.resolve(*method_name), inner_ty
            ));
            return Some(cg.i64_type.const_int(0, false));
        }
    };

    // Try exact lookup first
    if let Some(fn_val) = cg.module.get_function(&mangled_name) {
        let mut call_args = vec![inkwell::values::BasicMetadataValueEnum::IntValue(receiver_val)];
        for a in args {
            if let Some(v) = codegen_expr(cg, a, fctx) {
                call_args.push(inkwell::values::BasicMetadataValueEnum::IntValue(v));
            }
        }
        let result = cg.builder.build_call(fn_val, &call_args, "method_call").ok()?;
        return match result.try_as_basic_value() {
            inkwell::values::ValueKind::Basic(basic_val) => Some(basic_val.into_int_value()),
            _ => Some(cg.i64_type.const_int(0, false)),
        };
    }

    // Report error instead of silently returning 0
    cg.report_error(format!("method '{}' not found (mangled as '{}')", cg.interner.resolve(*method_name), mangled_name));
    Some(cg.i64_type.const_int(0, false))
}
```

## Summary of Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| `concretize` lives in the monomorphizer | Only the monomorphizer knows mangling. The typechecker works with logical types; the codegen works with concrete names. |
| `MonoResult` carries `expr_types: Vec<HirType>` | Replaces `type_overrides: HashMap<ExprId, HirType>`. The vec is indexed by output ExprIds, is complete, and requires no patching. |
| ExprId renumbering in output | Prevents collisions when the same generic function is specialized multiple times. The output HIR is self-contained. |
| Non-generic items also go through the substitution walker | Their call sites need callee names rewritten to mangled forms. The walker with empty substitution handles this. |
| `MethodCall` → `Call` desugaring in monomorphizer | The monomorphizer is the only component that knows the mangled method name. Desugaring here means the codegen sees simple calls. |
| Fallback `MethodCall` handling in codegen | For the simple compilation path (no typeck, no mono), the codegen still handles `MethodCall` directly. This is a pragmatic compromise. |
| No changes to the typechecker | The typechecker continues producing `expr_types` with `Generic` types and `call_type_args`. The monomorphizer bridges between logical and concrete types. |
