# v0.6.0 Phase 1: Foundation — Diagnostics, HirBuilder, Quote Syntax Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Scope check:** The v0.6.0 spec covers 7 phases across ~8 subsystems (quote/splice, macro stepper, rich errors, built-in derives, composability protocol, incremental expansion, attribute arguments). These MUST be separate plans. The codebase is at v0.1.0 level — no type checker, no structs, no enums, no generics, no macro expansion pipeline (`glyim-macro-core/src/lib.rs` is `//! TODO`). Every v0.6.0 phase depends on v0.3.0/v0.4.0/v0.5.0 landing first. This plan covers the only subset achievable at v0.1.0: multi-span diagnostic infrastructure, HirBuilder fluent API, and `quote! { ... }` syntax parsing. These three are independent of missing features and form the foundation that ALL subsequent v0.6.0 phases depend on.

**Blocked phases and what unblocks them:**
| Phase | Blocker |
|-------|---------|
| Phase 1 remainder (quote→HirBuilder expansion, `${join}`, `${str}`) | Macro expansion pipeline (v0.3.0+) |
| Phase 2 (rich errors in macros, extended MacroContext trait) | MacroContext trait impl (v0.3.0+). Multi-span diag part covered HERE. |

**Goal:** Add multi-span diagnostic support to `glyim-diag`, a fluent `HirBuilder` API to `glyim-hir`, and `quote! { ... }` syntax recognition to `glyim-parse` — all purely additive with zero behavior change to existing compilation pipeline.

**Architecture:** `Diagnostic` gains `labels`, `notes`, and `expansion_trace` fields with full backward compatibility. `HirBuilder` provides ergonomic constructors for all existing `HirExpr`/`HirStmt` variants (the target that `quote!` expansion will generate calls to). Parser recognizes `quote! { ... }` as an `ExprKind::Quote { body }` node storing the raw body text. No new behavior — every existing test passes unchanged.

```rust
use crate::Span;

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
pub enum Severity { Error, Warning, Note }

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Default)]
pub enum LabelStyle {
    #[default]
    Primary,
    Secondary,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct DiagnosticLabel {
    pub span: Span,
    pub message: String,
    pub style: LabelStyle,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct ExpansionFrame {
    pub macro_name: String,
    pub location: Option<Span>,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct Diagnostic {
    pub severity: Severity,
    pub message: String,
    pub span: Option<Span>,
    pub labels: Vec<DiagnosticLabel>,
    pub notes: Vec<String>,
    pub expansion_trace: Vec<ExpansionFrame>,
}

impl Diagnostic {
    pub fn error(message: impl Into<String>) -> Self {
        Self { severity: Severity::Error, message: message.into(), span: None, labels: vec![], notes: vec![], expansion_trace: vec![] }
    }
    pub fn warning(message: impl Into<String>) -> Self {
        Self { severity: Severity::Warning, message: message.into(), span: None, labels: vec![], notes: vec![], expansion_trace: vec![] }
    }
    pub fn note(message: impl Into<String>) -> Self {
        Self { severity: Severity::Note, message: message.into(), span: None, labels: vec![], notes: vec![], expansion_trace: vec![] }
    }
    pub fn with_span(mut self, span: Span) -> Self { self.span = Some(span); self }
    pub fn with_span_opt(mut self, span: Option<Span>) -> Self { self.span = span; self }
    pub fn is_error(&self) -> bool { self.severity == Severity::Error }

    /// Add a primary label at the given span.
    pub fn with_label(mut self, span: Span, message: impl Into<String>) -> Self {
        self.labels.push(DiagnosticLabel { span, message: message.into(), style: LabelStyle::Primary });
        self
    }

    /// Add a label with explicit style (primary or secondary).
    pub fn with_label_styled(mut self, span: Span, style: LabelStyle, message: impl Into<String>) -> Self {
        self.labels.push(DiagnosticLabel { span, message: message.into(), style });
        self
    }

    /// Add a note below the diagnostic.
    pub fn with_note(mut self, message: impl Into<String>) -> Self {
        self.notes.push(message.into());
        self
    }

    /// Add a macro expansion frame to the trace.
    pub fn with_expansion_frame(mut self, macro_name: impl Into<String>, location: Option<Span>) -> Self {
        self.expansion_trace.push(ExpansionFrame { macro_name: macro_name.into(), location });
        self
    }
}
```

```rust
use crate::{Diagnostic, ExpansionFrame, LabelStyle, Severity};
use ariadne::{Label, Report, ReportKind, Source};

pub fn render_diagnostics(source: &str, file_path: &str, diagnostics: &[Diagnostic]) -> String {
    let mut out = Vec::new();
    let cache = Source::from(source);
    for diag in diagnostics {
        let rendered = render_one(source, file_path, diag, &cache);
        out.extend(rendered);
    }
    String::from_utf8(out).unwrap_or_default()
}

pub fn render_single(source: &str, file_path: &str, diag: &Diagnostic) -> String {
    let cache = Source::from(source);
    let bytes = render_one(source, file_path, diag, &cache);
    String::from_utf8(bytes).unwrap_or_default()
}

fn render_one(source: &str, file_path: &str, diag: &Diagnostic, cache: &Source) -> Vec<u8> {
    let mut out = Vec::new();
    let kind = match diag.severity {
        Severity::Error => ReportKind::Error,
        Severity::Warning => ReportKind::Warning,
        Severity::Note => ReportKind::Advice,
    };

    // Determine the primary span for the report header
    let primary_span = if !diag.labels.is_empty() {
        diag.labels.iter().find(|l| l.style == LabelStyle::Primary)
            .or_else(|| diag.labels.first())
            .map(|l| l.span.start..l.span.end)
    } else {
        diag.span.map(|s| s.start..s.end)
    };
    let header_span = primary_span.unwrap_or(0..0);

    let mut report = Report::build(kind, (file_path, header_span)).with_message(&diag.message);

    // Add labels
    if !diag.labels.is_empty() {
        for label in &diag.labels {
            let ariadne_style = match label.style {
                LabelStyle::Primary => ariadne::LabelStyle::Primary,
                LabelStyle::Secondary => ariadne::LabelStyle::Secondary,
            };
            if label.span.start != label.span.end {
                report = report.with_label(
                    Label::new((file_path, label.span.start..label.span.end))
                        .with_message(&label.message)
                        .with_style(ariadne_style),
                );
            }
        }
    } else if let Some(span) = diag.span {
        // Backward compat: single span, no labels
        if span.start != span.end {
            report = report.with_label(
                Label::new((file_path, span.start..span.end)).with_message(&diag.message),
            );
        }
    }

    // Add notes
    for note_text in &diag.notes {
        report = report.with_note(note_text);
    }

    // Add expansion trace as notes
    if !diag.expansion_trace.is_empty() {
        let mut trace_lines = Vec::new();
        for frame in &diag.expansion_trace {
            let loc = frame.location
                .map(|s| format!(":{}", s.start))
                .unwrap_or_default();
            trace_lines.push(format!("  {}{}", frame.macro_name, loc));
        }
        report = report.with_note(format!("expansion trace:\n{}", trace_lines.join("\n")));
    }

    report
        .finish()
        .write((file_path, cache.clone()), &mut out)
        .unwrap();
    out
}
```

```rust
// crates/glyim-hir/src/hir_builder.rs
use glyim_interner::{Interner, Symbol};
use crate::{HirBinOp, HirExpr, HirStmt, HirUnOp};

/// Fluent builder for constructing HIR nodes.
/// This is the target API that `quote!` expansion will generate calls to.
pub struct HirBuilder;

impl HirBuilder {
    // ── Literals ────────────────────────────────────────

    pub fn int_lit(n: i64) -> HirExpr {
        HirExpr::IntLit(n)
    }

    pub fn str_lit(s: impl Into<String>) -> HirExpr {
        HirExpr::StrLit(s.into())
    }

    pub fn ident(sym: Symbol) -> HirExpr {
        HirExpr::Ident(sym)
    }

    pub fn ident_str(interner: &mut Interner, name: &str) -> HirExpr {
        HirExpr::Ident(interner.intern(name))
    }

    // ── Operators ───────────────────────────────────────

    pub fn binary(op: HirBinOp, lhs: HirExpr, rhs: HirExpr) -> HirExpr {
        HirExpr::Binary { op, lhs: Box::new(lhs), rhs: Box::new(rhs) }
    }

    pub fn add(lhs: HirExpr, rhs: HirExpr) -> HirExpr {
        Self::binary(HirBinOp::Add, lhs, rhs)
    }

    pub fn sub(lhs: HirExpr, rhs: HirExpr) -> HirExpr {
        Self::binary(HirBinOp::Sub, lhs, rhs)
    }

    pub fn mul(lhs: HirExpr, rhs: HirExpr) -> HirExpr {
        Self::binary(HirBinOp::Mul, lhs, rhs)
    }

    pub fn neg(operand: HirExpr) -> HirExpr {
        HirExpr::Unary { op: HirUnOp::Neg, operand: Box::new(operand) }
    }

    pub fn not(operand: HirExpr) -> HirExpr {
        HirExpr::Unary { op: HirUnOp::Not, operand: Box::new(operand) }
    }

    // ── Control flow ────────────────────────────────────

    pub fn block(stmts: Vec<HirStmt>) -> HirExpr {
        HirExpr::Block(stmts)
    }

    pub fn block_expr(exprs: Vec<HirExpr>) -> HirExpr {
        let stmts: Vec<HirStmt> = exprs.into_iter().map(HirStmt::Expr).collect();
        HirExpr::Block(stmts)
    }

    pub fn if_expr(condition: HirExpr, then_branch: HirExpr, else_branch: Option<HirExpr>) -> HirExpr {
        HirExpr::If {
            condition: Box::new(condition),
            then_branch: Box::new(then_branch),
            else_branch: else_branch.map(Box::new),
        }
    }

    // ── Builtins ────────────────────────────────────────

    pub fn println(arg: HirExpr) -> HirExpr {
        HirExpr::Println(Box::new(arg))
    }

    pub fn assert(condition: HirExpr, message: Option<HirExpr>) -> HirExpr {
        HirExpr::Assert { condition: Box::new(condition), message: message.map(Box::new) }
    }

    // ── Statements ──────────────────────────────────────

    pub fn let_stmt(name: Symbol, mutable: bool, value: HirExpr) -> HirStmt {
        HirStmt::Let { name, mutable, value }
    }

    pub fn let_immutable(name: Symbol, value: HirExpr) -> HirStmt {
        Self::let_stmt(name, false, value)
    }

    pub fn let_mutable(name: Symbol, value: HirExpr) -> HirStmt {
        Self::let_stmt(name, true, value)
    }
```

```rust
fn parse_quote_expr(&mut self) -> Option<ExprNode> {
    // We're positioned at Ident("quote")
    let name_tok = self.tokens.bump()?; // consume "quote"
    let name_start = name_tok.start;

    if !self.tokens.eat(SyntaxKind::Bang) {
        // Not a macro invocation — this shouldn't be reached since we check in parse_expr_atom
        // But handle gracefully: treat "quote" as an ident
        let sym = self.interner.intern(name_tok.text);
        return Some(ExprNode {
            kind: ExprKind::Ident(sym),
            span: Span::new(name_start, name_tok.end),
        });
    }

    if !self.tokens.eat(SyntaxKind::OpenBrace) {
        self.errors.push(ParseError::expected(SyntaxKind::OpenBrace, self.tokens.peek().map_or(SyntaxKind::Eof, |t| t.kind), self.tokens.peek().map_or(0, |t| t.start), self.tokens.peek().map_or(0, |t| t.end)));
        return None;
    }

    // Read the body as raw text, tracking brace depth
    let mut body = String::new();
    let mut brace_depth = 0u32;

    loop {
        match self.tokens.peek_raw() {
            None => {
                self.errors.push(ParseError::Message {
                    msg: "unexpected end of input in quote body".into(),
                    span: (name_start, name_start + 5),
                });
                break;
            }
            Some(tok) if tok.kind == SyntaxKind::OpenBrace => {
                brace_depth += 1;
                body.push_str(tok.text);
                self.tokens.bump_raw();
            }
            Some(tok) if tok.kind == SyntaxKind::CloseBrace => {
                if brace_depth > 0 {
                    brace_depth -= 1;
                    body.push_str(tok.text);
                    self.tokens.bump_raw();
                } else {
                    // Closing brace of quote body
                    self.tokens.bump_raw();
                    break;
                }
            }
            Some(tok) => {
                body.push_str(tok.text);
                self.tokens.bump_raw();
            }
        }
    }

    Some(ExprNode {
        kind: ExprKind::Quote { body },
        span: Span::new(name_start, self.tokens.peek().map_or(name_start + 5, |t| t.start)),
    })
}
```
