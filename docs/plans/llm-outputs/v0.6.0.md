# Glyim v0.6.0 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the complete macro system infrastructure for Glyim — diagnostics, HirBuilder, quote/splice syntax, extended MacroContext, attribute argument parsing, built‑in derive registry, composability checker, incremental expansion engine, and macro debugger — all working on top of the existing type‑checked, HIR‑lowering compiler.
